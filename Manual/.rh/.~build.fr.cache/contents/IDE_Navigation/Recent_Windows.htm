Janelasrecentes A lista de janelas recentes pode ser aberta a partir do Windows e mostrará uma pequena lista (por por padrão tem um máximo de 10 itens) das janelas que você já abriu anteriormente. Para começar com esta lista estará em branco, mas conforme você abrir diferentes janelas na IDE (como um sprite ou um script ou um objeto, etc...) a lista será preenchida. Deve você precisa voltar para qualquer uma destas janelas previamente abertas, então basta clicar com o botão esquerdo do mouse no item da lista e o espaço de trabalho irá para a janela. Se você clicar com o botão direito do mouse em um item, então você abrirá o seguinte menu: Aqui você pode escolher excluir um item (ou itens, como você pode usar + ou / + para selecionar múltiplos) da lista ou você pode tê-los movidos para o topo da lista. O menu RMB também tem uma opção adicional que lhe permite limpar completamente a lista. É importante notar que o A lista recente do Windows pode ser acoplada aos lados da IDE e você também pode ajustar certas Preferências para mudar o seu comportamento.
