O Menu de Ferramentas Este menu permite que você abra as diversas ferramentas stand alone que estão disponíveis dentro do GameMaker Studio 2, com as seguintes opções: Grupos de Textura - Veja a janela do Grupo de Textura. Grupos de Áudio - Veja a janela do Grupo de Áudio. Sound Mixer - Veja a janela do Sound Mixer. Gerar Imagens do Projeto - Abre a janela Gerar Imagens do Projeto. Criar Pacote Local - Abre a janela do Empacotador de Ativos, para que você possa fazer e salvar pacotes de ativos locais para uso futuro. Importar Pacote Local - Abre uma janela do explorador de arquivos onde você pode então selecionar um Pacote de Ativos Locais e importá-lo, seja adicionando-o ao projeto atual, ou criando um novo projeto para ele.
