O Menu Ajuda Este menu está relacionado à obtenção de ajuda ao utilizar o GameMaker Studio 2. Ele contém os seguintes itens de menu: Start Page - Isto abrirá um novo espaço de trabalho na Start Page. Página de boas-vindas - Esta página abrirá na primeira apresentação de slides da página de boas-vindas em um novo espaço de trabalho. Manual Aberto - Abre o documento de ajuda. Note que clicando no botão de ajuda na barra de menu, também abrirá o documento, assim como a imprensa . Base de Conhecimento - Isto o levará ao Centro de AjudaGameMaker Studio 2 onde você poderá encontrar artigos sobre configuração de plataformas, problemas conhecidos, usando o GameMaker Studio 2 e uma série de outros coisas. Report A Bug - Caso você encontre algum bug ou tenha algum problema durante o uso do GameMaker Studio 2, você pode relatá-lo aqui. YoYoGames.com - Abre a página inicial principal daYoYo Games. ComunidadeGameMaker - Isto abrirá os fóruns da ComunidadeGameMaker em seu navegador. Note que este é um fórum de execução comunitária , portanto, se você tiver um relatório de bug ou precisar de um assistência, você deve ir primeiro ao Site Oficial de Apoio (ver acima). Notas de Lançamento - Isto abrirá as Notas de Lançamento para GameMaker Studio 2 em seu navegador. Caso você precise reverter uma atualização, você pode obter instaladores de versões anteriores por clicando no link do número da versão nesta página. Notas de Lançamento em Tempo de Execução Atual - Desde o Tempo de execução pode ser atualizado independentemente da IDE, tem seu próprio conjunto de Notas de Lançamentoem Tempo de Execução. Selecionando esta opção, elas serão abertas em seu navegador, mas note que o tempo de execução atual pode não ser o que você está usando. Para saber em que tempo de execução você está, ou para informações sobre como adicionar, remover ou ativar tempos de execução, consulte a seção Preferências deTempo de Execução. SDKs necessários - Devido à natureza multiplataforma do GameMaker Studio 2, são necessários vários SDKs de terceiros. Clicando aqui, você será levado a um artigo do fórum de suporte que detalha o que você precisa em cada plataforma. Open Project In Explorer - Isto abrirá o local de salvamento do projeto atualmente carregado no OS file explorer. Sobre - Abre a janela "Sobre o GameMaker Studio 2" onde você pode obter informações sobre a versão do produto (essencial para reportar quaisquer bugs ou problemas).
