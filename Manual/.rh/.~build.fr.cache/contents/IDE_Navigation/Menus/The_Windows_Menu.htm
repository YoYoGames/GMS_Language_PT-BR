O Menu Windows Este menu é para abrir as janelas IDE acopladas em geral. Como as docas podem ser reorganizadas e o conteúdo pode ser removido como janelas autônomas ou mesmo totalmente fechadas, esta janela permite que você as abra novamente se achar que precisa delas para alguma coisa. As janelas que podem ser abertas a partir daqui são a janela do Navegador de Ativos e todas as diferentes janelas de Saída (Output, Search, Source Control, Pontos de parada e Erros de Compilação/Syntax). Em seguida, você pode abrir a janela do Inspetor. Esta janela é usada para inspecionar as propriedades do elemento IDE atualmente selecionado, e estas propriedades podem então ser editadas conforme necessário. Após a opção Inspetor você tem a lista de Janelas Recentes, e há também a opção de abrir a janela de Variáveis de Ambiente onde você pode ver os diferentes caminhos que os vários ambientes variáveis utilizadas nas Preferências apontam para. Estas variáveis não podem ser editadas (nem seu nome nem seu caminho/configuração), mas você pode selecionar e copiar/colar qualquer uma delas para uso em outro lugar. Finalmente, você tem a opção de abrir as janelas do Tutorial. Isto reabrirá quaisquer janelas que estejam relacionadas a um projeto tutorial você abriu, mas somente se o projeto foi criado como parte de um tutorial (se não, esta opção não estará disponível). Para obter informações sobre como criar seus próprios projetos de tutorial, consulte a seção sobre o Mercado (já que todos os tutoriais são criados, hospedados e baixados de lá).
