O Menu Arquivo O menu de arquivos é onde você pode escolher mudar, salvar ou importar projetos, bem como definir as preferências IDE do GameMaker Studio 2. As opções disponíveis são: Novo projeto - Isto é para criar um novo projeto a partir do zero no GameMaker Studio 2. Ele abrirá o diálogo "Salvar" e o levará a escolher um local para salvar o novo projeto. Uma vez que você tenha dado um nome ao projeto e um local para salvá-lo, você será levado a um novo espaço de trabalho, em branco, para começar a trabalhar nele. Os arquivos do projeto são salvos com a extensão *.yyp. Open Project - Isto abrirá um explorador de arquivos para que você selecione um arquivo de projeto previamente criado *.yyz para carregar. Observe que na importação, se o projeto foi criado utilizando uma versão inferior a 2.3.0, então você receberá a seguinte mensagem de aviso: Se você optar por prosseguir, então um explorador de arquivos será aberto e você poderá dar ao projeto um nome e uma localização atualizados para salvá-lo, para que o projeto original não seja exageradamente escrito. Uma vez convertido o projeto para o novo formato 2.3.0+, ele será aberto na IDE e você receberá uma mensagem dizendo que a conversão foi bem sucedido. Você deve proceder para verificar o relatório de compatibilidade que foi gerado pelo processo de conversão e ver se há algo que você precisa consertar antes de tentar executar ou compilar o projeto. Observe que se você receber repetidos erros ao tentar abrir um projeto feito com uma versão anterior à 2.3.0, então você deve contatar a YoYo Games (usando a opção "Reportar um bug" do Menu Ajuda) , fornecendo o arquivo original *.yyp que você está tentando converter. Projeto de importação - Aqui você pode importar projetos que tenham sido previamente exportados como um arquivo *.yyz (veja "Projeto de exportação", abaixo), ou - se você tiver um projeto feito com o GameMaker: Estúdio 1.4 software da YoYo Games - você pode selecionar esta opção para importar tanto o arquivo *.gmx ou o arquivo *.gmz dessa linha de produtos. Como na abertura de um projeto, se o *.yyz foi criado com uma versão do GameMaker Studio 2 antes da atualização 2.3.0, então ele precisará ser convertido conforme o formato do arquivo do projeto mudou com essa atualização. NOTA: Para importar um projeto GameMaker anterior para o GameMaker Studio 2 ele deve ter sido criado usando a versão 1.4, e mesmo assim ele pode não funcionar diretamente na importação. Muitas coisas são diferentes entre o GameMaker Studio 2 e o GameMaker Studio 2. GameMaker: Studio 1.4, e por isso seus projetos mais antigos podem precisar de alguns ajustes para que eles funcionem. Consulte o artigo Porting A GM:S 1.4 GameMaker Studio 2 no YoYo Games Base de Conhecimento e a seção do manual que cobre as Funções Obsoletas e os Scripts de Compatibilidade. IMPORTANTE! A importação de um projeto *.yyz que contenha recursos de fontes pode exigir que você "regenere" as fontes, removendo-as/adicionando-as, ou alterando a fonte no editor de fontes e depois ajustando de volta novamente. Isto porque as informações de fontes existentes são retidas pelo *.yyz quando são exportadas, e, na importação posterior ou em outra máquina, o projeto pode não ter as fontes necessárias já instaladas (ou pode têm uma fonte diferente com o mesmo nome). Isto só deve ser feito se você estiver vendo problemas com a forma como a fonte se torna após a importação. Projetos Recentes - Você pode usar esta opção para visualizar projetos usados recentemente. Selecionando-o, você reabrirá a página inicial (caso você a tenha fechado) e poderá selecionar um projeto recém-utilizado projeto da lista lá encontrada. Save Project - Isto salvará seu jogo no mesmo local que você definiu quando criou o projeto (ou de onde você o abriu). Salvar projeto como - O mesmo que acima, somente você especifica a localização e o novo nome do projeto. Projeto de Exportação - Este projeto será exportado como um único arquivo *.yyz. Muito útil para enviar cópias para pessoas ou para salvar backups. Preferências - Selecionando isto, você será levado para a Janela de Preferências onde você poderá configurar como o GameMaker Studio 2 se parece e se sente. Nova IDE - Quando você seleciona esta IDE, uma nova instância será aberta para você usar (ela será aberta na página inicial). Minha conta - Esta opção abrirá seu navegador padrão e o levará para sua página de conta YoYo. Logout - Selecionando isto, você sairá do GameMaker Studio 2. Você precisará fazer logon novamente com a mesma licença (ou qualquer outra) para poder usar o produto mais tarde. IMPORTANTE! Se você estiver usando uma licença de teste para o GameMaker Studio 2 e depois comprar uma das outras licenças (Desktop, Mobile, UWP, Web ou Console), então você pode precisar sair da IDE e reiniciar o GameMaker Studio 2 e fazer logon novamente para certificar-se de que a nova licença seja aplicada corretamente (embora isso deva acontecer automaticamente).
