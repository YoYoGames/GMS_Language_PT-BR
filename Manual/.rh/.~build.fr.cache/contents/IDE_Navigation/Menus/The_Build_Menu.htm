O Menu Construir O menu construir é onde você pode escolher construir seu projeto tanto para testes, depuração ou como executável final. As opções disponíveis são: Run - Execute seu projeto atual na plataforma alvo escolhida para os testes. Para as diferentes opções de compilação disponíveis ao compilar, consulte a seção sobre Compilação. Observe que esta é o mesmo que pressionar o botão Run no IDE. Debug - Execute seu projeto atual para testes com o depurador para que você possa ver informações detalhadas de desempenho e depurar quaisquer erros ou problemas que você possa ter. Para mais informações sobre depuração, por favor, consulte a seção Depuração. Note que este comando é o mesmo que pressionar o botão Debug no IDE. Clean - GameMaker Studio 2 mantém um Cache de arquivos para ajudar a acelerar o tempo de compilação durante os testes. No entanto, às vezes um cache "velho" pode levar a erros estranhos e inesperados. em seu jogo, caso em que a primeira coisa que você deve fazer é limpar o cache daqui e fazer um novo teste. Também é uma boa idéia limpar o cache antes de criar um executável final para a plataforma alvo. Note que este comando é o mesmo que pressionar o botão Clean Cache na IDE. Criar Executável - Isto compilará o projeto atual e criará um pacote executável (ou pacotes) para a plataforma alvo. Primeiro abrirá o explorador de arquivos para que você possa selecionar onde deseja salvar o projeto e depois compilar o jogo. Consulte a seção sobre Compilação para mais informações sobre como estabelecer os objetivos da compilação. Observe que este comando é o o mesmo que pressionar o botão Create Executable no IDE. Show Remote Worker Installer - Isto abrirá uma janela do explorador de arquivos na localização do instalador para a ferramenta Remote Worker (veja aqui para mais informações sobre a ferramenta Remote Worker Trabalhadores).
