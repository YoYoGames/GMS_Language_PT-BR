O Menu de Layouts Este menu está relacionado a salvar e carregar diferentes layouts IDE. Você pode configurar a IDE como quiser, com certos elementos (como o Navegador de Ativos, ou as Janelas de Saída) dispostas de forma diferente do padrão. Isto permite que você configure o GameMaker Studio 2 de acordo com seu fluxo de trabalho e com as opções neste menu você pode salvar os layouts atuais do projeto como um arquivo *.xml que pode então ser carregado em qualquer projeto para configurar os layouts para serem os mesmos. As opções neste menu são as seguintes: Load Layout - Isto abrirá um explorador de arquivos onde você poderá selecionar um arquivo *.xml de layout previamente salvo para carregamento. Save Layout - Isto abrirá um explorador de arquivos onde você pode nomear e salvar o layout atual da IDE como um arquivo *.xml. Reset Layout - Isto irá redefinir o layout atual do IDE para as configurações padrão. Voltar: Menus A seguir: O Menu Ajuda © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
