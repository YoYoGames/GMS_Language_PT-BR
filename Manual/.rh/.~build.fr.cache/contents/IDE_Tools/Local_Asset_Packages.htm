Pacotes de bens locais No Menu de Ferramentas você tem duas opções para criar e importar Pacotes de Ativos Locais. Um Pacote de Ativos Locais é um grupo de recursos salvos usando o formato de arquivo *.yymp que pode ser facilmente instalado em projetos existentes. Isto significa que você pode criar sua própria biblioteca de pacotes para recursos comumente usados, como bibliotecas de scripts, intros de tela de splash, sistemas de menu, etc... sem ter que usar os Jogos YoYo Mercado. Criando um pacote de ativos Para criar um pacote de recursos, você deve primeiro ter criado os recursos que deseja economizar em um projeto, e depois a partir do menu Ferramentas selecione a opção Criar pacote local. Isto abrirá a seguinte janela: Nesta janela você pode fornecer um Nome da Editora opcional e um Nome de Exibição obrigatório e ID do pacote. O ID do pacote será usado para gerar o nome do arquivo com o qual o pacote será salvo, e será automaticamente anexado com a extensão do arquivo *.yymp no momento da criação. Você também pode definir o Número da versão do pacote, e optar por assiná-lo ou não (isto está desativado por padrão). A assinatura de um pacote exige que você tenha criado um certificado de assinatura (você pode fazer isto a partir das Preferências do Mercado), e significa que qualquer bem que tenha sido assinado não poderá ser importado se tiver sido adulterado depois criação. Depois de preencher os detalhes do pacote, você pode então ir em frente e selecionar os ativos que deseja incluir na seção "Projeto" à direita (usando o botão esquerdo do mouse ou uma combinação de / e junto com o botão esquerdo do mouse). Os recursos selecionados podem ser movidos para a visão "Pacote de Ativos", usando os botões Adicionar ou Adicionar Todos e podem ser removidos novamente usando os botões Remover e Remover Todos. Quando você estiver satisfeito com a seleção de recursos e tiver preenchido os detalhes necessários, clicando no botão Ok , abrirá um explorador de arquivos onde você poderá escolher o local para salvar o pacote. Uma vez selecionado, o pacote será criado. e uma janela de confirmação será exibida, com a opção de abrir a localização do pacote no OS file explorer. Importação de um pacote de bens Há duas maneiras de importar um Pacote de Ativos Locais: o primeiro é simplesmente arrastar o arquivo *.yymp de uma janela do explorador para a IDE GameMaker Studio 2 o segundo é selecionar a opção "Importar Pacote Local" no Menu de Ferramentas . Ao utilizar o Menu de Ferramentas, uma janela do explorador se abrirá e você poderá navegar até o local do pacote e então selecioná-lo para importação. De qualquer forma, abrirá a seguinte janela no GameMaker Studio 2: À esquerda estão o conteúdo do pacote e à direita as partes do pacote que você deseja importar. Você pode selecionar uma ou mais recursos do pacote (usando o botão esquerdo do mouse ou uma combinação de / e junto com o mouse esquerdo ) e depois use o botão Adicionar para adicioná-los para importação, ou use o botão Adicionar tudo para adicionar tudo para importação. Quando tiver tudo pronto, você pode então clicar no botão Importar para adicionar os recursos selecionados ao seu projeto. Observe que você também tem a opção de Importar todos os recursos para um novo projeto, o que criará um novo projeto para você (e lhe pedir para dar um local para salvá-lo) e então adicionar os recursos do pacote a isso. Observe que os pacotes de ativos criados usando uma versão anterior ao GameMaker Studio 2.3 exigirão conversão, uma vez que o formato do arquivo do projeto mudou com a atualização 2.3. Isto significa que pode ser preferível importar o projeto inteiro e converter tudo depois exportá-lo como um novo pacote para um novo local se os bens forem importados freqüentemente para outros projetos.
