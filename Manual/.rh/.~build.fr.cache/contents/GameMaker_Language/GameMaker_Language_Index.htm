Índice deidiomas GameMaker A linguagem GameMaker (também chamada simplesmente GML) é a proprietária GameMaker Studio 2 linguagem de scripting. Esta linguagem é estruturada para permitir que os usuários criem seus jogos de forma intuitiva e flexível, enquanto oferecendo todo o poder de qualquer outra grande linguagem de programação. Ela também é a base para Drag e Drop™ e pode ser usada em conjunto com ela, se necessário.
