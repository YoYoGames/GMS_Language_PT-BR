chr Esta função retorna uma string contendo o caractere que se relaciona ao código Unicode de entrada para exibição. Este caractere depende da página de código das fontes de desenho atual e se nenhuma fonte for definida, ele usará a página de código padrão para a máquina. Sintaxe: chr(val); Argumento Descrição val O valor do código Unicode para obter a seqüência a partir de Devoluções: Cordão Exemplo: mystring = chr(53) + chr(48); Isto definiria mystring como "50" (um fio, não um inteiro), pois chr(53) é "5" e chr(48) é "0".
