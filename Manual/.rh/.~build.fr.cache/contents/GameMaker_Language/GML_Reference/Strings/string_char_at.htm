string_char_at Você pode usar esta função para retornar um caractere específico em uma posição específica dentro de uma cadeia, com o índice começando em 1 para o primeiro caractere. Se nenhum caractere for encontrado ou a cadeia for menor que o valor dado ao índice, uma cadeia vazia "". é retornada. Sintaxe: string_char_at(str, index); Argumento Descrição str O fio a ser verificado. índice A posição para obter o personagem. Devoluções: Cordão Exemplo: str1 = "Olá Mundo"; str2 = string_char_at(str1, 7); Isto colocará str2 no sétimo caractere ("H" contando como o 1º) em str1, neste caso "W".
