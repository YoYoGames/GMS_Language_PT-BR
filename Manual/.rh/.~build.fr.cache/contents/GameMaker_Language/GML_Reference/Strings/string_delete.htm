string_delete Você pode usar esta função para remover uma parte específica de um fio. Assim, você fornece a string de entrada e a posição inicial e final dentro dessa string para remover caracteres (o índice começa em 1) e a função retornará uma nova string sem essa seção nela. Sintaxe: string_delete(str, index, count); Argumento Descrição str O fio para copiar e excluir de. índice A posição do primeiro personagem a ser removido. contar O número de caracteres a serem removidos. Devoluções: Cordão Exemplo: str1 = "Helloo World"; str2 = string_delete(str1, 5, 1); Isto colocará str2 em "Hello World", pois remove o "o" extra de "Hello", ou seja: a corda conta junto com a primeira letra 5 lugares, depois deleta 1 caracter).
