string_count Esta função retornará a quantidade de vezes que o substrato dado aparecer dentro de uma cadeia específica. Desta forma, você pode verificar quantas vezes uma única letra ou frase é repetida em uma seção do texto armazenado. Sintaxe: string_count(substrato, str); Argumento Descrição substr O substrato para verificar o cordel. str O fio a ser verificado. Devoluções: Real Exemplo: str1 = "Hello World"; ocount = string_count( "o", str1 ); Isto definirá a variável com o número de "o"s na str1, neste caso 2.
