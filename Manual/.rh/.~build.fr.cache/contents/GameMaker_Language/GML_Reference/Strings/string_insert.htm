string_insert Com esta função você pode criar uma nova corda composta de duas cordas, onde uma foi inserida na outra em uma determinada posição. Pode ser útil, por exemplo, adicionar um nome de usuário em um texto pré-definido e assim fazer o jogador de seu jogo sentir mais envolvidos na ação. Tenha em mente ao calcular a posição a ser inserida nas cordas que são indexadas a partir de 1, portanto o primeiro caractere em uma corda é 1 e não 0, como você pode esperar. Sintaxe: string_insert(substrato, str, index); Argumento Descrição substr O substrato a ser inserido. str O fio a ser copiado. índice A posição em caracteres da corda para inserir o substrato. Devoluções: Cordão Exemplo: str2 = string_insert(nome de usuário, "Olá, , como vai?", 8); Isto irá inserir a string na variável "nome de usuário" na frase dada com a string resultante parecendo assim: "Olá, NOME, como você está?
