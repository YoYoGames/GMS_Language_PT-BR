clipboard_get_text Esta função retornará uma seqüência de texto contida na prancheta. se nenhum texto for armazenado, retornará uma seqüência vazia "". NOTA: Esta função só é válida para a Windows alvo. Sintaxe: clipboard_get_text(); Devoluções: Cordão Exemplo: se clipboard_has_text() { str = clipboard_get_text(); clipboard_set_text(""); } O código acima verifica o texto na prancheta e se ela contém algum, ele é lido como uma string na variável "str".
