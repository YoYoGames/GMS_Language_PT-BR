string_upper Com esta função você pode forçar uma cadeia a conter apenas caracteres maiúsculos. NOTA: Esta função detecta apenas o alfabeto inglês de 26 letras de A a Z. Sintaxe: string_upper(string); Argumento Descrição corda O fio para converter para maiúsculas. Devoluções: Cordão Exemplo: str1 = "Olá Mundo"; str2 = string_upper(str1); O código acima definirá a variável str2 como "HELLO WORLD".
