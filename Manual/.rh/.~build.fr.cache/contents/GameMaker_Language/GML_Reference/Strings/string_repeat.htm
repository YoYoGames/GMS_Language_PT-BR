string_repeat Esta função simplesmente retorna a mesma seqüência repetida um determinado número de vezes sobre si mesma. Sintaxe: string_repeat(str, count); Argumento Descrição str O fio a repetir. contar O número de vezes para repetir o fio. Devoluções: Cordão Exemplo: str1 = "Hello World"; str2 = string_repeat(str1, 3); O código acima definirá str2 como "Olá WorldHello WorldHello World".
