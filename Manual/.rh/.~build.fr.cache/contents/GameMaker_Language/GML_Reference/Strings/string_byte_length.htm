string_byte_length Esta função retorna o número de bytes em uma corda, mas você deve observar que, devido ao fato de serem mantidos como UTF8, isto não será igual ao comprimento de sua corda. Sintaxe: string_byte_length(string); Argumento Descrição corda O fio para medir o número de bytes de. Devoluções: Real Exemplo: bytesize = string_byte_length("Olá Mundo"); Isto definiria o número de bytes no "Hello World".
