Altura_de_filtro Esta função retornará a altura (em pixels) da cadeia de entrada, levando em conta a separação de linha e a largura da quebra de linha (que é definida como o número de pixels que a cadeia pode ocupar antes que uma quebra de linha seja inserida). É muito útil para calcular distâncias entre elementos de texto com base na mais alta das letras que compõem a corda, bem como o próprio comprimento real da corda quando espalhada por várias linhas, pois seria desenhada com draw_text_ext() usando a fonte atualmente definida. A separação e a largura podem ser definidas como -1 para obter o espaçamento padrão. Sintaxe: string_height_ext(string, sep, w); Argumento Descrição corda O fio para medir a altura de. sep A distância em pixels entre linhas de texto como se a corda estivesse sendo desenhada. w A largura máxima (em pixels) da corda antes de uma quebra de linha, como se a corda fosse trazida desenhada. Devoluções: Real Exemplo: var hh = string_height_ext(str_Story_Text[1], -1, 100); draw_text_ext(32, 32, str_Story_Text[1], -1, 100); draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100); O código acima obterá a altura da corda dada, levando em conta a separação de linha e a largura da quebra de linha, e então desenhará duas linhas de texto, usando a altura total da corda retornada como um separador.
