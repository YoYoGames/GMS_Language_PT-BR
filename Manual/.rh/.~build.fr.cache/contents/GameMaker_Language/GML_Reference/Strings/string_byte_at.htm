string_byte_at Devolve o valor do byte bruto como um valor real em uma determinada posição na cadeia dada. Sintaxe: string_byte_at(str, index); Argumento Descrição str O fio a ser verificado. índice A posição para obter o byte. Devoluções: Real Exemplo: newbyte = string_byte_at("Olá Mundo", 5); Isto irá definir o novo byte ao valor do byte bruto da sexta carta de "Olá Mundo".
