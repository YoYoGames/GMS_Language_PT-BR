largura_de_filtro Esta função retornará a largura (em pixels) da cadeia de entrada, levando em conta quaisquer quebras de linha que o texto possa ter. É muito útil para calcular as distâncias entre elementos de texto com base na largura total das letras que compõem a cadeia de caracteres como seria desenhado com draw_text() usando a fonte atualmente definida. Sintaxe: string_width(string); Argumento Descrição corda O fio para medir a largura de. Devoluções: Real Exemplo: var ww = string_width(str_Name + " "); draw_text(32, 32, str_Name)); draw_text(32 + ww, 32, "ganhou o jogo!"); O código acima obterá a largura da corda dada e então desenhará duas linhas de texto, usando a largura da corda retornada como separador.
