clipboard_has_text Esta função retornará verdadeiro se a prancheta contiver texto ou falso se não contiver. NOTA: Esta função só é válida para a Windows alvo. Sintaxe: clipboard_has_text(); Devoluções: Booleano Exemplo: se clipboard_has_text() { str = clipboard_get_text(); clipboard_set_text(""); } O código acima verifica o texto na prancheta e se ela contém algum, ele é lido como uma string na variável "str". Finalmente, a prancheta é limpa, definindo-a como uma string vazia. Voltar: Cordas Próximo: clipboard_get_text © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
