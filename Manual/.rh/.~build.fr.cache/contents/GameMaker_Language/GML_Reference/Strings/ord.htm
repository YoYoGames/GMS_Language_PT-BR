ord Esta função pega uma cadeia de entrada de um único caractere e retorna o valor Unicode (UTF8) para esse caractere. Note que quando usado com a função teclado_check* funções, a cadeia de entrada pode ter apenas um caráter e só pode ser um número de 0 a 9 ou um caráter romano capitalizado de A a Z. Sintaxe: ord(string); Argumento Descrição corda O fio com o qual se encontra o código Unicode. Devoluções: Real Exemplo: se teclado_check(ord("W")) { y -= 4; } Isto moverá a instância de chamada quatro pixels para cima se a chave W for mantida para baixo.
