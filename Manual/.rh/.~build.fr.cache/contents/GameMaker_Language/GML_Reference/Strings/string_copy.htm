string_copy Com esta função você pode selecionar facilmente um número de caracteres de dentro de uma cadeia para ser copiado para outra. O primeiro caractere de uma cadeia é sempre indexado como 1 e não 0 como você pode esperar, portanto, para copiar (por exemplo) os primeiros cinco caracteres de corda você teria string_copy(str, 1, 5). Um outro exemplo é fornecido abaixo. Sintaxe: string_copy(str, index, count); Argumento Descrição str O fio para copiar de. índice A posição do primeiro caractere na cadeia a ser copiado de (numerado a partir de 1) contar O número de caracteres, partindo da posição do primeiro, para copiar Devoluções: Cordão Exemplo: nome = keyboard_string; if (string_length(name) > 15) { keyboard_string = string_copy(name, 1, 15); } O código acima permitirá que o jogador insira uma string através do teclado, que é então armazenada no nome da variável. Esta variável é então verificada para ver se ela excede o comprimento máximo de 15 caracteres e se excede, a string_do_título do teclado é substituída por uma cópia de 15 caracteres da cadeia de nomes.
