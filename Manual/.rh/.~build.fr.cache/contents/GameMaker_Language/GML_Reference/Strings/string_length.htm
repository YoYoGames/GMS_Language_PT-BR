comprimento_de_filtro Esta função retorna o número de caracteres que compõem uma determinada cadeia. Ela pode ser útil para coisas como trabalhar quando se deve limitar o comprimento de um texto personalizado (por exemplo: limitar o nome de um jogador a 10 caracteres). Lembre-se de que isto é diferente de string_width() na medida em que mede o número de caracteres na corda, não sua largura como desenhada na tela em pixels. Sintaxe: string_length(string); Argumento Descrição corda O fio para medir o número de caracteres de. Devoluções: Real Exemplo: if (string_length(name) > 10) { nome = string_copy(nome, 1, 10); } Isto verificará se a cadeia de nomes é maior que dez caracteres e, se for, copiará e utilizará apenas os primeiros dez caracteres.
