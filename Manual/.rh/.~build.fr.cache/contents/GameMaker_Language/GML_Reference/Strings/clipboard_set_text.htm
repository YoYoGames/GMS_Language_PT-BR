clipboard_set_text Esta função ajustará a prancheta para segurar a corda definida. Você pode configurá-la para uma corda vazia "" para limpar efetivamente a área de transferência do texto. NOTA: Esta função só é válida para a Windows alvo. Sintaxe: clipboard_set_text(string); Argumento Descrição corda O texto (como um fio) para definir a prancheta a ser segurada. Devoluções: N/A Exemplo: se clipboard_has_text() { str = clipboard_get_text(); clipboard_set_text(""); } O código acima verifica o texto na prancheta e se ela contém algum, ele é lido como uma string na variável "str". Finalmente, a prancheta é limpa, definindo-a como uma string vazia. Voltar: Cordas Próximo: ansi_char © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
