string_last_pos Esta função devolverá a posição de caracteres de uma instância de uma sub-cordelinha dentro de uma corda, procurando desde o final da corda até o início (portanto, o inverso de string_pos(). A função retornará 0 se a cadeia de busca não for encontrada, ou a posição do primeiro caractere da cadeia de busca, se for. Tenha em mente que para as cadeias de suporte herdadas são indexadas a partir de 1, portanto 1 é a primeira posição na cadeia, não 0 como você pode esperar. Sintaxe: string_last_pos(substr, str); Argumento Descrição substr O substrato a ser procurado no cordão. str O fio. Devoluções: Inteiro Exemplo: if (string_last_pos(",", texto) != 0) { string_insert(nome, texto, string_last_pos(",", texto)); } O código acima procura na string armazenada na variável "texto" por uma vírgula que inicia a busca a partir do final da string.
