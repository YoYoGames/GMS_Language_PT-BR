densidade_física_set Quando você amarra um acessório a uma instância usando physics_fixture_bind() isto retorna um "id" para a fixação do limite. Você pode usar este id para definir o valor de densidade do dispositivo vinculado, e não o valor de densidade do dispositivo vinculado. fixação "base", a qualquer momento, usando esta função. Sintaxe: physics_set_density(fixação, densidade) Argumento Descrição fixação a identificação da fixação do limite densidade o novo valor de densidade a ser aplicado Devoluções: N/A Exemplo: var densidade = physics_get_density(fix_id); physics_set_density(fix_id, densidade - 0,1); O código acima obtém o valor da densidade atual para as propriedades físicas vinculadas da instância e depois as define para um valor diferente.
