fisica_fixture_set_set_circle_shape Esta função define uma forma de círculo para sua fixação com um raio definido pelo argumento "rad". Sintaxe: physics_fixture_set_circle_shape(fixture, rad) Argumento Descrição fixação o índice da fixação rad raio do círculo Devoluções: N/A Exemplo: physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2); O código acima aplicará uma forma de círculo à fixação indexada na variável "fix_Ball" com um raio igual ao da largura do duende "spr_Ball" dividido por 2.
