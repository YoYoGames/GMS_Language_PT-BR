physics_get_restitution Quando você amarra um acessório a uma instância usando fixação_física() isto retorna um "id" para a fixação do limite. Você pode usar este id para obter a restituição (a propriedade "bounciness") valor da fixação vinculada, não a fixação "base", a qualquer momento usando esta função. Sintaxe: physics_get_restitution(fixture) Argumento Descrição fixação a identificação da fixação do limite Devoluções: Real Exemplo: var descanso = physics_get_restitution(fix_id); physics_set_restitution(fix_id, descanso * 2); O código acima obtém o valor de restituição atual para as propriedades físicas vinculadas da instância e depois as define para um valor diferente.
