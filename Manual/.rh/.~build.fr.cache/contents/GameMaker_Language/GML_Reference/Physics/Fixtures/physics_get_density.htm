physics_get_density Quando você amarra um acessório a uma instância usando fixação_física() isto retorna um "id" para a fixação do limite. Você pode usar este id para obter o valor de densidade do dispositivo vinculado(e não o valor de densidade do dispositivo vinculado). "base") a qualquer momento usando esta função. Sintaxe: physics_get_density(fixture) Argumento Descrição fixação a identificação da fixação do limite Devoluções: Real Exemplo: var densidade = physics_get_density(fix_id); physics_set_density(fix_id, densidade - 0,1); O código acima obtém o valor da densidade atual para as propriedades físicas vinculadas da instância e depois as define para um valor diferente.
