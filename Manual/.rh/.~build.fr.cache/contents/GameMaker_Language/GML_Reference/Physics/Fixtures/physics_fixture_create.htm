physics_fixture_create O primeiro passo para a criação de uma fixação é criá-la com esta função. O índice de retorno deve ser armazenado em uma variável para ser usado em todas as outras funções que são usadas para definir e usar esta fixação. Sintaxe: physics_fixture_create() Devoluções: Índice da fixação Exemplo: fix_Ball = physics_fixture_create(); O código acima criará uma fixação e armazenará seu índice na variável "fix_Ball".
