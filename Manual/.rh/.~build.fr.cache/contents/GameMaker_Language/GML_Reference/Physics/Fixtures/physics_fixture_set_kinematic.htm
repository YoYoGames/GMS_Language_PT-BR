fisica_fixture_set_kinematic Haverá certas instâncias em um jogo baseado na física que você quer mover mas não deseja ser agido por forças como a gravidade, nem por forças incorridas por colisões com objetos dinâmicos (pense em plataformas móveis em um jogo de plataforma, pois exemplo). Para tais objetos, a simples definição da densidade de uma fixação a 0 significará que a física assumirá que o objeto se destina a ser estático. Entretanto, caso você defina a phy_speed_x, phy_speed_y ou f_velocidade_angular variáveis para uma instância que tenha tido tal fixação, ela será convertida em um objeto cinemático e iniciar o movimento. Mudar a instância de estática para cinemática de simulação média pode ser caro em termos de processamento (dependendo do estado das instâncias no mundo da física) e, portanto, é recomendável definir o fixação para ser cinemático usando esta função antes de ligá-lo a uma instância, para que não seja afetado por colisões e forças (como a gravidade), mas possa se mover com velocidade linear e/ou angular constante. Sintaxe: fisic_fixture_set_kinematic(fixture) Argumento Descrição fixação o índice da fixação Devoluções: N/A Exemplo: physics_fixture_set_kinematic(fix_Cloud); O código acima indicará que a fixação indexada em "fix_Cloud" deve ser marcada como cinemática.
