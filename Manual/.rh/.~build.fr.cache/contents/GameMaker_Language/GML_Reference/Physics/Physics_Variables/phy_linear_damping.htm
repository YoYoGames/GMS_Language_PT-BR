f_linear_damping Esta variável pode ser usada para definir o amortecimento linear da instância, ou pode ser usada para obter o amortecimento linear atual. O amortecimento é a quantidade de "resistência" ao movimento de avanço que a instância habilitada pela física tem, com um valor mais baixo permitindo que a instância se mova e acelere mais rápido e um valor mais alto, o que exige um empurrão mais forte Sintaxe: f_linear_damping; Devoluções: Real (valor de ponto flutuante de precisão único, ou indefinido se a instância não estiver habilitada fisicamente) Exemplo: if place_meeting(phy_position_x, phy_position_y, obj_Water) { phy_linear_damping = 10; } mais { phy_linear_damping = 3; } O código acima verificará se há uma colisão entre a instância de chamada e instâncias de "obj_Water" e alterará o amortecimento linear de acordo.
