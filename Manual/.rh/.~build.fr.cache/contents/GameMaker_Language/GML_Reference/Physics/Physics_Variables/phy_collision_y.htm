f_collisão_y Esta matriz só de leitura retorna a posição y de todos os pontos detectados em uma colisão entre duas instâncias habilitadas pela física. NOTA: Esta variável só está disponível no evento de colisão de uma instância física habilitada. Sintaxe: f_collisão_y; Devoluções: Real (valor de ponto flutuante de precisão único, ou indefinido se a instância não estiver habilitada fisicamente) Exemplo: for(var i = 0; i < phy_collision_points; i += 1;) { part_particles_create(global.sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5); } O código acima cria partículas em todos os pontos definidos de uma colisão entre duas instâncias físicas habilitadas.
