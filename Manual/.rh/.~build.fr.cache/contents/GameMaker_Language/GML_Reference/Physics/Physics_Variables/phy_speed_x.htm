phy_speed_x Esta variável pode ser usada para obter ou alterar o componente x do vetor de velocidade linear da instância e é definida em pixels por etapa (para pixels por segundo, veja phy_linear_velocity_x). Alterando isto para uma instância estática (ou seja: uma instância com densidade 0) a transformará em uma instância cinemática. Sintaxe: phy_speed_x; Devoluções: Real (valor de ponto flutuante de precisão único, ou indefinido se a instância não estiver habilitada fisicamente) Exemplo: se phy_speed_x != 0 { phy_speed_x = 0; } O código acima verificará o componente x do vetor de velocidade linear e se ele for qualquer coisa diferente de 0 ele o ajustará para 0.
