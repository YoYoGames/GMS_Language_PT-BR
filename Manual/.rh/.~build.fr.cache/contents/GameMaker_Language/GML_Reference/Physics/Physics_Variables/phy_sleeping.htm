f_sleeping Esta variável somente leitura retorna se a instância está ou não "dormindo" (verdadeira) ounão (falsa), Uma instância"dormindo" é aquela que não está ativamente envolvida em nenhuma simulação física. O GameMaker Studio 2 colocará objetos para dormir para salvar ciclos de simulação quando uma instância está em repouso e não em colisão com outra instância. Sintaxe: f_sleeping; Devoluções: Booleano (ou indefinido se a instância não estiver habilitada para física) Exemplo: se f_sleeping { instance_destroy(); } O código acima verifica se o objeto está sendo ativamente simulado ou não e se não está sendo destruído.
