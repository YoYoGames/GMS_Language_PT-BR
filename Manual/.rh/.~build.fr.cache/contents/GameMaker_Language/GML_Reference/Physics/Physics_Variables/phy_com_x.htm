phy_com_x Esta variável somente leitura retornará a posição x do centro de massa da instância. Isto é calculado automaticamente com base na densidade, inércia e massa da instância, conforme definido pelas funções apropriadas. Sintaxe: phy_com_x; Devoluções: Real (valor de ponto flutuante de precisão único, ou indefinido se a instância não estiver habilitada fisicamente) Exemplo: xx = phy_com_x; yy = phy_com_y; O código acima estabelece duas variáveis para a posição x e y do centro de massa das instâncias.
