phy_fixed_rotation Esta variável pode ser usada para definir se a instância pode ou não ser afetada por forças rotacionais (o padrão é falso). Se esta for definida como verdadeira, nenhuma força externa (seja de impulsos ou forças codificadas, ou de colisões) afetará o valor de rotação da instância e isso teria de ser definido manualmente usando o phy_rotation variável Sintaxe: phy_fixed_rotation; Devoluções: Booleano (ou indefinido se a instância não estiver habilitada para física) Exemplo: phy_rotation = 0; phy_fixed_rotation = true; O código acima mudará a instância para ter uma rotação fixa e, em seguida, definirá o ângulo de rotação.
