f_linear_velocidade_y Esta variável pode ser usada para obter ou alterar o componente y do vetor de velocidade linear da instância e é definida em pixels por segundo (para pixels por etapa, veja phy_speed_y). Alterando isto para uma instância estática (ou seja: uma instância com densidade 0) a transformará em uma instância cinemática Sintaxe: f_linear_velocidade_y; Devoluções: Real (valor de ponto flutuante de precisão único, ou indefinido se a instância não estiver habilitada fisicamente) Exemplo: if (phy_linear_velocity_y != 0) { phy_linear_velocity_y = 0; } O código acima verificará o componente y do vetor de velocidade linear e se for algo diferente de 0, ele o colocará em 0.
