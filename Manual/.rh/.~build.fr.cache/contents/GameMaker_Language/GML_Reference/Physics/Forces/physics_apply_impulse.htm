física_aplicar_impulso Você não só pode aplicar força e gravidade a um objeto com a física no GameMaker Studio 2, mas também pode aplicar um impulso. Isto é ligeiramente diferente de uma força na medida em que quando ela é aplicada afetará imediatamente a velocidade, e, potencialmente o torque (ou "spin") do objeto, particularmente se o ponto escolhido para aplicar o impulso não estiver alinhado com o centro de massa (nota: o centro de massa não é necessariamente o mesmo que a origem!). Aqui está uma ilustração: Como você pode ver, o jogador clicou na bola e isso lhe deu um impulso para passar para a parte superior direita, girando à medida que avança. O impulso exato é definido pelo vetor que obtemos dos componentes ximpulso/yimpulso em relação às coordenadas xpos/ypos - o que significa simplesmente que o impulso é calculado como a distância de xpos/ypos para ximpulso/yimpulso em Newtons, e a direção é o ângulo que obtemos de xpos/ypos para ximpulso/yimpulso. NOTA: O mundo da física usa medidores para todos os seus cálculos, e a proporção de pixels por metro é definida no editor da sala ou usando physics_world_create(). Sintaxe: physics_apply_impulse(xpos, ypos, ximpulse, yimpulse) Argumento Descrição xpos A coordenada x na sala onde o impulso será aplicado ypos A coordenada y na sala onde o impulso será aplicado ximpulse o componente x do vetor de impulso yimpulse o componente y do vetor de impulso Devoluções: N/A Exemplo: if mouse_check_button_pressed(mb_left) { com (place_meeting(mouse_x, mouse_y, all)) { physics_apply_impulse(mouse_x, mouse_y, -10 + irandom(20), -10 + irandom(20)); } } O código acima aplicará um impulso com um vetor aleatório a uma instância que está na posição do mouse quando o botão esquerdo é pressionado.
