Juntas No mundo da física GameMaker Studio 2, as juntas são usadas para restringir instâncias para o mundo ou umas para as outras. Exemplos típicos em jogos incluem ragdolls, teeters e roldanas, mas as articulações podem ser combinadas de muitas maneiras diferentes para criar moções ou acrescente realismo ao seu mundo de jogo. NOTA: Uma articulação não precisa ser apagada quando se destrói uma instância unida a outra instância, nem precisa ser apagada no final de uma sala. Exceto em casos muito específicos (ver a junta de engrenagens). já que isto é tratado automaticamente pelo GameMaker Studio. As páginas seguintes explicam as juntas disponíveis e como elas podem ser criadas: fisica_distância_distância_criar physics_joint_revolute_create physics_joint_prismatic_create physics_joint_pulley_create physics_joint_gear_create physics_joint_rope_create physics_joint_wheel_create física_joint_weld_create física_fricção_junta_criar physics_joint_delete Uma vez criada uma junta, muitas vezes pode ser necessário conhecer suas propriedades a fim de mudá-las em momentos específicos de seu jogo. As seguintes funções (com suas constantes internas correspondentes) estão disponíveis para testar e alterar os valores das juntas: physics_joint_enable_motor fisica_joint_get_value valor_junto_físico Além disso, você pode usar várias constantes de dentro das funções do GameMaker Studio 2 para obter (ou definir) diferentes propriedades de certas juntas.
