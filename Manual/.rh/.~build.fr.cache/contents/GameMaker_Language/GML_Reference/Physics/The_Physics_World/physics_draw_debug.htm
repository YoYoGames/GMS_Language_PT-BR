physics_draw_debug Ao criar um sistema de física no GameMaker Studio 2, muitas vezes é importante ter uma representação visual do que está realmente acontecendo com uma instância. Isto pode ser alcançado usando physics_draw_debug(), que desenha um esquema esquemático das propriedades físicas associadas com a instância que executa o código. Aqui está uma imagem de como ela é: Deve-se notar que, para que isto funcione, deve ser no evento do sorteio do e será desenhado utilizando a cor de desenho atualmente definida. Sintaxe: physics_draw_debug() Devoluções: N/A Exemplo: draw_set_colour(c_red); physics_draw_debug(); O código acima irá desenhar uma representação das propriedades físicas associadas a essa instância usando a cor vermelha.
