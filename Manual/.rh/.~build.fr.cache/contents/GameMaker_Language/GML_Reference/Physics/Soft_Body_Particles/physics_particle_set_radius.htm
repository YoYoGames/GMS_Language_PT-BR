physics_particle_set_radius Com esta função você pode definir o raio (em pixels) para as partículas em uma simulação física. Esta função é global no escopo, na medida em que mudará o raio não apenas para novas partículas criadas após a mudança, mas também para aquelas já presente na simulação. Sintaxe: physics_particle_set_radius(radius) Argumento Descrição raio O raio (em pixels) da fixação da partícula. Devoluções: N/A Exemplo: physics_particle_set_radius(15); physics_particle_set_density(0.5); physics_particle_set_damping(1); physics_particle_set_gravity_scale(1); O código acima definirá as propriedades básicas para todas as partículas na simulação.
