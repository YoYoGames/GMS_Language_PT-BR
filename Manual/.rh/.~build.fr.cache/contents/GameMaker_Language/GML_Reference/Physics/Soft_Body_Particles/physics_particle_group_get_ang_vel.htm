física_grupo_de_partículas_get_ang_vel Com esta função você pode recuperar a velocidade angular de um grupo de partículas. O valor do grupo é aquele que foi retornado quando você criou o grupo de partículas usando a função física_particle_group_end(), e a função retornará um valor que é o valor combinado das bandeiras atualmente definidas. Sintaxe: physics_particle_group_get_ang_vel(group) Argumento Descrição grupo O grupo de partículas a obter. Devoluções: Real Exemplo: ang_v = physics_particle_group_get_ang_vel(group1); O código acima obterá a velocidade angular do grupo de partículas indexada na variável "grupo1" e a armazenará em uma variável.
