física_particle_group_polygon Esta função irá definir a forma do grupo de partículas que está sendo criado. Primeiro é preciso ter começado a definição do grupo usando a função physics_particle_group_begin()e então você use esta função para dizer ao GameMaker Studio 2 que você vai adicionar nos pontos aproximados que serão usados para definir a forma do polígono - aproximada porque a forma exata dependerá do tamanho das partículas base, como definido por o physics_particle_set_radius() A simulação física tenta "encaixar" o maior número possível de partículas na forma definida. Você então definiria os pontos do polígono que chama a função grupo_de_partículas_físico_agrupamento_add_ponto()(mínimo de 3, máximo de 8, e eles pode ser definido como côncavo, mas os pontos dentro da forma serão ignorados). Finalmente, é preciso chamar física_particle_group_end() para criar o grupo de partículas na sala. Sintaxe: physics_particle_group_polygon() Devoluções: N/A Exemplo: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; var groupflags = phy_particle_group_flag_solid; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2); physics_particle_group_polygon(); physics_particle_group_add_point(200, 200); physics_particle_group_add_point(300, 300); physics_particle_group_add_point(100, 300); mLastGroup = physics_particle_group_end(); O código acima armazena as bandeiras para o tipo de partícula e as propriedades do grupo de partículas em variáveis e depois as utiliza para criar um grupo de partículas poligonais de três lados na posição do mouse.
