physics_particle_set_max_count Esta função definirá o número total permitido de partículas em uma simulação física. Se você definir este valor como 0, então não há limite para as partículas criadas, e defini-lo para qualquer outro valor impedirá que mais partículas sejam criadas quando o O número total de partículas é igual ao valor. Observe que o número mínimo de partículas que você pode criar em uma simulação é 128, portanto, a definição deste valor qualquer valor inferior não terá efeito (a menos que seja definido como 0). Sintaxe: physics_particle_set_max_count(count) Argumento Descrição contar O número máximo de partículas a permitir. Devoluções: N/A Exemplo: physics_particle_set_max_count(500); O código acima definirá o limite de partículas da simulação física para 500.
