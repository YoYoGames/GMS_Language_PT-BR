contagem_de_partículas_físicas Esta função retornará o número de partículas que estão ativas em uma sala habilitada para a física. Sintaxe: Physics_particle_count() Devoluções: Real Exemplo: if (physics_particle_count() < physics_particle_get_max_count()) { physics_particle_create(0, x, y, 0, 0, 0, c_white, 1, 1) } O código acima verificará se há menos do que o número máximo de partículas permitidas na sala e, em caso afirmativo, criará mais uma.
