gc_enable Com esta função, você pode ativar ou desativar o coletor de lixo. Chamando a função com true como o argumento o permite e usando falso o desativa (não recomendado). Ela é ativada por padrão. Sintaxe: gc_enable(enable); Argumento Descrição habilitar habilitar(verdadeiro) ou desabilitar(falso) o coletor de lixo. Devoluções: N/A Exemplo: if (global.debug == true) { gc_enable(false); } O código acima desabilita a coleta de lixo se a variável global em questão for verdadeira.
