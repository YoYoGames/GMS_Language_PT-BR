gc_is_enabled Com esta função, você pode verificar se o coletor de lixo está ou não habilitado. A função retornará verdadeira se estiver ativada ou falsa de outra forma. Sintaxe: gc_is_enabled(); Devoluções: Bool Exemplo: se (!gc_is_enabled())) { gc_enable(true); } O código acima verifica se o coletor de lixo está habilitado e se não está habilitado.
