steam_get_stat_avg_rate Com esta função você pode obter o valor de uma estatística média específica. A estatística deve ter sido previamente definida na página de contas do painel de controle da Steamworkspara seu jogo e a seqüência que é passada para a função deve corresponder que era usado como o nome API no painel de controle. Sintaxe: steam_get_stat_avg_rate(stat_name); Argumento Descrição stat_name O nome da estatística a ser obtida (um fio). Devoluções: Real Exemplo: var avg = steam_get_stat_avg_rate("PointsPerHour"); draw_text(8, 8, "PPH = " + string(avg); O código acima obtém o valor atual para a estatística média "PointsPerHour" e o desenha na tela.
