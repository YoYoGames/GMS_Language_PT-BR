vapor_clear_achievement Com esta função você pode dizer ao Steam API para limpar (resetar) uma conquista específica. A realização deve ter sido previamente definida na página de contas do painel de controle Steamworkspara seu jogo e a seqüência que é passada para a função deve corresponde àquele usado como o nome API no painel de controle. Sintaxe: steam_clear_achievement(ach_name); Argumento Descrição ach_name O nome da realização a limpar (string). Devoluções: N/A Exemplo: if mouse_check_button_pressed(mb_left) { steam_clear_achievement("Ach_Game_Win"); steam_clear_clear_achievement("Ach_Died_10_Times"); steam_clear_clear_achievement("Ach_Killed_100_Enemies"); steam_clear_clear_achievement("Ach_Beat_Boss_Level_1"); } O código acima irá redefinir as realizações do jogo quando o usuário clicar com o botão esquerdo do mouse.
