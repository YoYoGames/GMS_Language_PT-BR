steam_is_user_logged_on Esta função retornará verdadeiro se o cliente Steam tiver atualmente uma conexão ao vivo com os servidores Steam. Se ela retornar falsa, significa que não há conexão ativa devido a um problema de rede na máquina local, ou ao Steam servidor estar em baixo ou ocupado. Sintaxe: steam_is_user_logged_on(); Devoluções: Cordão Exemplo: if steam_is_user_logged_on() { global.user_id = steam_get_user_account_id(); } O código acima verificará se o usuário está logado no servidor Steam e se armazena o ID do usuário em uma variável global.
