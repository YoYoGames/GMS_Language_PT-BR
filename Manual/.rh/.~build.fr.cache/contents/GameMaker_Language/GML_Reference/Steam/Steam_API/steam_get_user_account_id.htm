steam_get_user_account_id Esta função é usada para recuperar o ID de usuário único que a Steam atribui a cada usuário, que é necessário para usar algumas das funções de Conteúdo Gerado pelo Usuário. Sintaxe: steam_get_user_account_id(); Devoluções: Real Exemplo: global.user_id = steam_get_user_account_id(); O código acima obtém o ID de usuário único para a pessoa que possui o jogo e o armazena em uma variável global.
