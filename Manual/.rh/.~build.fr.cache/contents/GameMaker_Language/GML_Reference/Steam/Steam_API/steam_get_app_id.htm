steam_get_app_id Esta função é usada para recuperar o ID exclusivo do aplicativo que a Steam atribui ao seu jogo, que é necessário para usar algumas das funções de Conteúdo Gerado pelo Usuário. Sintaxe: steam_get_app_id(); Devoluções: Real Exemplo: global.app_id = steam_get_app_id(); O código acima obtém o ID exclusivo do aplicativo para seu jogo no Steam e o armazena em uma variável global.
