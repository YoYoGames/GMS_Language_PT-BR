vapor_inicializado Ao utilizar o Steam API, esta função pode ser chamada para verificar se o Steam API do cliente foi inicializado corretamente antes de qualquer outra chamada para funções específicas do Steam em seu jogo. Sintaxe: vapor_inicializado(); Devoluções: Booleano Exemplo: global.steam_api= falso; if steam_initialised() { if steam_stats_ready() && steam_is_overlay_enabled() { global.steam_api= true; } } O código acima definirá uma variável global como verdadeira se a API do cliente Steam for inicializada corretamente, juntamente com as estatísticas e a funcionalidade de sobreposição do Steam, ou definirá a variável como falsa de outra forma.
