steam_get_persona_name Você pode usar esta função para retornar o nome de usuário do usuário atualmente conectado ao cliente Steam. Este é o nome de tela visível e não a identificação única do usuário (isto pode ser encontrado usando a função steam_get_user_steam_id()). Sintaxe: steam_get_persona_name(); Devoluções: Cordão Exemplo: se vapor_inicializado() { global.p_name = steam_get_persona_name(); } O código acima definirá uma variável global para o nome da tela dos usuários atuais se a API do cliente Steam for inicializada corretamente.
