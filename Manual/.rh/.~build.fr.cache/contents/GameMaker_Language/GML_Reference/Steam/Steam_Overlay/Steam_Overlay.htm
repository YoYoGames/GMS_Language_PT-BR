Vapor O Steam Overlay é o display visual que pode ser trazido para exibir a interface Steam para o usuário. Isto normalmente é feito pelo próprio usuário usando uma combinação de teclas, mas você também tem a possibilidade de fazendo isso de dentro de seu jogo, para que você possa mapear um botão ou um evento para mostrar a sobreposição, usando as seguintes funções: vapor_é_apto_a_sobre_camada vapor_é_activado_sobre_camada vapor_ativar_sobreposição vapor_ativar_ativar_sobre_browser vapor_activar_sobre_loja vapor_ativar_overlay_user Voltar: Vapor A seguir: Painéis de Liderança © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
