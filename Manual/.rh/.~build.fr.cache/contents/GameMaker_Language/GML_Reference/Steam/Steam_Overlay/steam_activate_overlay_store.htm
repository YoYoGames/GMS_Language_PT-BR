vapor_activar_sobre_loja Com esta função você pode abrir o Steam overlay na página da loja para um jogo, para que os usuários possam comprar ou baixar DLC (por exemplo). Você precisa fornecer o App ID exclusivo para o jogo ou DLC que você obteria do painel de controle do Steam quando o ajustasse. para cima. Sintaxe: vapor_activar_sobre_lote_overlay(app_id); Argumento Descrição app_id O único App ID para o seu jogo. Devoluções: N/A Exemplo: if keyboard_check_pressed(vk_f1) { steam_activate_overlay_store(global.DLC_id); } O código acima pesquisa o teclado para a tecla F1 e se for, então a sobreposição de vapor será aberta na página para o conteúdo do jogo usando o ID do aplicativo armazenado na variável global.
