vapor_ativar_ativar_sobre_browser Com esta função você pode abrir o jogo Steam overlay para seu navegador da web e depois fazer com que ele carregue a URL especificada. você precisa usar a URL completa como uma string para que isto seja resolvido corretamente, por exemplo: "http://www. steamgames.com/". Sintaxe: vapor_activar_overlay(url); Argumento Descrição url A URL (completa) para a sobreposição a ser aberta. Devoluções: N/A Exemplo: if keyboard_check_pressed(vk_f1) { steam_activate_overlay_browser("http://www. steamgames.com/"); } O código acima pesquisa o teclado para a tecla F1 e, se for, então a sobreposição de vapor será aberta e resolvida para a URL dada.
