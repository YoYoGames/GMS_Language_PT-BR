steam_get_quota_free Com esta função você pode descobrir quanto espaço livre é deixado para o usuário da cota do Steam Cloud. O valor retornado está em bytes. Sintaxe: steam_get_quota_free(); Devoluções: Real Exemplo: if steam_is_cloud_enabled_for_app() { quota = steam_get_quota_free(); } O código acima verifica se a nuvem de vapor está habilitada para o jogo e, em caso afirmativo, obtém o tamanho do espaço livre de armazenamento e o armazena em uma variável.
