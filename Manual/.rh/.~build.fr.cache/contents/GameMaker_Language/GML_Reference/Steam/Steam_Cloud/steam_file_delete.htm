steam_file_delete Esta função excluirá o arquivo dado da nuvem de vapor. A função retornará um valor 0 se falhar por qualquer razão e um valor maior que 0 se for bem sucedida. Sintaxe: steam_file_delete(filename); Argumento Descrição nome do arquivo O nome do arquivo excluir. Devoluções: Real Exemplo: se o steam_file_exists("Save.txt") { steam_file_delete("Save.txt"); } O código acima verificará se um arquivo existe, e se existe, ele apaga o arquivo da nuvem de vapor.
