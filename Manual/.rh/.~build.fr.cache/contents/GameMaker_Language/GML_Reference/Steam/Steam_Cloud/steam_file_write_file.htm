steam_file_write_file Com esta função você pode copiar o conteúdo de um arquivo salvo localmente para um arquivo que está sincronizado com o Steam Cloud. O arquivo local deve existir antes de usar esta função, e ele retornará um valor 0 se falhar por qualquer motivo e um valor maior que 0 se for bem sucedido. Sintaxe: steam_file_write_file(steam_filename, local_filename); Argumento Descrição nome_do_filme_vapor O arquivo Steam Cloud para copiar. local_filename O arquivo local a ser utilizado para copiar. Devoluções: real Exemplo: steam_file_write_file("rm_koala.png", "Koala2.png"); O código acima copiará o conteúdo do arquivo "Koala2.png" para o arquivo Steam Cloud "rm_koala.png".
