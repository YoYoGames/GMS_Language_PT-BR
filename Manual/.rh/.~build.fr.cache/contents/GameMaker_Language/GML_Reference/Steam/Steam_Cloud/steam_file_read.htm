steam_file_read Esta função lerá o conteúdo do arquivo dado em uma corda que pode ser posteriormente analisada em seu jogo. Sintaxe: steam_file_read(filename); Argumento Descrição nome do arquivo O nome do arquivo a ser lido. Devoluções: Cordão Exemplo: se o steam_file_exists("Save.txt") { save_str = steam_file_read("Save.txt"); } O código acima verifica se um arquivo existe no Steam Cloud e se existe, ele o abre e lê seu conteúdo na variável "save_str".
