steam_ugc_get_item_update_progress Esta função pode ser usada para rastrear o status de atualização de um item. Você dá o controle do item (como retornado pela função steam_ugc_start_item_update()) e um mapa DS vazio que será então preenchido com os seguintes pares de chaves/valores: "status_code" - O código de status do vapor "status_string" - Uma cadeia para o status atual " bytes_processados" - Os bytes processados até agora " bytes_total" - O número total de bytes na atualização Se houver um erro, a função retornará falsa e o mapa estará vazio, caso contrário, a função retornará verdadeira. Sintaxe: steam_ugc_get_item_update_progress(ugc_update_handle, info_map); Argumento Descrição ugc_update_handle O cabo único para a UGC a ser atualizado. info_map Um índice do mapa DS (anteriormente criado).
