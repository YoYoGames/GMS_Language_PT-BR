bolsa_de_vapor_conquista_add_requerida_e_etiqueta Esta função pode ser usada para filtrar ainda mais qualquer consulta UGC, especificamente para pesquisar somente aqueles itens UGC com a etiqueta dada. O controle da consulta é o valor retornado quando você criou a consulta (usando, por exemplo, steam_ugc_create_query_user()) e o segundo argumento é um fio que você quer usar como etiqueta a ser incluída. A função retornará verdadeiro se o filtro de consulta tiver sido configurado corretamente, ou falso de outra forma. Sintaxe: steam_ugc_query_add_required_tag(ugc_query_handle , tag_name); Argumento Descrição ugc_query_handle O cabo de consulta a ser usado. tag_name O nome da etiqueta a incluir (um fio). Devoluções: Booleano Exemplo: var query_handle = steam_ugc_create_query_all(ugc_query_RankedByTrend, ugc_match_Items, 1); steam_ugc_query_add_required_tag(query_handle, "RPG"); steam_ugc_query_set_return_long_description(query_handle, true); steam_ugc_query_set_allow_cached_response(query_handle, true), verdadeiro); query_ID = steam_ugc_send_query(query_handle); O código acima cria um pedido de consulta e armazena seu tratamento em uma variável local para uso futuro nas demais funções que definem melhor o pedido de consulta antes de enviar a consulta.
