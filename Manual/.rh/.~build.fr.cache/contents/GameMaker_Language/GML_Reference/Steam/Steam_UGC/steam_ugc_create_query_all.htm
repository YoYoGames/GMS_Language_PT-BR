vapor_ugc_criate_query_all Esta função pode ser usada para consultar a base de dados UGC usando alguns tipos de consulta pré-definidos. A função requer as seguintes constantes para o tipo de consulta a ser criada, o tipo de item a ser consultado e o número de página a ser consultado - observe que uma consulta retornará um número máximo de 50 itens. Tipo de consulta Constante Descrição ugc_query_RankedByVote Veja a Documentação doVapor. ugc_query_RankedByPublicationDate - ugc_query_AcceptedForGameRankedByAcceptanceDate - ugc_query_RankedByTrend - ugc_query_FavoritedByFriendsRankedByPublicationDate - ugc_query_CreatedByFriendsRankedByPublicationDate - ugc_query_RankedByNumTimesReported - ugc_query_CreatedByFollowedUsersRankedByPublicationDate - ugc_query_NotYetRated - ugc_query_RankedByTotalVotesAsc - ugc_query_RankedByVotesUp - ugc_query_RankedByTextSearch - Constante do tipo de partida Descrição ugc_match_Items Veja a Documentação doVapor. ugc_match_Items_Mtx - ugc_match_Items_ReadyToUse - ugc_match_Collections - ugc_match_Artwork - ugc_match_Videos - ugc_match_Screenshots - ugc_match_AllGuides - ugc_match_WebGuides - ugc_match_IntegratedGuides - ugc_match_UsableInGame - ugc_match_ControllerBindings - A função retorna um valor único de manuseio de consulta que deve ser armazenado em uma variável para uso nas outras funções de consulta.
