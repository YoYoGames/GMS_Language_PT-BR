steam_ugc_subscribe_item Esta função pode ser usada para subscrever um item da UGC. Você fornece o ID do arquivo publicado e então a função retorna um valor único de ID async que pode ser usado para verificar quando o item subscrito foi baixado no eventoSteam Async. Este evento terá a função async_load Mapa DS que será povoado com os seguintes pares de chaves/valores: " id" - A identificação assimétrica retornada pela função que acionou o evento "resultado" - O resultado da operação (um valor real). Este será ou a constante GML ugc_result_success ou algum outro número real. Portanto, você deve verificar esta constante para garantir que a chamada foi bem sucedida, e se caso contrário, alguma coisa não funcionou corretamente. Os demais valores possíveis retornados são mostrados como resultado do valor EResult " Steam " e você deve ver o steamclientpublic .h nos cabeçalhos do SDK para todos os 89 valores possíveis. " event_type" - A string " ugc_subscribe_item". "published_file_id" - A identificação única para o item publicado. Sintaxe: steam_ugc_subscribe_item(published_file_id); Argumento Descrição published_file_id A identificação de arquivo única para a UGC a ser subscrita. Devoluções: Identificação Async Exemplo: steam_sub= steam_ugc_subscribe_item(global.pubFileID); O código acima assinará (e baixará) o item com o ID do arquivo armazenado na variável global "pubFileID".
