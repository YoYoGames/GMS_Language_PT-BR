steam_ugc_get_item_update_info Esta função pode ser usada para recuperar informações sobre o estado atual do download para a ID do arquivo em questão. Você dá o ID do item e fornece o índice para um mapa DS vazio que será então preenchido com os seguintes pares de chaves/valores, se o item existir: "needs_update" - Retornará verdadeiro ou falso, dependendo se o item precisa ou não de uma atualização. "is_downloading" - Retornará verdadeiro ou falso, dependendo se o item está ou não sendo baixado no momento. " bytes_downloaded" - O número de bytes que foram baixados. " bytes_total" - O tamanho total (número de bytes) necessário para o item em disco. Se o item existir, então a função retornará verdadeiro e povoará o mapa, caso contrário, retornará falso e o mapa permanecerá vazio. Sintaxe: steam_ugc_get_item_update_info(published_file_id, info_map); Argumento Descrição published_file_id O ID do arquivo único para a UGC a ser verificado. info_map Um índice do mapa DS (anteriormente criado). Devoluções: Booleano Exemplo: var info_map = ds_map_create(); var info = steam_ugc_get_item_update_info(global.fileID, info_map); if info { draw_text(32, 15, "needs_update: " + string(info_map[? "needs_update"])); draw_text(32, 30, "is_downloading: " + string(info_map[? "is_downloading"])); draw_text(32, 45, "bytes_downloaded: "+ string(info_map[? "bytes_downloaded"])); draw_text(32, 60, "bytes_total: " + string(info_map[? "bytes_total"])); } O código acima irá consultar o status de download do item indexado na variável global "fileID", usando um DS Map para armazenar as informações.
