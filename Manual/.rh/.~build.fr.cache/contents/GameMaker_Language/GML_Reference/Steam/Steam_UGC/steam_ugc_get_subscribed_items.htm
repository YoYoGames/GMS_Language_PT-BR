steam_ugc_get_subscribed_items Esta função irá preencher uma lista DS com todas as IDs de arquivo publicadas para os itens que o usuário está inscrito no momento. É necessário primeiro criar a lista e armazenar o índice em uma variável, depois passar isto para a função. A função retornará verdadeiro se tudo estiver correto e o Steam API for inicializado, ou falso se houver um erro. Sintaxe: steam_ugc_get_subscribed_items(item_list); Argumento Descrição item_list Um índice de lista DS (anteriormente criado). Devoluções: Booleano Exemplo: steam_list= ds_list_create(); steam_ugc_get_subscribed_items(steam_list); O código acima criará uma lista DS vazia e depois a preencherá com os IDs de arquivo para todos os itens subscritos para o usuário.
