Vapor O Steam API suporta placas de liderança persistentes com entradas ordenadas automaticamente. Estes quadros de líderes podem ser usados para exibir quadros de líderes globais e de amigos em seu jogo e na página web da comunidade para seu jogo. Cada jogo pode ter até 10.000 quadros de classificação, e cada tabela de classificação pode ser recuperada imediatamente após a pontuação de um jogador ter sido inserida nela, mas observe que para cada tabela de classificação, um jogador pode ter apenas uma entrada, embora não haja limite para o número de jogadores por quadro de líderes. Cada entrada do quadro de líderes contém um nome, uma pontuação e uma classificação para o quadro de líderes, e esses dados serão substituídos quando uma nova entrada do quadro de líderes for criada para o usuário, e as seguintes funções podem ser usadas para adicionar e recuperar esses dados do quadro de líderes para o seu jogo: steam_create_leaderboard steam_upload_score steam_upload_buffer_score_buffer vapor_download_scores vapor_download_scores_around_utilizador vapor_download_friends_scores Voltar: Vapor A seguir: Conquistas e estatísticas © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
