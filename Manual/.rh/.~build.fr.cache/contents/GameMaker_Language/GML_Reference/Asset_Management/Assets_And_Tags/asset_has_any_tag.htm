ativos_etags_de_benefícios Com esta função você pode verificar se uma ou mais cadeias de tags são atribuídas a qualquer ativo a partir do navegador de ativos. Você fornece o nome do ativo (como uma string) ou seu índice de ativos, assim como uma única tag string ou uma matriz onde cada item é uma única cadeia de etiquetas. Se você fornecer um valor de índice de ativos, então você precisará fornecer o argumento do tipo de ativo opcional (uma constante), pois ativos de tipos diferentes podem ter o mesmo índice, mesmo que não possam ter o mesmo nome. Os ativos disponíveis Os tipos de ativos estão listados na tabela abaixo: Constante Descrição asset_object O nome dado refere-se a um objeto. asset_sprite O nome dado refere-se a um duende. asset_sound O nome dado refere-se a um som. asset_room O nome dado refere-se a uma sala. asset_tiles O nome dado refere-se a um conjunto de azulejos. asset_path O nome dado refere-se a um caminho. asset_script O nome dado refere-se a um roteiro. asset_font O nome dado refere-se a uma fonte. asset_timeline O nome dado refere-se a uma linha de tempo. asset_shader O nome dado refere-se a um shader. asset_animationcurve O nome dado refere-se a uma Curva de Animação. asset_sequence O nome dado refere-se a uma sequência. Se a função for bem sucedida e uma ou mais das etiquetas estiver presente para o bem, então ela retornará verdadeira, caso contrário, retornará falsa. Se você precisar verificar uma correspondência exata com qualquer tag ou conjunto de tags, então use a função asset_has_tags(). Sintaxe: asset_has_any_tag(name_or_index, tags, [asset_type]); Argumento Descrição nome_ou_index O nome do bem (uma cadeia) ou seu valor de índice (um número inteiro). etiquetas Uma única cadeia de tags de ativos ou uma matriz com várias tags de ativos. [asset_type] OPCIONAL! O tipo de bem a ser verificado pelas etiquetas, usado somente quando se fornece um índice para o primeiro argumento. Devoluções: Booleano Exemplo: var _a = array_create(3); _a[0] = "chefe"; _a[1] = "chefe_principal"; _a[2] = "chefe_final"; if asset_has_any_tag(object_index, _a, asset_object) { instance_create_layer(0, 0, "Overlay", obj_Boss_Text); } O código acima criará um conjunto de tags e então verificará se alguma delas é aplicada ao objeto em questão, e se são, criará uma instância de outro objeto.
