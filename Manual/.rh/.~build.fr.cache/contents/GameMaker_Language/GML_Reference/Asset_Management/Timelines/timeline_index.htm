timeline_index Esta variável contém o índice da linha do tempo atualmente associada à instância. Você pode definir isto para uma determinada linha de tempo para usar aquela, ou defini-la como -1 para parar de usar uma linha de tempo para a instância (se nenhuma linha de tempo for definida para a instância, -1 é devolvido também). Note que isto não inicia a linha do tempo - para isso use a variável linha do tempo_executando. Sintaxe: timeline_index; Devoluções: Real (número inteiro - valor do índice de recursos) Exemplo: se a linha do tempo_existir(global.tl) { timeline_index = global.tl; } O código acima atribuirá à instância executando o código uma linha de tempo indexada na variável "global.tl", se essa linha de tempo existir.
