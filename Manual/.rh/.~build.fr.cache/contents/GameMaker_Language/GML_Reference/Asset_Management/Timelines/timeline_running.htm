linha do tempo_executando Esta variável mantém o estado atual da linha de tempo atribuída e retornará verdadeira se estiver em funcionamento e falsa se não estiver. Você também pode definir esta variável como verdadeira ou falsa para deve-se observar que uma linha de tempo parada não é reiniciada e, portanto, recomeçá-la em um momento posterior a partir do momento exato em que ela foi parada. Sintaxe: cronograma_executado; Devoluções: Booleano Exemplo: se !timeline_running { timeline_position = 0; timeline_running = true; } O código acima verificará se a instância está executando uma linha de tempo, e se não estiver, então ela repõe a linha de tempo atribuída para começar no primeiro momento e depois a inicia.
