shader_current Esta função retornará o valor de identificação do índice do sombreador que está sendo usado atualmente para renderização, ou retornará -1 se nenhum sombreador estiver sendo usado. Sintaxe: shader_current(); Devoluções: Real (valor de identificação do sombreador, ou -1 se não houver sombreador) Exemplo: if(shader_current() == -1) { shader_set(sh_warp) } O código acima verificará qual é o sombreador atual e se ele retornar -1 (nenhum sombreador sendo usado), então um sombreador é definido.
