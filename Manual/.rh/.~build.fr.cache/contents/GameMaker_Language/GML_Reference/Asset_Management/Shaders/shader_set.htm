shader_set Com esta função você pode definir a meta de desenho para o sombreador dado e todos os outros desenhos serão feitos usando isso. Você pode terminar o uso do sombreador com a função shader_reset(). NOTA: Você só pode definir um sombreador no evento de sorteio de uma instância. Sintaxe: shader_set(shader); Argumento Descrição shader O índice do sombreador a ser usado. Devoluções: N/A Exemplo: shader_set(shader_Glass); draw_self(); shader_reset(); O código acima definirá um shader top a ser usado para desenhar, depois desenhar o sprite atual usado para o exemplo usando-o.
