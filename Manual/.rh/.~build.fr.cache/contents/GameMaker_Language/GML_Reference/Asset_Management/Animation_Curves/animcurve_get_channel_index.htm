animcurve_get_channel_index Esta função retornará o valor do índice para qualquer canal da curva de animação. Você fornece o ID ou estrutura da curva de animação, onde o ID da curva é o nome da curva de animação como foi definida no Navegador de Ativos, ou o ponteiro da estrutura como retornado. pela função animcurve_create(). Você então fornece o nome do canal, como um fio, e a função retornará o valor do índice associado a esse canal. Note que se a curva ou canal fizer não existir, então você receberá um erro. Sintaxe: animcurve_get_channel_index(curve_struct_or_id, channel_name); Argumento Descrição curva_ou_e_extracto O identificador ou ponteiro estrutural da curva de animação para o alvo nome_do_canal O nome do canal (uma cadeia). Devoluções: Estrutura Exemplo: var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos") var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex); var _points = _channeldata.points; for (var i = 0; i < array_length(_points); ++i;) { _points[i].valor += 1; } O código acima recupera a estrutura do canal chamado "x_pos" no ativo da curva "ac_ButtonTween", depois percorre os pontos da curva do canal e adiciona um ao seu valor.
