visível Uma instância pode ser marcada como visível ou não, definindo esta variável como verdadeira (visível) ou falsa (invisível). Isto funciona dizendo ao GameMaker Studio 2 para pular o evento do sorteio para esta instância, portanto, é preciso tomar cuidado ao usar isto, pois significa que nenhum código colocado no evento do sorteio será executado enquanto a instância não for visível. Observe também que se a camada à qual a instância é atribuída for marcada como invisível, definindo esta variável como true não terá efeito até que a própria camada seja sinalizada como visível também. Sintaxe: visível; Devoluções: Booleano Exemplo: se outro.visível = verdadeiro { x = xprevious; y = yprevious; } O código acima verificará a bandeira visível da "outra" instância em um evento de colisão e, se ela estiver configurada para verdadeira mudança dessa instância de volta à sua posição anterior.
