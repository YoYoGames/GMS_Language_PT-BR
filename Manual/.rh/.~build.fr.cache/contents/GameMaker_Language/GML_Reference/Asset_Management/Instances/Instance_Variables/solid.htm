sólido Uma instância pode ser marcada como sólida através das propriedades do objeto no Editor de Objetos, ou alterando o valor desta variável incorporada. Se sólido é definido como verdadeiro, então, quando uma colisão é detectada, a instância de colisão é devolvida automaticamente à posição em que se encontrava na etapa anterior à colisão (e então qualquer código ou ações são executadas no evento da colisão). Se for definida como falsa, todos os posicionamentos deve ser tratado durante o evento da colisão. Sintaxe: sólido; Devoluções: Booleano Exemplo: se outro.sólido = verdadeiro { x = xprevious; y = yprevious; } O código acima verificará a bandeira sólida da "outra" instância em uma colisão e se ela está configurada para mover verdadeiramente essa instância de volta à sua posição anterior.
