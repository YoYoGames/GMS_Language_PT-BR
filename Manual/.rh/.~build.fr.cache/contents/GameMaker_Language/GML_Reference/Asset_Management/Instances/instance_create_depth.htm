instance_create_depth Com esta função você pode criar uma nova instância do objeto especificado em qualquer ponto dentro da sala e na profundidade especificada. A profundidade pode ser qualquer valor, onde quanto menor a profundidade, mais "perto" da câmera as coisas serão desenhadas e quanto maior a profundidade, mais longe, então uma instância em profundidade -200 será arrastada sobre uma instância em profundidade +300 (por exemplo). Note que esta função realmente criará uma camada na sala, já que todas as instâncias devem estar em uma camada na sala, mas como esta é uma camada administrada (ou seja: não uma que você criou através do código ou na sala, mas uma que GameMaker Studio 2 foi criado automaticamente) você não pode acessá-lo, e o camada a variável de instância retornará -1. IMPORTANTE! Há uma profundidade mínima e máxima de camada de -16000 a 16000. Qualquer coisa colocada em uma profundidade fora dessa faixa não será desenhada, embora todos os eventos ainda funcionem normalmente. Esta função retorna o id da nova instância, que pode então ser armazenada em uma variável ou usada para acessar essa instância. Note que esta função também chamará o Create Event da instância sendo criada antes de continuar com o código ou ações para o evento que chamou a função. Sintaxe: instance_create_depth(x, y, depth, obj); Argumento Descrição x A posição x o objeto será criada em y A posição y do objeto será criada em profundidade A profundidade para atribuir a instância criada a obj O índice de objetos do objeto para criar uma instância de Devoluções: Real (valor de identificação da instância) Exemplo: var inst = instance_create_depth(x, y, -10000, obj_Bullet); com (inst) { speed = other.shoot_speed; direction = other.image_angle; } O código acima cria uma nova instância do objeto "obj_Bullet" e armazena a identificação da instância em uma variável.
