vspeed vspeed é uma das propriedades "embutidas" que todas as instâncias têm e define a velocidade de movimento vertical (ao longo do eixo y) da instância em pixels por degrau. Assim, um vs-velocidade de 3 significa 3 pixels de movimento para o fundo (+y) a cada passo, e uma velocidade de -3 significaria 3 pixels de movimento para o topo (-y) a cada passo. Observe que se você definir as variáveis speed e/ou direction, então o valor vspeed será atualizado automaticamente para refletem essas mudanças e, da mesma forma, a mudança do valor vspeed também afetará os valores speed e direction. Observe também que gravity, gravity_direction e friction podem todos modificar o valor desta variável quando eles são usados em seus jogos. Sintaxe: vs. velocidade; Devoluções: Real (valor de ponto flutuante de precisão único) Exemplo: if keyboard_check(vk_up) vspeed = -5; if keyboard_check(vk_down) vspeed = 5; O código acima mudará a velocidade vertical, dependendo de quais teclas são pressionadas.
