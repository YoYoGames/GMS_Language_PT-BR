exemplo_id Esta matriz de leitura contém todas as ids de cada instância ativa dentro da sala. Isto significa que se você tiver usado qualquer uma das funções Desativar Instância, aquelas instâncias que foram desativadas não serão incluídas neste array (se você tiver usado um valor deste array anteriormente, ele agora retornará a palavra-chave noone). Sintaxe: instance_id[num]); Devoluções: Real (valor de identificação da instância) Exemplo: for (var i = 0; i < instance_count; i ++;) { com (instance_id[i]) velocidade += 0.1; } O código acima passará por todas as instâncias dentro da sala e adicionará 0,1 à sua velocidade.
