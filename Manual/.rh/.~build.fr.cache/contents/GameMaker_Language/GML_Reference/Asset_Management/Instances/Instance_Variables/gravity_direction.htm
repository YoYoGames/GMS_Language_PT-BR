direção_da_gravidade A direção_da_gravidade é uma das propriedades "embutidas" que todas as instâncias têm e podem ser usadas para definir a direção do movimento quando as instâncias gravidade Note que as direções no GameMaker Studio 2 são normalmente calculadas como 0° sendo a direita, 90° sendo para cima, 180° sendo para a esquerda e 270° sendo para baixo. Sintaxe: direção_da_gravidade; Devoluções: Real (valor de ponto flutuante de precisão único) Exemplo: if place_meeting(x, y, obj_switch) { gravity_direction += 180; } O código acima mudará a direção da gravidade se o objeto jogador encontrar uma instância de "obj_switch".
