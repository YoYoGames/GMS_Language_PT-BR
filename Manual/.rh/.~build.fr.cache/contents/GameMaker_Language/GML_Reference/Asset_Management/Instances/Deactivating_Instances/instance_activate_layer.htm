instance_activate_layer Com esta função você pode ativar uma camada que tenha sido desativada anteriormente. Você precisa fornecer o ID da camada, que pode ser o nome da camada como escrito no editor de código (como uma string) ou o valor real do ID da camada como retornado. pela layer_create() e todas as instâncias desativadas nessa camada serão ativadas mais uma vez. Note que se você desativou uma camada que tem instâncias persistentes, você irá precisam reativar a camada novamente com esta função antes do vestiário, caso contrário, quaisquer instâncias persistentes não serão transportadas e serão descartadas. Note também que a ativação não é instantânea, e uma instância que tenha sido ativada Desta forma, não será considerado ativo até o final do evento em que a função foi chamada.
