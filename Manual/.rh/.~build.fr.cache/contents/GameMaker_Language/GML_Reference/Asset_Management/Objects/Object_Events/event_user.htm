event_user Com esta função você diz à instância para executar as ações ou código que foi colocado dentro de um dos 16 eventos definidos pelo usuário. Estes eventos só podem ser chamados desta forma, ou usando o event_perform função. Sintaxe: event_user(numb); Argumento Descrição dormente O número do evento a ser chamado, entre 0 e 15. Devoluções: N/A Exemplo: event_user(4); Isto chamaria Evento Definido pelo Usuário 4.
