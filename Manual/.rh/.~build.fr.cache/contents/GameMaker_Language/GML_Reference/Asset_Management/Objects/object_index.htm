object_index Esta variável lida retorna apenas o índice do objeto a partir do qual a instância foi criada. Isto não é o mesmo que o nome do objeto, que é uma string e pode ser encontrada usando object_get_name(), já que esta função retorna o número índice, que é um valor único que o GameMaker Studio 2 atribui a cada objeto no momento da criação. Sintaxe: object_index; Devoluções: Real (número inteiro - valor do índice de recursos) Exemplo: obj_name = object_get_name(object_index); O código acima usará o object_index para encontrar o nome do objeto a partir do qual a instância atual foi criada.
