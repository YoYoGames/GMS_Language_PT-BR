object_get_physics Esta função lhe dirá se o objeto que você está verificando foi assinalado como "physics enabled" - nesse caso ele retornará verdadeiro, - ou não - nesse caso ele retornará falso. Sintaxe: object_get_physics(obj); Argumento Descrição obj O índice do objeto a ser verificado. Devoluções: Booleano Exemplo: if object_get_physics(object_index) { phy_active = true; } O código acima verificará a instância em execução para ver se o objeto a partir do qual é criado está habilitado para a física, e se é ele que ativa a simulação física para a instância.
