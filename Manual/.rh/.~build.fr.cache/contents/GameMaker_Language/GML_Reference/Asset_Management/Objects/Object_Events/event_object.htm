event_object Esta variável só de leitura retorna o índice de objetos da instância que está executando o evento que está sendo verificado. Sintaxe: event_object; Devoluções: Real Exemplo: global.obj = event_object; O código acima armazena o índice de objetos da instância que realiza o evento em uma variável global.
