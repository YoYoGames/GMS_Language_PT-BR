object_get_name Esta função devolverá o nome como um fio do objeto especificado. Este nome é o que foi especificado para o objeto no Navegador de Ativos da janela principal do GameMaker Studio 2. Observe que isto é apenas uma cadeia de caracteres e não pode ser usado para referenciar diretamente o objeto - para isso você precisaria do índice de objetos. Você pode, entretanto, usar esta string para obter o índice do objeto usando a string retornada junto com a função asset_get_index(). Sintaxe: object_get_name(obj); Argumento Descrição obj O índice do objeto a ser verificado. Devoluções: Cordão Exemplo: str = object_get_name(object_index); O código acima obterá o nome do índice do objeto para a instância executando o código e armazenará o valor de retorno na variável "str".
