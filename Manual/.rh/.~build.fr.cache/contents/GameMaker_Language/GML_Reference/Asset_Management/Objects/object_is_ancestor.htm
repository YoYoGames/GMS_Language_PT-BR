object_is_ancestor Esta função pode ser usada para verificar um objeto e ver se ele é um ancestral ou não de outro objeto. Portanto, ambos os argumentos precisam ser índices de objetos e não a identificação da instância, e o primeiro é sempre o objeto que você querem descobrir se é um filho do segundo argumento, que é sempre o objeto que quer verificar como o antepassado (pai). Sintaxe: object_is_ancestor(obj, par); Argumento Descrição obj O objeto que está sendo verificado como a criança. par O objeto que está sendo verificado como o antepassado (pai). Devoluções: Booleano Exemplo: if object_is_ancestor(object_index, obj_Enemy) { instance_destroy(); } O código acima verifica se a instância que executa o código é uma criança do objeto "obj_Enemy" e, se for o caso, é destruída.
