event_action (depreciado) Esta variável somente leitura retornou o índice da ação que está sendo executada atualmente, começando em 0 nas versões anteriores do GameMaker. Entretanto, esta é agora uma variável obsoleta no GameMaker Studio 2. Ela foi deixada apenas para suporte legado, e sempre retornará 0, pois todas as ações são concatenadas para melhorar a velocidade de execução. Sintaxe: evento_ação; Devoluções: Real Exemplo: num = evento_ação; O código acima armazena o número da ação atual na variável "num".
