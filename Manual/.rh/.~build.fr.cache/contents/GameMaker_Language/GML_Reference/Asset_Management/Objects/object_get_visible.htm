object_get_visible Esta função lhe dirá se o objeto que você está verificando foi marcado como "visível" (realiza seu evento de sorteio) ou não (não realiza seu evento de sorteio). Note que esta não é uma função de instância! Portanto, você pode ter um objeto visível e uma instância invisível do mesmo objeto e vice-versa. Você pode definir uma visibilidade de instâncias individuais usando o visível variável de instância. Sintaxe: object_get_visible(obj); Argumento Descrição obj O índice do objeto a ser verificado. Devoluções: Booleano Exemplo: if (!visible) && (object_get_visible(object_index)) { visible = true; } O código acima verificará a instância em execução para ver se ela está visível ou não, bem como o índice de objetos da instância para ver se ela está sinalizada como visível ou não. Se a instância ainda não estiver visível, o índice de objetos é sinalizado como em, ele se tornará "visível" para o caso em questão.
