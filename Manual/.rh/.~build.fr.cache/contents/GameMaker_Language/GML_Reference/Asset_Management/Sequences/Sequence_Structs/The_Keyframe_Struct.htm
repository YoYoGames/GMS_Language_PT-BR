A estrutura do Keyframe Cada trilha (conforme definido na página que cobre as estruturas das trilhas) terá uma ou mais estruturas de quadros-chave atribuídas a ela (que você pode obter no quadros-chave propriedade da estrutura da pista), e cada uma delas terá as seguintes propriedades: moldura: A posição (em quadros) ao longo da linha do tempo para o quadro-chave. O valor padrão é 0. comprimento: O comprimento do quadro-chave. O valor padrão é 1, e quando definido para valores maiores, então a propriedade da pista a que o quadro-chave se refere será mantida no valor inicial durante a duração do comprimento dado. Note que a propriedade de trecho anulará isto, se for definido como verdadeiro. esticar: Se esta propriedade for definida como verdadeira, então o chaveiro se estende para o próximo chaveiro da pista ou para o final da pista, se for o último chaveiro . Você pode obter ou definir este valor, e o valor padrão é falso. canais: Esta propriedade permite o acesso à lista de estruturas de dados do quadro-chave para os canais da pista. Ao obter esta propriedade, uma matriz de A estrutura de dados do quadro-chave é retornada e, ao definir esta propriedade, deve ser especificada uma série de estruturas de dados do quadro-chave.
