sequence_get Com esta função, você pode recuperar a estrutura do objeto sequencial a partir de um índice de ativos sequencial. Você fornece o índice de ativos para a seqüência, conforme definido no navegador de ativos, e o retornará a estrutura de objetos da seqüência que pode então ser acessada, ou -1 se a seqüência não existir ou se o índice de ativos dado não for uma seqüência. Sintaxe: sequence_get(sequence_index); Argumento Descrição sequence_index O índice de seqüência conforme definido no navegador de ativos. Devoluções: estrutura ou -1 Exemplo: var _seq = sequence_get(seq_Logo); seq.loopmode = seqplay_pingpong; O código acima recupera a estrutura do objeto sequencial do objeto sequencial "seq_Loop" e então define o modo loop para a seqüência (todas as instâncias sequenciais criadas subseqüentemente usarão agora este modo loop).
