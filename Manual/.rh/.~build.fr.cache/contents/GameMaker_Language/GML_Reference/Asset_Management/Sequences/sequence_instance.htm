sequence_instance Esta é uma variável incorporada que faz parte das variáveis de instância criadas para cada instância de objeto em seu jogo. Se a instância estiver sendo controlada por uma seqüência, esta variável manterá a estrutura da instância da seqüência que controla a instância, caso contrário, ela será indefinida. Esta é uma variável somente de leitura e não pode ser alterado. Note que esta variável ficará indefinida após o fim da Sequência de controle, mesmo que seu Elemento Sequencial ainda exista, e obterá a estrutura novamente se esse Elemento Sequencial for reproduzido. Sintaxe: sequence_instance Devoluções: Booleano Exemplo: se (in_sequence) { sequence_instance.speedScale = 2; } O código acima verifica o in_sequence e se for true (o que significa que a instância está sendo controlada por uma seqüência), então ela mudará a escala de velocidade daquela Sequência a 2.
