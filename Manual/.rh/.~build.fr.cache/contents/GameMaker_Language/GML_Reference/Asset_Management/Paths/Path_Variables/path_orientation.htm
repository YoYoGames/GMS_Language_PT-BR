orientação_pista Esta variável mantém a orientação atual do caminho que foi atribuído à instância quando a função path_start() foi chamada. Quando um caminho é criado, sua orientação é o padrão 0 graus, mas você pode definir este valor para qualquer coisa que desejar usando isto. Lembre-se de que no GameMaker Studio 2 (a menos que você esteja usando física) os ângulos são calculados no sentido anti-horário, portanto definindo a orientação do caminho a 90° giraria o caminho para a esquerda. Sintaxe: orientação_pista; Devoluções: Real Exemplo: mypath = path_duplicate(choose(path_1, path_2, path_3, path_4)); path_start(path, 4, path_action_reverse, 0); path_orientation = 90; O código acima duplica um ativo de caminho pré-fabricado aleatório na variável "mypath". Este novo caminho é então iniciado e girado a 90°. Voltar: Variáveis de caminho Próximo: path_endaction © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
