caminho_existe sua função retorna se existe ou não um caminho com o índice dado. Note que se você verificar a existência de um caminho através de uma variável que ainda não foi declarada, isto irá lançar um erro. Sintaxe: path_exists(index); Argumento Descrição índice O índice do caminho a ser verificado. Devoluções: Booleano Exemplo: if (path_exists(my_path)) { path_delete(my_path); } Este código verifica se a variável em questão armazena um índice de caminho e se o faz, então o caminho é excluído da memória.
