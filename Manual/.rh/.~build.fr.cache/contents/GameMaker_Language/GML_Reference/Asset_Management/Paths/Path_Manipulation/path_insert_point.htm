caminho_ponto_de_inserção Com esta função você pode inserir um novo ponto em um caminho (o caminho pode ter sido criado no editor de caminhos ou através de código usando path_add()). O ponto será adicionado no caminho antes do ponto "n". que é especificado na função. Sintaxe: path_insert_point(index, n, x, y, speed); Argumento Descrição índice O índice do caminho para inserir o ponto. n O ponto definidor para inserir o novo ponto ANTES. x A coordenada x (relativa ao caminho) do novo ponto. y A coordenada y (relativa ao caminho) do novo ponto. velocidade O fator velocidade do ponto. Devoluções: N/A Exemplo: path_insert_point(mypath, 0, 50, 50, 100); Isto inserirá um ponto logo no início do caminho indexado na variável "mypath", a (50,50), com um fator de velocidade de 100%.
