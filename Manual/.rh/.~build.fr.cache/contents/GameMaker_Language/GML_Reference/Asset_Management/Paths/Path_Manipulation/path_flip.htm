path_flip Esta função pega todos os pontos do caminho e os desloca ao longo do eixo horizontal. Esta função muda o ativo do caminho real, e assim afetará permanentemente como o caminho é usado por todas as instâncias no jogo a partir do momento em que a função Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do caminho primeiro, então chame esta função no ativo duplicado (não esqueça de ligar para path_delete() sobre o ativo quando ele não for mais necessário). Sintaxe: path_flip(index); Argumento Descrição índice O índice do caminho a ser percorrido. Devoluções: N/A Exemplo: path_flip(mypath); Isto inverteria o caminho indexado na variável "mypath" ao longo do eixo horizontal.
