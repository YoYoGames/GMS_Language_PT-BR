path_get_length Você pode usar esta função para obter o comprimento exato de um caminho em pixels. este não é um comprimento aproximado de ponto a ponto, mas sim um comprimento exato ao longo da forma do caminho, mesmo quando o caminho é liso com uma alta precisão curvada. Sintaxe: path_get_length(index); Argumento Descrição índice O índice do caminho a ser medido. Devoluções: Real Exemplo: path_len = path_get_length(pth_AI); Isto definirá "path_len" para o comprimento total do caminho indexado em "pth_AI" em pixels.
