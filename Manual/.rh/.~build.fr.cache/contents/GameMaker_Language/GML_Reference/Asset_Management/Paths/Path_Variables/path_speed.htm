caminho_velocidade Você pode usar esta função para obter ou definir a velocidade de um caminho depois de ter sido iniciado, usando a função path_start(). Você pode usar valores negativos para significar que a instância deve seguir o caminho ao contrário. Sintaxe: caminho_velocidade; Devoluções: Real Exemplo: path_speed = -1 + random(2); O código acima estabelecerá a instância para percorrer o caminho a uma velocidade aleatória entre -1 e 1.
