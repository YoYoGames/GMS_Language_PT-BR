path_index Todos os recursos no GameMaker Studio 2 têm um número de identificação único, e o path_index é uma variável somente de leitura que contém o valor do índice para um determinado recurso de caminho que foi atribuído a uma instância usando a função path_start(). Se a instância não tiver um caminho atribuído, a variável será definida como -1. Sintaxe: path_index; Devoluções: inteiro (Path ID) Exemplo: if (path_index == -1) { path_start(pth_enemy3, 4, path_action_reverse, 0); } O código acima verifica se um caminho foi atribuído à instância e, caso contrário, inicia um novo caminho (atribuindo-o ao path_index ao mesmo tempo).
