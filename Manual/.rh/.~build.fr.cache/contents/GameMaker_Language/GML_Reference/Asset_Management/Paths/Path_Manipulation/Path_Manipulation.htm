Manipulação do caminho No GameMaker Studio 2 você não está limitado apenas aos caminhos que você faz usando o Path Editor. Há uma série de funções que podem ser usadas para manipular e mudar esses caminhos em tempo de execução, ou mesmo para criar novos e depois mudar e manipular esses caminhos.
