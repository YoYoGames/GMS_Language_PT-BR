sprite_width Esta variável lida retorna apenas a largura do duende que foi atribuído à instância. Esta largura é devolvida em pixels e será dependente do image_xscale. Se você precisar da largura não escalonada você deve usar sprite_get_width(). Sintaxe: sprite_width; Devoluções: Real (valor de ponto flutuante de precisão único) Exemplo: se sprite_width != sprite_get_width(sprite_index) { image_xscale = 1; } O código acima verifica a largura do duende atribuído à instância executando o código contra a largura do recurso duende e, se não for o mesmo, redefine a escala image_x para 1, Voltar: Variáveis de Sprite Instance Próximo: sprite_height © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
