esqueleto_animação_get_frame Essa função retornará o número do frame da animação na faixa de animação especificada. A função retornará -1 se nenhuma animação for atribuída à faixa de animação específica dada. Sintaxe: esqueleto_animação_get_frame(track); Argumento Descrição faixa A trilha de animação para obter o quadro de. Devoluções: Inteiro Exemplo: var frame = skeleton_animation_get_frame(0); if frame > 0 && frame < 2 { var box = skeleton_get_minmax(); var ww = (box[2] - box[0]) / 2; var hh = (box[3] - box[1]) / 2; part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10); } O código acima verificará o quadro atual da animação designado para a faixa 0 e então explodirá algumas partículas de um ponto no meio da caixa de delimitação total para o duende.
