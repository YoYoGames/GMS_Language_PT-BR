esqueleto_animação_get_ext Um único duende de animação esquelético pode ter vários conjuntos de animação, e a esses conjuntos podem ser atribuídas faixas diferentes para que você possa "misturar e combinar" conjuntos de animação. Essa função retornará o nome do conjunto de animação atualmente utilizado pelo dado número de faixa (conforme definido pela função esqueleto_animação_set_ext_sext). Sintaxe: esqueleto_animação_get_ext(pista); Argumento Descrição faixa O número da faixa para obter o nome do conjunto de animação. Devoluções: Cordão Exemplo: if skeleton_animation_get_ext(1) != "bodyfight" { skeleton_animation_set_ext("bodyfight", 1); } O código acima mudará o conjunto de animação sendo usado pela faixa 1 para "bodyfight" se ainda não o for.
