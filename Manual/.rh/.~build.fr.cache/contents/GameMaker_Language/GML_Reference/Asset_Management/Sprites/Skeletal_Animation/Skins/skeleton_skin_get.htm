esqueleto_esqueleto_esqueleto Com os sprites de animação esquelética, você pode atribuir texturas separadas (chamadas "peles") à animação, usando assim uma animação para várias coisas diferentes. Esta função devolverá o nome da pele (como uma corda) que é atualmente atribuído ao sprite de animação esquelética que sua instância está usando. O nome retornado é o que você definiu quando criou o duende em seu programa de animação. Sintaxe: skeleton_skin_get(); Devoluções: Cordão Exemplo: if skeleton_skin_get() == "skin_Enemy1" { skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3"); } O código acima verificaria a pele atualmente designada para a animação e se for "skin_Enemy1", ele escolhe e define uma nova pele a partir de uma das três opções.
