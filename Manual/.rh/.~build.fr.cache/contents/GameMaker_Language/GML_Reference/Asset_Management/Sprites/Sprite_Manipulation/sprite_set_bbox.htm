sprite_set_bbox Esta função pode ser usada para definir os valores da caixa de delimitação para um duende. Você fornece o índice sprite para usar e depois os valores esquerdo, superior, direito e inferior para as posições da caixa delimitadora. As posições são valores absolutos, onde a posição (0, 0) corresponde ao canto superior esquerdo do duende, independentemente do offset para o duende, quaisquer pixels "vazios" que o duende possa ter, ou onde ele está sendo desenhado na sala. OBSERVAÇÃO: Esta função afeta o patrimônio do duende, de modo que todas as outras instâncias com este duende terão a mesma caixa de delimitação. Sintaxe: sprite_set_bbox(ind, left, top, right, bottom); Argumento Descrição ind O índice do duende para colocar a caixa delimitadora. esquerda O lado esquerdo da caixa de delimitação topo A parte de cima da caixa de delimitação. à direita O lado direito da caixa de delimitação inferior O fundo da caixa delimitadora. Devoluções N/A Exemplo: if sprite_get_bbox_mode(sprite_index) == 2 { esquerda = irandom(sprite_width / 2); direita = irandom((sprite_width / 2) + irandom(sprite_width / 2))); topo = irandom(sprite_height / 2); fundo = irandom((sprite_height / 2) + irandom(sprite_height / 2))); sprite_set_bbox(sprite_index, esquerda, em cima, direita, em baixo); } O código acima verificará o modo de caixa delimitadora do sprite atribuído ao sprite_index, e se estiver definido para manual, então terá sua caixa delimitadora alterada.
