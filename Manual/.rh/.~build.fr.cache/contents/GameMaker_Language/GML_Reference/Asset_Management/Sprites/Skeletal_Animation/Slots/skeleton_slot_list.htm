lista_esqueleto_de_squema Com esta função você pode preencher uma lista DS (pré-criada) com todos os nomes dos slots criados como parte do sprite de animação esquelética. Os nomes serão strings e poderão então ser usados nas outras funções do esqueleto do esqueleto de animação para estes tipos de duendes. Sintaxe: skeleton_slot_list(sprite, list); Argumento Descrição sprite O índice sprite da animação do esqueleto da coluna vertebral para obter a lista. lista A identificação da lista DS a ser preenchida com os nomes dos ossos. Devoluções: N/A Exemplo: bone_list = ds_list_create(); skeleton_slot_list(sprite_index, bone_list); O código acima cria uma lista DS e depois a preenche com os nomes dos slots para uso posterior.
