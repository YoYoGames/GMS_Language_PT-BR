bbox_top Esta variável lida apenas retorna a posição dentro da sala (ao longo do eixo y) do topo da caixa de delimitação, onde a caixa de delimitação é definida pela largura e altura máximas da máscara para a instância (conforme estabelecido pelo sprite_index ou pelo mask_index). Mesmo quando um duende tem uma máscara de colisão precisa, a caixa de delimitação existe e é usada para certas coisas, e assim você pode usar esta variável para encontrá-la. Observe que quando a instância não tem sprite atribuído o valor retornado será o mesmo que a posição Y da instância. Sintaxe: bbox_top; Devoluções: Real (inteiro) Exemplo: se bbox_top < 0 { y = sprite_yoffset; } O código acima verifica se a caixa delimitadora da instância está fora da sala e se é ela que coloca a instância em uma nova posição.
