image_angle Este valor define o ângulo (rotação) do duende e é medido em graus, com a direita sendo 0º, acima sendo 90º, esquerda sendo 180º e abaixo sendo 270º. Ajuste esta variável para 0 para reiniciar o duende a ser desenhado como foi definido no editor do duende. Por favor, observar que para que as mudanças nesta variável sejam visíveis, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker Studio 2 irá desenhar por padrão o sprite) ou ser desenhada usando uma das funções de desenho estendido como draw_self() ou draw_sprite_ext(). Sintaxe: image_angle; Devoluções: Real (valor de ponto flutuante de precisão único) Exemplo: image_angle = point_direction(x, y, mouse_x, mouse_y); O código acima irá girar o duende da instância para sempre apontar para a posição do mouse.
