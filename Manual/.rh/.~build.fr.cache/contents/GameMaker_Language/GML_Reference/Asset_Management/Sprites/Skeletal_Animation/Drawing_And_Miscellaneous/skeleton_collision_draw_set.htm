esqueleto_colisão_desenho_de_squadrilho Com esta função, você pode ativar(true) ou desativar(false) o desenho dos dados de colisão para a animação esquelética atual que está sendo usada pela instância. Se esta for ligada, a caixa de delimitação e a máscara de colisão precisa ser desenhado como contornos em torno do duende. Sintaxe: skeleton_collision_draw_set(flag); Argumento Descrição bandeira Ajustar para verdadeiro para ligar o desenho, e falso para desligá-lo. Devoluções: N/A Exemplo: if debug_mode == true { skeleton_collision_draw_set(true); } O código acima verifica se o jogo está sendo executado no modo de depuração e, se estiver, alterna os dados de colisão do esqueleto para a instância a ser mostrada.
