Nove funções em fatias Além da seção de nove fatias no editor Sprite, Estúdio GameMaker 2 fornece funções para definir as propriedades de Nove fatias para sprites em tempo de execução. Antes de ler sobre estas funções, é recomendável ler esta página para obter uma compreensão da técnica das Nove Fatias. As seguintes funções podem ser usadas para recuperar e alterar as propriedades de Nove fatias para um ativo sprite em tempo de execução: sprite_nineslice_create sprite_get_nineslice sprite_set_nineslice Estas funções fazem uso de uma estrutura de nove fatias, e você pode ler sobre o conteúdo dessa estrutura nesta página.
