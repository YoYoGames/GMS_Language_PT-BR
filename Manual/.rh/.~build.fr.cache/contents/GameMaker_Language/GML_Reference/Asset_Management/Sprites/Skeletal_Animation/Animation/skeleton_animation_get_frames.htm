esqueleto_animação_get_frames Esta função pode ser usada para recuperar o número de quadros que qualquer animação de esqueleto possui. Você fornece o nome da animação do esqueleto (como uma string, como definido no programa usado para fazer a animação, ou como retornado usando a função esqueleto_animação_gete a função retorna os quadros que tem como um valor inteiro. A função retornará 0 se a animação especificada não existir. Sintaxe: esqueleto_animação_get_frames(anim_name); Argumento Descrição nome_anim_name O nome da animação para obter os quadros de. Devoluções: Inteiro Exemplo: var num = skeleton_animation_get_frames(skeleton_animation_get()); image_index = num -1; image_speed = 0; O código acima obterá o número de quadros na animação e então definirá o sprite para o último quadro e parará de animar.
