bbox_right Esta variável lida somente retorna a posição dentro da sala (ao longo do eixo x) do lado direito da caixa de delimitação, por exemplo, onde a caixa de delimitação é definida pela largura e altura máximas da máscara por exemplo (como estabelecido pelo sprite_index ou pelo mask_index). Mesmo quando um duende tem uma máscara de colisão precisa, a caixa de delimitação existe e é usada para certas coisas, e assim você pode usar esta variável para encontrá-la. Observe que quando a instância não tem sprite atribuído o valor retornado será o mesmo que a instância X posição. Sintaxe: bbox_right; Devoluções: Real (inteiro) Exemplo: se bbox_right > room_width { x = room_width - sprite_xoffset; } O código acima verifica se a caixa delimitadora da instância está fora da sala e se é ela que coloca a instância em uma nova posição.
