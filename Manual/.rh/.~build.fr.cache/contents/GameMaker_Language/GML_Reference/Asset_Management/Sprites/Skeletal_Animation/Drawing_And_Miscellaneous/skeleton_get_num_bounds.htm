esqueletos_get_num_bounds Esta função retornará o número de caixas de delimitação associadas ao sprite de animação do esqueleto atribuído à instância que executa o código. Isto pode então ser usado junto com a função skeleton_get_bounds() para Recuperar dados sobre cada uma das caixas de delimitação. Sintaxe: skeleton_get_num_bounds(); Devoluções: Inteiro Exemplo: var num = skeleton_get_num_bounds(); var yy = 60; for(var i = 0; i < num; i++;) { var b_info = skeleton_get_bounds(i); if b_info[0] > 0 { var data = b_info[1] + ":"; for(var j = 0; j < b_info[0]; j++;) { data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")"; } draw_text(20, yy, data); yy += 20; } } O código acima passará por cada uma das caixas de delimitação associadas ao sprite atualmente atribuído e então desenhará esses dados como um fio para a tela.
