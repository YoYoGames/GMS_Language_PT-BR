sprite_height Esta variável lida retorna apenas a altura do duende que foi designado para a instância. Esta altura é retornada em pixels e será dependente do image_yscale. Se você precisar da altura não escalonada, você deve usar sprite_get_height(). Sintaxe: sprite_height; Devoluções: Real (valor de ponto flutuante de precisão único) Exemplo: se sprite_height != sprite_get_height(sprite_index) { image_yscale = 1; } O código acima verifica a altura do duende atribuído à instância executando o código contra a altura do recurso duende e, se não for o mesmo, repõe a imagem_escala para 1, Voltar: Variáveis de Sprite Instance Próximo: sprite_xoffset © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
