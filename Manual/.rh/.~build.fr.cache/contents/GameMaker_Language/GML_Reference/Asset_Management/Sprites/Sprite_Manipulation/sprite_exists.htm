sprite_existe Esta função retorna se um sprite com o índice especificado existe ou não no projeto em execução. Sintaxe: sprite_exists(index); Argumento Descrição índice O índice do sprite a ser verificado. Devoluções N/A Exemplo: if sprite_exists(spr_array[0]) { sprite_index = spr_array[0]; } mais { sprite_index = spr_BaseSprite; } O código acima verifica um array para ver se ele contém um índice sprite válido e, se assim for, atribui esse sprite à instância, mas se não, atribui um sprite a partir dos recursos incluídos.
