Bones Como o nome indica, os esqueletos de animação são compostos de uma série de "ossos", sobre os quais uma pele é adicionada para criar a animação final do personagem. Normalmente, você gostaria de montar esses ossos no software de animação (Spine) e depois exportar o sprite com um conjunto de animações já predefinidas para que você possa simplesmente alternar entre elas conforme necessário.
