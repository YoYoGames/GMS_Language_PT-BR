esqueleto_dados_esqueleto_de_instância Com esta função você pode preencher uma lista DS (pré-criada) com dados para cada um dos slots de anexo disponíveis para o duende da coluna vertebral associado à instância atual (incluindo qualquer modificações de anexos). Estes dados são devolvidos no fórum de um mapa DS que contém os seguintes pares chave/valor: "nome": O nome do espaço. "osso": O nome do "osso". "anexo": O nome do anexo se for usado ou "(nenhum)" se não houver um atribuído. Os valores para cada chave serão cordas e podem então ser usados nas outras funções de fixação do esqueleto para estes tipos de duende. Observe que o DS Map criado não é destruído, portanto você precisará fazer loop através da lista DS e destruir cada um dos mapas criados você mesmo. Sintaxe: skeleton_slot_data_instance(lista); Argumento Descrição lista A identificação da lista DS a ser preenchida com os mapas DS.
