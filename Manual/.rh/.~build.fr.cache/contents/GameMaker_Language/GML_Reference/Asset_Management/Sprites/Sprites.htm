Sprites Os Sprites são geralmente as representações visuais dos objetos dentro do jogo que você está criando, embora possam ser usados para muitos outros fins. Um duende é uma imagem única, ou um conjunto de imagens que, quando jogado um após o outro, se parece com um animação de looping. Sprites podem ser criados em muitos formatos, sendo o sprite "tradicional" composto de quadros de imagens pixel, importados para o GameMaker Studio 2 como arquivos *.png. Entretanto, você também pode importar sprites vetoriais que tenham sido salvos no formato *.swf, ou sprites de animação esquelética que tenham sido salvos como arquivos *.json com arquivos *.atlas e *.png acompanhantes (para mais informações sobre esses tipos de arquivos de imagem e como importá-los para o GameMaker Studio 2, consulte a seção Importação de Sprites Não-Bitmap). Como os sprites são parte integrante de qualquer jogo, o GameMaker Studio 2 tem um grande número de funções e variáveis que podem ser usadas para obter informações sobre sprites específicos, ou para mudar aqueles que estão em seu jogo, ou até mesmo para carregar customizados. sprites em tempo de execução a partir de um arquivo de imagem de origem.
