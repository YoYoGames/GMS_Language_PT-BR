draw_skeleton_collision Esta função desenhará as máscaras de colisão associadas com a animação esquelética dada. Você fornece o sprite base, o conjunto de animação a ser usado e o frame para obter as informações, e você também pode definir as propriedades de transformação para se adequar. Sintaxe: draw_skeleton_collision(sprite, animname, frame, x, y, xscale, yscale, yscale, rotineiramente, colorido) Argumento Descrição sprite O índice do duende a desenhar. animname O nome da animação para obter o quadro de (um fio). quadro O quadro de animação a desenhar (de 0 a image_number - 1). x A coordenada x de onde desenhar o duende. y A coordenada y de onde desenhar o duende. xscale A escalada horizontal do duende, como um multiplicador: 1 = escalada normal, 0,5 é metade, etc... yscale A escalada vertical do duende, como um multiplicador: 1 = escalada normal, 0,5 é metade etc... podridão A rotação do duende. 0=normal, 90=voltada 90 graus no sentido anti-horário, etc. cor A cor com a qual se deve misturar o duende. Devoluções: N/A Exemplo: draw_skeleton_collision(sprite_index, "jump", image_index, x, y, image_xscale, image_yscale, image_angle, c_white); O código acima desenhará os dados da máscara de colisão para o duende atual, usando as transformações atuais, para o conjunto de animação "salto".
