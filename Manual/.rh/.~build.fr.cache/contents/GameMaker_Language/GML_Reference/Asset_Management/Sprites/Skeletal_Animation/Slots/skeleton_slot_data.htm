dados_esqueleto_dados_de_squadra Com esta função você pode preencher uma lista DS (pré-criada) com dados para cada um dos slots de anexo disponíveis que uma animação contém. Estes dados são retornados no fórum de um mapa DS que contém os seguintes pares chave/valor: "nome": O nome do espaço. "osso": O nome do "osso". "anexo": O nome do anexo se for usado ou "(nenhum)" se não houver um atribuído. Os valores para cada chave serão cordas e podem então ser usados nas outras funções de fixação do esqueleto para estes tipos de duende. Observe que o mapa DS criado não é destruído, portanto você precisará percorrer a lista DS e destruir cada um dos os próprios mapas criados. Sintaxe: skeleton_slot_data(sprite, lista); Argumento Descrição sprite O índice sprite da animação do esqueleto da coluna vertebral para obter os dados. lista A identificação da lista DS a ser preenchida com os mapas DS.
