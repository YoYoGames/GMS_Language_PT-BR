sprite_get_number Um duende tem que ter pelo menos uma subimagem e esta função devolverá quantos ele tem. Lembre-se, uma subimagem sprite começa a ser numerada a partir de 0, portanto, se esta função retornar 3 (a sprite que está sendo verificada tem 3 subimagens), elas serão numeradas 0, 1 e 2. Sintaxe: sprite_get_number(index); Argumento Descrição índice O índice do duende. Devoluções Inteiro Exemplo: image_index = floor(random(sprite_get_number(sprite_index))); O código acima definirá o índice de imagem para um quadro aleatório com base no número de sub-imagens que o sprite tem.
