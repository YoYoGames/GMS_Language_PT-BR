audio_emitter_get_z Esta função retorna a posição z atual do emissor de áudio dado. Sintaxe: audio_emitter_get_z(emissora); Argumento Descrição emissora O índice do emissor a ser utilizado. Devoluções: Real Exemplo: if audio_emitter_get_z(emitter_player) != 0 { var ex = audio_emitter_get_x(emissor_player); var ey = audio_emitter_get_y(emissor_player); audio_emitter_position(emissor_player, ex, ey, 0); } O código acima verifica a posição z atual de um determinado emissor e, se não for igual a 0, é ajustado para 0.
