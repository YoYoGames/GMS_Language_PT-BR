audio_emitter_create Esta função cria um novo emissor de áudio e retorna o índice para ele. Este índice deve ser armazenado em uma variável para todas as outras funções relacionadas a este emissor, e então, quando não for mais necessário, deve ser removido da memória usando o função audio_emitter_free() para evitar vazamentos de memória que podem acabar por travar seu jogo. Sintaxe: audio_emitter_create(); Devoluções: Número inteiro (identificação do emissor) Exemplo: s_emit = audio_emitter_create(); O código acima cria um novo emissor de áudio e atribui seu índice à variável "s_emit".
