audio_emitter_get_y Esta função retorna a posição y atual do emissor de áudio dado. Sintaxe: audio_emitter_get_y(emissora); Argumento Descrição emissora O índice do emissor a ser utilizado. Devoluções: Real Exemplo: if audio_emitter_get_y(emitter_player) != x { audio_emitter_position(emitter_player, x, y, 0); } O código acima verifica a posição y atual de um determinado emissor e, se não for igual à posição y da instância, é definido para a posição da instância.
