audio_listener_get_data Esta função criará um mapa DS e o preencherá com os valores de posição, velocidade e orientação para o ouvinte dado. O índice padrão do ouvinte é 0, mas você pode usar a função audio_get_listener_info() para obter os diferentes índices disponíveis para a plataforma alvo. Se você fornecer um índice de ouvinte incorreto, então a função retornará -1. NOTA: Você é responsável pela destruição do mapa DS devolvido usando a função apropriada. O mapa DS conterá as seguintes chaves: " x" - A posição x do ouvinte " y" - A posição y do ouvinte "z" - A posição z do ouvinte "vx" - A velocidade do eixo x do ouvinte "vy" - A velocidade do eixo y do ouvinte "vz" - A velocidade do eixo z do ouvinte "lookat_x" - O componente x do olhar para o vetor do ouvinte "lookat_y"- O componente y do olhar para o vetor do ouvinte "lookat_z" - O componente z do olhar para o vetor do ouvinte "up_x"- O componente x do vetor up do ouvinte " up_y"- O componente y do vetor up do ouvinte " up_z"- O componente z do vetor up do ouvinte Sintaxe: audio_listener_get_data(index); Argumento Descrição índice O ouvinte para obter os dados para (padrão 0).
