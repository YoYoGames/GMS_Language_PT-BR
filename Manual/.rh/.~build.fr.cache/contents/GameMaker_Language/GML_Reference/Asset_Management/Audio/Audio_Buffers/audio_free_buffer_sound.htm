audio_free_buffer_sound Com esta função você pode liberar o valor do índice do ponteiro associado à identificação do som. Os sons livres não estarão disponíveis para reprodução, e se várias instâncias do som estiverem sendo reproduzidas, todas elas serão interrompidas. Note que antes que você possa apagar o buffer em si, você deve primeiro liberar todas as identificações sonoras associadas a ele. Sintaxe: audio_free_buffer_sound(index); Argumento Descrição índice O índice do som amortecido para liberar. Devoluções: N/A Exemplo: audio_free_buffer_sound(soundID); O código acima libera o som de tamponamento indexado na variável "soundID".
