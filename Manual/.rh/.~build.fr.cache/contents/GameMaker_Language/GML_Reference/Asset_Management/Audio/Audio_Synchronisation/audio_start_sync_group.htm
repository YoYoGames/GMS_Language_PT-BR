audio_start_sync_group Com esta função, você pode começar a tocar um grupo de sincronização previamente criado. Você precisa fornecer o índice do grupo que é o valor retornado quando você criou o grupo usando a função audio_create_sync_group(). NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5. Sintaxe: audio_start_sync_group(group_index); Argumento Descrição group_index O índice do grupo para jogar. Devoluções: N/A Exemplo: sg = audio_create_sync_group(true); audio_play_in_sync_group(sg, sound1); audio_play_in_sync_group(sg, sound2); audio_sound_gain(sound2, 0, 0); audio_play_in_sync_group(sg, sound3); audio_sound_gain(sound3, 0, 0); audio_play_in_sync_group(sg, sound4); audio_sound_gain(sound4, 0, 0); audio_start_sync_group(sg); O acima exposto cria um novo grupo de sincronização e atribui o índice do grupo à variável "sg". Quatro sons são então adicionados ao grupo, com o ganho para três deles definidos como 0. Finalmente, o grupo de sincronização é tocado.
