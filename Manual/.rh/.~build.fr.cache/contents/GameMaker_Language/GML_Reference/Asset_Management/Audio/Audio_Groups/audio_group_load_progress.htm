audio_group_load_progress Esta função verificará o progresso de carga de um grupo de áudio e retornará um valor (aproximado) entre 0 e 100. Sintaxe: audio_group_load_progress(groupID); Argumento Descrição groupID O índice do grupo de áudio a verificar (como definido na Janela Grupos de Áudio) Devoluções: Real Exemplo: se carregar { var pc = audio_group_load_progress(audiogroup_level1); draw_text(32, 32, "Loading: " + string(pc)); } O código acima verifica uma variável e se ela retornar verdadeiro, então algum texto será desenhado mostrando o progresso do áudio sendo carregado.
