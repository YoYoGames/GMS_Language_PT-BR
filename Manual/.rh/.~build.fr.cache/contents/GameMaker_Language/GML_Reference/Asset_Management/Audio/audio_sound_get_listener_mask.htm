audio_sound_get_listener_mask Esta função retornará os dados da máscara de bits que define de quais ouvintes de áudio um som deve ser reproduzido. Consulte a seção sobre os ouvintes de áudio para mais informações. Sintaxe: audio_sound_get_listener_mask(soundID); Argumento Descrição soundID A identificação única do som para obter a máscara de Devoluções: Real Exemplo: var snd = audio_play_sound(snd_PlayerDead, 10, false); if audio_sound_get_listener_mask(snd) != global.PlayerMask { audio_sound_set_listener_mask(snd, global.PlayerMask); } O código acima toca um som e então verifica os dados da máscara do ouvinte quanto ao som, e se não for o mesmo que o armazenado em uma variável global, ele define o(s) ouvinte(s) para tocar a partir do uso dos dados da máscara armazenados na variável global.
