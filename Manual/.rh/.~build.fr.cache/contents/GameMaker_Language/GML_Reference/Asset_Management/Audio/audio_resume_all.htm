audio_resume_tudo Com esta função você pode retomar todos os sons que já foram pausados anteriormente. Sintaxe: audio_resume_all(); Devoluções: N/A Exemplo: se keyboard_check_pressed(ord("P")) { global.Pause = !global.Pause; if global.Pause { audio_pause_all(); } else { audio_resume_all(); } } O código acima verifica se a tecla do teclado "P" está pressionada e se detecta uma, define a variável global "Pausa" como verdadeira ou falsa e então ou pausa todos os sons ou reinicia tudo anteriormente sons pausados.
