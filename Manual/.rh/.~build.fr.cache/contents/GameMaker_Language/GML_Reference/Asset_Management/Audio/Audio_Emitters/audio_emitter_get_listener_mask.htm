audio_emitter_get_listener_mask Esta função retornará os dados da máscara de bits que define de quais ouvintes de áudio um emissor deve reproduzir os sons. Para mais informações, consulte a seção sobre os ouvintes de áudio. Sintaxe: audio_emitter_get_listener_mask(emitterID); Argumento Descrição emissorID A identificação única do emissor para obter a máscara de Devoluções: Real Exemplo: s_emit = audio_emitter_create(); if audio_emitter_get_listener_mask(s_emit) != global.PlayerMask { audio_emitter_set_listener_mask(snd, global.PlayerMask); } O código acima cria um emissor e então verifica os dados da máscara do ouvinte, e se não for o mesmo que o armazenado em uma variável global, ele define o(s) ouvinte(s) para jogar a partir do uso dos dados da máscara armazenados na variável global.
