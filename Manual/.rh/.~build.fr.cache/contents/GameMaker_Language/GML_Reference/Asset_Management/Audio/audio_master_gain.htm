audio_master_gain Com esta função você pode definir o valor absoluto para o volume global de todos os sons e músicas. É baseado em uma escala linear de 0 (silencioso) a qualquer valor maior que 0, embora normalmente você consideraria o volume máximo como 1. Qualquer coisa acima de 1 pode ser usado, mas, dependendo do som usado e da plataforma que está sendo compilada, você pode obter distorção ou recorte quando o som é reproduzido. Esta função afetará o volume relativo de todos os sons e músicas reproduzidos a partir de seu jogo. Sintaxe: audio_master_gain(gain); Argumento Descrição ganhar Valor para o volume global (0 a 1). Devoluções: N/A Exemplo: if keyboard_check(vk_up) { if vol < 1 vol += 0,05; audio_master_gain(vol); } if keyboard_check(vk_down) { if vol > 0 vol -= 0.05; audio_master_gain(vol); } O código acima verifica se as teclas das setas para cima e para baixo estão pressionadas, o que aumenta ou diminui a variável "vol".
