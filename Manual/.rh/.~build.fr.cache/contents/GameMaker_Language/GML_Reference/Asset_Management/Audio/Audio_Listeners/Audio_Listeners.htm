Ouvintes de áudio Ao lidar com situações complexas de áudio, pode ser útil definir onde no ambiente de áudio o ouvinte deve estar situado. Ao trabalhar com as funções de áudio simples, normalmente não é necessário que o ouvinte seja alterado de qualquer forma, no entanto, ao utilizar emissores para gerar áudio realista dentro do espaço de áudio 3D, é importante que o ouvinte seja trocado e movido corretamente, caso contrário, o som estará na posição errada para a pessoa que está tocando seu jogo. GameMaker Studio 2 fornece as seguintes funções de ouvinte: posição_de_ouvinte_audio audio_listener_velocity audio_listener_orientation audio_listener_get_data audio_listener_set_position audio_listener_set_velocity audio_listener_set_orientation Certas plataformas alvo podem permitir mais de um ouvinte, com cada uma correspondendo a um dispositivo de saída diferente, ou seja: sua plataforma alvo poderia permitir dois ouvintes, um para os fones de ouvido e outro para o monitor. Portanto, o GameMaker O Studio 2 fornece uma série de funções extras com as quais você pode encontrar os ouvintes disponíveis, assim como configurá-los para sons sendo tocados: audio_get_listener_count audio_get_listener_info audio_get_listener_mask audio_set_listener_mask Há também funções para obter e definir dados de máscara sobre sons individuais: audio_sound_get_listener_mask audio_sound_set_listener_mask Assim como para obter e definir dados de máscara sobre emissores: audio_emitter_get_listener_mask audio_emitter_set_listener_mask Voltar: Áudio A seguir: Grupos de Áudio © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
