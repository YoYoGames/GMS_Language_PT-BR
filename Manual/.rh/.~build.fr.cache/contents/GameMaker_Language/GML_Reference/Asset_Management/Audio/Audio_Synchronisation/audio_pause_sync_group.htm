audio_pause_sync_group sua função irá pausar o grupo de sincronização dado se estiver jogando, sendo o índice do grupo o valor retornado quando você criou o grupo usando a função audio_create_sync_group(). Isto não pára o som, e a vocação audio_resume_grupo_sync()A partir da mesma posição em que foi pausada novamente. NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5. Sintaxe: audio_pause_sync_group(group_index); Argumento Descrição group_index O índice do grupo a pausar. Devoluções: N/A Exemplo: if (keyboard_check_pressed((ord)"P")) { global.Pause = !global.Pause if global.Pause { audio_pause_sync_group(sg); } else { audio_resume_sync_group(sg); } } O código acima verifica se a tecla "P" está pressionada e, se for detectada, alterna a variável "global.Pause" e, em seguida, verifica-a para pausar ou retomar o grupo de sincronização indexado na variável "sg".
