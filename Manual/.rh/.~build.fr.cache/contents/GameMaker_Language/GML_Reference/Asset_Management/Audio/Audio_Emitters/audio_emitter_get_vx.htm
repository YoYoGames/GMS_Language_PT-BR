audio_emitter_get_vx Esta função retorna a velocidade da corrente ao longo do eixo x para o emissor de áudio dado. Sintaxe: audio_emitter_get_vx(emissora); Argumento Descrição emissora O índice do emissor a ser utilizado. Devoluções: Real Exemplo: if audio_emitter_get_vx(emitter_player) != 0 { audio_emitter_velocity(emissor_player, 0, 0, 0, 0); } O código acima verifica a velocidade atual x de um determinado emissor e, se não for igual a 0, é ajustado para 0.
