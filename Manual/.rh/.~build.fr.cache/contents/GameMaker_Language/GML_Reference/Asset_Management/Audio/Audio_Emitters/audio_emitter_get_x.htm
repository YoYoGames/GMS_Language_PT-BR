audio_emitter_get_x Esta função retorna a posição atual x do emissor de áudio dado. Sintaxe: audio_emitter_get_x(emissora); Argumento Descrição emissora O índice do emissor a ser utilizado. Devoluções: Real Exemplo: if audio_emitter_get_x(emitter_player) != x { audio_emitter_position(emitter_player, x, y, 0); } O código acima verifica a posição atual x de um determinado emissor e, se não for igual à posição x da instância, é definido para a posição da instância.
