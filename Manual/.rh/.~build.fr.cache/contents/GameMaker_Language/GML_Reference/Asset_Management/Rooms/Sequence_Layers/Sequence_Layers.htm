Sequência de camadas Nesta seção temos as funções que são usadas para adicionar, remover e editar como as seqüências se comportam em uma sala de jogos em relação à camada em que se encontram. Deve-se observar que Seqüências são semelhantes a objetos, na medida em que quando um é criado em uma camada em uma sala, é considerado uma instância do objeto de seqüência principal do Navegador de Ativos. Portanto, quando você adiciona uma seqüência a uma camada em uma sala, você está criando uma instância do objeto de seqüência base, e esta instância de seqüência é adicionada como um elemento dentro de uma camada da sala. A instância de seqüência é o que controla coisas como reprodução velocidade e direção, e ela mesma conterá outra estrutura de dados de seqüência que é o que contém os dados de seqüência real da pista. Isto é importante, pois as funções listadas abaixo farão referência ao elemento seqüencial ID (o ID da seqüência na camada), bem como a identificação da instância de seqüência (a identificação real da seqüência que está sendo referenciada pelo elemento) e a estrutura de dados da seqüência (que contém todos os dados da seqüência). Abaixo está uma lista de todas as funções que podem ser usadas para editar seqüências que foram adicionadas a uma camada de ativos: layer_sequence_exists layer_sequence_create layer_sequence_destroy layer_sequence_x layer_sequence_y layer_sequence_angle camada_seqüência_escala_x camada_seqüência_escala layer_sequence_headpos layer_sequence_headdir layer_sequence_pause layer_sequence_play escala_de_série_de_camadas layer_sequence_get_x layer_sequence_get_y layer_sequence_get_angle camada_seqüência_escala_get_x camada_seqüência_agenda_escala_de_série layer_sequence_get_headpos layer_sequence_get_headdir escala_de_série_de_série_de_camadas layer_sequence_get_length layer_sequence_get_instance layer_sequence_get_sequence layer_sequence_is_paused layer_sequence_is_is_finished Há também funções adicionais disponíveis para lidar com o conteúdo das seqüências e para definir suas próprias seqüências.
