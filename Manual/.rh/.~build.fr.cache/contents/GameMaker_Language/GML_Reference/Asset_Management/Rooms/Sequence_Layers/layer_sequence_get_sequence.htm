layer_sequence_get_sequence Com esta função você fornece a identificação do elemento de seqüência - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e ele retornará a estrutura do objeto da seqüência. Esta função supera a necessidade de primeiro obter a estrutura da instância de seqüência e permite o acesso direto aos dados da seqüência. Você pode saber mais sobre o formato da estrutura do objeto de seqüência nesta página. Sintaxe: layer_sequence_get_sequence(sequence_element_id) Argumento Descrição sequence_element_id O valor de identificação único do elemento de seqüência para o alvo Devoluções: Estrutura Exemplo: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); var _struct = layer_sequence_get_sequence(_seq); _struct.playbackSpeedType = spritespeed_framespersecond; _struct.playbackSpeed = 30; O código acima cria um novo elemento de seqüência e depois recupera sua estrutura de dados de seqüência. Isto é então usado para alterar a velocidade de reprodução e o tipo da seqüência.
