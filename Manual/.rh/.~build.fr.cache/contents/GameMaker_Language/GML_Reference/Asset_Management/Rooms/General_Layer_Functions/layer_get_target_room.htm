layer_get_target_room Esta função devolverá a sala atual que está sendo visada pelas funções da camada. Ver layer_set_target_room() para mais informações. Sintaxe: layer_get_target_room() Devoluções: Identificação do quarto (real) Exemplo: se layer_get_target_room() != quarto { layer_reset_target_room(); } O código acima verifica a sala de destino atual e se não for a sala atual, então a sala de destino da camada é reinicializada.
