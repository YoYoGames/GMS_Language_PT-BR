room_speed (depreciado) Esta variável mantém a velocidade de funcionamento de todas as salas (e do jogo) em quadros de jogo por segundo. Note que isto NÃO é o FPS (frames por segundo), mas sim o número de etapas do jogo que o GameMaker Studio 2 tentará manter em cada segundo. IMPORTANTE! Esta variável é mantida somente para o Legacy Support, e não deve ser usada, pois ela não define mais a velocidade para uma única sala, mas para TODAS as salas do jogo. Para mudar a velocidade do jogo você deve, ao invés disso, estar usando a função game_set_speed(). Sintaxe: room_speed; Devoluções: Real Exemplo: alarme[0] = room_speed * 10; O código acima usa a velocidade room_speed para definir um alarme para funcionar por 10 segundos. O uso da variável desta forma garantirá que o alarme funcione por 10 segundos, não importando a velocidade definida para a sala.
