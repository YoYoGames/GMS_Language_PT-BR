tilemap_x Esta função controla a posição ao longo do eixo x da sala do elemento do mapa de bens azulejos na camada. Você dá o ID do elemento do mapa de azulejos (que você obtém quando cria um elemento do mapa de azulejos usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e então definir o valor x a ser usado (com base nas coordenadas da sala). Sintaxe: tilemap_x(tilemap_element_id, x); Argumento Descrição tilemap_element_id O valor de identificação único do elemento do mapa de azulejos a ser mudado x A posição x para o mapa de azulejos Devoluções: N/A Exemplo: var lay_id = layer_get_id("Asset_sky"); var map_id = layer_tilemap_get_id(lay_id); tilemap_x(map_id, irandom(room_width)); O código acima obtém o valor de identificação do ativo do mapa de azulejos atribuído à camada "Asset_sky" e então define sua posição x para um valor aleatório entre 0 e a largura da sala.
