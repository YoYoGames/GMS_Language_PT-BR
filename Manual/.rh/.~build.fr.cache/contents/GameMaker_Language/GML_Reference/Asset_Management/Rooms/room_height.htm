altura_do_quarto Esta variável mantém a altura da sala atual em pixels. Você pode mudar esta variável para mudar a altura da sala a qualquer momento, e as mudanças serão aplicadas na parte inferior da sala, já que a origem é considerada como o canto superior esquerdo. Portanto, a mudança será feita no canto superior esquerdo, por exemplo, se a sala tiver 480px de altura e você a ajustar para 640px, a sala será expandida para baixo com um acréscimo de 180px a mais no fundo. Sintaxe: altura_do_quarto; Devoluções: Real Exemplo: se bbox_bottom > room_height { y += room_height - bbox_bottom; } O código acima verifica se a caixa de delimitação do duende da instância atual é maior que a altura da sala, e se é ela que move a instância para cima.
