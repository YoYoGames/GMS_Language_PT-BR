room_restart Esta função reiniciará a sala atual, como se ela tivesse acabado de ser inserida. Note que a sala não reiniciará até o final do evento onde a função foi chamada, portanto, qualquer código após esta ter sido chamada ainda será executado se no mesmo evento. O A ressalva a isto é que qualquer código que crie instâncias chamadas depois disto, não criará a instância e assim deve ser chamado antes do vestiário. Esta função também acionará o evento Room End. Sintaxe: room_restart(); Devoluções: N/A Exemplo: se vive < 1 quarto_restart(); O código acima verifica se a variável "vive" é inferior a 1 e se é ela irá reiniciar a sala.
