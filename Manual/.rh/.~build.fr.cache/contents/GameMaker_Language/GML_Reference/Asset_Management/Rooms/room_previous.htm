room_previous Com esta função você pode recuperar o índice da sala antes da entrada da sala na função. Por exemplo, você pode usar a variável sala para obter o índice da sala atual e depois usar esta função para encontrar a sala que vem antes dela, conforme listado no Room Manager. Se não houver sala antes da entrada, então -1 é devolvido. Note que esta função não irá reconhecer ou levar em consideração quartos que foram adicionados dinamicamente usando room_add() ou room_duplicate(). Sintaxe: room_previous(numb); Argumento Descrição dormente O índice da sala para obter o de antes. Devoluções: Índice Exemplo: if room_previous(room) != -1 { room_goto_previous()); } O código acima verificará se a sala anterior existe e, em caso afirmativo, irá até ela.
