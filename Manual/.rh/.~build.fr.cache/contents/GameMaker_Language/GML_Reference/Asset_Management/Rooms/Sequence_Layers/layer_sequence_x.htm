layer_sequence_x Com esta função você pode definir a posição ao longo do eixo X (horizontal) da sala para o elemento de seqüência dado. Você fornece o ID do elemento seqüencial conforme retornado por layer_sequence_create() ou por um do elemento camada funciona junto com a posição X a ser definida e a seqüência será movida para a nova posição. Sintaxe: layer_sequence_x(sequence_element_id, pos_x) Argumento Descrição sequence_element_id O valor de identificação único do elemento de seqüência para o alvo pos_x A posição X para mover o elemento de seqüência para Devoluções: N/A Exemplo: if layer_sequence_exists(my_seq) { layer_sequence_x(my_seq, x); layer_sequence_y(my_seq, y); } O código acima verifica se o elemento de seqüência referenciado na variável "my_seq" existe, e se ele define a posição x/y para a da instância que está executando o código.
