camada_seqüência_escala_get_x Com esta função você fornece a identificação do elemento de seqüência - como retornado por layer_sequence_create() ou por uma das funções dos elementos de camada - e retornará a escala atual ao longo do eixo X do elemento de seqüência na sala de jogos. Sintaxe: layer_sequence_get_xscale(sequence_element_id) Argumento Descrição sequence_element_id O valor de identificação único do elemento de seqüência para o alvo Devoluções: Real Exemplo: var _xs = layer_sequence_get_xscale(title_sequence) if _xs < 1 { _xs += 0,01; layer_sequence_xscale(title_sequence, _xs); } O código acima recupera a escala atual ao longo do eixo X do elemento de seqüência com o ID armazenado na variável "title_sequence", e se for menor que 1, então 0,01 é adicionado à escala X atual.
