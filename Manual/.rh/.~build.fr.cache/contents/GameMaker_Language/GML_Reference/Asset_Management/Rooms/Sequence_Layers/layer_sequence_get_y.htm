layer_sequence_get_y Com esta função você fornece a identificação do elemento de seqüência - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e retornará a posição Y atual na sala de jogo para a seqüência. Sintaxe: layer_sequence_get_y(sequence_element_id) Argumento Descrição sequence_element_id O valor de identificação único do elemento de seqüência para o alvo Devoluções: Real Exemplo: if layer_sequence_get_y(title_sequence) != room_height / 2 { layer_sequence_y(title_sequence, room_height / 2); } O código acima verifica a posição Y do elemento de seqüência ID armazenado na variável "title_sequence", e se não estiver definido para metade da altura da sala, está definido para este valor.
