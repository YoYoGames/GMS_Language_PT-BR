layer_get_hspeed Você pode usar esta função para recuperar a velocidade horizontal (em pixels por frame de jogo) da camada dentro da sala atualmente escopada. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como uma string - isto terá um impacto no desempenho) e a função retorna um número real para a velocidade horizontal, onde um valor positivo é para a direita e um valor negativo para a esquerda. O valor padrão é 0 (a menos que seja definido no editor da sala). Sintaxe: layer_get_hspeed(layer_id) Argumento Descrição layer_id O valor de identificação único da camada para obter a velocidade horizontal a partir Devoluções: N/A Exemplo: var lay_id = layer_get_id("Sprites"); if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0 { layer_hspeed(lay_id, 0); layer_vspeed(lay_id, 0); } O código acima verifica as velocidades horizontal e vertical da camada dada e, se ambas não estiverem ambas ajustadas a 0, então é ajustado para 0.
