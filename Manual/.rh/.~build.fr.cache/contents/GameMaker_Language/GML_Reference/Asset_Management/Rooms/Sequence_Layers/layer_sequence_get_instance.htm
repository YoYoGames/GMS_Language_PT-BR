layer_sequence_get_instance Com esta função você fornece a identificação do elemento de seqüência - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e ele retornará a estrutura da instância de seqüência. Você pode saber mais sobre o formato da estrutura da instância de seqüência nesta página. Sintaxe: layer_sequence_get_instance(sequence_element_id) Argumento Descrição sequence_element_id O valor de identificação único do elemento de seqüência para o alvo Devoluções: Estrutura Exemplo: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); var _struct = layer_sequence_get_instance(_seq); _struct.speedScale = 0,5; O código acima cria um novo elemento de seqüência e depois recupera sua identificação de instância de seqüência. Isto é então usado para alterar a propriedade da escala de velocidade de reprodução da instância de seqüência.
