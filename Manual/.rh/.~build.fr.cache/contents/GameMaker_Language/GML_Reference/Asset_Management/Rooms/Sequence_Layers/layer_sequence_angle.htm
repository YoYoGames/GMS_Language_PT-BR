layer_sequence_angle Com esta função você pode definir o ângulo de rotação para o elemento de seqüência dado. Você fornece o ID do elemento seqüencial conforme retornado por layer_sequence_create() ou por uma das funções dos elementos da camada junto com o novo ângulo a ser definido e a seqüência será girada sobre sua origem para a nova posição. Os ângulos no GameMaker são calculados com 0º para a direita, e vão no sentido anti-horário - assim 90º é para cima, 180º para a esquerda e 270º para baixo - e o ângulo padrão para uma seqüência seria de 0º. Note que você não deve alterar as variáveis image_xscale image_yscale image_angle x y para qualquer instância em uma seqüência que utiliza esta função, pois serão sobrescritas assim que a seqüência for atualizada após a chamada da função. Sintaxe: layer_sequence_angle(sequence_element_id, angle) Argumento Descrição sequence_element_id O valor de identificação único do elemento de seqüência para o alvo ângulo O novo ângulo para girar o elemento de seqüência para Devoluções: N/A Exemplo: if current_angle < 90 { current_angle += 1; layer_sequence_angle(my_seq, current_angle); } O código acima verifica o valor mantido na variável current_angle, e se for inferior a 90, então ele adiciona o valor para definir o ângulo do elemento de seqüência referenciado na variável my_seq.
