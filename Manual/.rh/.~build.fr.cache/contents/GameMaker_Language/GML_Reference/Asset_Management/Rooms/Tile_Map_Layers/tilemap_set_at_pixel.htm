mapa_do_cartão_de_fixos_na_pixel Esta função pode ser usada para definir uma célula dentro do elemento do mapa de azulejos na camada para um novo azulejo usando a posição real do azulejo para mudar dentro da sala. Você dá o ID do elemento do mapa de azulejo (que você obtém quando cria um elemento do mapa de azulejo usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e depois os dados dos azulejos a serem fixados, bem como a posição dentro da sala. Você pode recuperar os dados dos azulejos usando a função tilemap_get_at_pixel() e depois usar as funções tile_ para mudá-lo antes de configurá-lo novamente usando esta função. A função voltará a ser verdadeira se o azulejo foi ajustado com sucesso e falsa se houve um problema e ele não foi ajustado. Sintaxe: tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y) Argumento Descrição tilemap_element_id O valor de identificação único do elemento do mapa de azulejos a ser mudado tiledata Os dados do conjunto de azulejos a definir xcell A posição x (na sala) ycell A posição y (na sala) Devoluções: Booleano Exemplo: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var data = tilemap_get_at_pixel(map_id, 4, 4); data = tile_set_flip(data, true); tilemap_set_at_pixel(map_id, data, 4, 4); O código acima obtém a identificação para o mapa de azulejos na camada "Tiles_Sky" e depois a usa para obter os dados do azulejo na posição (4, 4). Estes dados do azulejo são então virados antes de serem usados para colocar o azulejo novamente no mapa de azulejos.
