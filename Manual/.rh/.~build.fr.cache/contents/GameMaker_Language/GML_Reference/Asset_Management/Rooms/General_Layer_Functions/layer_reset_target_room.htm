layer_get_target_room Esta função é usada para repor o alvo da camada para a sala atual. Veja a função layer_set_target_room() para maiores informações. Sintaxe: layer_reset_target_room() Devoluções: N/A Exemplo: layer_set_target_room(rm_Game); var l = layer_get_id("SpriteAssets"); repeat(50) { layer_sprite_create(l, random(room_get_width(rm_Game)), random(room_get_height(rm_Game)), spr_Trees); } layer_reset_target_room(); O código acima define a sala alvo para a sala "rm_Game" e então obtém a identificação da camada chamada "SpriteAssets" naquela sala. Este ID de camada é então usado para adicionar 50 ativos sprite aleatórios à camada, antes do alvo da camada é reajustado para a sala atual.
