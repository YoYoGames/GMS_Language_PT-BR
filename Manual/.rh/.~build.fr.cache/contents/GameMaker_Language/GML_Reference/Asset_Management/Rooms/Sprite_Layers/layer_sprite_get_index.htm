layer_sprite_get_index Esta função pode ser usada para obter o valor atual do índice de imagem do elemento sprite. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usar a função layer_sprite_get_id()), e a função retornará um valor real que representa o índice de imagem que está sendo mostrado para o duende. A função retornará -1 se ou o elemento sprite não existir ou o elemento não tiver um sprite válido atribuído a ele. Sintaxe: layer_sprite_get_index(sprite_element_id); Argumento Descrição sprite_element_id O valor de identificação único do elemento sprite para obter as informações de Devoluções: Real (o índice de imagem sprite atual ou -1) Exemplo: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_index(spr_id) < 4 { layer_sprite_index(spr_id, 4); } O código acima obterá a identificação da camada chamada "sprite_sky" e depois a utilizará para obter a identificação do elemento sprite naquela camada. Este ID é então usado para verificar se o índice de imagem para o elemento é inferior a 4, e se assim for definido a 4.
