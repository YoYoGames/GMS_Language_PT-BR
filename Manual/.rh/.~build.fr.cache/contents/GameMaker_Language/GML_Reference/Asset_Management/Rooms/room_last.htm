quarto_last Esta variável lida apenas retorna o índice da última sala do jogo (isto é definido pela ordem em que as salas aparecem no Room Manager e não pela ordem em que eles foram criados). Note que esta variável não reconhecerá ou levará em consideração salas que tenham sido adicionadas dinamicamente usando room_add() ou room_duplicate(). Sintaxe: quarto_last; Devoluções: Índice Exemplo: se keyboard_check_pressed(ord("Q")) { room_goto(room_last); } O código acima verifica se uma tecla foi pressionada e se sim vai para a última sala do jogo.
