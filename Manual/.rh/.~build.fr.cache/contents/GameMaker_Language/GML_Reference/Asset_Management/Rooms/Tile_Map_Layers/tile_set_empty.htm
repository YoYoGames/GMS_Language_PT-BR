tile_set_empty Esta função pode ser usada para definir um determinado conjunto de dados de azulejos como sendo um azulejo vazio. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e a função voltará a ser verdadeira se o azulejo está vazio, ou falso, se houver um índice de azulejo. A função retornará um conjunto modificado de dados de azulejos que pode então ser aplicado usando o tilemap_set() função. Sintaxe: tile_set_empty(tiledata) Argumento Descrição tiledata os dados da telha a definir Devoluções: Real (tileata) Exemplo: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); for (var i = 0; i < tilemap_get_width(map_id); i++;) { for (var j = 0; j < tilemap_get_height(map_id); j++;) { var data = tilemap_get(map_id, i, j); if !tile_get_empty(data) { data = tile_set_empty(data) tilemap_set(map_id, data, i, j); } } } O código acima obtém o ID do mapa de azulejos da camada dada e então procede à verificação de cada célula do mapa de azulejos para ver se ela tem ou não dados.
