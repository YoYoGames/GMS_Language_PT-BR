layer_sprite_get_y Esta função pode ser usada para obter a posição y do elemento sprite na sala. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usar a função layer_sprite_get_id()), e a função retornará o valor da posição y. Sintaxe: layer_sprite_get_y(sprite_element_id); Argumento Descrição sprite_element_id O valor de identificação único do elemento sprite para obter as informações de Devoluções: Real Exemplo: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_y(spr_id) < 0 { layer_sprite_y(spr_id, room_height); } O código acima obterá a identificação da camada chamada "sprite_sky" e depois a utilizará para obter a identificação do elemento sprite naquela camada. Este ID é então usado para verificar a posição do elemento y e se for inferior a 0, então o elemento da camada é movido para uma posição y diferente.
