fundo_da_camada_da_velocidade_do_get Esta função pode ser usada para obter o valor atual do multiplicador de velocidade do elemento de fundo. Você dá o ID do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e a função retornará um valor real que representa o multiplicador de velocidade que está sendo usado para animar o duende. O valor padrão é 1. Sintaxe: layer_background_get_speed(background_element_id); Argumento Descrição background_element_id O valor único de identificação do elemento de fundo para obter as informações de Devoluções: Real Exemplo: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_speed(back_id) > 0 { layer_background_speed(back_id, 0); } O código acima obterá a identificação da camada chamada "Background_sky" e depois a utilizará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para verificar a velocidade de animação do elemento e se ele é maior que 0, está ajustado a 0.
