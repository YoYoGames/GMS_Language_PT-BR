layer_get_all Esta função retornará uma matriz preenchida com os valores de identificação exclusivos de cada camada da sala. Sintaxe: layer_get_all() Devoluções: Array (1D, povoado com identificações por camadas) Exemplo: var a = layer_get_all(); for (var i = 0; i < array_length(a); i++;) { layer_destroy(a[i]); } O código acima recupera todas as camadas em uma sala e acrescenta seus valores de identificação a uma matriz.
