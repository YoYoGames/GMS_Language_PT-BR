layer_background_get_blend Esta função pode ser usada para obter a cor da mistura do elemento de fundo. Você dá a identificação do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e a função retornará um valor real que representa a cor atribuída. Sintaxe: layer_background_get_blend(background_element_id); Argumento Descrição background_element_id O valor único de identificação do elemento de fundo para obter as informações de Devoluções: Valor da cor (Real) Exemplo: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_blend(back_id) == c_white { layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255)); } O código acima obterá a identificação da camada chamada "Background_sky" e depois a utilizará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para verificar a cor da mistura do elemento e se ela é equivalente à constante c_branco, então a mistura de camadas é definida para uma cor aleatória.
