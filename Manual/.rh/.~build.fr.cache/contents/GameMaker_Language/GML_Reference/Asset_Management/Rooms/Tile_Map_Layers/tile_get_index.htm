tile_get_index Esta função pode ser usada para obter o índice de azulejos (a posição do azulejo dentro da imagem do conjunto de azulejos) a partir de um conjunto de dados de azulejos. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e a função retornará um valor inteiro para o índice ou -1 se houver um erro. Sintaxe: tile_get_index(tiledata) Argumento Descrição tiledata os dados dos azulejos a serem verificados Devoluções: Inteiro Exemplo: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var ind = tile_get_index(data); data = tile_set_index(data, irandom(23)); tilemap_set(map_id, data, mx, my); O código acima obtém o ID do mapa de azulejos da camada dada e depois usa-o para obter os dados do azulejo para a célula sob a posição do mouse. Estes dados são então usados para definir o índice de azulejos para um número aleatório.
