room_goto_previous Com esta função você pode fazer seu jogo ir para o anterior, conforme listado no Gerente de Sala no momento em que o jogo foi compilado. Se esta sala não existir, um erro será lançado e o jogo será forçado a fechar. Note que a sala não mudará até o final do evento onde a função foi chamada, portanto, qualquer código após esta ter sido chamada ainda será executado se no mesmo evento. A ressalva a isto é que qualquer código que crie instâncias que é chamado depois disto, não criará a instância e assim deve ser chamado antes do vestiário. Esta função também acionará o evento Room End. Sintaxe: room_goto_previous(); Devoluções: N/A Exemplo: se room_exists(room_previous(room)) { room_goto_previous(); } O código acima verificará se há outra sala antes da atual e, se houver, irá para aquela sala.
