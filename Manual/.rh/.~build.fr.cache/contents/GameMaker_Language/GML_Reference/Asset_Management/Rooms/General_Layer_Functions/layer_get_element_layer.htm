layer_get_element_layer Você pode usar esta função para obter o Layer ID que o elemento dado está ligado. Você fornece o valor único de identificação do elemento (por exemplo, como retornado pela função que criou o elemento ou pelo editor da sala) e a função retornará o valor único de identificação do elemento (por exemplo, como retornado pela função que criou o elemento ou pelo editor da sala). Identificação da camada em que o elemento é encontrado. Se a identificação do elemento dado não for válida, então a função retornará -1. Sintaxe: layer_get_element_layer(element_id) Argumento Descrição element_id O valor de identificação único do elemento para obter a camada de Devoluções: ID da camada ou -1 se o elemento for inválido (ou seja: não existe) Exemplo: element = layer_get_element_layer(asset_1); O código acima obtém o ID da camada para o elemento com o ID armazenado na variável "asset_1" e o armazena em uma variável.
