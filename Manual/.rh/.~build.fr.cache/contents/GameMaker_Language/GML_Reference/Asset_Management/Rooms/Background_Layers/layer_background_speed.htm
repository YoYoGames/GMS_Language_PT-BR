fundo_da_camada_velocidade Esta função pode ser usada para definir a velocidade de um duende animador que tenha sido atribuído a um elemento de fundo. Você dá a identificação do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e então definir a velocidade da animação. Se a IDE definiu a velocidade de animação para o duende costumava ser calculada como Frames Por Frame de Jogo, então o valor da velocidade dada é um multiplicador, sendo 1 o valor padrão, e definindo-o como 0,5 será metade da velocidade da animação, enquanto que definindo-o como 2 será o dobro. Entretanto, se a IDE tiver definido a velocidade de animação para o sprite usado para ser calculado como Frames Por Segundo, então o valor da velocidade dada é um valor literal de frames por segundo, então definindo este valor como 5, por exemplo, definirá a animação como loop 5 frames a cada segundo que o jogo for executado. Se o elemento de fundo não tiver sprite anexado ou o duende utilizado não tem sub-imagens, esta função não terá efeito. Sintaxe: layer_background_speed(background_element_id, speed); Argumento Descrição background_element_id O valor de identificação único do elemento de fundo a ser mudado velocidade O multiplicador de velocidade/valor a ser usado para o elemento Devoluções: N/A Exemplo: var lay_id = layer_get_id("Background_id_water"); var back_id = layer_background_get_id(lay_id); layer_background_speed(back_id, 0.5); O código acima obterá a identificação da camada chamada "Background_water" e depois a utilizará para obter a identificação do elemento de fundo nessa camada.
