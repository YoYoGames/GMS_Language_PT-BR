layer_is_draw_depth_forced Esta função pode ser usada para recuperar o conjunto de profundidade Z para as camadas de renderização dentro da sala. Veja layer_force_draw_depth() para mais informações. Sintaxe: layer_get_forced_depth() Devoluções: Real Exemplo: se layer_get_forced_depth() != 0 { layer_force_draw_depth(true, 0); } O código acima verifica se a profundidade da camada Z é forçada a um valor 0 ou não e, se não for, define a profundidade da camada Z como 0.
