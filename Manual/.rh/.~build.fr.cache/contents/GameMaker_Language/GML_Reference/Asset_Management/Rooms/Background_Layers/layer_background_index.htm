layer_background_index Esta função pode ser usada para definir o índice de imagem do sprite de fundo que tem múltiplas subimagens. Você dá o ID do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e depois definir o índice de imagem a ser utilizado. Se você definir um valor fora do intervalo de sub-imagens, então o índice de imagem irá circular em volta. Sintaxe: layer_background_index(background_element_id, image_index); Argumento Descrição background_element_id O valor de identificação único do elemento de fundo a ser mudado índice O índice de imagem a utilizar para o fundo Devoluções: N/A Exemplo: var lay_id = layer_get_id("Background_trees"); var back_id = layer_background_get_id(lay_id); layer_background_index(back_id, 1); O código acima obterá a identificação da camada chamada "Background_trees" e depois a utilizará para obter a identificação do elemento de fundo nessa camada.
