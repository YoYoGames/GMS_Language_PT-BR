layer_get_name Você pode usar esta função para obter o nome da camada dada. Você fornece o valor único de identificação da camada e se a camada for uma das camadas nomeadas criadas no editor da sala, então a função retornará uma string com o nome da camada. Se a camada não é um dos editores da sala (ou seja: foi criado usando layer_create()) então um fio vazio será devolvido.
