font_get_italic Com esta função você pode verificar qualquer fonte ativa para ver se ela tem ou não a bandeira itálica. Se ele fizer a função retornará verdadeiro, caso contrário, retornará falso. Sintaxe: font_get_italic(ind); Argumento Descrição ind Índice da fonte a ser verificada. Devoluções: Booleano Exemplo: if font_get_italic(fnt_Main) { draw_set_font(fnt_Main); } Isto definirá a fonte de desenho ativa para fnt_Main se ela for definida como itálica em suas propriedades de fonte.
