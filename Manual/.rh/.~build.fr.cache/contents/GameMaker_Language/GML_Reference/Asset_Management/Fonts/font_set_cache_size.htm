font_set_cache_size_size Ao realizar a mistura e coloração de imagens, o HTML5 não pode fazê-lo dinamicamente da mesma forma que um executável poderia ser executado. Portanto, o GameMaker Studio 2 tem que salvar temporariamente uma cópia misturada das imagens e carregar quando necessário. Esta função define quantas cópias mescladas da fonte dada podem ser colocadas em cache antes que as antigas sejam sobrescritas. O valor padrão é 4. NOTA: para as fontes sprite você deve estar usando a função equivalente para sprites, sprite_set_cache_size(). Sintaxe: font_set_cache_size_size(ind, max); Argumento Descrição ind O índice da fonte para alterar o tamanho do cache. max O número máximo de cópias em cache da fonte que podem ser armazenadas. Devoluções: N/A Exemplo: font_set_cache_size(fnt_MainMenu, 2); Isto irá definir o cache da fonte indexada na variável "fnt_MainMenu" para 2 cópias.
