font_get_name Esta função pode ser usada para obter o nome (como um fio) que foi dado à fonte quando ela foi adicionada como um ativo ao GameMaker Studio 2 Asset Browser. Observe que este é apenas um e não pode ser usado para referenciar diretamente a fonte - para isso você precisaria do índice de fontes. Você pode, entretanto, usar esta string para obter o índice da fonte usando a string retornada junto com a função asset_get_index(). Sintaxe: font_get_name(ind); Argumento Descrição ind Índice da fonte a ser verificada. Devoluções: Cordão Exemplo: fnt_Name = font_get_name(font0); O código acima receberá o nome de um recurso de fonte como aparece no Navegador de Ativos e o armazenará como uma string na variável "fnt_Name".
