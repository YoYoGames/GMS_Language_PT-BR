font_get_bold Com esta função você pode verificar qualquer recurso de fonte para ver se ele tem ou não a bandeira em negrito. Se ele fizer a função retornará verdadeiro, caso contrário, retornará falso. Sintaxe: font_get_bold(ind); Argumento Descrição ind Índice da fonte a ser verificada. Devoluções: Booleano Exemplo: if font_get_bold(fnt_Main) { draw_set_font(fnt_Main); } Isto definirá a fonte de desenho ativa para fnt_Main se ela for definida como ousada em suas propriedades de fonte.
