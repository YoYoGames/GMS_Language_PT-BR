font_addd_get_enable_aaa Esta função pode ser usada para verificar se o anti-aliasing (AA) está habilitado para fontes adicionadas usando font_add(). A função retornará verdadeira se AA for ativado, e falsa se não é. Note que AA é ativado por padrão, mas você pode mudar o estado de AA para fontes adicionadas usando a função font_add_enable_aaa()desde que seja chamado antes de adicionar a fonte. Sintaxe: font_addd_get_enable_aaa(); Devoluções: Booleano Exemplo: if !font_addd_get_enable_aaa() { font_add_enable_aaa(true); } O código acima verifica o status do anti-aliasing para fontes adicionadas e se ele não estiver ativado, então nós o habilitamos.
