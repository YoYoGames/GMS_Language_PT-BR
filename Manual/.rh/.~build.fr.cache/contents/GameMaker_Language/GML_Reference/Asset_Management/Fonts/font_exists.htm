font_existe Esta função retorna se existe ou não uma fonte com o índice especificado. Você pode verificar índices de fontes como definidos no Navegador de Ativos, ou fontes que tenham sido adicionadas usando funções como font_add(). Sintaxe: font_exists(ind); Argumento Descrição ind Índice da fonte a ser verificada. Devoluções: Booleano Exemplo: se font_existir(fnt_Main) { draw_set_font(fnt_Main); } Isto definirá a fonte de desenho ativa para fnt_Main se ela existir.
