highscore_name Com esta função você pode recuperar a cadeia de nomes que foi armazenada na lista de pontuação alta na posição determinada. Se nenhum nome tiver sido inserido, a cadeia de nomes "Desconhecido" será devolvida. Sintaxe: highscore_name(lugar); Argumento Descrição lugar O lugar sobre a mesa (1-10). Devoluções: Cordão Exemplo: var i = 9; repeat(10) { nome[i] = highscore_name(i + 1); i -= 1; } O código acima irá percorrer a lista de pontuação alta e armazenar todos os nomes em uma matriz.
