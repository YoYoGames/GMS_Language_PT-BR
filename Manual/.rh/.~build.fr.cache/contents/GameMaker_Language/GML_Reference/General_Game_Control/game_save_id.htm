game_save_id Esta variável somente leitura retornará a identificação completa do caminho do diretório que é usado por seu jogo para salvar arquivos. Este diretório pode ou não ser visível para outras aplicações, dependendo da plataforma, e do alvo HTML5, ele retornará um fio vazio. NOTA: Esta variável é meramente informativa e não deve ser usada para obter um caminho de arquivo ou qualquer outra coisa semelhante, pois não funcionará corretamente em todas as plataformas. Sintaxe: game_save_id Devoluções: Cordão Exemplo: save_dir = jogo_save_id; Isto irá armazenar o diretório para salvar arquivos em uma variável para uso futuro.
