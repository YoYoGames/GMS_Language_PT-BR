nome_do_projeto_do_jogo Esta variável lida apenas retorna o nome de exibição de seu jogo para a plataforma alvo em um formato "save-friendly" para a plataforma alvo. Se o nome de exibição contiver quaisquer caracteres que não sejam permitidos para um nome de arquivo, eles ser substituído automaticamente por "_". O nome do visor pode ser definido nas Opções de Jogo. Note que como não há restrições de nomes de arquivo para HTML5, esta string provavelmente será a mesma como aquela devolvida por nome_do_jogo_display. Sintaxe: nome_do_projeto_do_jogo Devoluções: Cordão Exemplo: var file = game_project_name + ".ini"; ini_open(file); ini_write_real("Pontuação", "Pontuação", pontuação); ini_close(); O código acima recebe o nome do projeto e o usa para abrir (ou criar) um arquivo ini que é então escrito.
