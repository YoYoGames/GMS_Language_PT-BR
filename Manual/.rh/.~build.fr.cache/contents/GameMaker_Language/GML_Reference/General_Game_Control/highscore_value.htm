highscore_value Com esta função você pode recuperar o valor da pontuação que foi armazenado na lista de pontuação alta na posição determinada. Se nenhuma pontuação tiver sido inserida, a função retornará 0. Sintaxe: highscore_value(local); Argumento Descrição lugar O lugar sobre a mesa (1-10). Devoluções: N/A Exemplo: var i = 9; repeat(10) { scr[i] = highscore_value(i + 1); i -= 1; } O código acima irá percorrer a lista de pontuação alta e armazenar todas as pontuações em uma matriz.
