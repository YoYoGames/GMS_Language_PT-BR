is_string Esta função retorna se uma determinada variável é um número de string ou não. Em alguns casos você quer verificar e ver se uma variável no GameMaker Studio 2 contém uma string e não uma real e é quando você usaria esta função. Ela não retorna o fio, mas sim verdadeiro ou falso, portanto, um valor de, por exemplo, "peixe" para n retornará verdadeiro, mas um valor de 200 por n retornará falso. Sintaxe: is_string(n); Argumento Descrição n O argumento para verificar. Devoluções: Booleano Exemplo: if is_string(val) { name = "Player: " + val; } O código acima verifica a variável "val" para ver se ela contém uma string e se ela a adiciona a outra string que é então atribuída à variável "name".
