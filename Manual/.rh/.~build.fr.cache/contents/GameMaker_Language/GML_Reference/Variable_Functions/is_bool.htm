is_bool Esta função retorna se uma determinada variável é booleana(verdadeira ou falsa) ou não. Em alguns casos você deseja verificar e ver se uma variável no GameMaker Studio 2 contém um booleano valor, e é quando você usaria esta função. Sintaxe: is_bool(n); Argumento Descrição n O argumento para verificar. Devoluções: Booleano Exemplo: if is_bool(val) { global.Sound = val; } mais { global.Sound = true; } O código acima verifica a variável "val" para ver se é um número real e se é um número global.
