ptr Esta função tentará converter um determinado valor em um tipo de dado ponteiro, onde o valor deve ser ou um real, um string, um int64, um int32, ou um ptr. Qualquer outra coisa fará com que o jogo trave com uma mensagem de erro. Sintaxe: ptr(n); Argumento Descrição n O valor a converter. Devoluções: Apontador Exemplo: se !is_ptr(val) { val = ptr(application_surface); } O código acima verifica a variável "val" para ver se ela contém um ponteiro e se não contém, então um é atribuído a ela.
