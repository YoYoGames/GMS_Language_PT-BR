array_pop Com esta função você pode recuperar um valor do final de uma matriz sem ter que saber o comprimento da matriz. Você fornece a variável que contém o array, e a função retornará o conteúdo da última entrada no array. Se o array for vazia, a função retornará undefined. Sintaxe: array_pop(variável); Argumento Descrição variável A variável que sustenta a matriz. Devoluções: Valor (qualquer tipo de dado) ou indefinido Exemplo: var _lastscore = array_pop(score_array); draw_text(32, 32, "Last Score = " + string(_lastscore)); O código acima irá estourar o último valor da matriz dada e então o desenhará junto com algum texto para a tela.
