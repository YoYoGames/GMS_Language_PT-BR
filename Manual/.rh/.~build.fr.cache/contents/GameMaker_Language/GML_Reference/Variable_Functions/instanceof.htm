exemplo de Esta função pode ser usada para obter o nome da função que foi usada para criar uma estrutura quando a estrutura foi criada usando uma função construtora e o novo operador. Você fornece a variável com a referência estrutural para verificação e a função retornará ou uma string com o nome da função ou indefinida. Observe que se você passar a função uma estrutura literal (ou seja: uma estrutura que foi criada sem utilizar uma função construtora) então ela simplesmente retornará a string "estrutura". Esta função também pode ser usada para verificar se uma referência estrutural é uma referência fraca ou não, neste caso a função retornará a string "weakref" em vez do nome da função que criou a estrutura para mais informações, veja a função weak_ref_create()). Sintaxe: instanceof(struct_id); Argumento Descrição estrutura A referência estrutural a ser usada.
