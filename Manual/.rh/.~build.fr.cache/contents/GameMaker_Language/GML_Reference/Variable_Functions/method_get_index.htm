método_get_index Com esta função você pode recuperar o índice do Script para o script onde o método foi definido. Se o método não foi definido em um script, então a função retornará -1, caso contrário, ele retornará o valor do índice para o roteiro. Sintaxe: método_get_index(método); Argumento Descrição método O método variável a ser verificado Devoluções: Inteiro Exemplo: var _index = método_get_index(light_setup); if _index != -1 { show_debug_message(script_get_name(_index)); } O código acima verificará o índice do script para o método "light_setup" e então, se não for -1, emitirá o nome do script para o console.
