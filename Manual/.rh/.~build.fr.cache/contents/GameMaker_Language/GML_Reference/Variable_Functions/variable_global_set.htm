variável_global_set Com esta função você pode definir o valor de uma determinada variável global. Você fornece o nome da variável global para definir o valor de uma string (ver código de exemplo abaixo), e depois o valor a ser definido (pode ser qualquer tipo de dado válido). Se a variável global ainda não existir no jogo, ela será criada e então lhe será atribuído o valor. Sintaxe: variável_global_set(nome, val); Argumento Descrição nome O nome da variável global a ser definida (como uma string) val O valor para definir a variável global para Devoluções: N/A Exemplo: se !variable_global_exists("enemy_num") { variable_global_set("enemy_num", 0); } O código acima verificará se existe uma variável global e se ela não existe, então ela é criada e definida como 0.
