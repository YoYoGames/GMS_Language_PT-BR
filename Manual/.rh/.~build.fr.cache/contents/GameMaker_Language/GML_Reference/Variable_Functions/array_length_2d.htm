array_length_2d(Deprecated) ADVERTÊNCIA! Esta função é depreciada, pois as matrizes não são mais limitadas a apenas 1 ou 2 dimensões, e como tal, esta função é fornecida apenas para apoio a projetos legados. Todos os novos projetos devem usar a forma atual de criar e acessar projetos multi-dimensionais arrays (veja aqui para mais informações). Com esta função você pode obter o comprimento (número de entradas) de uma segunda dimensão de uma matriz. Você fornece o número de entradas para a primeira dimensão e a função retornará o número de entradas da segunda dimensão que a matriz tem (para encontrar o comprimento da primeira dimensão utilizar a função array_height_2D()). A função retornará 0 se a variável dada não for um array ou 1 se a variável for um array 1D (já que ainda há 1 linha). Sintaxe: array_length_2d(array_index, n); Argumento Descrição array_index O índice da matriz a ser verificada. n A entrada da matriz para obter o comprimento de. Devoluções: Real Exemplo: para (var i = 0; i < altura_da_matriz_2d(a); ++i;) { for (var j = 0; j < array_length_2d(a, i); ++j;) { a[i, j] = -1; } } O código acima passará por uma matriz 2D e definirá cada entrada como -1.
