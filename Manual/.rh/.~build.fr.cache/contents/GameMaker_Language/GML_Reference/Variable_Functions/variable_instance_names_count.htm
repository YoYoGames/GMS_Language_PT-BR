nomes_de_instância_variável Com esta função você pode encontrar o número total de variáveis definidas para uma instância. Você fornece o ID da instância para verificar, e a função retornará um valor inteiro para o número de variáveis encontradas, ou (se nenhuma instância do dado Existe identificação) -1. Sintaxe: variável_instance_names_count(instance_id); Argumento Descrição exemplo_id O valor único de identificação da instância a ser verificada. Devoluções: Inteiro Exemplo: var _num = variable_instance_names_count(myinst); show_debug_message("Instance Variables = " + string(_num)); O código acima recuperará o número de variáveis na instância dada e mostrará uma mensagem de debug na saída do console com esse valor.
