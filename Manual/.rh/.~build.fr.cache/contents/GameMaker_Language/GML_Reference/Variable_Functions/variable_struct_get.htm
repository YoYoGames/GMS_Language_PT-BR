variável_struct_get Com esta função você pode obter o valor de uma determinada variável nomeada dentro de uma estrutura. Você fornece a referência da estrutura bem como o nome da variável para obter o valor de uma string (veja o código de exemplo abaixo). A função irá retornar o valor detido pela variável ou indefinido se a variável nomeada não existir. IMPORTANTE! Se a variável que você está recebendo não existir, então a função retornará a palavra-chave indefinida e você poderá receber erros que impedirão o funcionamento do jogo, portanto, em caso de dúvida, use a função variável_struct_exists() primeiro.
