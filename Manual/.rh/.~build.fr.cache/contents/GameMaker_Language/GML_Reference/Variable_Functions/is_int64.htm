is_int64 Esta função retorna se uma determinada variável é um inteiro de 64 bits ou não. Em alguns casos, você quer verificar e ver que tipo de dado uma variável contém no GameMaker Studio 2 e é quando você usaria esta função. Ela retorna verdadeiro ou falso, dependendo se o valor é um int64 ou não. Sintaxe: is_int64(n); Argumento Descrição n O argumento para verificar. Devoluções: Booleano Exemplo: if !is_int64(val) { show_debug_message("Not a 64bit integer!"); } O código acima verifica a variável "val" para ver se ela contém uma int64 e se não contém, então ela mostra uma mensagem no console de depuração.
