array_length Com esta função você pode obter o comprimento (número de entradas) de uma dimensão de matriz. Você fornece o valor do índice da matriz e a função retornará um valor inteiro representando o número de entradas que a matriz contém. Esta função também pode ser usada para matrizes multidimensionais, desde que você especifique qual dimensão você deseja obter o comprimento de quando você fornece o índice da matriz, seguindo este padrão: // Retorna a primeira dimensão da matriz val = array_length(my_array); // Retorna a segunda dimensão da matriz val = array_length(my_array[0]); // Retorna a terceira dimensão da matriz val = array_length(my_array[0][0]); // etc... Sintaxe: array_length(array_index); Argumento Descrição array_index O índice da matriz a ser verificada. Devoluções: Inteiro Exemplo: para (var i = 0; i < array_length(a); ++i;) { a[i] = -1; } O código acima passará por uma matriz e definirá cada entrada como -1.
