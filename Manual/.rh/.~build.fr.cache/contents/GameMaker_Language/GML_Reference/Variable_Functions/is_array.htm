is_array Esta função pode ser usada para verificar e ver se uma variável contém uma matriz (ela retornará verdadeira) ou não (neste caso, retornará falsa). Sintaxe: is_array(n); Argumento Descrição n A variável a ser verificada. Devoluções: Booleano Exemplo: if is_array(a) { a = -1; } O código acima verifica uma variável para ver se ela é um array, e se a função retorna verdadeiro, o array é apagado definindo a variável como -1.
