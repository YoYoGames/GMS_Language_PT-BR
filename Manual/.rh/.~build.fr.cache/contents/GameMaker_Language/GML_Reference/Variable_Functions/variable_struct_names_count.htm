nomes_de_estrutura_variável Com esta função você pode encontrar o número total de variáveis definidas para uma estrutura. Você fornece o ID da estrutura para verificar, e a função retornará um valor inteiro para o número de variáveis encontradas, ou (se não existir uma estrutura do ID dado) -1. Sintaxe: variável_nomes_de_estrutura_conta(struct_id); Argumento Descrição struct_id O valor único de identificação da estrutura a ser verificada. Devoluções: Inteiro Exemplo: var _num = variable_struct_names_count(mystruct); show_debug_message("Struct Variables = " + string(_num)); O código acima recuperará o número de variáveis na estrutura dada e mostrará uma mensagem de debug na saída do console com este valor.
