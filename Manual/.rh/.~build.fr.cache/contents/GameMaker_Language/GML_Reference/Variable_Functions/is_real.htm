é_real Esta função retorna se uma determinada variável é um número real (simples, duplo ou inteiro) ou não. Em alguns casos você quer verificar e ver se uma variável possui um número real, e é quando você usaria esta função. Ela não retorna o número real, mas sim verdadeiro ou falso, portanto um valor de, por exemplo, "peixe" retornaria falso, porém um valor de 200 retornaria verdadeiro. NOTA: Esta função retornará falso para quaisquer valores enuméricos, pois eles são armazenados como int64s. A função correta para verificar esse tipo de dado é is_int64(). Sintaxe: is_real(n); Argumento Descrição n O argumento para verificar. Devoluções: Booleano Exemplo: if is_real(val) { pontuação += val; } O código acima verifica a variável "val" para ver se é um número real e se é, então, adiciona-a à pontuação.
