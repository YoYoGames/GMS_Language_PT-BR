array_create Com esta função você pode criar uma matriz 1D de um determinado tamanho. Você informa à função o comprimento da matriz a ser criada, e ela retorna o "cabo" para a matriz que você pode então atribuir a uma variável. Arrays criados desta forma terão cada entrada rubricada a 0 a menos que você especifique um valor de inicialização (opcional). Se você fornecer o valor extra para inicialização da matriz, então todos os índices dentro da nova matriz serão definidos para isso ao invés de 0, mas note que a função têm uma maior sobrecarga de desempenho neste caso. Sintaxe: array_create(tamanho, [valor]); Argumento Descrição tamanho O tamanho da matriz a ser criada. valor O valor a ser usado para rubricar todos os índices de matriz (opcional). Devoluções: Referência da matriz Exemplo: instance_array = array_create(100, noone); O código acima criará uma nova matriz de 100 entradas, cada uma definida para a palavra-chave noone, e então a atribuirá à variável "instance_array".
