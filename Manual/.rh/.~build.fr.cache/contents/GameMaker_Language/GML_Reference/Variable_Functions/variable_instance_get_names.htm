nomes_de_instância_variável_get_names Com esta função você pode recuperar uma matriz preenchida com os nomes das variáveis de instância para uma instância, ou as variáveis globais para um jogo. Quando você passa em uma instância de valor de ID, cada A entrada na matriz será uma string do nome da variável que corresponde a uma variável de escopo da instância que foi criada na instância. Entretanto, se você passar na palavra-chave global, cada entrada na matriz será uma seqüência do nome da variável que corresponde a uma variável de escopo global. Sintaxe: variável_instance_get_names(instance_id/global); Argumento Descrição exemplo_id/global O valor único de identificação da instância a ser verificada ou a palavra-chave global Devoluções: Array (cada entrada é um fio) Exemplo: var str = ""; var array = variable_instance_get_names(id); show_debug_message("Variables for " + object_get_name(object_index) + string(id)); for (var i = 0; i < array_length(array); i++;) { str = array[i] + ":" + string(variable_instance_get(id, array[i])); show_debug_message(str); } O código acima recuperará um array de todas as variáveis de escopo da instância executando o bloco de código e então exibirá estes junto com seus valores na saída de depuração.
