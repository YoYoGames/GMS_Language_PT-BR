is_int32 Esta função retorna se uma determinada variável é um número inteiro de 323 bits ou não. Em alguns casos, você quer verificar e ver que tipo de dado uma variável contém no GameMaker Studio 2 e é quando você usaria esta função. Ela retorna verdadeiro ou falso, dependendo se o valor é um int 32 ou não. Sintaxe: is_int32(n); Argumento Descrição n O argumento para verificar. Devoluções: Booleano Exemplo: if !is_int32(val) { show_debug_message("Not a 32 bit integer!"); } O código acima verifica a variável "val" para ver se ela contém uma int32 e se não contém, então mostra uma mensagem no console de depuração.
