is_vec4 Em alguns casos, você deseja verificar e ver que tipo de dados uma variável contém no GameMaker Studio 2 e é quando você usaria esta função. Ela retorna verdadeira ou falsa, dependendo do tipo de variável. sobre se o valor é ou não um vetor de 4 valores. Sintaxe: is_vec4(n); Argumento Descrição n O argumento para verificar. Devoluções: Booleano Exemplo: if !is_vec4(val) { show_debug_message("Not a 4 value vector!"); } O código acima verifica a variável "val" para ver se ela contém um vec4 e se não contém, então ela mostra uma mensagem no console de depuração.
