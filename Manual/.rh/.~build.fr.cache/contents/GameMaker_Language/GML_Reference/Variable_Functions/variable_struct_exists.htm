variável_estruto_existe Com esta função você pode verificar se existe ou não uma variável dentro da estrutura dada. Você fornece a referência estrutural a ser usada, bem como o nome da variável a ser verificada como uma string (veja o código de exemplo abaixo). A função retornará verdadeiro se existir uma variável com o nome dado para a estrutura e falso caso contrário. Sintaxe: variable_struct_exists(structure, name); Argumento Descrição estrutura A referência estrutural a ser verificada nome O nome da variável estrutural a ser verificada (como uma string) Devoluções: Booleano Exemplo: if !variable_struct_exists(mystruct, "shields")) { mystruct.shields = 0; } O código acima verificará se a variável chamada "shields" existe na estrutura dada e se não existe, então ela é criada e rubricada a 0.
