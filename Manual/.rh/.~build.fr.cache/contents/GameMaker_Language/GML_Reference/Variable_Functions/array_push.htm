array_push Com esta função você pode empurrar um valor (ou valores) para o final de uma matriz sem ter que saber o comprimento da matriz. A função exige que você forneça uma variável que contenha o array, bem como pelo menos um valor para empurrar, embora você pode opcionalmente fornecer outros argumentos e todos eles serão empurrados para a matriz em ordem consecutiva. Sintaxe: array_push(variável, valor, [valor], [valor], [valor], [etc...]); Argumento Descrição variável A variável que sustenta a matriz. valor O valor para empurrar para o fim da matriz [valor], [valor], [valor], [etc...] OPCIONAL! Outros valores a serem empurrados para a matriz Devoluções: N/A Exemplo: array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr); O código acima irá empurrar quatro valores para o final da matriz dada.
