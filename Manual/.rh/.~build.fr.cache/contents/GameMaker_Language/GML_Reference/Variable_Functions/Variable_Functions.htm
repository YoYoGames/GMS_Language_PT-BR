Funções variáveis As funções desta página são projetadas para lidar com todas as diferentes variáveis e tipos de variáveis ao usar a linguagem GameMaker em seus jogos. Para mais informações sobre variáveis, consulte a página sobre Variáveis e Escopo de Variáveis. Elas são projetadas para lhe dar flexibilidade em seu código e lhe permitirão criar arrays mais funcionais e dinâmicos ao fazer seus jogos.
