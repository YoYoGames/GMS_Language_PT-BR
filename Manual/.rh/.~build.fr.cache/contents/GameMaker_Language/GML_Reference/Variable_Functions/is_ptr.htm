is_ptr Esta função retorna se uma determinada variável é ou não um ponteiro. Em alguns casos, você quer verificar e ver que tipo de dado uma variável contém no GameMaker Studio 2 e é quando você usaria esta função. Ela retorna verdadeiro ou falso, dependendo do tipo de variável. sobre se o valor é ou não um ponteiro. Sintaxe: is_ptr(n); Argumento Descrição n O argumento para verificar. Devoluções: Booleano Exemplo: se !is_ptr(val) { show_debug_message("Não é uma textura válida!"); } O código acima verifica a variável "val" para ver se ela contém um ponteiro e se não contém, então mostra uma mensagem no console de depuração.
