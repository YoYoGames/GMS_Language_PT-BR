network_resolve Esta função retornará o endereço IP do URL dado. Sintaxe: network_resolve(url); Argumento Descrição url A URL para obter o IP de (uma cadeia). Devoluções: Cordão Exemplo: game_ip = network_resolve("www.macsweeneygames.com"); O código acima retornará o endereço IP do URL dado e o armazenará na variável "game_ip".
