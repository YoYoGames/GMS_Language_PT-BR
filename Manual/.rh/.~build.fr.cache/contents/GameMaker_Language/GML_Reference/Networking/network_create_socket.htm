network_create_socket Esta função é utilizada para criar um novo soquete de cliente para que seu jogo se comunique através da rede. Você deve definir o tipo de socket (veja a lista de constantes abaixo) e a função retornará uma identificação única para esse socket, que deve ser usada em todas as outras funções exige esse soquete, ou um valor inferior a 0 se a conexão falhar. NOTA: Você pode usar o IP "127.0.0.1" para conectar de volta ao mesmo dispositivo que está rodando o jogo. Constante Descrição network_socket_tcp Criar um soquete usando TCP. rede_socket_udp Criar um soquete utilizando o UDP. network_socket_ws Criar um soquete web(somente para projetos rodando em HTML5), usando TCP. rede_socket_bluetooth Criar um soquete bluetooth(atualmente indisponível!). Sintaxe: network_create_socket(tipo); Argumento Descrição tipo O tipo de conexão de soquete a ser criado (ver as constantes listadas abaixo). Devoluções: Real Exemplo: if os_browser == browser_not_a_browser { cliente = network_create_socket(network_socket_tcp); network_connect( cliente, "192.134.0.1", 6510 ); } mais { cliente = network_create_socket(network_socket_ws); network_connect( cliente, "192.134.0.1", 6520 ); } O código acima verificará se o jogo está rodando em um navegador ou não e criará um novo TCP ou socket Web antes de tentar se conectar através dele ao endereço IP dado na porta dada.
