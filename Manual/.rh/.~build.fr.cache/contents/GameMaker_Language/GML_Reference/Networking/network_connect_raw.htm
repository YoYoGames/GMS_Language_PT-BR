network_connect_raw Com esta função, você pode enviar um pedido para se conectar a um servidor. A função leva a identificação do soquete para se conectar (veja network_create_socket()) e exige que você forneça o endereço IP para se conectar para (um fio), bem como a porta de passagem, e se a conexão falhar, um valor inferior a 0 será devolvido. A diferença entre esta função e network_connect() é que esta função pode se conectar a qualquer servidor e não faz nada com os dados brutos, o que significa que você mesmo tem que implementar os protocolos no final do servidor. Note que por padrão a função é síncrona, o que significa que seu jogo pode parecer "pendurado" como o a conexão é feita. Você pode definir um valor de timeout para conexão, ou alternativamente fazer a criação assíncrona, usando a função network_set_config()ou, alternativamente, usar a função network_connect_async_raw() ao invés disso. Sintaxe: network_connect_raw(socket, url, porta); Argumento Descrição soquete A identificação do soquete a ser utilizado. url A URL ou IP a que se conectar (uma cadeia). porto A porta a ser conectada. Devoluções: Real Exemplo: cliente = network_create_socket(network_socket_tcp); network_connect_raw(client, "www.macsweeneygames.com", 6510); O código acima criará um novo soquete TCP e então tentará criar uma conexão "bruta" através dele para a URL dada na porta 6510.
