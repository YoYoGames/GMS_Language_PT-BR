network_set_timeout Com esta função, você pode definir o tempo limite para ler e escrever dados de/para um servidor através do soquete dado. Note que o timeout não gera nenhum tipo de evento, portanto você mesmo terá que lidar com os timeouts usando alarmes (por exemplo). Note que este valor afeta apenas o envio e recebimento de dados, e caso você deseje alterar o valor de timeout da conexão, então você deve estar usando a função network_set_config(). Sintaxe: network_set_timeout(socket, read_timeout, write_timeout); Argumento Descrição soquete A identificação do soquete a ser utilizado. read_timeout Os milissegundos em que uma transferência de um servidor terá um timeout. write_timeout Os milissegundos em que uma transferência para um servidor terá um timeout. Devoluções: N/A Exemplo: network_set_timeout(sock, 3000, 3000); O código acima definirá o tempo limite para leitura e escrita de dados através do soquete indexado na variável "meia" a 3 segundos.
