ds_queue_copy Esta função pode ser usada para copiar o conteúdo de uma fila para outra. Note que isto NÃO remove o conteúdo da fila original, nem destrói a fila original. Ao usar esta função, a fila para a qual está sendo copiado deve ter foi previamente criada e se continha algum item antes da cópia, então estes serão limpos primeiro (significando que esta informação será perdida). Sintaxe: ds_queue_copy(id, fonte); Argumento Descrição id A identificação da NOVO fila. fonte A fila ORIGINAL para copiar para a identificação. Devoluções: N/A Exemplo: com (instance_create_layer(x, y, "Enemies", obj_Enemy)) { queue = ds_queue_create(); ds_queue_copy(queue, other.queue); } A função acima cria uma nova instância e, nessa instância, cria uma nova fila de DS e copia o conteúdo da fila na instância que executa o bloco de código, para a fila da instância recém-criada.
