ds_queue_head Esta função só lerá o primeiro valor da fila (aquela que está "à cabeça"). Ela não irá decifrar o valor, o que significa que ainda poderá ser lido no futuro por esta função ou pelo ds_queue_dequeue(). Se a fila estiver vazia, então a função retornará a constante indefinida, caso contrário, retornará o valor real ou de cordel contido na fila. Sintaxe: ds_queue_head(id); Argumento Descrição id A identificação da estrutura de dados a ser lida. Devoluções: Valor do tipo de dados armazenados na fila Exemplo: num = ds_queue_head(control_queue); O código acima irá ler o valor da fila indexada na variável "control_queue" e armazenar o valor de retorno na variável "num".
