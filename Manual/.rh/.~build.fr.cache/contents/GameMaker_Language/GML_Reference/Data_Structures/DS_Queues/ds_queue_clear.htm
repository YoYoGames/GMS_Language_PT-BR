ds_queue_clear Com esta função você pode limpar todos os dados da estrutura de dados da fila em questão. Isto NÃO destrói a estrutura de dados (para isso você deve usar ds_queue_destroy()) ele só limpa todos os dados dele e devolve uma fila vazia. Sintaxe: ds_queue_clear(id); Argumento Descrição id A identificação da estrutura de dados a limpar. Devoluções: N/A Exemplo: if (count = 15) && (!ds_queue_empty(command_queue)) { ds_queue_clear(command_queue); alarm[0] = room_speed; ai_count = 0; } O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a fila DS indexada na variável "command_queue", define um alarme e redefine a variável para 0.
