ds_grid_get_disk_mean Esta função pode ser usada para encontrar o valor médio para todas as células encontradas dentro da área definida do disco de uma grade (todos os valores das células são somados e depois divididos pelo número total de células que compõem o disco), como mostrado na imagem abaixo: Sintaxe: ds_grid_get_disk_mean(index, xm, ym, r); Argumento Descrição índice O índice da grade. xm A posição x do disco na grade. ym A posição y do disco na grade. r O raio do disco na grade. Devoluções: Real ou String Exemplo: val = ds_grid_get_disk_mean(grid, 5, 5, 2) O código acima definirá a variável "val" para o valor médio contido dentro do disco dado da grade DS indexada na variável "grid".
