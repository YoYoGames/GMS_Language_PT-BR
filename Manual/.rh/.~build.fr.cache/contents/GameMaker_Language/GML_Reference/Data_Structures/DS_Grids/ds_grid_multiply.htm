ds_grid_multiply Esta função multiplicará o valor de uma determinada célula da grade pela quantidade especificada. NOTA: Esta função só funcionará com números reais, não com cordas. Sintaxe: ds_grid_multiply(index, x, y, val); Argumento Descrição índice O índice da grade. x A posição x da célula na grade. y A posição y da célula na grade. val O valor para multiplicar com a célula. Devoluções: N/A Exemplo: ds_grid_multiply(mygrid, 5, 5, 2) O código acima multiplicará o valor armazenado na célula de grade DS dada por 2.
