ds_grid_sort Esta função pode ser usada para classificar uma grade DS com base nos valores de uma determinada coluna (assim como você classificaria os arquivos por data, tamanho, etc... no explorador de arquivos do SO). A imagem a seguir mostra um exemplo: Sintaxe: ds_grid_sort(índice, coluna, ascendente); Argumento Descrição índice O índice da grade para classificar. coluna A coluna a ser usada para ordenar as filas ascendente Seja para classificar o mais baixo para o mais alto(verdadeiro), ou o mais alto para o mais baixo(falso). Devoluções: N/A Exemplo: ds_grid_sort(grid, 3, false) Isto pegaria todos os valores da grade DS indexados na variável "grid" e os classificaria de acordo com os valores encontrados na 3ª coluna da grade (como mostrado na imagem acima).
