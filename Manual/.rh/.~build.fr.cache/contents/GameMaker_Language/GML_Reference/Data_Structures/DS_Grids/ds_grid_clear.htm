ds_grid_clear Esta função pode ser usada para limpar uma determinada grade DS para um valor específico. Todas as células dentro da grade conterão então este valor, que pode ser um número real ou uma string. A imagem abaixo ilustra como isto funciona: Sintaxe: ds_grid_clear(index, val); Argumento Descrição índice Este índice da grade deve ser limpo. val O novo valor para todas as células da grade. Devoluções: N/A Exemplo: ds_grid_resize(global.Grid, room_width / 32, room_height / 32); ds_grid_clear(global.Grid, -1) O código acima redimensionará a grade DS indexada na variável global "Grid" e então a limpará para que cada célula mantenha o valor -1.
