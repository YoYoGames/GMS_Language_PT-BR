ds_grid_value_disk_existe Com esta função, você pode verificar se um valor específico (real ou string) está presente dentro de uma área circular de uma determinada grade DS. Se estiver presente, a função retornará verdadeiro, caso contrário, retornará falso. Sintaxe: ds_grid_value_disk_exists(index, xm, ym, r, val); Argumento Descrição índice O índice da grade. xm A posição x do disco na grade. ym A posição y do disco na grade. r O raio do disco na grade. val O valor a encontrar. Devoluções: Booleano Exemplo: if ds_grid_value_disk_exists(grid, 5, 5, 5, val) { xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val); ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val); } O código acima verifica uma grade DS para um valor específico dentro de uma região de disco. se for encontrado, ele então armazena a posição x e y do valor em duas variáveis para uso posterior.
