ds_grid_value_existe Com esta função você pode verificar se um valor específico (real ou string) está presente dentro de uma área retangular de uma determinada grade DS. Se estiver presente, a função retornará verdadeiro, caso contrário, retornará falso. Sintaxe: ds_grid_value_exists(index, x1, y1, x2, y2, val); Argumento Descrição índice O índice da grade. x1 A posição x da esquerda da região na grade. y1 A posição y da parte superior da região na grade. x2 A posição x da direita da região na grade. y2 A posição y da parte inferior da região na grade. val O valor a encontrar. Devoluções: Booleano Exemplo: if ds_grid_value_exists(grid, 0, 1, 5, 6, val) { xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val); ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val); } O código acima verifica uma grade DS para um valor específico dentro de uma região retangular. se for encontrado, ele então armazena a posição x e y do valor em duas variáveis para uso posterior.
