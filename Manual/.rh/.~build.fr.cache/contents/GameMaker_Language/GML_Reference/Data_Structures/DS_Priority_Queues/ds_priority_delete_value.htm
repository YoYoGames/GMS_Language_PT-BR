ds_priority_delete_delete_value Esta função irá simplesmente apagar o valor dado, juntamente com sua prioridade, da fila de prioridade indexada. Sintaxe: ds_priority_delete_value(id,val); Argumento Descrição id A identificação da fila prioritária a ser usada. val O valor a excluir da fila prioritária. Devoluções: N/A Exemplo: se (ai_move == falso) { ds_priority_delete_value(ai_priority, AI_Move); } O código acima verifica uma variável de instância e, se ela retornar falso, removerá a função do script indexado da fila de prioridade.
