ds_priority_delete_max Esta função retornará o valor que tem maior prioridade na fila e então removerá o valor (e prioridade) da estrutura de dados. Se mais de um valor tiver a mesma prioridade, então qualquer um deles poderá ser devolvido em qualquer ordem, mas todos outros valores com a mesma prioridade ainda estarão na fila de espera. Sintaxe: ds_priority_delete_delete_max(id); Argumento Descrição id A identificação da fila prioritária a ser usada. Devoluções: Tipo de dados armazenados na prioridade Exemplo: se ai_move { script_execute(ds_priority_delete_max(ai_priority)); } O código acima verifica uma variável de instância e, se ela retornar verdadeiro, executará um script indexado na fila de prioridade com o valor de prioridade mais alto e, em seguida, removerá esse script da fila.
