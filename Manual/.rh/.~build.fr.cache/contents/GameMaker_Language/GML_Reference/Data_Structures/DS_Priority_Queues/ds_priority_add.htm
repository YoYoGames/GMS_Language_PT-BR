ds_priority_add Com esta função você pode adicionar um valor (seja um número real ou uma string) a uma fila de prioridade, ao mesmo tempo em que lhe atribui um valor de prioridade. Sintaxe: ds_priority_add(id, val, priority); Argumento Descrição id A identificação da fila prioritária a ser acrescentada. val O valor a ser acrescentado à fila prioritária. prioridade A prioridade do valor a agregar. Devoluções: N/A Exemplo: ds_priority_add(ai_priority, AI_Search, 5); O código acima acrescenta uma função de script à fila de prioridade indexada na variável "ai_priority" e atribui a ela uma prioridade de 5.
