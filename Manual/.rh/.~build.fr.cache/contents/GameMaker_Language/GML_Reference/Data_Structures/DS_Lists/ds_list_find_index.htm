ds_list_find_index Com esta função você pode verificar a lista dada para um valor e a posição dentro da lista para esse valor será devolvida. Observe que se houver mais de uma entrada na lista com o mesmo valor, a posição de qualquer uma delas poderá ser devolvida, e que, se o valor não existir, então -1 será devolvido. Note que o valor pode ser um array, que você pode verificar com a função is_array(). Sintaxe: ds_list_find_index(id, val); Argumento Descrição id O id da lista a ser utilizada. val O valor a encontrar. Devoluções: Real Exemplo: pos = ds_list_find_index(list, "Player1"); O código acima verifica a lista indexada na variável "lista" para o valor "Player1" e armazena a posição retornada na variável "pos".
