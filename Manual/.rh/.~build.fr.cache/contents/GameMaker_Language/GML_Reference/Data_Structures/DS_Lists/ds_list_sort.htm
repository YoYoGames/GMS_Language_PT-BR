ds_list_sort Com esta função você pode ordenar todos os valores dentro de uma lista, seja em ordem ascendente ou descendente. Se a lista contiver cordas, estas serão ordenadas alfabeticamente, com base no alfabeto inglês de 26 letras. Sintaxe: ds_list_sort(id, ascend); Argumento Descrição id O id da lista para classificar. ascender Se os valores devem ser de ordem ascendente(verdadeiro) ou descendente(falso). Devoluções: N/A Exemplo: se newgame { ds_list_sort(name_list, true); } O código acima ordenará a lista indexada na variável "name_list" se a variável "newgame" estiver marcada como verdadeira.
