ds_list_add Esta função pode ser usada para adicionar um novo valor (real ou string) à lista, que será adicionado no final. A função pode levar mais argumentos opcionais (quantos forem necessários), permitindo adicionar vários valores consecutivamente à lista. em uma única chamada. Sintaxe: ds_list_add(id, val1 [, val2, ... max_val]); Argumento Descrição id A identificação da lista a ser acrescentada. val O valor a ser acrescentado à lista. [val2, ... max_val] Valores opcionais a serem adicionados à lista. Devoluções: N/A Exemplo: ds_list_add(sc_list, score); O código acima adicionará o valor armazenado na variável "score" à lista indexada na variável "sc_list".
