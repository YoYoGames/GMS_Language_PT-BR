ds_list_find_value Com esta função, você pode verificar a posição dada na lista e o valor mantido dentro da lista para aquela posição será devolvido. Note que se você der uma posição que esteja fora do tamanho da lista dada (ou seja: posição 11 em uma lista de 10 valores) então a função Isto porque quando você cria a lista, internamente as primeiras entradas da lista são definidas como 0 para minimizar os problemas de desempenho ao adicionar itens à lista inicialmente (embora o ds_list_size() ainda retornará 0 em uma lista recém-criada). Se você deseja garantir que a lista esteja "verdadeiramente" vazia ao criar, então você deve chamar ds_list_clear() depois de criar a lista, que então significa que quaisquer valores retornados para os slots da lista não-populosa serão indefinidos, o que você pode então verificar consistentemente usando o is_undefined() função. Sintaxe: ds_list_find_value(id, pos); Argumento Descrição id O id da lista a ser utilizada. pos A posição a ser observada, onde 0 corresponde ao início da lista e a posição final é ds_list_size(id)-1. Devoluções: Real, Cordão ou Indefinido Exemplo: val = ds_list_find_value(list, ds_list_size(list) - 1); O código acima verifica a lista indexada na variável "list" na última posição da lista e armazena o valor retornado na variável "val".
