ds_list_copy Com esta função você pode copiar o conteúdo de uma lista para outra. Ambas as listas devem ter sido criadas anteriormente e se a lista que está sendo copiada já tiver informações dentro dela, esta lista será limpa primeiro. O resultado final são duas listas independentes listas que contêm as mesmas informações. Sintaxe: ds_list_copy( id, fonte ); Argumento Descrição id A identificação da lista sendo copiada para. fonte A identificação da lista a ser copiada. Devoluções: N/A Exemplo: if !ds_list_empty(main_list) { old_list = ds_list_create(); ds_list_copy(old_list, main_list); ds_list_clear(main_list); } O código acima verificará uma lista DS para ver se ela está vazia. Se não estiver vazia, é copiada para outra lista DS (que foi criada anteriormente) e então a lista original é limpa.
