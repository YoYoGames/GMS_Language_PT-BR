ds_stack_pop Esta função irá estourar o valor superior da pilha DS, retirando-o da pilha e retornando o valor a ser armazenado em uma variável. Se a pilha estiver vazia, então a função retornará a constante indefinida, caso contrário, retornará o valor real ou de cordel contido na pilha. Sintaxe: ds_stack_pop(id); Argumento Descrição id A identificação da estrutura de dados a partir da qual os dados devem aparecer. Devoluções: Valor do tipo de dado que é armazenado na pilha ou indefinido Exemplo: if !ds_stack_empty(move_stack) { var xx = ds_stack_pop(move_stack); var yy = ds_stack_pop(move_stack); move_towards_point(xx, yy, 4); } O código acima verifica a pilha DS indexada na variável "move_stack" para ver se ela está vazia, e se não estiver, ela então estala os dois valores superiores da pilha e os usa para definir uma direção para o movimento.
