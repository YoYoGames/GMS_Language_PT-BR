ds_map_existe Esta função retornará verdadeiro se a chave especificada existir no mapa DS (previamente criado), e falso se não existir. Sintaxe: ds_map_exists(id, key); Argumento Descrição id a identificação da estrutura de dados a verificar chave a chave para verificar Devoluções: Booleano Exemplo: se !ds_map_existir(inventário, "poções") { ds_map_add(inventário, "poções", 1); } O código acima verificará o mapa DS indexado na variável "inventário" para a chave "poções" e, se não existir, o adicionará ao mapa.
