ds_map_empty Esta função simplesmente retornará falso se o mapa DS especificado (previamente criado) contiver qualquer par chave/valor, ou verdadeiro se não contiver. Sintaxe: ds_map_empty(id); Argumento Descrição id A identificação da estrutura de dados a ser verificada. Devoluções: Booleano Exemplo: if ds_map_empty(inventory) { peso = 0; } O código acima verifica se o mapa DS indexado na variável "inventário" tem algum par chave/valor e se não define a variável "peso" como 0.
