ds_map_size Com esta função você pode encontrar quantos pares de chaves/valores o mapa DS (anteriormente criado) contém. Sintaxe: ds_map_size(id); Argumento Descrição id A identificação da estrutura de dados a ser verificada. Devoluções: Real Exemplo: if (ds_map_size(inventory) > 49) { completo = verdadeiro; } O código acima verificará o tamanho do mapa DS (ou seja: número de pares chave/valor) e se for maior que 49, define a variável "cheia" como verdadeira.
