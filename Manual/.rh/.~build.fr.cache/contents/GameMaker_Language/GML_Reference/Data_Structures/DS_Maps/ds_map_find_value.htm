ds_map_find_value Com esta função você pode obter o valor a partir de uma chave especificada. Os valores de entrada da função são o mapa DS (previamente criado) a ser usado e a chave a ser verificada. NOTA: Se tal chave não existir, a função retornará indefinida. Você deve sempre verificar isto usando a função is_undefined() função Sintaxe: ds_map_find_value(id, key) Argumento Descrição id A identificação do mapa a ser utilizado. chave A chave para encontrar. Devoluções: Real, de corda ou indefinido Exemplo: quantidade = ds_map_find_value(inventário, "alimento"); Ou, usando o acessor de mapa "?": quantidade = estoque[? "alimentos"]; O código acima obterá o valor da chave "alimento" e o armazenará na variável "quantidade".
