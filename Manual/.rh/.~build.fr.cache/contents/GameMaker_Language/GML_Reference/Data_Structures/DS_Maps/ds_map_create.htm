ds_map_create Esta função é usada para criar um novo mapa DS vazio e retornará sua identificação que será então usada para acessar a estrutura de dados em todas as outras funções do mapa DS. IMPORTANTE! Quando você cria uma estrutura de dados, o valor do índice para identificá-lo é um valor inteiro começando em 0. Isso significa que estruturas de dados de diferentes tipos podem ter o mesmo valor de índice, portanto, em caso de dúvida, você deve estar usando o ds_exists() antes de acessá-los. Observe também que os índices são reutilizados, de modo que um valor de índice de estrutura de dados destruída pode ser usado por um índice recém-criado posteriormente. Sintaxe: ds_map_create(); Devoluções: Número inteiro (DS map ID) Exemplo: inventário = ds_map_create(); O código acima criará um novo mapa DS vazio e armazenará seu índice de identificação na variável "inventário".
