status_de_login_realização Esta função é atualmente útil apenas para o Google Play na plataforma Android. Ela verificará se o usuário já está logado nos serviços do Google, caso em que retornará verdadeiro, ou não, retornando falso. Se ele retornar verdade, não há necessidade de chamar o achievement_login() função. Sintaxe: achievement_login_status(); Devoluções: Booleano Exemplo: se !achievement_login_status() { achievement_login(); } O código acima o registrará no quadro de líderes escolhido e no serviço de realizações.
