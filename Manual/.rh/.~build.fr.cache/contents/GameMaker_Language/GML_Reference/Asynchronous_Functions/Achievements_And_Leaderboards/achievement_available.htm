realizações_disponíveis Esta função retornará verdadeiro se o usuário estiver atualmente conectado à Internet e se o leaderboard escolhido e o sistema de realização estiver disponível, caso contrário, retornará falso. Sintaxe: achievement_available() Devoluções: Booleano Exemplo: se achievement_available() { achievement_post("ach_Level1", 100); } O código acima verificará se o serviço de quadro de líderes está disponível e, em caso afirmativo, afixará para ele.
