achievement_get_pic Esta função enviará um pedido ao servidor para a imagem de um jogador ou de um amigo e desencadeará um Evento Assíncrono Social de retorno que contém o async_load mapa povoado com os pares chave/valor relevantes. A chave de id deste DS Map é usada para identificar o retorno de chamada correto (pode haver mais de uma função de acionamento para qualquer evento assíncrono), e será emparelhada com a constante realização_pic_carregada, bem como um número de outros pares de chaves/valores. O conteúdo exato do mapa é mostrado abaixo: NOTA: Esta função é para iOS somente. " id" - Para esta função deve ser realizado_pic_loaded "playerid" - A identificação do jogador ou amigo de quem estamos recebendo a imagem. "spriteid" - O id do duende para a imagem que pode então ser usado para desenhá-la na tela usando qualquer uma das funçõesdo duende ADVERTÊNCIA! Esta função cria sprites e como tal, seus ids devem ser armazenados em variáveis e depois removidos da memória usando o sprite_delete() Caso contrário, o uso repetido lhe dará um vazamento de memória que eventualmente diminuirá a velocidade ou prejudicará seu jogo. Sintaxe: achievement_get_pic(char) Argumento Descrição char O id da pessoa para obter a imagem de (seja de uma lista de amigos ou de um quadro de líderes) Devoluções: N/A Exemplo ampliado: O seguinte código seria chamado depois que você obtivesse uma lista de amigos ou uma lista de líderes usando achievement_load_friends() ou achievement_load_leaderboard(). Essas funções retornarão valores "id" únicos para cada pessoa, que são então utilizados na função para recuperar a imagem do perfil para eles: achievement_get_pic(global.player_id[0]); Isto enviará uma solicitação para a imagem associada à identificação do jogador armazenada na matriz global e gerará uma chamada de retorno assíncrona com o mapa especial de DS de carga assimétrica contendo os seguintes dados: var ident = ds_map_find_value(async_load, "id"); if ident == achievement_pic_loaded { var picid = ds_map_find_value(async_load, "playerid"); var spriteid = ds_map_find_value(async_load, "spriteid"); if global.playerid[0] == picid) { global.playerpic[0] = spriteid; } } O código acima verifica o Mapa DS retornado no Evento Assíncrono Social e se seu "id" corresponde à constante a ser verificada, então verifica se o id associado à imagem é o mesmo que está sendo solicitado e se é, ele estabelece outra matriz global para manter a identificação única para a imagem (que pode ser usada para desenhar a imagem como um duende).
