Facebook O Facebook é um dos sites de mídia social mais importantes da Internet e muitos jogos e desenvolvedores atualmente têm suas próprias páginas para postar jogos e mídia. O GameMaker Studio 2 tem uma extensão que permite a você se conectar com o Facebook e realizar certas ações e tarefas que seu API permite. É recomendável que você visite a página Desenvolvedores do Facebook antes de começar a usar a extensão como um firme entendimento de como o O Facebook Graph funciona e as chamadas e permissões disponíveis são essenciais para você obter o máximo destas funções. Você pode descobrir como configurar e usar o ramal do Facebook para todas as plataformas disponíveis no link a seguir: Android, iOS e HTML5: Integrando o Facebook As seguintes são funções obsoletas que só eram válidas para a API antiga do Facebook. Como tal, você não pode mais usar estas funções em seus projetos e elas são listadas aqui caso você tenha importado um projeto antigo com elas. Para funcionalidade atual do Facebook, você precisa ter ativado a API do Facebook nas Opções de Jogo apropriadas para a plataforma alvo, o que o levará a baixar e instalar uma extensão. A extensão vem com um "mini-manual" em formato *.pdf explicando como utilizá-la, além de mostrar todas as definições e exemplos de funções. Todas as referências às antigas funções do facebook_*() precisarão ser substituídas (e atualizado) para usar a extensão fb_*() equivalentes.
