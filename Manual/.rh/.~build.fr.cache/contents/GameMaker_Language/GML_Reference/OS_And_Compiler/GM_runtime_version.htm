GM_runtime_version Esta constante mantém o número da versão em tempo de execução, conforme definido nas Preferências, como o tempo de execução que está sendo usado para construir o projeto. O valor é armazenado como um fio. Sintaxe: GM_runtime_version; Devoluções: Cordão Exemplo: draw_text(32, 32, date_time_string(GM_build_date)); draw_text(32, 64, "v" + GM_version); draw_text(32, 96, " Runtime " + GM_runtime_version); O código acima pega a data GM, versão e constantes de tempo de execução e as desenha para a tela.
