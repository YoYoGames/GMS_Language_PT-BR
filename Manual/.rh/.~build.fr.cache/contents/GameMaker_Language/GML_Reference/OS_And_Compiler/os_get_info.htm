os_get_info Esta função retorna um Mapa DS com informações detalhadas sobre o sistema operacional em que o jogo está sendo executado. As informações exatas retornadas dependerão do sistema operacional e do dispositivo, portanto é recomendável que você teste esta função em todas as plataformas que você deseja direcionar e emitir os valores retornados usando uma função como show_debug_message(). Note que o mapa DS não é automaticamente apagado da memória e você deve usar o ds_map_destroy() quando você não precisa mais dos dados que ela contém. Abaixo você pode encontrar alguns detalhes sobre os dados extras que serão disponibilizados para o mapa DS específicos para as seguintes plataformas, e deve ser notado que em todas as plataformas exceto HTML5 o mapa terá a chave "is64bit". que será true ou false, dependendo se o sistema operacional da plataforma é 64bit ou não.
