os_browser Esta variável somente leitura contém uma das várias constantes (listadas abaixo) que o GameMaker Studio 2 tem que lhe dizer em qual navegador você está atualmente rodando o jogo (se houver). Sintaxe: os_browser Devoluções: Constante Constante Descrição browser_not_a_browser O jogo não está sendo jogado em um navegador browser_unknown Navegador desconhecido browser_ie Internet Explorer browser_ie_mobile Internet Explorer em um dispositivo móvel browser_firefox Mozilla Firefox browser_chrome Google Chrome browser_safari Safari browser_safari_mobile Safári em um dispositivo móvel browser_opera Ópera browser_tizen Navegador de dispositivos móveis Tizen browser_windows_store Windows App Exemplo: if (os_browser == browser_not_a_browser) { global.Config = 0; } mais { global.Config = 1; } O código acima verifica se o jogo está sendo executado em um navegador ou não e define uma variável global para um valor, dependendo do resultado da verificação.
