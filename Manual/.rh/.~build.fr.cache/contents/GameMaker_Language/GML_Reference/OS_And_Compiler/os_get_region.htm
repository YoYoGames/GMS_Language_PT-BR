os_get_region Esta função retorna uma cadeia com duas ou três letras Código Regional para o SO que está executando o jogo, conforme definido pela norma ISO3166-1. Se a informação não estiver disponível, ele segurará simplesmente um fio vazio "". NOTA: Este não é o código regional de localização que é devolvido, mas o código do idioma regional do sistema operacional.
