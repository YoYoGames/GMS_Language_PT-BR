os_version Esta variável lhe dirá o número da versão do sistema operacional que está rodando seu jogo. Por exemplo, se você estiver executando no Windows 10, os_version será igual a 655360. A tabela a seguir descreve os valores que são devolvidos para as versões mais recentes dos diferentes sistemas operacionais: Sistema operacional Nome da versão (número da versão) Valor de retorno (exemplos) Android Nougat (7.0 - 7.11) Oreo (8.0 - 8.11) Pie (9.0) Android X (10.0) 24 - 25 26 - 27 28 29 iOS O valor de retorno é calculado como: (major_version * 16777216) + (minor_version * 4096) + build_number iOS 10 (10.3) iOS 11 (11.4) iOS 12 (12.0) iOS 13 (13.0) iPhone iOS 13 (13.5) iPhone/Ipad 167784448 184565760 201326592 218103808 218124288 macOS X O valor de retorno é calculado como: (major_version * 16777216) + (minor_version * 4096) + build_number El Capitan (10.11) Sierra (10.12) High Sierra (10.13) Mojave (10.14) Catalina (10.15) 167817216 167821312 167825408 167829504 167833600 Windows (o valor de retorno é calculado como: majorVersion * 65536 + minorVersion) Windows 7 (6.1) Windows 8 (6.2) Windows 8.1 (6.3) Windows 10 (10.0) 393217 393218 393219 655360 NOTA: Caso você precise de mais informações sobre o sistema operacional Windows, você pode usar o ambiente_get_variable(). Sintaxe: os_version Devoluções: Real Exemplo: if (os_type == os_android) && (os_version > 10) { global.GFX = 1; } O código acima verifica o tipo e o número da versão e ambos estão corretos, então a variável global é definida como 1.
