conjunto_de_resolução_agendador Esta função é utilizada para alterar a resolução do agendador de fios do Windows. O valor da resolução fornecido no argumento precisa ser em milissegundos. Por padrão, GameMaker Studio 2 usa uma resolução de 1 milissegundo, que é a menor resolução permitida e dá o melhor desempenho. Mudar a resolução para um valor mais alto pode afetar negativamente o desempenho, mas também pode reduzir o consumo de energia. Você pode fornecer -1 como argumento para reverter a resolução para seu valor padrão (como definido pelo Windows). Note que mudar a resolução do agendador terá impacto em todos os processos em execução ao mesmo tempo que o seu jogo. NOTA: Esta função é para Windows somente. Sintaxe: programador_resolução_set(milissegundos); Argumento Descrição milissegundos O novo valor de resolução (em milissegundos) ou -1 para o padrão Devoluções: N/A Exemplo: programador_resolução_set(2); Este exemplo define a resolução do agendador de fios do Windows para 2 milissegundos.
