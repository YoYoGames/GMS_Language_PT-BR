move_outide_all Com esta função você pode dizer a uma instância para sair de uma colisão em qualquer direção e qualquer número de pixels a cada passo, com um valor de -1 ou 0 para o maxdist sendo um valor padrão de 1000px, ou seja: GameMaker Studio 2 irá mover a instância continuamente até 1000 pixels até que esteja fora de colisão. Sintaxe: move_outide_all(dir, maxdist); Argumento Descrição dir A direção para avançar. maxdist A distância máxima que o objeto pode percorrer (-1 ou 0 um valor padrão de 1000 pixels). Devoluções: N/A Exemplo: if place_meeting(x, y, all) { move_outside_all(90, 1); } O código acima moverá a instância para cima de 1 pixel a cada passo que ela estiver em colisão com qualquer outra instância.
