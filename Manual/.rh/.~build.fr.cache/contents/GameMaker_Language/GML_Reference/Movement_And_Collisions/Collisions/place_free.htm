place_free Você pode usar esta função para verificar e ver se a instância de chamada colidiria com qualquer instância marcada como sólida em seu jogo. Agora, deve ser observado que para que isto funcione, a instância que executa o código deve ter uma colisão válida máscara (seja para o próprio duende, seja através do máscara_index) e só registrará colisões com aquelas situações de bandeira sólida que também têm um máscara válida. A função em si funciona basicamente tomando a instância e testando para colisões apenas com instâncias marcadas sólidas quando colocadas na posição especificada pelos argumentos x/y. A verificação de colisão pode ser precisa ou baseada na delimitação caixa da instância, dependendo de que tipo de máscara de colisão foi selecionada. Observe que as coordenadas x/y dadas serão niveladas com o inteiro mais próximo antes da verificação ser realizada. Sintaxe: place_free(x, y); Argumento Descrição x A posição x a ser verificada. y A posição y a ser verificada. Devoluções: Booleano Exemplo: if place_free(mouse_x, mouse_y) { x = mouse_x; y = mouse_y; } O código acima verificará se há uma colisão com qualquer instância sólida sinalizada se a instância de chamada for colocada na mesma posição que o mouse. Se não for detectada nenhuma colisão, então a instância tem suas coordenadas x/y definidas para as do mouse.
