retângulo_em_círculo Esta função irá verificar uma área retangular que você define para ver se ela não está em colisão, completamente dentro dos limites de destino, ou se está simplesmente tocando, uma área circular definida. Se eles não estiverem tocando em nada, a função retornará 0, se a fonte estiver completamente dentro do destino ela retornará 1, e se estiverem simplesmente sobrepostos então ela retornará 2. A imagem abaixo ilustra isto: Sintaxe: retângulo_em_círculo(sx1, sy1, sx2, sy2, x, y, rad); Argumento Descrição sx1 A coordenada x do lado esquerdo do retângulo da fonte. sy1 A coordenada y do lado superior do retângulo da fonte. sx2 A coordenada x do lado direito do retângulo da fonte. sy2 A coordenada y do lado inferior do retângulo da fonte. x A coordenada x do centro do círculo y A coordenada y do centro do círculo. rad O raio em torno do ponto central no qual se deve verificar uma colisão. Devoluções: Real Exemplo: inst = instance_nearest(x, y, obj_Bullet); se instance_exists(inst) { if rectangle_in_circle(inst.x - 5, inst.y - 5, inst.x + 5, inst.y + 5, x, y - 25, 20) > 0 { hit = true; } } O código acima usa a função retângulo_em_circulo para verificar se há uma colisão dentro de uma área circular e o retângulo em torno de uma instância encontrada. Se houver uma colisão (uma borda sobreposta ou englobada), então uma variável será definida como verdadeira.
