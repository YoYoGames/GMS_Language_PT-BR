position_destroy Esta função simplesmente destrói todas as instâncias que se encontram em colisão com a posição especificada. As colisões são baseadas na máscara das instâncias, e se qualquer parte da máscara se sobrepõe ao ponto-alvo, então a função destruirá esse exemplo. Instâncias destruídas desta forma desencadearão seus eventos de Destruição e Limpeza. Sintaxe: position_destroy(x, y); Argumento Descrição x A coordenada x de onde destruir as instâncias de colisão. y A coordenada y de onde destruir as instâncias de colisão. Devoluções: N/A Exemplo: if mouse_check_button_pressed(mb_left) { position_destroy(mouse_x, mouse_y); } Isto destruirá todas as instâncias que colidirem com a posição do cursor do mouse quando o botão esquerdo do mouse for pressionado.
