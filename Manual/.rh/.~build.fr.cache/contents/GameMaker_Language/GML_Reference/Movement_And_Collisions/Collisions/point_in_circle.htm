point_in_circle Ao utilizar esta função, você define uma área circular e o GameMaker Studio 2 irá determinar se o ponto dado se enquadra ou não em seus limites. Se o ponto se enquadrar no círculo definido, a função retornará verdadeiro, caso contrário a função retornará falso. Sintaxe: point_in_circle(px, py, x1, y1, rad); Argumento Descrição px A coordenada x do ponto a ser verificado. py A coordenada y do ponto a verificar. x1 A coordenada x do centro do círculo. y1 A coordenada y do centro do círculo. rad O raio do círculo. Devoluções: Booleano Exemplo: if point_in_circle(mouse_x, mouse_y, x, y, 16) { over = verdadeiro; } mais { over = falso; } O código acima usa a função point_in_circle para verificar se a posição do mouse está dentro da área circular definida, definindo a variável "over" como verdadeira se estiver, ou falsa de outra forma.
