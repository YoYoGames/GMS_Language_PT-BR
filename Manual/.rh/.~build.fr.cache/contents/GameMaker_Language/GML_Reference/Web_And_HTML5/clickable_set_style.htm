clickable_set_style Esta função permite definir as propriedades do estilo CSS para o botão dado através dos pares chave/valor nos pares fornecidos Mapa DS. Você precisa ter criado previamente os dois elementos do botão (usando o clickable_add()) e o DS Map e fornecer os índices armazenados a cada um como argumentos. Sintaxe: clickable_set_style(index, map) Argumento Descrição índice O índice do ícone clicável ao estilo. mapa O índice do DS Map para definir o estilo a partir de. Devoluções: N/A Exemplo: var map = ds_map_create(); ds_map_add(map, "opacity", "0.5"); clickable_set_style(button, map); ds_map_destroy(); O código acima mudará o estilo do botão clicável com o índice armazenado na variável "botão".
