url_open Isto abrirá a URL especificada no navegador do dispositivo alvo escolhido, ou, se você estiver usando o módulo HTML5, no navegador atualmente aberto. Sintaxe: url_open(url); Argumento Descrição url O URL (endereço do site) para o qual o link deve ser criado. Devoluções: N/A Exemplo: url_open("http://yoyogames.com"); Isto abriria a página inicial dos YoYo Games na janela atual.
