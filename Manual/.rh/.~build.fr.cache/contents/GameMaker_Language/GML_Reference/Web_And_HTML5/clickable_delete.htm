clickable_delete Esta função deve ser usada para remover um ícone clicável previamente criado com clickable_add() da janela do jogo. Sintaxe: clickable_delete(index); Argumento Descrição índice Índice do ícone clicável a ser removido. Devoluções: N/A Exemplo: if clickable_exists(global.Help_Icon) { clickable_delete(global.Help_Icon); } O código acima remove o ícone clicável indexado na variável global "Help_Icon" da janela do jogo, se ela existir.
