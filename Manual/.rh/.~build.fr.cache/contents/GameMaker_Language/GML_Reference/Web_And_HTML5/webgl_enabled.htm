webgl_enabled Esta variável somente leitura retornará se a WebGL estiver habilitada(verdadeira) ou não(falsa) para seu jogo. Ela só funcionará para aqueles jogos rodando através de um navegador (ou seja: HTML5), e para todas as outras plataformas ela retornará true. Sintaxe: webgl_enabled Devoluções: Booleano Exemplo: se habilitado para webgl_enabled { global.quality = 1; } mais global.quality = 0; O código acima verifica a bandeira WebGL e então define a variável global "qualidade" de acordo.
