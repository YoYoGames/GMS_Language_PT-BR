browser_input_capture Com esta função você pode definir se a janela do navegador deve capturar toda a entrada (defini-la como falsa) ou se o jogo deve capturar a entrada (defini-la como verdadeira). Note que esta função é apenas para uso com o módulo HTML5. Sintaxe: browser_input_capture(enable) Devoluções: N/A Exemplo: browser_input_capture(true); O código acima define o jogo para capturar todas as entradas do navegador na plataforma alvo HTML5.
