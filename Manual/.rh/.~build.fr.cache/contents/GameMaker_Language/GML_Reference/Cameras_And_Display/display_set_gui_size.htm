display_set_gui_size Esta função só afetará as coisas desenhadas no evento de desenho da GUI e pode ser usada para definir uma largura e altura específicas para todos os componentes da GUI nesse evento, e não importa o que tamanho do display ou janela é usado para exibir o GUI, ele será escalado para caber corretamente. A posição da GUI (ou seja: a posição (0,0)) dependerá de se você usou a função display_set_gui_maximise() ou não. Se você não usar esta função (a configuração padrão), então a camada GUI será posicionada com seu (0,0) ponto na mesma posição da superfície de aplicação (a área onde seu jogo está sendo desenhado) e a largura e altura ser esticado para se ajustar a esta superfície, de modo que o ajuste de uma largura GUI de 256px com esta função quando a superfície de aplicação é de 512px esticaria aqueles 256 pixels para se ajustar, tornando efetivamente os pixels duas vezes maiores. Isto é ideal para aqueles jogos que têm correção de proporção definida nas configurações globais do jogo. Se você tiver usado a função display_set_gui_maximise(), então a posição (0,0) para a camada GUI será padrão no canto superior esquerdo da janela do jogo, ou display (quando em tela cheia ou em dispositivos), com o fator de escala definido por essa função, e a largura e a altura definidas por este. Você pode redefinir a camada GUI a qualquer momento para a configuração padrão chamando esta função usando -1 como largura e altura - isto definirá a posição (0, 0) para a superfície da aplicação e a largura e a altura será redefinida para 1:1 também com a superfície. Sintaxe: display_set_gui_size(width, height); Argumento Descrição largura A largura da GUI altura A altura da GUI Devoluções: N/A Exemplo: display_set_gui_size(768, 1024); O código acima bloqueará o evento de desenho GUI na largura e altura determinadas, escalonando todos os componentes para caber tanto na superfície da aplicação, quanto no display, usando essa proporção.
