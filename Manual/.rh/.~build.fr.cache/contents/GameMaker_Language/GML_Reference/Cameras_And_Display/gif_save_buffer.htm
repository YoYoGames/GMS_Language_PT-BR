gif_save_buffer Com esta função, você pode salvar uma animação GIF. Você fornece o índice GIF (como retornado pela função gif_open()) e a função retornará um buffer de crescimento de 1 byte alinhado com os dados do GIF. Note que a função final Os dados GIF terão sido paletizados utilizando a Paleta Universal 884 (veja aqui para mais informações).
