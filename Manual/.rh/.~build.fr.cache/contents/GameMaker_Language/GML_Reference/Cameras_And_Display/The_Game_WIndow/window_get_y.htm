window_get_y Com esta função você pode obter a posição y (em pixels) dentro do navegador se for um jogo HTML5 ou dentro da tela se for um jogo Windows, Ubuntu(Linux) ou macOS. Sintaxe: window_get_y(); Devoluções: Real Exemplo: wy = window_get_y(); O código acima armazena a posição atual da janela do eixo y na variável "wy".
