largura_máxima_de_janela Esta função pode ser usada para definir uma largura máxima de janela para seu jogo. Se você ativar a opção de redimensionamento da janela nas Opções de Jogo para a plataforma alvo, então o jogador pode redimensionar a janela do jogo para o tamanho que desejar, entretanto, usando esta função você pode limitar a largura máxima ao tamanho especificado. Se você desejar voltar ao comportamento padrão (ou seja: sem mínimo), então use um valor -1. NOTA: Esta função só está disponível no Windows, macOS e Linux plataformas. Sintaxe: window_set_max_width(width); Argumento Descrição largura A largura máxima em pixels para a janela do jogo. Devoluções: N/A Exemplo: window_set_min_width(640); window_set_min_height(480); window_set_max_width(1280); window_set_max_height(960); O código acima definirá a largura e a altura mínima e máxima para a janela do jogo.
