window_center Com esta função você pode centralizar a janela do jogo na tela quando o módulo alvo for Windows, Ubuntu(Linux) ou macOS, ou você pode centralizá-la no navegador se o módulo alvo for HTML5. Esta função não tem efeito em nenhum outro dispositivo. NOTA: Se você redimensionou a janela do jogo (por exemplo, mudando de tela cheia para janela), e deseja centralizar a nova janela, esta função deve ser chamada em um alarme[0] pelo menos um passo depois da mudança do tamanho da janela, caso contrário não vai funcionar corretamente. Sintaxe: window_center(); Devoluções: N/A Exemplo: if !window_get_fullscreen() { window_center(); } O código acima centralizará a janela do jogo dentro da tela se ela não estiver no modo tela cheia.
