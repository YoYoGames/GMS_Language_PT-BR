window_set_caption Com esta função você pode mudar ou definir a legenda das janelas para a sala em que você se encontra atualmente. Esta legenda aparece na parte superior da janela, ao lado do ícone do jogo, quando o jogo não está no modo tela cheia. Sintaxe: window_set_caption(caption); Argumento Descrição legenda A nova legenda. Devoluções: N/A Exemplo: se window_get_caption() != "" { window_set_caption(""); } O código acima verificará a legenda da janela para ver se ela tem texto ou não, e se a coloca em uma cadeia vazia para que nenhuma legenda seja exibida.
