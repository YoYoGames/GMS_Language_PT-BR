window_set_colour Esta função pode definir a cor de fundo da janela do jogo. Esta cor representa aquela que será usada para as áreas da janela do jogo que não são ocupadas por nenhuma vista. A imagem a seguir ilustra isto: A imagem acima tem duas vistas com duas portas de visualização, cada uma desenhada em posições diferentes. Esta estica a janela do jogo para acomodar ambas as portas e usa a cor da janela para colorir o fundo onde não é mostrada nenhuma vista. Sintaxe: window_set_colour(cor); Argumento Descrição cor A cor para definir a região. Devoluções: N/A Exemplo: se window_get_colour() != c_black { window_set_colour(c_black); } O código acima verificará a cor da janela para ver se ela está definida como preta ou não, e se não está definida como preta.
