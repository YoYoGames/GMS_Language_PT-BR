window_mouse_get_y Com esta função você pode obter a posição y do cursor do mouse (em pixels) dentro do navegador se for um jogo HTML5 ou dentro da tela se for um jogo Windows, Ubuntu(Linux) ou macOS. NOTA: Para funções regulares do mouse, veja a seção sobre Entrada do Rato Sintaxe: window_mouse_get_y(); Devoluções: Real Exemplo: wy = window_mouse_get_y(); O código acima armazena a posição atual da janela do eixo y do mouse na variável "wy".
