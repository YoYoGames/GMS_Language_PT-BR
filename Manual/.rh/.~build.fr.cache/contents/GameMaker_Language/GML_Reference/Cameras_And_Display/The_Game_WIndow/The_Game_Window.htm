A Janela de Jogo O jogo que você cria acontece em uma janela (mesmo quando em tela cheia), e esta janela tem uma série de propriedades, como posição, tamanho, se é em tela cheia, etc... Esses detalhes são normalmente definidos automaticamente para você com base no tamanho da sala e nas portas de visualização ativado em combinação com as configurações das Opções de Jogo para a plataforma alvo, mas você pode alterá-las durante o jogo usando as funções listadas nesta página. NOTA: Estas funções são para Windows, Ubuntu(Linux), macOS e HTML5 apenas e pode não funcionar em nenhum outro dispositivo.
