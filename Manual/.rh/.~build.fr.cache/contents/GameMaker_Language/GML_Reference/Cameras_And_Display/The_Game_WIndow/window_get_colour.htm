window_get_colour Esta função retorna a cor de fundo da janela do jogo. Esta cor representa aquela que será usada para as áreas da janela do jogo que não são ocupadas por nenhuma vista. A imagem a seguir ilustra isto: A imagem acima tem duas vistas com duas portas de visualização, cada uma desenhada em posições diferentes. Isto estica a janela do jogo para acomodar ambas as portas e usa a cor da janela para colorir o fundo onde nenhuma vista é mostrada. Sintaxe: window_get_colour(); Devoluções: N/A Exemplo: se window_get_colour() != c_black { window_set_colour(c_black); } O código acima verificará a cor da janela para ver se ela está definida como preta ou não, e se não está definida como preta.
