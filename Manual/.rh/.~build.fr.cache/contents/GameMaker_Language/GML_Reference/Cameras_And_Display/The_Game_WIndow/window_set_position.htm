window_set_position Com esta função você pode configurar a janela do jogo para uma posição específica dentro da tela (no MacOS, Linux(Ubuntu) e Windows) ou dentro do navegador (HTML5). NOTA: Se seu jogo HTML5 usa um index.html personalizado e que define a tela para uma posição fixa, então esta função não terá efeito sobre a posição da janela. Sintaxe: window_set_position(x, y); Argumento Descrição x A coordenada x de onde posicionar a janela. y A coordenada y de onde posicionar a janela. Devoluções: N/A Exemplo: window_set_position(0, 0); O código acima posicionará a janela do jogo no canto superior esquerdo do navegador ou da tela (dependendo do módulo alvo que estiver sendo usado).
