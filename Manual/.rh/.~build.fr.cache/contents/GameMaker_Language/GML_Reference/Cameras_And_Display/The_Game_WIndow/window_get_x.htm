window_get_x Com esta função você pode obter a posição x (em pixels) dentro do navegador se for um jogo HTML5 ou dentro da tela se for um jogo Windows, Ubuntu(Linux) ou macOS. Sintaxe: window_get_x(); Devoluções: Real Exemplo: wx = window_get_x(); O código acima armazena a posição atual da janela do eixo x na variável "wx".
