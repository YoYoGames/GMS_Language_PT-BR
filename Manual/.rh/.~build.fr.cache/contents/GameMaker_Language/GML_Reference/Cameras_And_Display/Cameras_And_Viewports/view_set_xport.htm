view_set_xport Esta função pode ser usada para definir a posição x da porta de visão dada. Você dá o índice da porta de visão (de 0 a 7) e a nova posição para colocá-la. Sintaxe: view_set_xport(view_port, x) Argumento Descrição view_port O porto de visão para o alvo (0 - 7) x A nova posição x Devoluções: N/A Exemplo: se view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2) { view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2)); } O código acima verificará a posição x da porta de visualização[0] e, se não estiver onde é necessário estar, será ajustado para essa posição.
