camera_get_view_mat Esta função pode ser usada para recuperar a matriz de visualização. A função retorna o valor de identificação da matriz que pode ser usado em outras funções da matriz ou para definir a matriz de visão de outra câmera (usando camera_set_view_mat()). Note que esta função é geralmente para uso com câmeras criadas usando camera_create()mas também pode ser utilizado sobre aqueles criados no editor da sala (ou com camera_create_view()) - neste caso, a matriz devolvida só será válida após a utilização da câmera dada em uma porta de visualização pela primeira vez. Sintaxe: camera_get_view_mat(camera_id) Argumento Descrição camera_id O valor único de identificação da câmera retornou quando você criou a câmera Devoluções: Matriz de identificação (real) Exemplo: mat = camera_get_view_mat(camera_view[0]); O código acima obtém a matriz de visualização para a câmera designada para visualizar a porta[0].
