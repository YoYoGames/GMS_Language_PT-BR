view_get_hport Esta função pode ser usada para recuperar a altura da porta de visualização dada. Sintaxe: view_get_hport(view_port) Argumento Descrição view_port O porto de visão para o alvo (0 - 7) Devoluções: Real Exemplo: se view_get_hport(0) != (display_get_height () / 2) { view_set_hport(0, display_get_height() / 2); } O código acima verificará a altura da porta de visualização[0] e se não for metade da altura do visor, será ajustado para esse valor.
