camera_get_view_y Esta função pode ser usada para recuperar a posição y da vista de uma determinada câmera. Note que esta função só é válida para câmeras criadas usando camera_create_view() ou para aqueles adicionados no editor da sala. Sintaxe: camera_get_view_y(camera_id) Argumento Descrição camera_id O valor único de identificação da câmera retornou quando você criou a câmera Devoluções: Real Exemplo: var vx = camera_get_view_x(view_camera[0]); var vy = camera_get_view_y(view_camera[0]); draw_text(vx + 5, vy + 5, "SCORE" + string(score)); O código acima recupera a posição da câmera designada para visualizar a porta[0] e depois desenha o texto relativo a essa posição.
