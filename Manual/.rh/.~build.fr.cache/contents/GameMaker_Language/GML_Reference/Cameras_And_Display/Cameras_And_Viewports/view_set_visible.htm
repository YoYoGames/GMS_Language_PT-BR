view_set_visible Esta função pode ser usada para definir a visibilidade do porto de vista dado. A função toma o índice da porta de visualização (de 0 a 7) e um booleano verdadeiro se for visível e falso se não for. Sintaxe: view_set_visible(view_port, visível) Argumento Descrição view_port O porto de visão para o alvo (0 - 7) visível Alternância de visibilidade(verdadeiro é visível e falso é invisível) Devoluções: N/A Exemplo: if !view_get_visible(0) { view_set_visible(0, true); } O código acima verificará se a porta de visualização[0] está visível e se não está definida para visível.
