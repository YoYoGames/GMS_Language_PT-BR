câmera_set_view_angle Você pode usar esta função para atualizar o ângulo da câmera de visão dentro da sala. Você dá o valor único de identificação da câmera (como retornado pelos diferentes camera_create() funções) e então dar o ângulo que você deseja ajustar a visão da câmera. O valor padrão é 0° com valores positivos girando a câmera no sentido anti-horário, ou seja: ajustando o valor para 90° girará a câmera 90° para a esquerda. Sintaxe: camera_set_view_angle(camera_id, angle) Argumento Descrição camera_id O valor único de identificação da câmera retornou quando você criou a câmera ângulo O novo ângulo para definir a visão da câmera para Devoluções: N/A Exemplo: camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y)) O código acima definirá o ângulo de visão da câmera designada para a porta de visão[0].
