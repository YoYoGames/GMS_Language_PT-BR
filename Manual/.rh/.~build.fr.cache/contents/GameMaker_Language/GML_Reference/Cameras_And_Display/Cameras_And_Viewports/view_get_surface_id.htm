view_get_surface_id Esta função pode ser usada para recuperar o valor de identificação único para a superfície atribuída à porta de visualização dada (retornará -1 se nenhuma superfície tiver sido atribuída). Sintaxe: view_get_surface_id(view_port) Argumento Descrição view_port O porto de visão para o alvo (0 - 7) Devoluções: Real Exemplo: if view_get_surface_id(0) == -1 { view_set_surface_id(0, global.vSurf); } O código acima verificará se uma superfície foi atribuída à porta de visualização[0] e se não foi atribuída, então uma é atribuída.
