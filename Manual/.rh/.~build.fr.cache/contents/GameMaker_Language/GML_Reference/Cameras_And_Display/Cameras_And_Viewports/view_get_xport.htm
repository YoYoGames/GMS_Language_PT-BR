view_get_xport Esta função pode ser usada para recuperar a posição x da porta de visualização dada. Sintaxe: view_get_xport(view_port) Argumento Descrição view_port O porto de visão para o alvo (0 - 7) Devoluções: Real Exemplo: se view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2) { view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2)); } O código acima verificará a posição x da porta de visualização[0] e, se não estiver onde é necessário estar, será ajustado para essa posição.
