view_set_wport Esta função pode ser usada para definir a largura da porta de visualização dada. Você dá o índice da porta de visualização (de 0 a 7) e a nova largura a ser usada. Sintaxe: view_set_wport(view_port, w) Argumento Descrição view_port O porto de visão para o alvo (0 - 7) w A nova largura (em pixels) Devoluções: Real Exemplo: se view_get_wport(0) != (display_get_width () / 2) { view_set_wport(0, display_get_width() / 2); } O código acima verificará a largura da porta de visualização[0] e, se não for metade da largura do visor, será ajustado para esse valor.
