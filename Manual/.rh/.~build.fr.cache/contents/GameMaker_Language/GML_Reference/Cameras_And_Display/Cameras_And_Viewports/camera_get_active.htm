camera_get_active Esta função pode ser usada para recuperar o valor único de identificação da câmera atualmente ativa. Sintaxe: camera_get_active() Devoluções: Índice da câmera (real). Exemplo: var active = camera_get_active(); if active != view_camera[0] { view_camera[0] = ativo; } O código acima obtém a identificação da câmera ativa e define a câmera de visualização para a porta [0] para usá-la.
