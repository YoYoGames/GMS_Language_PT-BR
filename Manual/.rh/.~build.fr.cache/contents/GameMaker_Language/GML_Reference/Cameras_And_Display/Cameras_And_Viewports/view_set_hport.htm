view_set_hport Esta função pode ser usada para definir a altura da porta de visão dada. Você dá o índice da porta de visão (de 0 a 7) e a nova altura a ser usada. Sintaxe: view_set_hport(view_port, h) Argumento Descrição view_port O porto de visão para o alvo (0 - 7) h A nova altura (em pixels) Devoluções: Real Exemplo: se view_get_hport(0) != (display_get_height () / 2) { view_set_hport(0, display_get_height() / 2); } O código acima verificará a altura da porta de visualização[0] e se não for metade da altura do visor, será ajustado para esse valor.
