camera_get_view_speed_speed_y Esta função pode ser usada para recuperar a velocidade de movimento da câmera dada ao longo do eixo y (movimento vertical). O valor de retorno será em pixels por quadro de jogo. Sintaxe: camera_get_view_speed_speed_y(camera_id) Argumento Descrição camera_id O valor único de identificação da câmera retornou quando você criou a câmera Devoluções: Real Exemplo: var xs = camera_get_view_speed_x(view_camera[0]); var ys = camera_get_view_speed_y(view_camera[0]); if xs != 5 |||| ys != 5 { camera_set_view_speed_speed(view_camera[0], 5, 5); } O código acima recupera a velocidade x e y da câmera designada para visualizar a porta[0] e então verifica se ela corresponde ao valor dado. Se não corresponder, a câmera de visualização é ajustada para essa velocidade.
