view_visible Esta variável pode ser usada para descobrir se uma determinada porta de visualização está visível ou não no momento. Você também pode definir esta variável para efetivamente "ligar" ou "desligar" uma vista definindo o valor como verdadeiro (visível) ou falso (invisível) . Observe que mesmo que você tenha uma porta de visualização definida para visível, se as portas de visualização não estiverem ativadas (usando a variável embutida view_enabled ou habilitá-los no editor da sala), então eles não serão atraídos para a tela. Sintaxe: view_visible[0 ... 7]; Devoluções: Booleano(verdadeiro: habilitado, falso: desabilitado) Exemplo: se !view_visible[0] { view_visible[0] = true; } O código acima verifica se a vista[0] é visível ou não e, se não for, então a define para ser visível.
