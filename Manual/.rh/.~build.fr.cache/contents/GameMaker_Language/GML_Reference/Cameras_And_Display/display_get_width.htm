largura_de_visualização Esta função retornará a largura do visor (em pixels). Observe que no alvo HTML5, este valor é a largura da janela do navegador, ao invés do tamanho da tela física. Sintaxe: display_get_width(); Devoluções: Real Exemplo: mywidth = display_get_width(); Isto ajustaria a variável "mywidth" para a largura do display.
