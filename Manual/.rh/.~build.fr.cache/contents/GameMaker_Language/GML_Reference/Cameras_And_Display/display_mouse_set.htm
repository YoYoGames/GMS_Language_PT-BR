display_mouse_set Com esta função você pode mudar ou definir a posição do mouse dentro da exibição do jogo, o que pode ser útil para jogos FPS, por exemplo. A função só funcionará enquanto o jogo estiver em foco e, usando ALT + Tab, o mouse será desbloqueado. Sintaxe: display_mouse_set(x, y); Argumento Descrição x A coordenada x no visor. y A coordenada y no display. Devoluções: N/A Exemplo: display_mouse_set(display_get_width() / 2, display_get_height() / 2); O código acima centralizaria o mouse na tela do jogo.
