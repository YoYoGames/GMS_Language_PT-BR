display_get_sleep_margin Esta função pode ser usada para obter o valor atual da margem de sono usada para o tempo de renderização de seu jogo, e retornará um valor de milissegundos. Para mais informações sobre o tempo de exibição, por favor veja display_set_timing_method(). Sintaxe: display_get_sleep_margin(); Devoluções: Real Exemplo: se display_get_timing_method() != tm_sleep { display_set_set_timing_method(tm_sleep); if display_get_sleep_margin() != 20 { display_set_sleep_sleep_margin(20); } } O código acima verificará o método de tempo e, se não estiver ajustado para tm_sleep , será ajustado para isso e a margem de sono será ajustada para 20.
