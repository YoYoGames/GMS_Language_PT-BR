gif_save Com esta função, você pode salvar uma animação GIF. Você fornece o índice GIF (como retornado pela função gif_open()), bem como um nome de arquivo para salvá-lo. Note que o GameMaker Studio 2 não é automaticamente anexe a extensão do arquivo .gif, portanto você deve incluir isto como parte da cadeia de nomes do arquivo se desejar que o arquivo salvo seja identificado como um GIF. O GIF criado será paletizado usando a Paleta Universal 884 (veja aqui para mais informações). Observe que se a função for bem sucedida e o GIF for salvo corretamente, então ele retornará 0, caso contrário retornará -1.
