display_mouse_get_x Esta função devolverá a posição x do mouse dentro da tela. Deve-se notar que esta função só funciona corretamente quando usada no alvo do Windows. Ela também pode ser usada para HTML5, mas só retornará um valor relativo ao valor 0,0 da própria tela, e não devolverá nenhum valor enquanto o mouse estiver fora da tela. Para outros dispositivos, ele retornará 0, e você deve usar o device_mouse_raw_x()e dispositivo_mouse_raw_y() funções em seu lugar. Sintaxe: display_mouse_get_x(); Devoluções: Real Exemplo: my_x = display_mouse_get_x(); Isto ajustaria meu_x para a coordenada x do mouse em relação à tela.
