xboxlive_gamertag_for_user Esta função obterá a tag Gamer para o usuário atualmente inscrito. Note que esta função só é válida se um usuário estiver conectado e, como tal, você deve fazer uma verificação para isso usando a função xboxlive_user_is_signed_in() antes de solicitar o Gamer Tag. Se você chamar esta função quando nenhum usuário estiver conectado, você receberá uma string vazia"" retornado. Sintaxe: xboxlive_gamertag_for_user(); Devoluções: Cordão Exemplo: if !xboxlive_user_is_signed_in() { if !xboxlive_user_is_signing_in() { xboxlive_show_account_picker(); } } else { global.GamerTag = xboxlive_gamertag_for_user(); } O código acima verifica se um usuário está conectado ao XBox Live e se ele está conectado, o código irá obter o gamertag para o usuário e armazená-lo em uma variável global para uso futuro. Se não estiverem assinados, o código verificará se eles estão no processo de assinatura, caso em que nada mais acontecerá, e se não estiverem, então abrirá o selecionador de contas, levando-os a assinar.
