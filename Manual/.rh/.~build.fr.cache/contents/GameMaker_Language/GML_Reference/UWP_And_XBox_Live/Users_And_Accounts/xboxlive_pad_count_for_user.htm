xboxlive_pad_count_for_user Com esta função você pode descobrir quantos "slots" de bloco estão ativos para o usuário atual (veja xboxlive_pad_for_user() para mais detalhes). Sintaxe: xboxlive_pad_count_for_user(user_id, slot); Argumento Descrição user_id O ID do usuário (um ponteiro) para verificar Devoluções: Inteiro Exemplo: global.slots[0] = xboxlive_pad_count_for_user(user_id[0]) O código acima armazena o número de blocos associados com o ponteiro de identificação do usuário dado.
