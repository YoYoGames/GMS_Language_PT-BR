xboxlive_id_de_utilizador_para_utilizador Esta função converte o ponteiro de identificação do usuário em um fio. Sintaxe: xboxlive_user_id_id_for_user(user_id); Argumento Descrição user_id O ponteiro de identificação do usuário a ser verificado. Devoluções: Cordão Exemplo: var _uid = xboxlive_user_for_pad(0); uid_string = xboxlive_user_id_id_for_user(_uid); O código acima obtém o ponteiro de identificação do usuário para o usuário atribuído ao slot do gamepad [0], e então salva essa identificação do usuário como uma string para uma variável de instância.
