xboxlive_sponsor_para_utilizador Esta função recupera a identificação do patrocinador do usuário em questão. Sintaxe: xboxlive_sponsor_para_utilizador(user_id); Argumento Descrição user_id O ponteiro de identificação do usuário a ser verificado. Devoluções: Real Exemplo: var _uid = xboxlive_user_for_pad(0); sponsor = xboxlive_sponsor_sponsor_for_user(_uid); O código acima obtém o ponteiro de identificação do usuário para o usuário atribuído ao slot gamepad [0], e então usa essa identificação para recuperar a identificação do patrocinador do usuário.
