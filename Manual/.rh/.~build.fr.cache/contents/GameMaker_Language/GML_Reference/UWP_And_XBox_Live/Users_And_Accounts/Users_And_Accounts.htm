Usuários e contas Esta seção é para usuários direcionados às plataformas UWP ou Xbox One e descreve as funções do XBox Live disponíveis para lidar com usuários e contas de usuários. O Xbox One permite que vários usuários se conectem a um console simultaneamente. Cada usuário tem um controlador específico associado a ele e qualquer dado de jogo salvo também é específico do usuário. Para aproveitar ao máximo este sistema, o GameMaker Studio 2 introduz um número da XBox One somente funções relacionadas com a associação entre controladores e usuários, permitindo-lhe consultar informações sobre o usuário e também controlar como são tratados os dados salvos por usuário. Você pode encontrar mais informações do desenvolvedor sobre este sistema do white paper oficial no site do desenvolvedor Microsoft. NOTA: Note que o ID de usuário é único para cada indivíduo e é perpetuado em consoles diferentes, portanto, se o mesmo usuário assinar em dois dispositivos diferentes, este ID de usuário será o mesmo.
