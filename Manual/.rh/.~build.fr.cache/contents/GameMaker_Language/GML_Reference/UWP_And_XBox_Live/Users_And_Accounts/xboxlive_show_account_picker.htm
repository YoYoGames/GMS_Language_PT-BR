xboxlive_show_account_picker A função abrirá a tela do XBox Account Picker para que o usuário possa entrar se ainda não tiver entrado (se já tiver entrado, a função não fará nada). Note que esta função só é aplicável quando se utiliza o UWP construído em PC e não no Xbox, para lançar um jogo no Xbox, o usuário já deve ter se inscrito. Sintaxe: xboxlive_show_account_picker(); Devoluções: N/A Exemplo: if !xboxlive_user_is_signed_in() { if !xboxlive_user_is_signing_in() { xboxlive_show_account_picker(); } } else { global.GamerTag = xboxlive_gamertag_for_user(); } O código acima verifica se um usuário está conectado ao XBox Live e se ele está conectado, o código irá obter o gamertag para o usuário e armazená-lo em uma variável global para uso futuro. Se não estiverem assinados, o código verificará se eles estão no processo de assinatura, caso em que nada mais acontecerá, e se não estiverem, então abrirá o selecionador de contas, levando-os a assinar.
