xboxlive_matchmaking_stop Esta função pode ser usada para encerrar uma sessão de matchmaking para o usuário em questão. Sintaxe: xboxlive_matchmaking_stop(user_id); Argumento Descrição user_id O ponteiro de identificação do usuário a ser usado Devoluções: N/A Exemplo: if !xboxlive_user_is_signed_in(user_id[0]) { xboxlive_matchmaking_stop(user_id[0]); } O código acima terminará a sessão de matchmaking para o usuário em questão.
