uwp_license_trial_version Esta função pode ser usada para verificar se o jogo está ou não sob uma licença de teste. Se for a função retornará verdadeiro, ou falso de outra forma. Sintaxe: uwp_license_trial_version(); Devoluções: Booleano Exemplo: se uwp_license_trial_version() { global.LevelCap = 10; } O código acima verifica se o aplicativo está sob uma licença de teste e, em caso afirmativo, define uma variável global.
