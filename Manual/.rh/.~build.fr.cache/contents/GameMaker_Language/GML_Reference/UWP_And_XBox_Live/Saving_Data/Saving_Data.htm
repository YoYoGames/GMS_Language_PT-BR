Salvando dados As seguintes funções têm especificamente a ver com a economia de dados usando a exportação UWP para o Xbox One e usando as funcionalidades do XBox Live. Enquanto os dados de gravação relacionados ao progresso do jogador devem estar sempre associados a um determinado usuário, você pode salvar dados genéricos na área de "armazenamento da máquina", fornecendo o valor -1 como argumento do usuário. Esta área é específica para o título, mas não associado a um determinado usuário. NOTA: Nenhum arquivo salvo pode ser maior que 16 megabytes!
