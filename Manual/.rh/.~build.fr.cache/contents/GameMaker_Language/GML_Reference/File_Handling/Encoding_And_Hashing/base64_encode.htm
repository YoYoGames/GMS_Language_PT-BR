base64_código Esta função converterá uma string em uma string codificada no formato base64. Este é um esquema de codificação comumente utilizado para qualquer mídia que precise ser armazenada ou transferida pela Internet como texto, e torna a saída ilegível para o olho humano. Sintaxe: base64_encode(string) Argumento Descrição corda O fio a ser codificado. Devoluções: Cordão Exemplo: var str, file; str = base64_encode(game_data); file = file_text_open_write("save.txt"); file_text_write_string(file, str); file_text_close(file); O código acima irá encobrir a string armazenada em "game_data" em uma string codificada base64 que é então armazenada em um arquivo de texto externo.
