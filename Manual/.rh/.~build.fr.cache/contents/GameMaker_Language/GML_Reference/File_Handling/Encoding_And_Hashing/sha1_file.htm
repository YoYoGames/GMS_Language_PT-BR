sha1_file Na criptografia, SHA-1 é uma função de hash criptográfico projetada pela Agência de Segurança Nacional dos Estados Unidos e é empregada em diversas aplicações amplamente utilizadas Git onde é usado para verificar alterações de arquivos, e os protocolos TLS e SSL, PGP, SSH, S/MIME, e IPsec. Esta função pegará um arquivo de entrada e retornará um resumo de 160 bit de mensagem em formato ASCII exclusivo a esse arquivo para ser usado para verificação de integridade em qualquer data posterior. NOTA: O SHA-1 não é completamente seguro e pode ser quebrado. Veja esta página para mais informações. Sintaxe: sha1_file(nome do arquivo) Argumento Descrição nome do arquivo O arquivo para gerar o hash sha1. Devoluções: Cordão Exemplo: hash = sha1_file(directório_de_trabalho + "game_data.ini") O código acima irá gerar um hash sha1 para o arquivo especificado e armazenar o valor retornado na variável "hash".
