diretório_criar Esta função criará um diretório com o nome dado na área de salvamento. NOTA: Esta função não funcionará em HTML5, pois não é possível criar ou remover diretórios no armazenamento local do navegador. ATENÇÃO! Esta função pode não funcionar como você espera devido ao GameMaker Studio 2 ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações. Sintaxe: directory_create(dname) Argumento Descrição dname O nome do diretório a ser criado. Devoluções: N/A Exemplo: se !directory_exists("Jogos") { directory_create("Jogos"); } Isto verificará se o diretório especificado existe na pasta de dados local e, se não existir, o criará para você.
