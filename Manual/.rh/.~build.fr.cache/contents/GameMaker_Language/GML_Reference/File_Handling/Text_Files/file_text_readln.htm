file_text_readln Com esta função você pode pular o restante da linha atual de um determinado arquivo de texto aberto e passar para o início do próximo arquivo. A função também retornará a linha completa como um fio, tornando-a uma maneira fácil de ler "pedaços" completos. de dados para análise posterior. Sintaxe: file_text_readln(fileid); Argumento Descrição fileid A identificação do arquivo a ser lido. Devoluções: corda Exemplo: var file = file_text_open_read(working_directory + "hiscore.txt"); for (var i = 0; i < 10; ++i;) { scr[i] = file_text_read_real(file); file_text_readln(file); scr_name[i] = file_text_read_string(file); file_text_readln(file); } file_text_close(file); O código acima abre um arquivo para leitura e depois percorre as linhas da leitura do arquivo alternadamente um valor numérico real e uma cadeia em duas matrizes diferentes para uso futuro.
