file_exists Esta função retornará verdadeiro se o arquivo especificado existir e falso se não existir. Note que a função só pode ser usada para verificar arquivos locais, mas não quaisquer arquivos armazenados em um servidor remoto. Sintaxe: file_exists(fname); Argumento Descrição fname O nome do arquivo a ser verificado. Devoluções: Booleano Exemplo: se file_exists("level.txt") { file = file_text_open_read("level.txt"); } Isto verificaria a existência de um arquivo e, se existir, ele está aberto para leitura.
