filename_drive Esta função retorna as informações de acionamento do nome do arquivo. Sintaxe: filename_drive(fname); Argumento Descrição fname O arquivo a ser utilizado. Devoluções: Cordão Exemplo: drive = nome_do_arquivo(file_find_first(directório_de_trabalho + "*.doc", 0))); O código acima obtém as informações da unidade (como uma string) do arquivo especificado.
