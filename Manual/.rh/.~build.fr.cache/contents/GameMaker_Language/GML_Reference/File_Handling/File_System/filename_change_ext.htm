nome_de_arquivo_mudar_extra Esta função retorna o nome do arquivo indicado com a extensão (incluindo o ponto) alterado para a nova extensão. Ao usar uma string vazia como nova extensão, você pode remover a parte da extensão em conjunto. Sintaxe: filename_change_ext(fname, newext); Argumento Descrição fname O arquivo a ser utilizado. newext A nova extensão de uso. Devoluções: Cordão Exemplo: ext = filename_change_ext(file_find_first(file_find_first(directório_de_trabalho + "*.*", 0), ""); O código acima recebe o nome do arquivo (como uma string) com a parte da extensão removida.
