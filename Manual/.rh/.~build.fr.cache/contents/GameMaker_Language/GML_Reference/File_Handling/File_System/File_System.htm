Sistema de arquivo Existe no GameMaker Studio 2 uma série de funções que permitem a manipulação de arquivos de todos os tipos de diferentes maneiras: file_exists file_delete file_rename file_copy file_find_first file_find_next file_find_close file_attributes As seguintes funções podem ser usadas para mudar os nomes dos arquivos: NOTA: Estas funções não funcionam nos arquivos reais! Elas lidam apenas com as cordas relacionadas aos diferentes elementos do caminho e devem ser usadas em conjunto com as outras funções dos arquivos. nome_nome_do_arquivo filename_path filename_dir filename_drive nome do arquivo_ext nome_de_arquivo_mudar_extra Finalmente há algumas funções especiais para obter nomes de arquivos que podem ser usados para carregar e salvar arquivos com filtros do tipo arquivo. Estas funções abrirão uma janela exploradora e, portanto, farão com que o jogo perca o foco. Isto significa que qualquer entrada no O jogo pode não estar corretamente registrado e, como tal, recomendamos que (se usar estas funções em um evento de teclado) você use o evento de tecla para cima e não os eventos pressionados ou para baixo. NOTA: Estas funções são Windows e macOS somente. ADVERTÊNCIA! Estas funções podem não funcionar como você espera devido ao GameMaker Studio 2 ser lixado!
