filename_dir Esta função retorna a parte do diretório do nome do arquivo indicado, que normalmente é o mesmo que o caminho, exceto pela contrabarra final. Sintaxe: filename_dir(fname); Argumento Descrição fname O arquivo a ser utilizado. Devoluções: Cordão Exemplo: dir = filename_dir("Test.ini"); O código acima recebe o diretório (como uma string) do arquivo especificado.
