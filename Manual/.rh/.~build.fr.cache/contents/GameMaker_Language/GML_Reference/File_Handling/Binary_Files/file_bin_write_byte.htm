file_bin_write_byte Esta função irá escrever um byte de dados no arquivo identificado pelo ID do arquivo na posição de escrita atual. Você fornece o valor do ID do arquivo, conforme retornado pela função file_bin_open() e o byte de dados para escrever. NOTA: Estas funções não funcionam quando o módulo alvo é HTML5. Sintaxe: file_bin_write_byte(binfile, byte); Argumento Descrição binfile O ID do arquivo para o qual escrever. byte Os dados a serem escritos. Devoluções: real Exemplo: file_bin_write_byte(file, data); Isto escreveria um byte para o arquivo selecionado.
