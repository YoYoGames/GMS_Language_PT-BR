Arquivos binários Em situações raras você pode precisar ler dados de arquivos binários. Essas funções podem ser usadas para esse fim, embora, em geral, recomendamos que você use as funções de buffer dedicadas para criar, salvar e carregar dados binários, em vez destas funções. NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.
