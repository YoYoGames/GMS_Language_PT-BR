buffer_get_alignment Com esta função você pode obter o alinhamento de um byte para o ID do buffer dado. Sintaxe: buffer_get_alignment(buffer); Argumento Descrição tampão O índice do buffer a ser verificado. Devoluções: Real Exemplo: alinhamento = buffer_get_alignment(buff); O código acima obterá o alinhamento do buffer a partir do valor indexado na variável "buff" e o armazenará em uma variável.
