buffer_read Esta função pode ser usada para ler dados de um buffer previamente criado. O valor de retorno dependerá do tipo de dados que você estiver lendo, o que em si mesmo é definido pelas seguintes constantes: Constante Descrição buffer_u8 Um número inteiro não assinado de 8 bits. Este é um valor positivo de 0 a 255. buffer_s8 Um número inteiro de 8 bits assinado. Este pode ser um valor positivo ou negativo de -128 a 127 (0 é classificado como positivo). buffer_u16 Um inteiro de 16 bits, não assinado. Este é um valor positivo de 0 - 65,535. buffer_s16 Um número inteiro de 16 bits assinado. Este pode ser um valor positivo ou negativo de -32.768 a 32.767 (0 é classificado como positivo). buffer_u32 Um número inteiro não assinado de 32 bits. Este é um valor positivo de 0 a 4.294.967.295. buffer_s32 Um número inteiro de 32 bits assinado. Este pode ser um valor positivo ou negativo de -2.147.483.648 a 2.147.483.647 (0 é classificado como positivo). buffer_u64 Um número inteiro não assinado de 64 bits. buffer_f16 Um flutuador de 16 bits. Este pode ser um valor positivo ou negativo dentro da faixa de +/- 65504. (Não suportado atualmente!) buffer_f32 Um flutuador de 32bit. Este pode ser um valor positivo ou negativo dentro da faixa de +/-16777216. buffer_f64 Um flutuador de 64 bits. buffer_bool Um valor booleano. Pode ser apenas 1 ou 0(verdadeiro ou falso) cordão_tampão Um fio de qualquer tamanho. texto_tampão Uma cadeia de qualquer tamanho, sem o caráter final de terminação nula. Se a função for bem sucedida, ela retornará um valor do tipo dado, porém, se falhar, causará um erro de corredor. NOTA: O uso do tipo de dados incorretos para os dados que estão sendo lidos resultará em valores errôneos que serão devolvidos. Sintaxe: buffer_read(buffer, tipo) Argumento Descrição tampão O índice do buffer a ser lido. tipo O tipo de dados que devem ser lidos do buffer (ver a lista de constantes acima). Devoluções: Real/Calça Exemplo: var cmd = buffer_read(buff, buffer_s16); O código acima lê do buffer com o id armazenado na variável "buff" um valor de 16 bits assinado na variável local "cmd".
