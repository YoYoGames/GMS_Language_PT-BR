tamanho_utilizado_do_soltampão Esta função é principalmente para uso dentro de extensões, e permite definir o tamanho "usado" do buffer dado, que é o número de bytes que foram escritos a ele. Quando você escreve dados em um buffer de uma extensão, o GameMaker não sabe quanto do buffer foi preenchido pelo código da extensão e não é capaz de ler esses dados. Esta função pode ser chamada pela extensão para dizer ao mecanismo quantos bytes de dados foi escrito no buffer, para que o motor possa ler esses dados. Sintaxe: buffer_set_use_size(buffer, tamanho); Argumento Descrição tampão O índice do buffer a ser utilizado. tamanho O número de bytes a definir como o tamanho "usado". Devoluções: N/A Exemplo: buffer_write(_bufferAddress, buffer_u8, 1); buffer_write(_bufferAddress, buffer_u8, 2); buffer_write(_bufferAddress, buffer_u16, 400); buffer_set_used_size(_bufferAddress, 4); As funções acima seriam chamadas em uma extensão para escrever alguns dados em um buffer GameMaker (através de seu endereço de memória), e então definir o número de bytes que lhe foram escritos para que o motor possa ler esses dados.
