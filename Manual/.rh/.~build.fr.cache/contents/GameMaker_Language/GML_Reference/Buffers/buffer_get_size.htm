tamanho_do_tampão_de_servidor Com esta função, você pode obter o tamanho do buffer dado em bytes. Sintaxe: buffer_get_size(index); Argumento Descrição índice O índice do buffer para obter o tamanho de. Devoluções: Real Exemplo: var _tamanho = buffer_get_size(player_data); var _temp = buffer_create(_tamanho, buffer_fixed, 0); O código acima criará um novo buffer e armazenará seu índice na variável local "_temp", sendo o tamanho deste novo buffer o mesmo que o do buffer indexado na variável "player_data".
