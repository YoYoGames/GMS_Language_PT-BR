buffer_existe Esta função pode ser usada para verificar uma variável para ver se ela possui ou não um valor de identificação de buffer válido. Se ela fizer a função retornará verdadeira, caso contrário retornará falsa. Sintaxe: buffer_exists(buffer) Argumento Descrição tampão O índice do buffer a ser verificado. Devoluções: Booleano Exemplo: se buffer_exists(buff) { buffer_delete(buff); } O código acima verifica se a variável "buff" possui uma identificação de buffer e se possui, o buffer é apagado.
