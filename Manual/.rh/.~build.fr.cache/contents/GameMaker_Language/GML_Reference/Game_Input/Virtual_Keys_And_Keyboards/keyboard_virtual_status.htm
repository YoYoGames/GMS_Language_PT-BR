estado_virtual_do_ teclado Esta função pode ser usada para obter o status do teclado virtual no dispositivo que executa o jogo. A função retornará verdadeiro se o teclado virtual do SO estiver visível/em exibição ou falso se estiver oculto/esconde. Sintaxe estado_virtual_do_ teclado(); Devoluções: Booleano Exemplo: var _status = keyboard_virtual_status(); if _status == false { keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false); } O código acima mostrará o teclado virtual do sistema operacional se o status atual for falso.
