gesture_get_drag_distance Esta função é utilizada para obter a distância necessária para que um evento de gesto de arrastamento seja acionado pelo movimento de um toque ou clique. A distância é medida em polegadas e tem um valor padrão de 0,1. Sintaxe: gesture_get_drag_distance(); Devoluções: Real (polegadas) Exemplo: if gesture_get_drag_distance() != 0.1 { gesture_drag_distance(0.1); } O código acima verifica se a distância de arrasto para gestos está ajustada a 0,1 polegadas e se não está ajustada a esse valor.
