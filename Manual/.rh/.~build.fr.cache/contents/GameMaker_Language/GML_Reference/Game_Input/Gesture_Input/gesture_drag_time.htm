gesto_drag_time Esta função é usada para definir o tempo necessário para que um evento de Drag Gesture seja acionado por um toque ou clique. Este tempo também afetará a forma como o Evento de Toque é acionado como um toque/clique e liberação antes que este tempo acabe, será considerado um Tap. O tempo é medido em segundos e tem um valor padrão de 0,16. Sintaxe: gesture_drag_time(tempo); Argumento Descrição tempo O tempo (em segundos) a ser ajustado para a detecção do evento do gesto de arrasto. Devoluções: N/A Exemplo: if gesture_get_drag_time() != 0.16 { gesture_drag_time(0.16); } O código acima verifica se o tempo de arrastamento dos gestos está configurado para 0,16 segundos e se não está configurado para esse valor.
