gesture_get_double_tap_time Esta função é usada para obter o tempo que leva entre dois toques/clique para acionar um evento de gesto de duplo toque. O tempo é medido em segundos e tem um valor padrão de 0,16. Sintaxe: gesture_get_double_tap_time(); Devoluções: Real (segundos) Exemplo: if gesture_get_double_tap_time() != 0.16 { gesture_get_double_tap_time(0.16); } O código acima verifica se o tempo de toque duplo para gestos está configurado para 0,16 segundos e se não está configurado para esse valor.
