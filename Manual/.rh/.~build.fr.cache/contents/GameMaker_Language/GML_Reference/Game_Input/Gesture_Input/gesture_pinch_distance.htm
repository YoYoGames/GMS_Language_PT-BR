gesto_picch_distance Esta função é utilizada para definir a distância dentro da qual você tem que tocar/clicar a tela e mover-se com dois dedos antes de acionar um gesto de beliscão. A distância é medida em polegadas e tem um valor padrão de 0,1. Sintaxe: gesto_picch_distance(); Devoluções: Real (polegadas) Exemplo: if gesture_get_pinch_distance() != 0.1 { gesture_pinch_distance(0.1); } O código acima verifica se a distância de aperto para gestos está ajustada a 0,1 polegadas e se não está ajustada a esse valor.
