Entrada por gestos As funções de Entrada do Dispositivo são todas boas e bem para lidar com simples toques na tela ou obter orientação do dispositivo, mas às vezes você vai querer que seus projetos detectem coisas mais complexas, como ser capaz de arrastar ou "flick" uma instância por uma sala. Estas coisas podem ser feitas com as funções do Dispositivo, mas para facilitar as coisas, o GameMaker Studio 2 tem uma gama completa de eventos de gestos para detectar torneiras duplas, arrastamento, movimentos, pitadas e rotações, tanto a nível de instância como a nível global, como bem como informações de retorno sobre o evento. As funções listadas aqui são todas projetadas para lhe dar maior controle sobre como e quando estes eventos serão acionados. IMPORTANTE! Estas funções não funcionarão na plataforma de destino HTML5. Elas não darão um erro se usadas, mas serão simplesmente ignoradas.
