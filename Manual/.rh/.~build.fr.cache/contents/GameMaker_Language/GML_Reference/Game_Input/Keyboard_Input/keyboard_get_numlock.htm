teclado_get_numlock Você pode usar esta função para encontrar o status do bloqueio do número do teclado com o verdadeiro sendo devolvido para ligado, e o falso sendo devolvido para desligado. OBSERVAÇÃO: Esta funcionalidade só está disponível no exe do Windows e não funcionará em nenhum outro dispositivo. Sintaxe: teclado_get_numlock(); Devoluções: Booleano Exemplo: se teclado_get_numlock() { keyboard_set_numlock(false); } mais { keyboard_set_numlock(true); } O código de exemplo acima obterá o estado da chave de bloqueio numérico e, se estiver ligado (verdadeiro), o fará disparar (falso) e vice-versa.
