teclado_lastchar Esta variável armazena uma seqüência da última tecla pressionada. Esta variável não é somente lida e você pode modificá-la, por exemplo, para defini-la como "" (uma string vazia) se você já a manipulou. Sintaxe: teclado_lastchar; Devoluções: Cordão Exemplo: if keyboard_lastkey != -1 { str += keyboard_lastchar; keyboard_lastkey = -1; } O código acima verifica se a última variável chave é maior que -1, e se é ela adiciona qualquer que seja a última chave como uma string à variável "str".
