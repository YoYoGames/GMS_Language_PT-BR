teclado_chave Com esta variável você pode obter o código da tecla que está sendo pressionada atualmente e ele retornará 0 se nenhuma tecla estiver sendo pressionada quando a verificação for feita. Sintaxe: teclado_chave; Devoluções: Real Exemplo: switch (teclado_chave) { caixa vk_numpad1: arma = arma[0][0]; pausa; caixa vk_numpad2: arma = arma[1][0]; pausa; caixa vk_numpad3: arma = arma[2][0]; pausa; } O código acima usa o valor da variável keyboard_key para definir uma variável com o mesmo valor de um array.
