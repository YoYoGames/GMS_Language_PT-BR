mouse_wheel_down Esta função retorna verdadeiro se a roda do mouse estiver sendo girada para baixo e falso caso contrário. Sintaxe: mouse_wheel_down(); Devoluções: Booleano Exemplo: se mouse_wheel_down() { y += 10; } Isto move a instância atual para baixo da tela se a roda do mouse for girada para cima.
