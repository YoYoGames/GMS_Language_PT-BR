mouse_clear Esta função limpará o estado atual de determinado botão do mouse. Isto significa que as verificações de que ele está sendo mantido pressionado não retornarão à realidade até que o jogador libere o botão e pressione novamente (mas o estado de liberação ainda será detectado se a limpeza for feita enquanto o botão do mouse estiver sendo mantido pressionado). Você pode fornecer uma das constantes do botão do mouse listadas nesta página. Sintaxe: mouse_clear(botão); Argumento Descrição botão Qual a constante do botão do mouse a verificar. Devoluções: Booleano Exemplo: mouse_clear(mb_any); O código acima limpará o estado para baixo de todos os botões do mouse.
