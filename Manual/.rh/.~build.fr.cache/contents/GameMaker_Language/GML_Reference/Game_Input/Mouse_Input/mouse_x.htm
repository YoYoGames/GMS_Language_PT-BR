mouse_x Esta variável somente leitura retorna a posição atual do eixo x do mouse dentro da sala. Sintaxe: mouse_x; Devoluções: Real Exemplo: x = mediana(64, mouse_x, largura_de_sala - 64); O código acima manterá a instância na posição x do mouse enquanto estiver dentro dos limites de 64 pixels de cada lado da sala.
