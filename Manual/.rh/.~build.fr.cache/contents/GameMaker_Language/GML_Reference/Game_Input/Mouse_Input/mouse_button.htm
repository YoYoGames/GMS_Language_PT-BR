mouse_button Esta variável lida apenas retorna o botão do mouse que está sendo pressionado (atualmente, como nesta etapa) e pode retornar qualquer uma das constantes especiais do mouse, exceto mb_any. Sintaxe: botão_do_mouse; Devoluções: Constante Exemplo: se mouse_button = mb_left { x -= 1; } Isto move a instância atual para a esquerda se o botão esquerdo do mouse estiver para baixo.
