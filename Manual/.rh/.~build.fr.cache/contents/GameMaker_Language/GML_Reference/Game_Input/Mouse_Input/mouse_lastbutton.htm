último botão do mouse Esta variável retorna o último botão do mouse que foi pressionado e pode retornar qualquer uma das constantes especiais do mouse, exceto mb_any (você também pode definir esta variável para uma das constantes). Sintaxe: último botão do mouse; Devoluções: Constante Exemplo: se mouse_lastbutton = mb_left { x -= 1; } Isto move a instância atual para a esquerda se o último botão pressionado fosse o botão esquerdo do mouse.
