mouse_y Esta variável somente de leitura retorna a posição atual do eixo y do mouse dentro da sala. Sintaxe: mouse_y; Devoluções: Real Exemplo: y = mediana(64, mouse_y, altura_do_quarto - 64); O código acima manterá a instância na posição y do mouse enquanto estiver dentro dos limites de 64 pixels da parte superior e inferior da sala.
