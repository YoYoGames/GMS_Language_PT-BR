gamepad_remove_mapping Esta função pode ser usada para remover o mapeamento do dispositivo atual do índice de slots do gamepad dado. Uma vez chamado, o índice de slot precisará ser refazido usando a função gamepad_test_mapping() função, se você quiser ser capaz de usar as constantes do gamepad para detectar a entrada corretamente (a entrada direta pode sempre ser recuperada usando o eixo do gamepad/button/hat_count() e gamepad_axis/button/hat_value() funciona em conjunto). Sintaxe: gamepad_remove_mapping(index); Argumento Descrição índice De qual "ranhura" do índice gamepad para remover o mapeamento. Devoluções: N/A Exemplo: se remap == verdadeiro { gamepad_remove_mapping(global.PadIndex); } O código acima removerá o mapeamento do slot de índice gamepad dado, com base no valor de uma variável.
