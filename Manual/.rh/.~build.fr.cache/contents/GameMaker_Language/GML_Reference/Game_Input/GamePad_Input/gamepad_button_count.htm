gamepad_button_count Esta função retornará o número total de botões disponíveis para o controle de jogo conectado ao "slot" do dispositivo em questão. Sintaxe: gamepad_button_count(dispositivo); Argumento Descrição dispositivo Qual dispositivo de controle de jogo "slot" a ser verificado. Devoluções: Real Exemplo: b_num = gamepad_button_count(0); O código acima retornará o número de botões disponíveis no controle de jogo conectado ao dispositivo "slot" 0 e armazenará o valor na variável "b_num".
