gamepad_axis_count Esta função retornará o número de controles de "eixo" no dispositivo que está sendo verificado. Estes controles são os "polegares" de direção analógica na maioria dos controladores. Sintaxe: gamepad_axis_count(dispositivo); Argumento Descrição dispositivo Qual dispositivo de controle de jogo "slot" a ser verificado. Devoluções: Real Exemplo: eixo = gamepad_axis_count(0)) O código acima armazena o número de eixos disponíveis para o controle de jogo conectado ao dispositivo "slot" 0 na variável "eixo".
