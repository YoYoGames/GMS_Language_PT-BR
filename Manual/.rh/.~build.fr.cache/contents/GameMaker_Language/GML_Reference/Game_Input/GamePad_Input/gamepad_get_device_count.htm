gamepad_get_device_count Esta função lhe dirá uma de duas coisas. Ou o número de game pads conectados, ou o número de "slots" disponíveis para os game pads a serem conectados. O valor real de retorno dependerá da plataforma e da configuração interna dessa plataforma e, como tal, essa função deve ser usada em conjunto com a função gamepad_is_connected() para garantir o número exato de almofadas conectadas a qualquer momento.
