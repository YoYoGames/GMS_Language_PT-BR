dia_actual Esta variável lida somente retornará o dia como um valor de 1 a 31, dependendo do mês. Sintaxe: dia_actual; Devoluções: Real Exemplo: draw_text(32, 32, "Hoje é " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +"); O código acima desenhará o dia, mês e ano.
