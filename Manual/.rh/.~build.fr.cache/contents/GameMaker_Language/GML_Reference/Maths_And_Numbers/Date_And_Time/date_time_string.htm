date_time_string Com esta função você pode criar uma seqüência contendo o tempo dado, formatado para o sistema ou dispositivo que está executando o jogo quando a função é chamada. Sintaxe: date_time_string( data ); Argumento Descrição data A data a ser usada. Devoluções: Cordão Exemplo: str = date_time_string(date_current_datetime())); Isto definiria a variável dada para algo como "11:36.00", dependendo das configurações do sistema para exibição de data/hora e da data e hora atuais.
