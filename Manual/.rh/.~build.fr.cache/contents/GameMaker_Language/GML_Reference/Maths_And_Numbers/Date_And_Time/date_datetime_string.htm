data_data_hora_string Com esta função você pode criar uma string contendo a data/hora determinada, formatada para o sistema ou dispositivo que está executando o jogo quando a função é chamada. Sintaxe: date_date_datetime_string(data); Argumento Descrição data A data a ser usada. Devoluções: Cordão Exemplo: str = date_date_datetime_string(date_current_datetime()); Isto definiria "str" para algo como "15 de setembro de 2011, 11:33.00", dependendo das configurações do sistema para exibição de data/hora e da data e hora atuais.
