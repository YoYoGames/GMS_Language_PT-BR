data_mês_de_inc Com esta função você pode incrementar um determinado valor de data/hora por um número específico de meses, e ele retornará o novo valor de data/hora. Sintaxe: date_inc_month( data, montante ); Argumento Descrição data A data a ser acrescentada. valor O número de meses (deve ser um número inteiro) a acrescentar. Devoluções: Real Exemplo: mynewdatetime = date_inc_month(date_current_datetime(), 12); Isto definiria o "mynewdatetime" para a data atual, mas com 12 meses adicionados.
