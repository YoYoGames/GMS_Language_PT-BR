date_day_span Com esta função você pode obter o número de dias entre duas datas. Este valor é sempre positivo, e dias incompletos serão devolvidos como uma fração. Sintaxe: date_day_span(data1, data2); Argumento Descrição data1 A primeira data. data2 A data e a hora para compará-la. Devoluções: Real Exemplo: diff = date_day_span(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime()); Isto definiria o número de dias entre 15 de setembro de 2011, 11:04.0 e a data e hora atual.
