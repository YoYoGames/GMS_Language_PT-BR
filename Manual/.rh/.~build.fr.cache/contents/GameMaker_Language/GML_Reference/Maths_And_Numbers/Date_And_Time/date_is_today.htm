data_é_data_data Esta função retornará verdadeiro se o valor de data/hora determinado for o dia em que está sendo verificado (ou seja: hoje), ou falso de outra forma. Esta pode ser uma função útil para coisas como Ovos de Páscoa em seus jogos, ou para desbloquear conteúdo sazonal. Note que esta função será afetada pelo fuso horário definido (o padrão é a hora local) que você pode alterar usando o data_set_timezone() função. Sintaxe: date_is_today(data); Argumento Descrição data A data a ser usada. Devoluções: Booleano Exemplo: if date_is_today(global.Halloween) { global.Max_Levels = 200; } O código acima verificará a data/hora armazenada na variável global "Halloween" para ver se ela é hoje, e se ela é outra variável global para um novo valor.
