data_actual_data hora Retorna o valor da data e hora do momento atual. A hora retornada é baseada no fuso horário padrão do sistema (ou seja: hora local). Você pode alterar o fuso horário base para usar com a função data_set_timezone() Sintaxe: data_actual_data/hora(); Devoluções: Real Exemplo: myhour = date_get_hour(date_current_datetime()); myday = date_get_day(date_current_datetime()); Isto definiria a variável local myhour para a hora atual, e myday para o dia da data atual.
