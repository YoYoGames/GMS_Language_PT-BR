tempo_actual Esta variável lida apenas retornará o número de milissegundos que passaram desde que o jogo foi iniciado. Sintaxe: tempo_actual; Devoluções: Real Exemplo: if (tempo_actual > 600000) { msg = show_question_async("Você gostaria de avaliar?"); } O código acima verifica se passaram mais de 10 minutos antes de fazer uma pergunta ao usuário.
