data_hora_esplanada Com esta função você pode obter o número de horas entre duas datas. Este valor é sempre positivo e as horas incompletas serão devolvidas como uma fração. Sintaxe: date_hour_span(data1, data2); Argumento Descrição data1 A primeira data. data2 A data para comparar o primeiro com o primeiro. Devoluções: Real Exemplo: diff = date_hour_span(date_create_datetime( 2011, 9, 15, 11, 11, 4, 0 ), date_current_datetime()); Isto definiria "dif" para o número de horas entre 15 de setembro de 2011, 11:04.0 e a data e hora atual.
