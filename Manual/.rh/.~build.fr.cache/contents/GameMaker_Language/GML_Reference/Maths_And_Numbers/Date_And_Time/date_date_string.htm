date_date_string Com esta função você pode criar uma seqüência contendo a data dada, formatada como dia/mês/ano. Sintaxe: date_date_string(data); Argumento Descrição data A data a ser usada. Devoluções: Cordão Exemplo: str = date_date_string(date_current_datetime()); draw_text(32, 32, str); Isto definiria "str" para manter uma string formatada da data e hora atual, como mostrado pelo sistema.
