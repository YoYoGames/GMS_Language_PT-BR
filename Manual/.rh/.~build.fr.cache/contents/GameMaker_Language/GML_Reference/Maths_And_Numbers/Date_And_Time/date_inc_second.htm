data_inc_segundo Com esta função você pode incrementar um determinado valor de data/hora por um número específico de segundos, e ele retornará o novo valor de data/hora. Sintaxe: date_inc_second(data, montante); Argumento Descrição data A data a ser acrescentada. valor O número de segundos (deve ser um número inteiro) a acrescentar. Devoluções: Real Exemplo: mynewdatetime = date_inc_second(date_current_datetime(), 60); Isto definiria "mynewdatetime" para a data atual, mas com 60 segundos adicionados.
