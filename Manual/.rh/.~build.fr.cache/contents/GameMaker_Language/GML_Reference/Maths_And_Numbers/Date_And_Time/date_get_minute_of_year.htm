data_get_minute_do_ano Esta função retorna o minuto do valor do tempo dado dentro do ano (do número total de minutos para o ano, levando em conta os anos bissextos). Sintaxe: date_get_minute_of_year(date); Argumento Descrição data A data a ser verificada. Devoluções: Real Exemplo: myminuteyear = date_get_minute_of_year(date_current_datetime()); Isto definiria o myminute ano para o minuto atual do ano.
