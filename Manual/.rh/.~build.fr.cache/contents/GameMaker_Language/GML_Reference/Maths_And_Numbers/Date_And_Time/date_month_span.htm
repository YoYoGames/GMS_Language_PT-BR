data_mês_espaço Com esta função você pode obter o número de meses entre duas datas. Este valor é sempre positivo, e meses incompletos serão devolvidos como uma fração. Sintaxe: date_month_span(data1, data2); Argumento Descrição data1 A primeira data. data2 A data e a hora para compará-la. Devoluções: Real Exemplo: diff = date_month_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime()); Isto definiria "dif" para o número de meses entre 15 de setembro de 2011, 11:04.0 e a data e hora atual.
