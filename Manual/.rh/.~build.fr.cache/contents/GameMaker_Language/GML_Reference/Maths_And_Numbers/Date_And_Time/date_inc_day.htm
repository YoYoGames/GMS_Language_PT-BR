data_inc_dia Com esta função você pode incrementar um determinado valor de data/hora por um número específico de dias, e ele retornará o novo valor de data/hora. Sintaxe: date_inc_day(data, quantidade); Argumento Descrição data A data a ser acrescentada. valor O número de dias (deve ser um número inteiro) a acrescentar. Devoluções: Real Exemplo: mynewdatetime = date_inc_day(date_current_datetime(), 365); Isto definiria "mynewdatetime" para a data atual, mas com 365 dias adicionados.
