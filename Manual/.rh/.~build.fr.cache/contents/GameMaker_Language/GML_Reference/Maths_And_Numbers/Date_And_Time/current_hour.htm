hora_actual Esta variável lida somente retornará a hora que corresponde ao momento atual com base no fuso horário padrão para o sistema (ou seja: hora local). Você pode mudar o fuso horário base para usar com a função data_set_timezone(). Sintaxe: hora_hoje; Devoluções: Real Exemplo: draw_text(32, 32, "O tempo é " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second)); O código acima desenharia a hora internacional atual na tela.
