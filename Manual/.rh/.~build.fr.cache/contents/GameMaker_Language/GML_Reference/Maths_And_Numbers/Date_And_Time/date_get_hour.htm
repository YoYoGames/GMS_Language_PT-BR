date_get_hour Esta função retorna a hora do valor da data e hora determinados. Sintaxe: date_get_hour(data); Argumento Descrição data A data a ser verificada. Devoluções: Real Exemplo: myhour = date_get_hour(date_current_datetime())); Isto definiria "myhour" para a hora atual.
