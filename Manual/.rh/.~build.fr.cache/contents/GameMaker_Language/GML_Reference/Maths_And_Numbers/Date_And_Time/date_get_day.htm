data_get_day Esta função retorna o dia (de 1 a 31) da data indicada. Sintaxe: date_get_day(data); Argumento Descrição data A data a ser verificada. Devoluções: Real Exemplo: myday = date_get_day( date_current_datetime() ); Isto definiria o "dia-a-dia" para o dia atual.
