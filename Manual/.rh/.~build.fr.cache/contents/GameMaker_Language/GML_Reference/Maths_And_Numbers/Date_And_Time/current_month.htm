mês_actual Esta leitura só retorna o mês atual como um valor numérico onde 1 é janeiro e 12 é dezembro. Sintaxe: mês_actual; Devoluções: Real Exemplo: draw_text((32, 32, "Hoje é " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +"); O código acima desenhará o dia, mês e ano.
