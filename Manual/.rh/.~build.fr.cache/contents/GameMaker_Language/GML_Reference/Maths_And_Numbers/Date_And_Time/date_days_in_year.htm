data_dias_no_ano Com esta função você pode obter o número de dias que o ano determinado tem, retornando 365 para um ano normal, e 366 para um ano bissexto. Sintaxe: data_dias_no_ano(data); Argumento Descrição data A data a ser usada. Devoluções: Real Exemplo: dias = data_dias_no_ano(date_current_datetime())); Isto definiria "dias" para o número de dias no ano corrente.
