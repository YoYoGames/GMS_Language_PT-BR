date_second_span Com esta função você pode obter o número de segundos entre duas datas. O valor de retorno é sempre positivo e será um número inteiro. Sintaxe: date_second_span(data1, data2); Argumento Descrição data1 A primeira data. data2 A data e a hora para compará-la. Devoluções: Real Exemplo: diff = date_second_span(date_create_datetime(2011, 9, 15, 11, 11, 4, 0 ), date_current_datetime()); Isto definiria "dif" para o número de segundos entre 15 de setembro de 2011, 11:04 e a data e hora atual.
