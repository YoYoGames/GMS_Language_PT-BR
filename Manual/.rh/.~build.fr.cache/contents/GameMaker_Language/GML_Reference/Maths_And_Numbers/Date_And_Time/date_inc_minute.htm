data_inc_minuto Com esta função você pode incrementar um determinado valor de data/hora por um número específico de minutos, e ele retornará o novo valor de data/hora. Sintaxe: date_inc_minute( data, quantidade ); Argumento Descrição data A data a ser acrescentada. quantia O número de minutos (deve ser um número inteiro) a acrescentar. Devoluções: Real Exemplo: mynewdatetime = date_inc_minute(date_current_datetime(), 60); Isto definiria "mynewdatetime" para a data atual, mas com 60 minutos adicionados.
