data_intervalo_de_tempo Com esta função você pode obter o número de minutos entre duas datas. Este valor é sempre positivo, e minutos incompletos serão devolvidos como uma fração. Sintaxe: date_minute_span(data1, data2); Argumento Descrição data1 A primeira data. data2 A data e a hora para compará-la. Devoluções: Real Exemplo: diff = date_minute_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime()); Isto definiria "dif" para o número de minutos entre 15 de setembro de 2011, 11:04.0 e a data e hora atual.
