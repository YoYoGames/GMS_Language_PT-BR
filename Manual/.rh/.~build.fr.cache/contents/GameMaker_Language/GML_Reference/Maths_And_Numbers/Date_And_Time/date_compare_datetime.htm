data_compare_datetime Com esta função você pode verificar duas datas e horários para ver qual é a mais cedo ou a mais tarde do que a outra. A função retorna -1 se a data1 for anterior, 1 se a data1 for posterior e 0 se forem as mesmas, embora como isto leva em conta o tempo como bem, um 0 para correspondência é muito raro (tem que ser literalmente o mesmo segundo do mesmo dia do mesmo mês do mesmo ano para retornar um 0). Sintaxe: date_compare_datetime(data1, data2); Argumento Descrição data1 A primeira data. data2 A data e a hora para compará-la. Devoluções: Real Exemplo: d = date_compare_datetime(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime()); Isto definiria d para o valor correspondente, dependendo de qual da data era a mais antiga, provavelmente -1 já que a data atual estaria mais adiantada do que 15 de setembro de 2011.
