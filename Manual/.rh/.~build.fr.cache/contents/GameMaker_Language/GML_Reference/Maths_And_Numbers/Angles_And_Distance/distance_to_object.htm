distância_a_objeto Esta função calcula a distância da borda da caixa delimitadora da instância chamadora até a borda mais próxima da instância mais próxima do objeto especificado. O objeto pode ser um índice de objeto ou um ID de instância específica, assim como o palavra-chave outros, e a distância é devolvida em pixels. Observe que se qualquer um dos objetos não tiver sprite ou nenhuma máscara definida, os resultados serão incorretos. Sintaxe: distance_to_object(obj); Argumento Descrição obj O objeto a ser verificado. Devoluções: Real. Exemplo: if (distance_to_object(obj_Player) < range) { canhoot = true; } O código acima verificará a distância até o objeto jogador e se for menor que o valor armazenado na variável "faixa", a variável "canhoot" é definida como verdadeira.
