ângulo_diferença Esta função retornará a menor diferença de ângulo entre dois ângulos como um valor entre -180 e 180 graus (onde um ângulo positivo é anti-horário e um ângulo negativo no sentido horário). Sintaxe: angle_difference(ang1, ang2) Argumento Descrição ang1 O primeiro ângulo a ser utilizado. ang2 O segundo ângulo a ser utilizado. Devoluções: Real Exemplo: var pd = point_direction(x, y, mouse_x, mouse_y); var dd = angle_difference(image_angle, pd); image_angle -= min(abs(dd), 10) * sinal(dd); O código acima irá obter o ângulo de direção da instância para o cursor do mouse, então obterá a diferença entre esse ângulo e a corrente image_angle, usando este valor para girar lentamente em direção ao mouse.
