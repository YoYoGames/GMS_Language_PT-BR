dot_product_normalised O produto ponto é um valor que expressa a relação angular entre dois vetores e é encontrado pegando dois vetores, multiplicando-os juntos e depois adicionando os resultados. O nome "produto de ponto" é derivado do ponto centralizado "-". que é freqüentemente usado para designar esta operação (o nome alternativo "produto escalar" enfatiza a natureza escalar em vez de vetor do resultado). A fórmula matemática real pode ser escrita desta forma: Portanto, em 2D o produto ponto dos vetores a[x1,y1] e b[x2,2] é x1x2 + y1y2, o que significa que o produto ponto_ no GameMaker Studio 2 é calculado como: a - b = (x1*x2)+(y1*y2); E quanto ao produto de ponto normalizado? O produto de ponto normalizado foi corrigido de forma a trazer o valor de retorno para a faixa de -1 e 1 (veja a seção sobre Vetores para informações mais detalhadas), o que é excepcionalmente útil em muitas circunstâncias, particularmente quando se trata de iluminação e outras funções 3D. Sintaxe: dot_product_normalised(x1, y1, x2, y2) Argumento Descrição x1 A coordenada x do primeiro vetor. y1 A coordenada y do primeiro vetor. x2 A coordenada x do segundo vetor. y2 A coordenada y do segundo vetor. Devoluções: Real Exemplo: var x1, y1, x2, y2; x1 = lengthdir_x(1, image_angle); y1 = lengthdir_y(1, image_angle); x2 = o_Player.x - x; y2 = o_Player.y - y; if dot_product_normalised(x1, y1, x2, y2) > 0 seen=true else see=false; O código acima cria um vetor utilizando o ângulo de imagem das instâncias, e então obtém o vetor do objeto jogador "o_Player" para si mesmo. Finalmente, ele calcula o produto de pontos desses dois vetores e, se for maior que 0, define a variável "visto" para verdadeiro, e se for igual ou inferior a 0, o define como "falso".
