distância_a_ponto Esta função calcula a distância da borda da caixa delimitadora da instância chamadora até a posição x/y especificada na sala, sendo o valor de retorno em pixels. Observe que se o objeto chamador não tiver sprite ou máscara definida, o os resultados serão incorretos. Sintaxe: distância_a_ponto(x, y); Argumento Descrição x A posição x a ser verificada. y A posição y a ser verificada. Devoluções: Real Exemplo: if (distance_to_point(obj_Player.x, obj_Player.y) < range) { canhoot = true; } O código acima verificará a distância até a posição x/y do objeto jogador e se for menor que o valor armazenado na variável "faixa", a variável "canhoot" é definida como verdadeira.
