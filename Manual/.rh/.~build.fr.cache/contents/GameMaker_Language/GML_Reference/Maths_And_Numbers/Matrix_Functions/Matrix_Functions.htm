Funções da Matriz Uma matriz é um conjunto de números dispostos em um número fixo de linhas e colunas. Normalmente os números são números reais, mas isso nem sempre tem que ser o caso. A imagem a seguir mostra em geral como uma matriz é construída com quatro filas e quatro colunas: A linha superior é a linha 1, a coluna mais à esquerda é a coluna 1, e esta matriz é uma matriz 4x4 porque tem quatro filas e quatro colunas (outras matrizes de tamanho podem ser construídas). Na descrição das matrizes, o formato é sempre linhas / colunas, e cada número que compõe uma matriz é chamado de um elemento da matriz. Os elementos de uma matriz têm locais específicos, descritos por sua posição de linha e coluna, onde o canto superior esquerdo da matriz é a linha 1 coluna 1. No exemplo da matriz acima, o elemento na linha 1 col 1 é o valor "1". O elemento na linha 2 coluna 3 é o valor "1". valor "4,8". As matrizes são muito importantes para muitas coisas, mas são mais usadas para lidar com o espaço 3D, por exemplo, podem ser usadas para definir a visão da câmera ou para traduzir ou transformar um modelo.
