andar Retorna o piso de n, ou seja, n arredondado para um número inteiro. Isto é semelhante ao redondo() mas apenas arredonda para baixo, não importa qual seja o valor decimal, então o piso (5,99999) retornará 5, assim como o piso(5.2), piso(5.6457) etc... Sintaxe: piso(n); Argumento Descrição n O número ao andar. Devoluções: Real Exemplo: val = piso( 3,9 ); Isto colocará Val a 3.
