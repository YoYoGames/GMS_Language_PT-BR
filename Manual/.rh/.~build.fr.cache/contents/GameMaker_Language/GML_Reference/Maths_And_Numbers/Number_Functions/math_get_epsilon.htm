math_get_epsilon Esta função retornará o valor atual do epsilon para a plataforma alvo. Para mais informações sobre o epsilon, por favor, veja a função matemático_set_epsilon(). Sintaxe: math_get_epsilon(); Devoluções: Real Exemplo: var e = math_get_epsilon(); if e != 0.000001 { math_set_epsilon(0.000001); } Isto irá recuperar o valor atual do epsilon e armazená-lo em uma variável local, que é então verificada e um novo conjunto de epsilon, se necessário.
