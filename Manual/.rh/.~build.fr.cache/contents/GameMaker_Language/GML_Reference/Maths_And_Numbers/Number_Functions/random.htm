aleatório Esta função é boa para probabilidades onde não é necessário devolver um número inteiro (número inteiro). Por exemplo, o aleatório(100) retornará um valor de 0 a 99, mas esse valor pode ser 22,56473! Você também pode usar números reais e não números inteiros em esta função - aleatória(0,5), que retornará um valor entre 0 e 0,499999999. NOTA: Esta função retornará o mesmo valor toda vez que o jogo for executado novamente devido ao fato de que o GameMaker Studio 2 gera a mesma semente inicial aleatória toda vez para fazer do código de depuração uma tarefa muito mais fácil. Para evitar este comportamento, use aleatoriedade() no início de seu jogo. Sintaxe: aleatório(n); Argumento Descrição n O intervalo superior a partir do qual o número aleatório será selecionado. Devoluções: Real Exemplo: if (random(10) >= 9) { pontuação += 100; } Isto produzirá aproximadamente uma em cada dez chances de adicionar 100 à pontuação.
