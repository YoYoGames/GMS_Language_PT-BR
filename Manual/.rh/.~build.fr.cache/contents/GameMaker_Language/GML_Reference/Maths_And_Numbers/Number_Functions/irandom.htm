irracional Esta função muito útil retorna apenas números inteiros (números inteiros). Assim, por exemplo, para obter um número aleatório de 0 a 9 você pode usar o irandom(9 ) e ele retornará um número de 0 a 9 inclusive. Note que também podem ser usados flutuadores, mas a função valor superior após o ponto será excluído, então o irracional(9,7) retornará um valor de 0 a 9 apenas. A função tem um limite superior de $7fffffffffffffLL, portanto, deve-se tomar cuidado se forem usados números muito grandes. NOTA: Esta função retornará o mesmo valor toda vez que o jogo for executado novamente devido ao fato de que o GameMaker Studio 2 gera a mesma semente inicial aleatória toda vez para fazer do código de depuração uma tarefa muito mais fácil. Para evitar este comportamento, use aleatoriedade() no início de seu jogo. Sintaxe: irandom(n); Argumento Descrição n O intervalo superior a partir do qual o número aleatório será selecionado. Devoluções: Inteiro Exemplo: if (irandom(9) == 1) { pontuação += 100; } Isto produzirá uma chance de um em dez (já que 0 está incluído) de adicionar 100 à pontuação.
