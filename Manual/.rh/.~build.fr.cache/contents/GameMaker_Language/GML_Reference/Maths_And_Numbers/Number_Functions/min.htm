min Esta função retorna o mínimo dos valores de entrada, dos quais ela pode ter tantos quantos forem necessários (note que mais argumentos significarão que a função será mais lenta de analisar). Por exemplo, min(12, 96, 32, 75) retornará 12, pois esse é o menor de todos os valores de entrada. Sintaxe: min(val1, val2, ... max_val); Argumento Descrição val0 ... max_val Os valores a serem comparados. Devoluções: Real Exemplo: x = min(x, largura_de_sala); Isto impedirá que o jogador saia pela direita da sala. Isto funciona ajustando constantemente seu x para si mesmo ou para a variável largura_da_câmara, o que for menor. Se o jogador sair pela direita, seu x será maior do que a largura_da_sala, então ele irá será recuado.
