Sintaxe: sinal(n); Argumento Descrição n O número para obter o sinal de. Devoluções: Inteiro Exemplo: y += sinal(y - mouse_y); O código acima adicionará 1, -1 ou 0 em y, dependendo do resultado de y - mouse_y.
