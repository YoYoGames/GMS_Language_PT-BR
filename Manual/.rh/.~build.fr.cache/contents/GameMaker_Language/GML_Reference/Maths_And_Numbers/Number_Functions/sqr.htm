sqr Multiplica um número por si só e assim retorna o quadrado desse número. EG: sqr(5) retornaria 25 desde 5*5=25. Sintaxe: sqr(val); Argumento Descrição val O número a ajustar. Devoluções: Real Exemplo: pontuação += sqr(dmg); O código acima irá adicionar no quadrado da variável "dmg" à pontuação.
