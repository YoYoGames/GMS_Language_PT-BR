gama_randômica Esta função retorna um número aleatório entre o intervalo especificado, e este valor de retorno não precisa ser um número inteiro. Por exemplo, o random_range(20,50) retornará um número aleatório de 20 a 50, mas o valor pode ser um número real como 38,65265. Números reais também podem ser usados como argumentos de entrada. NOTA: Esta função retornará o mesmo valor toda vez que o jogo for executado novamente devido ao fato de que o GameMaker Studio 2 gera a mesma semente inicial aleatória toda vez para fazer do código de depuração uma tarefa muito mais fácil. Para evitar este comportamento, use aleatoriedade() no início de seu jogo. Sintaxe: random_range(n1, n2); Argumento Descrição n1 A extremidade baixa do intervalo a partir do qual o número aleatório será selecionado. n2 O extremo alto do intervalo a partir do qual o número aleatório será selecionado. Devoluções: Real Exemplo: score += random_range(500, 600); Isto adicionará entre 500 e 600, ao total da pontuação.
