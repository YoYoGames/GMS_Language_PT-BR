show_message Esta função cria uma caixa de mensagem pop-up que exibe a seqüência dada e um botão marcado "Ok" para fechá-la. NOTA: Esta função é para uso de depuração apenas em alvos de Desktop e UWP, mas é depreciada em todos os outros alvos. Sintaxe: show_message(str); Argumento Descrição str O fio a ser mostrado na mensagem pop-up. Devoluções: N/A Exemplo: var tot = 0; for (var i = 0; i < 10; i += 1) { tot += inv[i]; } show_message("Total = " + string(tot)); O código acima irá percorrer os valores armazenados na matriz "inv" e adicioná-los à variável "tot" antes de mostrar uma mensagem com o total.
