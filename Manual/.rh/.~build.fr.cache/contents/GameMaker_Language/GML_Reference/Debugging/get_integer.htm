get_integer Isto cria uma janela pop-up mostrando uma mensagem personalizada, com um botão rotulado "Ok", e solicita que o usuário insira um valor inteiro. A função retornará o número inteiro digitado, ou o valor padrão, se nada tiver sido inserido. OBSERVAÇÃO: Esta função é para uso de depuração somente em alvos Desktop, mas é depreciada em todos os outros alvos. Sintaxe: get_integer(str, def); Argumento Descrição str O fio a ser mostrado na mensagem pop-up. def O valor padrão na caixa de texto. Devoluções: Real Exemplo: global.level = get_integer("Nível para testar?", 1); O código acima exibirá uma mensagem solicitando ao usuário que selecione um nível para testes.
