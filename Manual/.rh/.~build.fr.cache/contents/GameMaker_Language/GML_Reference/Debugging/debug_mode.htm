debug_mode Esta leitura só retorna verdadeira quando o jogo está sendo jogado no modo de depuração e falsa quando está sendo jogado normalmente. Sintaxe: debug_mode Devoluções: Booleano Exemplo: se debug_mode { ini_open("Cheats.ini"); } mais { ini_open("Game.ini"); } O código acima abre um arquivo ini diferente, dependendo do valor da variável somente leitura de debug_mode.
