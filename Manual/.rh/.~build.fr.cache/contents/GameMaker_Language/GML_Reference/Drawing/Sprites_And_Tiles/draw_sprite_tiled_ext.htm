draw_sprite_tiled_ext Esta função irá pegar um duende e então repetidamente colocá-lo em toda a vista (ou espaço se nenhuma vista estiver definida), começando a partir das coordenadas que você dá na função e com cada ladrilho escalonado, cor misturada e com o alfa que você define (estas propriedades são as mesmas que as utilizadas em draw_sprite_ext()). Esta função é apenas para projeções 2D (ortográficas), e não funcionará corretamente quando for usada uma projeção de câmera 3D. NOTA: A mistura de cores só é recomendada para o alvo HTML5 quando a WebGL está habilitada, embora você ainda possa definir a cor da mistura se ela não estiver habilitada e ela irá misturar o sprite como normal. Entretanto, toda mistura desta forma cria uma duplicata sprite, que é então armazenado no cache e utilizado quando necessário. Isto está longe de ser ideal e se você usar mudanças múltiplas de cor, isso irá diminuir o desempenho de seus jogos a menos que você ative o WebGL. se você não quiser usar o WebGL, então você pode definir o tamanho do cache da fonte para tentar limitá-lo, caso seja necessário, utilizando a função sprite_set_cache_size_size(). Sintaxe: draw_sprite_tiled_ext(sprite, subimg, x, y, xscale, yscale, cor, alfa); Argumento Descrição sprite O índice do duende a desenhar. subimg O subimg (frame) do duende a desenhar (image_index ou -1 correlacionado com o frame atual da animação no objeto). x A coordenada x de onde desenhar o duende. y A coordenada y de onde desenhar o duende. xscale A escalada horizontal do duende. Um multiplicador ' 1 = escalada normal, 0,5 é metade etc. yscale A escalada vertical do duende. Um multiplicador ' 1 = escalada normal, 0,5 é metade etc. cor A cor com a qual se deve misturar o duende. c_white é para exibi-lo normalmente. alfa O alfa do duende (de 0 a 1 onde 0 é transparente e 1 opaco). Devoluções: N/A Exemplo: draw_sprite_tiled_ext(sprite_index, image_index, x, y, 2, 2, c_red, 0,5); Isto desenhará as instâncias atribuídas ao sprite (sprite_index) e seu quadro atual de animação (image_index) na posição x e y das instâncias, mas escalado para o dobro do tamanho normal, misturado em vermelho e com metade do alfa normal.
