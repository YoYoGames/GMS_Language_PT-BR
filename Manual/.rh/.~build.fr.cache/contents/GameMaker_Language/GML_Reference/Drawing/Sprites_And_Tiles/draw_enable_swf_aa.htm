draw_enable_swf_aaa Com esta função, você pode ativar ou desativar o anti-aliasing (AA) para sprites vetoriais no formato SWF. AA simplesmente suaviza as bordas das imagens vetoriais para dar-lhes uma aparência mais agradável. A quantidade de AA usada dependerá do valor definido usando a função draw_set_swf_aa_level(). Por padrão, isto é desativado. Sintaxe: draw_enable_swf_aa(enable); Argumento Descrição habilitar Habilitar(verdadeiro) ou desabilitar(falso) AA para todos os sprites SWF. Devoluções: N/A Exemplo: if draw_get_swf_aa_level() == 0 { draw_enable_enable_swf_aa(true); draw_set_swf_aa_level(0.5); } O código acima verificará o valor AA para sprites no formato SWF, e se for 0, habilita AA e define o valor para 0,5.
