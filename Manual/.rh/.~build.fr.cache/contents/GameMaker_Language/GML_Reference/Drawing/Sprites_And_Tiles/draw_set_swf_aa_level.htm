nível_de_sorteio_swf_aa Esta função pode ser usada para definir o nível de anti-aliasing (AA) para sprites vetoriais no formato SWF. Este pode ser um valor real de 0 a 1 e irá "suavizar" as bordas destes sprites. Note que para ver este efeito, é necessário primeiro ter habilitado AA usando a função draw_enable_swf_aa(). Sintaxe: draw_set_swf_aa_level(AA); Argumento Descrição AA O valor anti-aliasing para usar de 0 a 1. Devoluções: N/A Exemplo: if draw_get_swf_aa_level() == 0 { draw_enable_enable_swf_aa(true); draw_set_swf_aa_level(0.5); } O código acima verificará o valor AA para sprites no formato SWF, e se for 0, habilita AA e define o valor para 0,5.
