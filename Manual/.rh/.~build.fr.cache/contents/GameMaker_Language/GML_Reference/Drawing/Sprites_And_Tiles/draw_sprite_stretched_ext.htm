draw_sprite_stretched_ext Esta função faz exatamente o mesmo que o draw_sprite_stretched() com a capacidade adicional de definir a mistura de cores e o valor alfa para o duende quando este é desenhado (semelhante à função draw_sprite_ext()). NOTA: A mistura de cores só é recomendada para o alvo HTML5 quando a WebGL está habilitada, embora você ainda possa definir a cor da mistura se ela não estiver habilitada e ela irá misturar o sprite como normal. Entretanto, toda mistura desta forma cria uma duplicata sprite, que é então armazenado no cache e utilizado quando necessário. Isto está longe de ser ideal e se você usar mudanças múltiplas de cor, isso irá diminuir o desempenho de seus jogos a menos que você ative o WebGL. se você não quiser usar o WebGL, então você pode definir o tamanho do cache da fonte para tentar limitá-lo, caso seja necessário, utilizando a função sprite_set_cache_sizee(). Sintaxe: draw_sprite_stretched_ext(sprite, subimg, x, y, w, h, cor, alfa); Argumento Descrição sprite O índice do duende a desenhar. subimg O subimg (frame) do duende a desenhar (image_index ou -1 correlacionado com o frame atual da animação no objeto). x A coordenada x de onde desenhar o duende. y A coordenada y de onde desenhar o duende. w A largura da área que o duende esticado irá ocupar. h A altura da área que o duende esticado irá ocupar. cor A cor com a qual se deve misturar o duende. c_white é para exibi-lo normalmente. alfa O alfa do duende (de 0 a 1 onde 0 é transparente e 1 opaco). Devoluções: N/A Exemplo: draw_sprite_stretched_ext(sprite_index, image_index, x, y, sprite_width, sprite_height / 2, c_white, 0,5); Isto atrairá as instâncias atribuídas ao duende e sua subimagem com o canto esquerdo na posição x/y da instância. Sua largura é igual à do duende, e a altura é a altura do duende dividida por dois. Ela também será misturada normalmente, mas terá um valor alfa parcialmente transparente.
