nível_get_swf_aa_de_sorteio Esta função pode ser usada para obter o nível anti-aliasing (AA) para sprites vetoriais no formato SWF. O valor de retorno estará entre 0 e 1 e mostra como "suavizar" as bordas destes sprites serão desenhadas. Você pode definir o nível AA usando a função draw_set_swf_aa_level(). Sintaxe: draw_get_swf_aa_level(); Devoluções: Real Exemplo: if draw_get_swf_aa_level() == 0 { draw_enable_enable_swf_aa(true); draw_set_swf_aa_level(0.5); } O código acima verificará o valor AA para sprites no formato SWF, e se for 0, habilita AA e define o valor para 0,5.
