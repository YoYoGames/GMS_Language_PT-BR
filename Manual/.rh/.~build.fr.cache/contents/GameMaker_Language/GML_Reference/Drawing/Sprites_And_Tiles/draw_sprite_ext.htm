draw_sprite_ext Esta função desenhará o duende dado como na função draw_sprite() mas com opções adicionais para mudar a escala, mistura, rotação e alfa do duende a ser desenhado. A mudança destes valores não altera o recurso de forma alguma (apenas como ele é desenhado), e você pode usar qualquer uma das variáveis sprite disponíveis em vez de valores diretos para todos os argumentos na função. A imagem abaixo ilustra como valores diferentes afetam o desenho do duende: NOTA: A mistura de cores só é recomendada para o alvo HTML5 quando a WebGL está habilitada, embora você ainda possa definir a cor da mistura se ela não estiver habilitada e ela irá misturar o sprite como normal. Entretanto, toda mistura desta forma cria uma duplicata sprite, que é então armazenado no cache e utilizado quando necessário. Isto está longe de ser ideal e se você usar mudanças múltiplas de cor, isso irá diminuir o desempenho de seus jogos a menos que você ative o WebGL. se você não quiser usar o WebGL, então você pode definir o tamanho do cache da fonte para tentar limitá-lo, caso seja necessário, utilizando a função sprite_set_cache_size_size(). NOTA: Esta função pode não funcionar como esperado ao usar sprites de animação de esqueleto, e você pode descobrir que a função apenas desenha o primeiro frame da pose padrão. Você deve usar as funções draw_skeleton_* em seu lugar. Sintaxe: draw_sprite_ext( sprite, subimg, x, y, xscale, yscale, podridão, cor, alfa ); Argumento Descrição sprite O índice do duende a desenhar. subimg O subimg (frame) do duende a desenhar (image_index ou -1 correlacionado com o frame atual da animação no objeto). x A coordenada x de onde desenhar o duende. y A coordenada y de onde desenhar o duende. xscale A escalada horizontal do duende, como um multiplicador: 1 = escalada normal, 0,5 é metade, etc... yscale A escalada vertical do duende como um multiplicador: 1 = escalada normal, 0,5 é metade, etc... podridão A rotação do duende. 0=direita para cima, 90=rotated 90 graus no sentido anti-horário, etc... cor A cor com a qual se deve misturar o duende. c_white é para exibi-lo normalmente. alfa O alfa do duende (de 0 a 1 onde 0 é transparente e 1 opaco). Devoluções: N/A Exemplo: draw_sprite_ext(sprite_index, image_index, x, y, image_xscale, image_yscale, image_angle, image_blend, image_alpha); Isto atrairá as instâncias atribuídas ao sprite com todos os seus valores padrão (essencialmente o mesmo que usar o draw_self).
