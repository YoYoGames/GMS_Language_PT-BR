draw_get_lighting Esta função retornará se a iluminação estiver habilitada ( verdadeira) ou não(falsa) para toda a cena. Sintaxe: draw_get_lighting() Devoluções: Booleano Exemplo: se !draw_get_lighting() { draw_set_lighting(true); } O código acima verificará se a iluminação está habilitada para a cena, e se não estiver habilitada.
