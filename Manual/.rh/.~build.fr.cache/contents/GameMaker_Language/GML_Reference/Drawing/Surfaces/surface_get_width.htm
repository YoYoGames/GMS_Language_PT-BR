largura_da_superfície Esta função simplesmente retorna a largura, em pixels, da superfície indexada. Deve-se notar que se você chamar isto para verificar a superfície da aplicação_ imediatamente após ter alterado seu tamanho usando tamanho_da_superfície() não devolverá o novo valor, pois a mudança precisa de um ou dois passos para ser totalmente processada. Depois de esperar uma etapa, deverá devolver o novo valor corretamente. NOTA: Ao trabalhar com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido ao fato de serem armazenadas na memória de textura. Você deve SEMPRE verificar se uma superfície existe usando superfície_existe() antes de referenciá-los diretamente. Sintaxe: surface_get_width(surface_id); Argumento Descrição surface_id A identificação da superfície para obter a largura de. Devoluções: Real Exemplo: sw = surface_get_width(surf); O código acima irá armazenar a largura da superfície indexada na variável "surf" na variável "sw".
