textura_get_texel_width Um texel, ou elemento de textura, é a unidade fundamental do espaço de textura utilizado na computação gráfica. As texturas são representadas por matrizes de textos, assim como as imagens são representadas por matrizes de pixels, e esta função retorna a largura de um único elemento de textura. texel da página de textura do bem de imagem utilizado. Sintaxe: textura_get_texel_width(tex); Argumento Descrição tex O ponteiro dos ativos da página de textura a usar Devoluções: Real Exemplo: var tex = sprite_get_texture(sprite_index, 0); tex_w = text_get_texel_width(tex); tex_h = text_get_texel_height(tex); O código acima obterá a largura e a altura do texel da textura tirada de um bem sprite.
