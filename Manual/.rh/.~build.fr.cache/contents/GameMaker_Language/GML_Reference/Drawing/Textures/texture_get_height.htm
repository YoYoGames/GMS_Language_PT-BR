Textura_esquadre_altura Retorna a altura da textura com o id dado, que é sempre um valor dentro da faixa 0 - 1. Isto pode então ser usado ao mapear texturas para modelos ou primitivas. Sintaxe: text_get_height(tex); Argumento Descrição tex O ponteiro dos ativos da página de textura a usar Devoluções: Real Exemplo: tex_h = text_get_height(surface_get_texture(global.Surf)); O código acima terá a altura da textura tirada de uma superfície previamente criada.
