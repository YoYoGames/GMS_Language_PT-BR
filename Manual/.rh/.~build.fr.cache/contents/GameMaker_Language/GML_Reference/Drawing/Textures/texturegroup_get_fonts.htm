grupo de textura_get_fonts Com esta função, você pode recuperar o índice de fontes de cada uma das fontes atribuídas às páginas de textura dentro de um determinado grupo de textura. Você fornece a cadeia de identificação do grupo de texturas (conforme definido no Editor de grupo de texturas) e a função retornará uma matriz 1D onde cada entrada contém o índice de fontes para um recurso de fontes. Se a função falhar - ou seja: um grupo inválido é dado, ou o grupo não tem textura atribuída a ele - então a matriz estará vazia (0 comprimento). Sintaxe: textgroup_get_fonts(tex_id); Argumento Descrição tex O nome do grupo de textura a verificar (um fio) Devoluções: 1D array Exemplo: var _tex_array = textgroup_get_fonts( "MainMenu"); for (var i = 0; i < array_length(_tex_array); ++i;) { show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i])); } O código acima recuperará os índices de fontes para o grupo de textura "MainMenu", depois exibirá essas IDs na janela de saída do console.
