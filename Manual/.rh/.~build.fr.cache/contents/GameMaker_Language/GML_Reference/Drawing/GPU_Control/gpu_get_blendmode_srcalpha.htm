gpu_get_blendmode_srcalpha Esta função pode ser usada para recuperar o fator alfa do modo de mistura estendida da fonte atual. O valor retornado será uma das seguintes constantes ("s" denota um valor retirado da fonte enquanto um "d" denota um valor de o destino) com apenas o componente "A" sendo usado no desenho: Constante Fator de mistura (Vermelho, Verde, Azul, Alfa) bm_zero (0, 0, 0, 0) bm_one (1, 1, 1, 1) bm_src_colour (Rs, Gs, Bs, As) bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As) bm_src_alpha (As, As, As, As, As) bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As) bm_dest_alpha (Anúncio, Anúncio, Anúncio, Anúncio) bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad) bm_dest_colour (Rd, Gd, Bd, Ad) bm_inv_dest_color (1-Rd, 1-Gd, 1-Bd, 1-Ad) bm_src_alpha_sat (f, f, f, 1) onde f = min(As, 1-Ad) Sintaxe: gpu_get_blendmode_srcalpha(); Devoluções: Constante (ver tabela acima) Exemplo: var bm; bm[0] = gpu_get_blendmode_srcalpha(); bm[1] = gpu_get_blendmode_destalpha(); gpu_set_blendmode_ext_sepalpha(bm_inv_src_alpha, bm_inv_dest_colour, bm[0], bm[1]); O código acima cria uma matriz local e obtém os fatores de mistura atuais da fonte e do destino para o componente alfa. Esta matriz é então usada para manipular o componente RGB dos fatores de mistura.
