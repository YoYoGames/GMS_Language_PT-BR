gpu_pop_state Esta função estala o estado anterior da GPU da pilha e a aplica. Veja gpu_push_state() para mais informações. Sintaxe: gpu_pop_state(); Devoluções: N/A Exemplo: gpu_push_state(); gpu_set_blendmode(bm_add); gpu_set_blendenable(false); gpu_set_cullmode(true); com (obj_Effect_Parent) { draw_self(); } gpu_pop_state(); O código acima "salvará" o estado atual da GPU na pilha, então mudará certas configurações da GPU e desenhará um grupo de instâncias antes de redefinir o estado da GPU para o que era anteriormente.
