gpu_get_blendenable Esta função pode ser usada para recuperar o estado de mistura alfa. Se retornar verdadeiro, a mistura alfa está habilitada e se retornar falso, está desabilitada. Por padrão, isto está ativado e, portanto, a função retornará verdadeiro. Sintaxe: gpu_get_blendenable(); Devoluções: Booleano Exemplo: if gpu_get_blendenable() == falso { gpu_set_blendenable(true); } O código acima verifica o estado da mistura alfa e, se estiver desativado, é ativado novamente.
