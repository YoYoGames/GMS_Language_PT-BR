gpu_set_set_state Esta função definirá o estado atual da GPU usando o Mapa DS aprovado. O mapa fornecido pode ser criado usando a função gpu_get_state(). Sintaxe: gpu_set_set_state(ds_map); Argumento Descrição ds_map O estado da GPU a ser definido como um Mapa DS. Devoluções: N/A Exemplo: gpu_set_state(gpu_map); O código acima define o estado da GPU usando o mapa fornecido na variável "gpu_map".
