gpu_get_texfilter Com esta função você pode verificar se a filtragem de textura (interpolação linear) está habilitada (retorna verdadeiro) ou não (retorna falso). Para mais informações sobre a filtragem de textura, veja a função gpu_set_texfilter(). NOTA: No alvo HTML5, esta função só funcionará com WebGL habilitado. Sintaxe: gpu_get_texfilter(); Devoluções: Booleano Exemplo: se gpu_get_texfilter() { gpu_set_texfilter(false); } mais { gpu_set_texfilter(true); } O código acima verifica se a filtragem de textura está ligada ou desligada e a liga de acordo.
