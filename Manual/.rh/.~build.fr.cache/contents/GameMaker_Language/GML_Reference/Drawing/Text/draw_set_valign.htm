draw_set_valign Esta função é usada para alinhar o texto ao longo do eixo vertical e mudar o alinhamento vertical mudará a posição e a direção na qual todo o texto adicional é desenhado, com o valor padrão sendo fa_top. As seguintes constantes são aceitas: Constante Alinhamento fa_top fa_middle fa_bottom Sintaxe: draw_set_valign(valign); Argumento Descrição valign Alinhamento vertical. Devoluções: N/A Exemplo: draw_set_halign(fa_center); draw_set_valign(fa_middle); draw_text(100, 32, "Pontuação: " + string(pontuação)); O código acima desenhará a pontuação centrada em torno do centro do texto.
