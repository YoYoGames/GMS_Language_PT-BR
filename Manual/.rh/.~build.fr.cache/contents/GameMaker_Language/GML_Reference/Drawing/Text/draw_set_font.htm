draw_set_font Esta função definirá a fonte a ser usada para todos os outros desenhos de texto. Esta fonte deve ter sido adicionada aos ativos da fonte do jogo ou ter sido criada usando ou o font_add() ou font_add_sprite(). Sintaxe: draw_set_font(font); Argumento Descrição fonte O nome da fonte a ser usada. Devoluções: N/A Exemplo: draw_set_colour(c_blue); draw_set_font(fnt_Game); draw_text(200, 200, "Olá Mundo" ); O código acima desenhará o texto dado usando a fonte indexada na variável "fnt_Game" e a cor azul.
