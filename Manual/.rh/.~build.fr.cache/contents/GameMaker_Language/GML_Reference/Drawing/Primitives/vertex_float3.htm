vértice_float3 Esta função adicionará três valores de ponto flutuante aos dados do vértice. O vértice deve ter sido formatado corretamente para aceitar isto usando a função vértice_formato_add_custom() função. Sintaxe: vertex_float3(buffer, float, float, float, float); Argumento Descrição tampão O buffer para escrever as informações. flutuador O primeiro valor de entrada. flutuador O segundo valor de entrada. flutuador O terceiro valor de entrada. Devoluções: N/A Exemplo: vertex_float3(buff, 0,05, 0,01, room_width / x); O código acima adicionará três valores de ponto flutuante aos dados de vértices que estão sendo definidos.
