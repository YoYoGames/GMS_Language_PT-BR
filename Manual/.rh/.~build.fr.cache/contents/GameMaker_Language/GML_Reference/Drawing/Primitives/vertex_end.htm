vertex_end Com esta função, você termina a construção do primitivo personalizado. Uma vez que você chama este comando, o primitivo pode ser usado no vértice_submit() para uso em um shader ou você pode congelar o buffer (fazendo a função buffer de vértices usado somente leitura e muito mais rápido) usando vértice_freeze(). Sintaxe: vertex_end(buffer); Argumento Descrição tampão O amortecedor para terminar de escrever. Devoluções: N/A Exemplo: vertex_end(buff); O código acima terminará a definição de um primitivo personalizado.
