vértex_texcoord Esta função definirá as coordenadas de textura a serem usadas para o vértice que está sendo definido atualmente para o primitivo personalizado. Você fornece o buffer para escrever os dados, bem como a posição UV dentro da textura a ser usada. Sintaxe: vertex_texcoord(buffer, u, v); Argumento Descrição tampão O buffer para escrever as informações. u A primeira coordenada de textura a ser utilizada (0 - 1). v A segunda coordenada de textura a ser utilizada (0 - 1). Devoluções: N/A Exemplo: vertex_texcoord(buff, 0, 0); O código acima definirá os valores UV do vértice que está sendo definido.
