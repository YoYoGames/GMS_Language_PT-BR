vértice_format_begin Antes de poder definir seu novo formato de vértice, você deve dizer ao GameMaker Studio 2 que está fazendo isso usando esta função. Você deve chamar isto primeiro, depois definir os valores do formato usando as funções apropriadas, e finalmente chamar vértice_format_end() para terminar a definição e devolver o novo formato "handle". Sintaxe: vértice_format_begin(); Devoluções: N/A Exemplo: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vértice_format_end(); O código acima criará um novo formato de vértice com apenas valores de cor e posição e então armazenará o id de formato na variável "my_format".
