vertex_create_buffer_ext_ext Como com a função vértice_criate_buffer()Esta função criará um novo tampão de vértice. Este é um buffer especial de crescimento criado pelo GameMaker Studio 2 que é pré-formatado para uso quando construindo primitivos para uso com shaders. Você pode especificar um tamanho inicial de partida para o buffer (em bytes) e ele retornará uma "alça" (índice) para o buffer que deverá então ser usada em todas as chamadas posteriores ao buffer. Sintaxe: vertex_create_buffer_ext(size); Argumento Descrição tamanho O tamanho inicial do buffer (em bytes). Devoluções: Apontador Exemplo: v_buff = vertex_create_buffer_ext(1024 * 1024); O código acima criará um novo buffer de vértices, inicialmente de 1MB de tamanho, e armazenará seu cabo na variável "v_buff".
