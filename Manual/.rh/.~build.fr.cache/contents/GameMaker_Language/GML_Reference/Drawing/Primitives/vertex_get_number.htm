vértice_get_number Com esta função você pode descobrir o número de vértices individuais definidos em um determinado buffer de vértices. Sintaxe: vertex_get_number(buffer); Argumento Descrição tampão O tampão de vértice a ser verificado. Devoluções: Real Exemplo: v_num = vertex_get_number(v_buffer); O código acima irá armazenar o número de vértices armazenados no buffer de vértices dado a uma variável.
