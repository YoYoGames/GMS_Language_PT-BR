vertex_format_add_texcoord Diga ao GameMaker Studio 2 para aceitar dados de posição de textura (u e v) como parte do novo formato de vértice que está sendo criado. Sintaxe: vertex_format_add_textcoord(); Devoluções: N/A Exemplo: vertex_format_begin(); vertex_format_add_textcoord(); vertex_format_add_position_3d(); my_format = vértice_format_end(); O código acima criará um novo formato de vértice com valores justos de textura e posição e então armazenará o id de formato na variável "my_format".
