vertex_format_end_end Esta função deve ser chamada após a definição de qualquer novo formato de vértice. Ela retorna o novo formato "handle" (índice) que deve ser usado em todas as outras funções de vértices que se referem a este novo formato. Sintaxe: vértice_format_end(); Devoluções: Real Exemplo: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vértice_format_end(); O código acima criará um novo formato de vértice com apenas valores de cor e posição e então armazenará o id de formato na variável "my_format".
