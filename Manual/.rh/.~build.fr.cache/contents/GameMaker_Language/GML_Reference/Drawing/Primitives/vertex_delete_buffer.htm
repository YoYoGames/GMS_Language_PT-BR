vértice_delete_buffer Esta função pode ser usada para remover um buffer de vértices previamente criado (ver vértice_criate_buffer()) a partir da memória do sistema. Isto também removerá todos os dados de vértices para quaisquer primitivos personalizados que contivesse. Sintaxe: vtex_delete_buffer(tampão); Argumento Descrição tampão O amortecedor a ser removido. Devoluções: N/A Exemplo: vertex_delete_buffer(v_buff); O código acima excluirá da memória o buffer indexado na variável "v_buff".
