vértice_float2 Esta função adicionará dois valores de ponto flutuante aos dados do vértice. O vértice deve ter sido formatado corretamente para aceitar isto usando a função vértice_formato_add_custom() função. Sintaxe: vertex_float2(buffer, float, float); Argumento Descrição tampão O buffer para escrever as informações. flutuador O primeiro valor de entrada. flutuador O segundo valor de entrada. Devoluções: N/A Exemplo: vértice_float2(buff, 0,05, 0,01); O código acima adicionará dois valores de ponto flutuante aos dados de vértices que estão sendo definidos.
