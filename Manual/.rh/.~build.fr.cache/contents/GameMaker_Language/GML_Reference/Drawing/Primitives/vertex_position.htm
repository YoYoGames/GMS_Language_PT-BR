vértice_posição Esta função adicionará dados de posição 2D ao vértice que está sendo definido atualmente para o primitivo personalizado. Você fornece o buffer para escrever os dados, bem como as posições x e y para o desenho. Sintaxe: vertex_position(buffer, x, y); Argumento Descrição tampão O buffer para escrever as informações. x A posição x. y A posição y. Devoluções: N/A Exemplo: vtex_position(buff, x - 100, y - 125); O código acima definirá a posição do vértice que está sendo definido.
