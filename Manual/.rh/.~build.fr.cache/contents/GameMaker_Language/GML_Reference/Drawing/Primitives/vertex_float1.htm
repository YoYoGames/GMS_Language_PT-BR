vértice_float1 Esta função adicionará um valor de ponto flutuante aos dados do vértice. O vértice deve ter sido formatado corretamente para aceitar isto usando a função vértice_formato_add_custom() função. Sintaxe: vertex_float1(buffer, float); Argumento Descrição tampão O buffer para escrever as informações. flutuador O valor de entrada. Devoluções: N/A Exemplo: vértice_float1(buff, 0,05); O código acima adicionará um valor de ponto flutuante aos dados de vértices que estão sendo definidos.
