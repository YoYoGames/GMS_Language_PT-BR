vértice_posição_3d Esta função adicionará dados de posição 3D ao vértice que está sendo definido atualmente para o primitivo personalizado. Você fornece o buffer para escrever os dados, bem como as posições x, y e z para o desenho. Sintaxe: vertex_position_3d(buffer, x, y, z); Argumento Descrição tampão O buffer para escrever as informações. x A posição x. y A posição y. z A posição z. Devoluções: N/A Exemplo: vertex_position_3d(buff, x - 100, y - 125, 0); O código acima definirá a posição do vértice que está sendo definido.
