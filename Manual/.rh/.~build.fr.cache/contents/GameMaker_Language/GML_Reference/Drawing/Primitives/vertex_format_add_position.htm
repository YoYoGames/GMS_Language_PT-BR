vértice_formato_add_posição Diga ao GameMaker Studio 2 para aceitar dados posicionais 2D (x e y) como parte do novo formato de vértice que está sendo criado. Sintaxe: vértice_format_add_position(); Devoluções: N/A Exemplo: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vértice_format_end(); O código acima criará um novo formato de vértice com apenas valores de cor e posição e então armazenará o id de formato na variável "my_format".
