make_colour_rgb OGameMaker Studio 2 fornece esta função (assim como outras) para permitir que o usuário faça suas próprias cores. Esta função em particular leva três componentes, o vermelho, o verde e os componentes azuis da cor que que você deseja fazer. Estes valores são tomados como estando entre 0 e 255 para que você possa fazer 16.777.216 (256*256*256) cores com isto! Abaixo você pode ver uma imagem de como estes componentes ficam quando separados: A imagem à esquerda é uma decomposição dos componentes individuais da função, e depois à direita é uma ilustração de como a mudança desses componentes afeta a cor final. Sintaxe: make_colour_rgb(vermelho, verde, azul); Argumento Descrição vermelho O componente vermelho da cor verde O componente verde da cor azul O componente azul da cor Devoluções: Int Exemplo: col = make_colour_rgb(100, 145, 255); O código acima usa a função para criar uma cor e armazenar seu valor na variável "col" para uso posterior.
