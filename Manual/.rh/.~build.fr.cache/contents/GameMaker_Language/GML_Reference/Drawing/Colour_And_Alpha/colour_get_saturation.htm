cor_get_saturação Esta função devolverá a saturação da cor dada. Esta é a quantidade do tom de cor que é misturada na cor final e faz parte do método de tonalidade, saturação e valor (luminosidade) para definir uma cor. A seguinte imagem ilustra como este valor corresponde à escala de cor HSV: Sintaxe: color_get_saturation(col); Argumento Descrição col A cor a ser verificada Devoluções: Int Exemplo: col = make_colour_hsv(random(255), color_get_sat(c_teal), 255); O código acima obtém a saturação usada para fazer a cor constante "c_teal" e então usa esse valor para definir uma cor aleatória para ter a mesma saturação, armazenando a cor resultante na variável "col".
