gpu_get_tex_min_mip Com esta função você pode obter o nível mínimo de mipmap que é usado atualmente, onde um valor de 0 é a maior resolução, 1 é usar o primeiro mipmap, 2 é o segundo etc... Sintaxe: gpu_get_tex_min_mip(); Devoluções: Real (padrão: 0) Exemplo: se gpu_get_tex_min_mip() != 0 { gpu_set_tex_min_mip(0); } O código acima verificará o nível mínimo atual do mipmap e, se não for 0, será ajustado para 0.
