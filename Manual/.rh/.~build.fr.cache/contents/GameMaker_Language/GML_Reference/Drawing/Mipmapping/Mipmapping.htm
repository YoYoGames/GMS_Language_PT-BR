Mipmapping Esta seção contém todas as funções relacionadas ao uso das funções de mipmapping. Antes de usar estas funções, você deve ter habilitado o mipmapping para o páginas de textura no Editor do Grupo de Textura e/ou habilitou o mipmapping para aqueles sprites que foram definidos para usar apenas uma única página de textura das Opções Gerais de Jogo. A imagem abaixo mostra a diferença que o mipmapping pode fazer ao renderizar seu projeto: Uma vez ativado o mipmapping para o projeto ou uma página de textura, você pode então usar as seguintes funções para obter e definir valores que mudarão a forma como as coisas ficarão em seu projeto quando forem executadas.
