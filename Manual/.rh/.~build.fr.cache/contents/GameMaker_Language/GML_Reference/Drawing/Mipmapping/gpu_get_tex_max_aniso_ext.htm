gpu_get_tex_max_aniso_ext Com esta função você pode obter o nível máximo atual de anisotropia ao usar o modo de filtro tf_anisotropic (veja gpu_get_tex_mip_filter() para maiores informações) em um amostrador de sombreador. Você fornecer o valor do índice para o amostrador de sombreamento (como retornado pela função shader_get_sampler_index()e a função retornará um valor dentro da faixa de 1 e 16. Sintaxe: gpu_get_tex_max_aniso_ext(sampler_index); Argumento Descrição índice_de_amostragem O índice do amostrador de sombreamento para obter Devoluções: Real (padrão: 16) Exemplo: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set (shd_Glass); if gpu_get_tex_max_aniso_ext(_sampleIndex) != 8 { gpu_set_tex_max_aniso_ext(_sampleIndex, 8); } text_set_set_stage(_sampleIndex , _spriteTex); shader_reset(); O código acima estabelece o nível máximo de anisotropia para 8 para o determinado amostrador de textura de sombreamento, caso ainda não tenha sido estabelecido para 8.
