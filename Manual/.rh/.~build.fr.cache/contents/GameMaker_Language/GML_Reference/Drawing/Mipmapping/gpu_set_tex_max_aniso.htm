gpu_set_tex_max_aniso Com esta função você pode definir o nível máximo atual de anisotropia ao usar o modo de filtro tf_anisotropic (veja gpu_get_tex_mip_filter() para mais informações). O valor de entrada deve variar entre 1 e 16. Sintaxe: gpu_set_tex_max_aniso(maxaniso); Argumento Descrição maxaniso O nível máximo para filtragem anisotrópica (padrão: 16) Devoluções: N/A Exemplo: if gpu_get_tex_max_aniso() != 8 { gpu_set_tex_max_aniso(8); } O código acima verificará o nível máximo atual de filtragem anisotrópica e, se não for 8, está ajustado para 8.
