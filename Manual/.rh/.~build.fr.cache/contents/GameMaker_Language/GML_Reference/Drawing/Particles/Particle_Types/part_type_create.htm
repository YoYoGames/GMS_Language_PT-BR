part_type_create Com esta função você pode criar um novo tipo de partícula e o valor de retorno deve ser armazenado em uma variável para uso em todas as funções de partícula subseqüentes. Sintaxe: part_type_create(); Devoluções: Inteiro (ID de partícula) Exemplo: mypart = part_type_create(); Isto criará um novo tipo de partícula, armazenando seu índice na variável "mypart".
