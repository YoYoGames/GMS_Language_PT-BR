tamanho_da_peça Esta função é usada para determinar o tamanho da partícula quando ela é criada e também pode ser usada para fazer a partícula aumentar ou diminuir de tamanho ao longo de sua vida útil. Os valores mínimo e máximo de tamanho padrão em 1 (o que significa que o tamanho é o mesmo como a partícula de base após a part_type_scale() escala foi definida) e se forem definidas para o mesmo valor, o tamanho inicial para todas as partículas será o mesmo, mas se forem definidas para valores diferentes, cada partícula começará com um tamanho aleatório dentro da faixa, por exemplo, se você tiver um mínimo de 0,5 e um máximo de 2, todas as partículas serão criadas com um tamanho entre a metade e o dobro da escala de base. Você também pode definir um valor de incremento que fará com que a partícula duende cresça (se um número positivo) ou encolha (se um número negativo) ao longo da vida útil da partícula, com a quantidade dada sendo adicionada ao tamanho da partícula em cada etapa. Este valor pode ser um mínimo de (+/-)0,01. Finalmente, você pode definir o fator "agitar". Este é um valor que será adicionado ou subtraído aleatoriamente ao tamanho de cada etapa da vida útil das partículas. Obviamente, valores maiores são mais pronunciados do que os menores, e este valor pode até ser um negativo, sendo a faixa máxima entre -20 e 20. Sintaxe: part_type_size(ind, size_min, size_max, size_incr, size_wiggle); Argumento Descrição ind O índice do tipo de partícula a ser mudado. tamanho_min O tamanho mínimo em que a partícula pode começar. tamanho_max O tamanho máximo em que a partícula pode começar. tamanho_emcr. O quanto a partícula deve aumentar ou diminuir por etapa. size_wiggle O quanto deve ser adicionado ou subtraído aleatoriamente do tamanho das partículas por etapa. Devoluções: N/A Exemplo: part_type_shape(particle1, pt_shape_explosion); part_type_size(particle1, 2, 2, 0.10, 0.01); part_type_scale(particle1, 1, 1); part_type_colour2(particle1, 16777215, 12632256); part_type_alpha3(particle1, 0, 0.20, 0); part_type_speed(part_type_speed(particle1, 1, 1), 0,20, 0,20, 0, 0, 0); part_type_direction(particle1, 0, 359, 0, 20); part_type_orientation(particle1, 0, 359, 0, 0, 1); part_type_blend(particle1, 0); part_type_life(particle1, 40, 60); O código acima definirá vários valores de partículas, incluindo o tamanho que é definido para dobrar o tamanho quando cada partícula é criada, e então incrementar o tamanho em 0,01 cada passo com um abanão de 0,1 também.
