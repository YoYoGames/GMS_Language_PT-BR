part_type_alpha1 Esta função é usada para definir um tipo de partícula para ter um único valor alfa (transparência) para a duração total da vida útil de cada partícula individual, e este pode ser de 0 (transparente) a 1 (opaco). Sintaxe: part_type_alpha1(ind, alfa1); Argumento Descrição ind O índice do tipo de partícula a ser mudado. alfa1 O alfa da partícula. Devoluções: N/A Exemplo: part_type_alpha1(global.Snow_Part, 0.5); O código acima definirá todas as partículas criadas do tipo de partícula indexada na variável global "Snow_Part" para ter um valor alfa de 0,5 (semi-transparente).
