part_type_blend Com esta função você pode fazer com que suas partículas sejam desenhadas com um modo de mistura aditiva(verdadeiro) ou não(falso). A mistura aditiva é um modo especial de mistura que adiciona os valores de luminosidade de cada partícula à medida que elas se sobrepõem, de modo que as cores claras aos poucos, vão ficando mais brilhantes (até parecerem brancos) à medida que se sobrepõem, e as cores escuras se tornam cada vez mais transparentes, sendo o preto quase invisível. Esta função pode dar alguns belos efeitos de partícula, mas nem sempre é necessária. Para Por exemplo, uma pista de fumaça teria uma mistura de aditivos para manter o efeito cinza, mas um efeito de chama provavelmente o teria para tornar as partículas mais translúcidas e mais brilhantes. Sintaxe: part_type_blend(ind, aditivo); Argumento Descrição ind O índice do tipo de partícula a ser mudado. aditivo Se as partículas devem ser misturadas aditivamente (verdadeiras) ou normalmente (falsas). Devoluções: N/A Exemplo: part_type_blend(part_Fire, true); O código acima definirá todas as partículas criadas do tipo de partícula indexada na variável "part_Fire" para ter um modo de mistura aditiva.
