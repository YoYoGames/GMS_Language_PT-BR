part_system_existe Com esta função você pode verificar se o dado sistema de partículas indexadas existe ou não no jogo. Note que se a variável a ser verificada for uma variável não-inicializada (que um sistema de partículas teria seu índice atribuído de outra forma), isto atirar um erro. Sintaxe: part_system_exists(ind); Argumento Descrição ind O índice do sistema de partícula a ser verificado. Devoluções: Booleano Exemplo: se !part_system_exists(global.sname) { global.Sname = part_system_create(); } O código acima verifica se o sistema de partícula referenciado na variável global existe e se não é criado.
