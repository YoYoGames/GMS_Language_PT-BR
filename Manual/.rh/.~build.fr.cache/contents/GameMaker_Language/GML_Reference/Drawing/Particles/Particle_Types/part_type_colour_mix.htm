part_type_colour_mix Com esta função, você pode definir o tipo de partícula dado para ser uma mistura aleatória de duas cores. Sintaxe: part_type_colour_mix(ind, color1, color2); Argumento Descrição ind O índice do tipo de partícula a ser mudado. cor1 A primeira cor que a mistura vai tomar. cor2 A segunda cor que a mistura tomará. Devoluções: N/A Exemplo: part_type_colour_mix(global.P_Damage, c_red, c_yellow); O código acima definirá a cor para cada partícula emitida do tipo de partícula indexada na variável global "P_Damage" para ser uma mistura aleatória entre as cores vermelha e amarela.
