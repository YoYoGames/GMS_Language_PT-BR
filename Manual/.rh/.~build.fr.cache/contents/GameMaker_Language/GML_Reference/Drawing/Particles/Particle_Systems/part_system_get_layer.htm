part_system_get_layer Esta função pode ser usada para recuperar o valor de identificação da camada única para o sistema de partículas em questão. Você fornece o valor de identificação do sistema de partícula (como retornado pela função part_system_create_layer()) e o retornará o valor de identificação da camada. Sintaxe: part_system_get_layer(ind); Argumento Descrição ind O valor de identificação do sistema de partículas para obter a identificação da camada de Devoluções: Real (valor único de ID da camada ou 0 se a camada for administrada internamente) Exemplo: if (part_system_get_layer(global.p_sys) != "effects_layer") { part_system_layer(global.p_sys, "effects_layer"; } O código acima verificará um sistema de partículas para ver em que camada está e, se não estiver na camada dada, será alterado.
