part_type_exists Com esta função você pode verificar se o determinado tipo de partícula indexada existe ou não no jogo. Note que se a variável a ser verificada for uma variável não-inicializada (que de outra forma um tipo de partícula teria seu índice atribuído), isto lançará um erro. Sintaxe: part_type_exists(ind); Argumento Descrição ind O índice do tipo de partícula a ser verificado. Devoluções: Booleano Exemplo: se !part_type_exists(global.p1) { global.p1 = part_type_create(); } O código acima verifica se a variável global "p1" armazena o índice de um tipo de partícula e se ela não cria um.
