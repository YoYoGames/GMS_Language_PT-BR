draw_point Com esta função você pode desenhar um único pixel em qualquer lugar da tela, usando a cor e o alfa atualmente definidos. Sintaxe: draw_point(x, y); Argumento Descrição x A coordenada x do ponto a ser traçado. y A coordenada y do ponto a ser traçado. Devoluções: N/A Exemplo: draw_set_colour(c_yellow); draw_point(100,100); Isto desenhará um pixel amarelo na posição (100.100).
