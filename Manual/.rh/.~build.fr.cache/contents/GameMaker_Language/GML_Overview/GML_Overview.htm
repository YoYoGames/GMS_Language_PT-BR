Visão geral dalinguagem GameMaker Esta seção do manual contém todas as informações necessárias para compreender e utilizar o GameMaker Studio 2 linguagem de script GML. As funções de tempo de execução nesta linguagem podem ser usadas para criar seus jogos e são adicionadas a objetos do Editor de objetosnormalmente quando você optou por fazer um novo Projeto delinguagem GameMaker embora também possa ser usado junto com DnD™(Arrastar e Soltar). Abaixo você pode ver uma imagem típica de um objeto com o editor de código aberto em um evento: Cada evento tem sua própria guia no editor e você pode adicionar, editar ou remover código deles a qualquer momento (para mais informações sobre eventos, veja Eventos Objeto). O código em si deve ter uma estrutura básica e pode conter índices de recursos, variáveis, funções, expressões, palavras-chave... etc... todos os quais são explicados nas seções abaixo. Se você é um novato em programação ou fazendo a troca de DnD™, é recomendável que você comece com a página em Estrutura básica do código e depois ler todas as outras páginas desta seção, testando o código de cada uma delas dentro GameMaker Studio 2 em si.
