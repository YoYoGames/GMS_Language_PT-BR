início / fim A linguagem GameMaker permite que você use as palavras-chave início e fim em vez das mais usuais entre colchetes {} ao criar blocos de código. O exemplo de código abaixo mostra como isto funciona: se (!visível) começar saída; fim while (place_meeting(x, y)) begin x -= lengthdir_x(1, direção - 180); y -= lengthdir_y(1, direção - 180); end Note que o uso dessas palavras-chave não é típico e é fornecido como parte do idioma mais para suporte legado do que para qualquer outra coisa, e a qualquer momento no futuro elas podem ser depreciadas.
