saúde Esta variável é global no escopo e é usada para manter um valor numérico que normalmente é usado para a saúde do jogador. Esta variável é projetada apenas para suportar projetos legados de versões anteriores de GameMaker e deve não ser utilizado em novos projetos como pode ser depreciado no futuro. Sintaxe: saúde; Devoluções: Real (valor de ponto flutuante de precisão único) Exemplo if (health <= 0) { global.state = "Game Over"; instance_destroy(); } O código acima verifica a variável health e se ela for menor ou igual a 0, uma variável global é definida e a instância é destruída.
