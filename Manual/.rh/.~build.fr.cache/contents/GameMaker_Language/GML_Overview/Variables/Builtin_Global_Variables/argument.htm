argumento Esta variável contém um array que é usado junto com a contagem de argumentos de variáveis somente de leitura em funções ou métodos de script. Cada posição de array contém um valor de entrada para a função e é especificamente para uso com funções de argumento de variável . Note que há também uma série de variáveis independentes de escopo global que também podem ser usadas em funções definidas pelo usuário para referenciar os diferentes argumentos de entrada: argument0, argument1, argument2, etc... até .argument15 NOTA: Isto não é mais estritamente necessário e é fornecido, mais do que qualquer outra coisa, para o apoio ao legado. Todas as funções definidas pelo usuário aceitarão argumentos variáveis, e qualquer argumento passado para uma função que não faça parte da função nomeada. as entradas serão inicializadas para indefinidas por padrão, o que pode ser verificado usando a função is_undefined().
