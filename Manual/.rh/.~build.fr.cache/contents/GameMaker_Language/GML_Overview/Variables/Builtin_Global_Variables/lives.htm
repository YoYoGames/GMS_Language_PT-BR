vive Esta variável é global no escopo e é usada para manter um valor numérico que normalmente é usado para a vida do jogador. Esta variável é projetada apenas para suportar projetos legados de versões anteriores de GameMaker e deve não ser utilizado em novos projetos como pode ser depreciado no futuro. Sintaxe: vidas; Devoluções: Real (valor de ponto flutuante de precisão único) Exemplo se (vidas < 5) { vidas += 1; } O código acima verifica a variável lives e se ela for inferior a 5, 1 é adicionado a ela.
