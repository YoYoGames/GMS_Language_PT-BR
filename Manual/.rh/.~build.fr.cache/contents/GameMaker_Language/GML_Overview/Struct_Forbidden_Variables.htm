Estruturas Variáveis Proibidas Ao criar uma estrutura, é possível utilizar certas variáveis embutidas como nomes de variáveis membros, por exemplo: mystruct = { object_index : obj_Player, speed : 3, image_blend : c_red } Entretanto, somente variáveis locais e de escopo de instância incorporadas podem ser utilizadas desta forma e valores atribuídos como se fossem variáveis de membros estruturantes regulares. O uso de variáveis embutidas de escopo global é proibido e causará problemas com seu jogo. Abaixo você pode encontrar uma lista completa destas variáveis para que você saiba quais evitar. As seguintes variáveis globais embutidas estão todas disponíveis para uso em seus projetos, mas não podem ser usadas como variáveis estruturais de membros: argumento[n] argumento0 ... argumento15
