A página inicial Quando você abrir o GameMaker Studio 2 pela primeira vez, você será solicitado a entrar em sua contaYoYo (ou se registrar para uma se ainda não o fez - veja aqui ). Uma vez registrado e logado, a IDE principal do GameMaker Studio 2 será aberta na página inicial: Na parte superior você pode encontrar os menus IDE gerais que são explicados nas seções seguintes do manual: O Menu Arquivo O menu de edição O Menu Construir O Menu Windows O Menu de Ferramentas O Menu do Mercado O Menu de Layouts O Menu Ajuda Observe que também aparecerão opções de menu específicas do contexto na barra de menu superior, dependendo da janela em que você se concentrou atualmente. Por exemplo, se você tiver a janela do Editor de Imagens em foco, então você terá um menu extra itens aqui para "Imagens", " Ver" e " Efeitos". Estes itens extras de menu são explicados nas seções relevantes do manual para o espaço de trabalho ou janela que gerou eles. O restante da página inicial é o seguinte: Home - Ao navegar pelas diferentes seções da Página Inicial, clique com o botão esquerdo do mouse sobre isso simplesmente o levará de volta para a página principal. Detalhes da Versão - Esta seção fornece detalhes sobre a versão IDE atual que está sendo utilizada, assim como a versão atual Tempo de execução versão. Você também será notificado de quaisquer mudanças disponíveis para a IDE ou para o Runtime nesta seção. Projetos recentes - Na esquerda da página inicial você pode ver uma lista de projetos anteriores que você pode abrir. Se você passar o mouse sobre eles, poderá ver o nome completo do projeto e o caminho, e clicar no botão esquerdo do mouse abrirá o projeto. Social - A barra superior da página inicial contém uma série de links para facilitar a navegação. Eles estão listados abaixo: Jogos YoYo - leva você para o siteprincipal daYoYo Games Ícones de Mídia Social - estes o levam aos vários sites de mídia social da YoYo Games onde você pode se manter atualizado com notícias sobre o GameMaker Studio 2 Opções de projetos - A seção Projetos é onde você pode criar, abrir ou importar projetos. Isto é explicado com mais detalhes mais abaixo nesta página. Informações - A seção de informações da página inicial permite que você navegue pelo Jogos YoYo Mercado bem como navegar e abrir Tutoriais e Demonstrações feitas pela equipe da YoYo Games para exibir aspectos específicos do GameMaker Studio 2 (veja abaixo para mais informações). Iniciando um novo projeto Você pode clicar no botão Novo para criar um novo projeto, no botão Abrir para abrir um projeto existente ou no botão Importar para abrir um GameMaker: Arquivo de projeto Studio 1.4 GMX ou para abrir um arquivo de projeto comprimido YYZ GameMaker Studio 2. Tanto o Open como o Import abrirão o explorador de arquivos para que você navegue até o arquivo do projeto que você precisa. NOTA: O GameMaker Studio 2 não é completamente compatível com o GameMaker ao contrário: Os projetos Studio 1.4, mas os projetos 1.4 importados ainda devem ser executados, pois a funcionalidade obsoleta foi recriada para você automaticamente usando scripts de compatibilidade. Para detalhes completos sobre os possíveis problemas e as mudanças feitas no GML, consulte o artigo do Help Center Porting A GMS 1.4 Project To GameMaker Studio 2, assim como a seção do manual sobre Funções Obsoletas. Para criar um novo projeto basta clicar no botão Novo para levá-lo para a tela seguinte: OGameMaker Studio 2 permite especificar que tipo de projeto você deseja criar, e você pode escolher com base em seus conhecimentos ou hábitos de codificação. As opções disponíveis são: Novo Projeto Drag e Drop™ - Isto criará um novo projeto no qual o Editor de Objetos (e quaisquer outros editores que permitam código) será configurado para usar o GameMaker Studio 2 Arrastar e Soltar™ interface. Para mais informações sobre Arrastar e Soltar(DnD™), por favor, veja aqui. Note que esta não é uma interface exclusiva DnD™, e você pode alternar entre DnD™e o código GML usando o mouse direito botão . Novo projeto de linguagem GameMaker: Isto criará um projeto onde a programação padrão é feita usando a linguagem GameMaker(GML). Você pode encontrar mais informações sobre o GML aqui. Note que esta não é uma interface GML exclusiva, e você pode alternar entre o código GML e DnD™usando o botão direito do mouse . Não importa o que você selecionar, você será então solicitado a escolher um local de salvamento para o projeto e então ele será aberto no GameMaker Studio 2. Tutoriais e Demonstrações Ao clicar nos botões Tutoriais ou Demonstrações, você será levado a uma tela que mostra os tutoriais/demonstrações que você possui e os tutoriais/demonstrações oficiais que estão disponíveis para download: Para baixar qualquer um dos tutoriais ou demos oficiais, basta clicar no ícone do que você deseja passar e ele será adicionado à sua seção " Próprio " no topo (note que você pode ser solicitado a fazer o login no Marketplace antes que isto possa ser feito). Uma vez que o tutorial ou demonstração esteja na seção "Propriedade", você pode passar o mouse sobre ele e instalá-lo ou visualizar as informações nele contidas a partir do Mercado : Uma vez instalado um tutorial ou demo, basta clicar sobre ele para abri-lo e você será solicitado a fornecer um local para salvar o projeto tutorial antes que a IDE abra para você começar. Note que os tutoriais e demos instalados terá um ícone adicional que pode ser usado para desinstalar o projeto de seu computador (isto não remove o tutorial ou a demonstração de sua lista de "Propriedade"). As seções de tutoriais/demonstrações da Página Inicial também tem um par de botões para ajudá-lo a usar e navegar melhor seus bens: Clicando aqui, você poderá atualizar a lista de projetos do Marketplace para verificar se algo foi atualizado e obter as versões mais recentes. Clicando nisto, você poderá alternar a visão entre os tutoriais ou demos que você possui e os que você instalou. Também vale a pena notar que você pode encontrar tutoriais e demonstrações de terceiros do Marketplace, e uma vez que estes tenham sido adquiridos e adicionados à sua conta, eles também aparecerão na seção "Possuídos" da página apropriada.
