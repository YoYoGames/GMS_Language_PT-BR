Criando um ativo no mercado Ao selecionar a opção Criar Pacote no Menudo IDE Marketplace você será levado para o seguinte espaço de trabalho: Aqui você deve primeiro selecionar a Editora identidade que você deseja publicar o pacote de bens como (que você deveria ter criado a partir de seu Painel da Editora), então dê um Nome de Exibição e um Número de Versão. Um ID de pacote será então criado para você com base no nome de exibição e na conta do editor utilizada, mas você pode editar isto para ser algo mais se desejar, desde que mantenha o formato de URL inversa. Em seguida, você precisará escolher o tipo de pacote de bens a fazer: Ativo: Esta é uma coleção de ativos (sprites, scripts, objetos, etc...) que você deseja vender no Mercado Demonstração: Esta é uma demonstração para demonstrar algum conceito ou idéia de jogo Tutorial: O recurso que está sendo carregado é um tutorial, projetado para ser executado a partir da IDE GameMaker Studio 2 (veja esta página para mais detalhes) Uma vez feito isso, você pode escolher quais ativos do projeto você deseja adicionar ao Pacote de Ativos usando a seção à direita. Você pode adicionar todos os seus bens, ou apenas alguns deles, bem como remover qualquer um que você tenha adicionado por engano. Presume-se que você tenha criado o ativo a ser carregado anteriormente no GameMaker Studio 2 e que o projeto atual contenha tudo o que você precisa para carregar. LEMBRE-SE! Para poder criar ou utilizar pacotes de ativos, você deve ter criado previamente um certificado de perfil de usuário. Você pode fazer isto a partir das Preferências do Mercado, que são explicadas aqui. Quando você estiver satisfeito com a seleção de ativos e quiser carregar seu pacote no Marketplace, basta clicar no botão Ok na parte inferior, e o pacote de ativos será criado e carregado para o Marketplace. Quando o upload estiver completo, um novo espaço de trabalho será aberto com uma visão do navegador da página de recursos para que você possa editar. Para mais informações sobre como configurá-lo, consulte a página seguinte: Listagem de um ativo Se você precisar atualizar um ativo do Marketplace, faça mais ou menos o mesmo que fez quando criou o pacote original, somente agora você precisa especificar o pacote a ser atualizado a partir de uma lista suspensa de pacotes disponíveis: Ao atualizar, você não poderá alterar o ID do pacote, e deve estar ciente de que o Número da Versão mostrado será o número da versão atual e você deve incrementá-lo conforme apropriado, caso contrário a atualização falhará (já que você não pode ter dois pacotes de ativos com o mesmo número de versão). Lá é também um campo onde você pode adicionar algum texto para o registro de mudanças do ativo, e este será carregado junto com o ativo e exibido na página do Marketplace. IMPORTANTE! Ativos feitos com GameMaker Studio versão 2.3+ só estarão disponíveis para aqueles usuários que estiverem usando a versão 2.3 e superior. Se você estiver atualizando um ativo de uma versão anterior, esse ativo ainda estará disponível para os usuários que não atualizaram e irão coexistir com a versão atualizada 2.3+. Carregando embora a página da Web Você pode fazer o upload de ativos através da página real do Marketplace em vez da IDE, algo que pode ser necessário se o upload através da IDE falhou por qualquer razão. Para começar, você criaria o pacote e tentaria fazer o upload através a IDE como mostrado acima, e se ela falhou, então vá para a Janela de Saída e verifique o log para uma linha que mostra onde o pacote que você está criando foi salvo antes de tentar o upload. Por exemplo: Em Windows: Arquivo do pacote criado: C:\Users\WINDOWSUSERNAME\AppData\Roaming\GameMakerStudio2\GMS2USERNAME\Marketplace\Packages\com.publishername.assetname.yymp Em macOS: Arquivo do pacote criado: /Users/MACUSERNAME/.config/GameMakerStudio2/GMS2USERNAME/Marketplace/Packages/com.publishername.assetname.yymp Você precisa navegar até esse local de arquivo e copiar o arquivo *.YYMP para um local seguro, pois este será o arquivo que você estará carregando para o Marketplace. Neste ponto você precisa abrir o Marketplace em seu navegador e navegar até a listagem de ativos para o ativo e depois ir para a seção Versões, e então clicar no botão Upload Version, como mostrado na imagem abaixo: Isto o levará para a página de upload da nova versão, onde você pode adicionar informações sobre a versão que está sendo carregada, se necessário e também fornecer o arquivo clicando no botão Choose FIle e selecionando o arquivo *.YYMP previamente criado: A adição do arquivo fará o upload das versões e mostrará uma barra de "progresso", e quando o upload estiver terminado e estiver satisfeito com os detalhes, você pode então clicar no botão Salvar para salvar o ativo na listagem. Observe que o bem estará na fase de Processamento por alguns minutos antes de poder ser publicado.
