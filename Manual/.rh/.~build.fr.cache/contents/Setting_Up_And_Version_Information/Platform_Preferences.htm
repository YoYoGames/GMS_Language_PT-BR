Preferências de plataforma Esta seção do manual cobre as diferentes Preferências específicas da plataforma que precisam ser configuradas antes que você possa compilar seus jogos para determinados objetivos. Estes geralmente só precisam ser configurados uma vez para começar (embora futuras atualizações para a tempo de execução pode exigir que você mude certas coisas, mas estas serão mencionadas nas Notas de Lançamento) e são usadas para apontar para quaisquer SDKsque sejam necessários e definir opções básicas de Compilação para como você deseja que o GameMaker Studio 2 compila seus projetos de jogo para qualquer plataforma específica. Note que as plataformas disponíveis dependerão da Licença que você possui e nem todas elas podem estar disponíveis. Abaixo está uma lista de todas as plataformas alvo disponíveis para licenças padrão: Android HTML5 iOS tvOS macOS Vapor Windows Windows UWP Observe que, além de configurar estas preferências, você também precisará configurar os dispositivos que algumas destas plataformas visam, caso contrário, você poderá não ser capaz de compilar ou testar seus projetos. Consulte a seção sobre Compilação e a seção sobre o Gerenciador de Dispositivos para obter mais informações. O manual não mostra as diversas preferências do Console (Xbox One, PS4 e Nintendo), devido às diferentes restrições legais mantidas pelas diferentes empresas. Se você tem uma Licença de Console, então você pode obter este formulário de informação o Helpdesk da YoYo Games, mas você deve primeiro contatar a YoYo Games com seu e-mail de identificação do console e solicitar permissão para visualizar as diferentes seções.
