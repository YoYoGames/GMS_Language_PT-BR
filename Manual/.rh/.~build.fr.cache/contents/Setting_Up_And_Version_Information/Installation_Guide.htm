Guia de Instalação GameMaker Studio 2 é projetado para uso em macOS e Windows computadores de mesa com as especificações mínimas mencionadas aqui. Você pode baixar o instalador do GameMaker Studio 2 de sua página de conta YoYo no site doYoYo Games. Uma vez instalado, a primeira vez que você executar o GameMaker Studio 2, você poderá ser solicitado a atualizar antes de ser capaz de para usar o produto. Após qualquer atualização ter sido instalada, você será solicitado a fornecer os detalhes da licença para sua conta YoYo Games (veja aqui para mais informações) antes de poder usar o produto, significando que a primeira vez que você usa o GameMaker Studio 2 você deve ter uma conexão com a Internet. Após o login inicial, você deverá ser capaz de usar o produto offline, porém periodicamente será necessário que você se conecte para verificar o conta de login. É importante notar que - dependendo do tipo de licença que você possui - você pode ser solicitado a instalar componentes adicionais ou seguir os passos adicionais para configurar e usar o GameMaker Studio 2 para plataformas específicas. Para informações completas sobre o que cada plataforma requer, favor ver este artigo no Helpdesk do YoYo Games: GMS2 SDKs requeridos. Versões Estável e Beta A versão que você baixa da página web da YoYo Games deve ser a versão estável atual do GameMaker Studio 2, mas também pode haver outra versão " Beta" disponível que é usada para mostrar ou testar novos funcionalidade antes de entrar em uma construção estável final. Os detalhes completos da versão são os seguintes: Estável: Esta versão é a que você estaria usando para desenvolvimento geral e é recomendada para uso com qualquer produto comercial. Pode não ter certas características disponíveis nas versões Beta, mas isto se deve ao fato de não terem sido totalmente testadas ou que requerem trabalho adicional. Beta: A versão Beta do GameMaker Studio 2 é para aqueles que desejam experimentar novas funcionalidades antes de serem adicionados ao Stable build, ou para aqueles que necessitam de uma correção para um problema. Não é recomendado que você use o Beta construir para a produção final do jogo, a menos que contenha uma correção importante para um bug que está atrasando seu projeto. Você pode obter a construção Beta do GameMaker Studio 2 na página "Downloads" de sua conta YoYo, e ele pode ser instalado e executado lado a lado. com a construção do Estável. Observe que se você estiver executando o GameMaker Studio 2 até Vapor então você não terá acesso à construção Beta do produto usando o próprio Steam, embora você ainda possa baixar e usar a versão desktop.
