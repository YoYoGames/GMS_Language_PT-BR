PreferênciasiOS As preferências do iOS têm as seguintes opções: Suprimir a construção e o funcionamento: Por padrão, quando você cria um projeto iOS e depois constrói um pacote executável final, o GameMaker Studio 2 constrói o pacote e depois tenta executá-lo em qualquer dispositivo conectado. Ao marcar esta opção, você suprimir a construção do aplicativo e sua execução no dispositivo e simplesmente criar um projeto XCode no Mac host MacOS Install Path: O caminho de instalação para os arquivos do projeto no Mac do build. Identificador da equipe padrão: Aqui você pode adicionar seu Identificador de Equipe padrão, como lhe foi designado pela Apple. Este ID de Equipe será usado quando seus arquivos de jogo forem enviados para Xcode para construir o aplicativo, e permitirá que Xcode gere os certificados de assinatura necessários. Note que esta configuração será aplicada por padrão a todos os jogos construídos para iOS, mas pode ser sobreposto em uma base por projeto a partir das Opções Gerais do Jogo iOS.
