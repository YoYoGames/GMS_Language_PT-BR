Preferências doWindows UWP As preferências do Windows UWP têm as seguintes opções: Localização doVisual Studio: Esta opção é para apontar o GameMaker Studio 2 para a instalação do Microsoft Visual Studio no PC construído. Ganhe a localização das ferramentas UWP: Esta opção é para apontar o GameMaker Studio 2 para o caminho de instalação das ferramentas UWP no PC construído. Arquitetura padrão do pacote: Por padrão, você verá uma janela de diálogo para escolher entre as arquiteturas x86, x64 e ARM ao compilar um executável UWP. Você pode definir esta opção a um desses três que serão então selecionados por padrão sempre que você optar por compilar um executável. Resposta automática à criação de pacote de loja: Na janela de diálogo mostrada na compilação, você pode optar por criar um pacote de loja. Você pode definir esta opção para Sim ou Não , que será usada por padrão. sempre que você optar por compilar um executável.
