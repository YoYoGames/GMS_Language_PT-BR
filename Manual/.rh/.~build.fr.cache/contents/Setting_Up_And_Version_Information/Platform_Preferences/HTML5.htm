Preferências HTML5 As preferências HTML5 têm as seguintes opções: Tipo de embalagem padrão: Ao construir um pacote executável HTML5 final, você pode escolher entre Pacote como arquivos soltos - que simplesmente gerará todos os arquivos necessários em uma pasta - ou Pacote como Zip - que criará um arquivo comprimido ZIP que terá o jogo e quaisquer arquivos adicionais dentro dele. Por padrão, você terá a escolha de qual destas duas opções você deseja escolher na compilação, mas você pode definir isto preferência de usar sempre um ou outro. Observe que quando esta opção for definida para Mostrar Diálogo, na janela de diálogo real que aparece na compilação, escolhendo "Lembrar Opção de Embalagem", esta preferência será definida para a opção você seleciona. Porta padrão do Servidor Web: Isto permite mudar a porta padrão para todos os dispositivos criados através do Gerenciador de Dispositivos. O valor padrão é 51264. Endereço padrão do servidor Web: Aqui você pode mudar o endereço padrão do servidor web para dispositivos que você adicionar ao Gerenciador de Dispositivos. O valor padrão é 127,0,0,0,1. Servidor Web padrão IPs permitidos: Aqui você pode adicionar quaisquer IPs permitidos para o Servidor Web e eles serão definidos como padrão para todos os dispositivos adicionados através do Gerenciador de destino.
