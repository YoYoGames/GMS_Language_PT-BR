Preferências IDE A Janela de Preferências é uma parte essencial do GameMaker Studio 2, pois é onde você pode montar e personalizar sua experiência usando as IDEs, bem como configurar qualquer SDK de plataforma específica, etc... Nesta seção nós cobrimos as áreas específicas da IDE da janela de preferências - para as plataformas alvo individuais, consulte a seção sobre Preferências da Plataforma Alvo. Note que após alterar quaisquer preferências desta janela você precisa clicar no botão Aplicar para que elas tenham efeito. Se, após mudar as opções aqui para personalizar o GameMaker Studio 2 para seus próprios gostos, você deseja respaldar as preferências que você definiu, você precisa ir para a pasta: %AppData%/GameMakerStudio2// no Windows, ou na pasta: ~/.config/GameMakerStudio2 no Mac, e depois copiar os dois arquivos a seguir para um local seguro: local_settings.json cloud_settings.json No entanto, se você deseja redefinir as preferências que você mudou, então você precisa clicar no botão Restaurar Padrões na parte superior da janela. Note que isto só restaura os padrões para a seção que está sendo editada atualmente.
