Guia de Atualização A IDE GameMaker Studio 2 verificará periodicamente se há atualizações e depois as baixará automaticamente. Algumas dessas atualizações serão opcionais e você pode optar por não instalá-las se não desejar ou pensar que algumas das mudanças podem quebrar uma projeto que você está prestes a publicar. Entretanto, há algumas atualizações que incluem correções ou melhorias importantes que não podem ser ignoradas e o GameMaker Studio 2 irá forçar uma atualização nestes casos. OGameMaker Studio 2 é mais do que apenas uma IDE e tem uma série de diferentes ferramentas de tempo de execução que ele usa nos bastidores para construir jogos para as diferentes plataformas alvo disponíveis. Estas ferramentas são principalmente os diferentes runners e compiladores que cada plataforma requer, e você pode realmente ter vários de cada um ao mesmo tempo. O que isto significa é que você pode "escolher" qual versão do compilador e qual versão do corredor a ser usada caso você tenha encontrado algum problema com a atual ou estão visando uma versão ou plataforma muito específica. As ferramentas serão automaticamente atualizadas quando houver uma nova versão disponível, mas você pode então usar as Preferências deFeedtime Feeds para selecionar quais você realmente deseja usar. Para mais informações sobre atualizações em tempo de execução, consulte Preferências - Runtime Feeds.
