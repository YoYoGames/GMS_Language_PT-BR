Preferências do editor de fontes As Preferências do Editor de Fontes são usadas para definir como o Editor de Fontes se comporta. As diferentes opções disponíveis estão listadas abaixo, juntamente com uma descrição do que cada uma faz. Atualizar fontes importadas dos projetos do GameMaker Studio 1: Quando isto for verificado, o GameMaker Studio 2 tentará atualizar as fontes e regenerar sua textura. Isto só acontecerá se: Esta preferência está habilitada A fonte está disponível na máquina O GameMaker Studio 2 pode localizar um rosto correspondente na família de fontes (esta é a causa mais provável de incoerência na importação) Note que isso também acontecerá se a fonte GameMaker Studio 1 não tiver uma textura de fonte em cache. Se uma fonte legada não puder ser atualizada, então o GameMaker Studio 2 usará por padrão o cached textura da fonte, e o Editor de Fonte não permitirá mais que você edite a fonte, mostrando uma opção para fazer backup e atualizar para outra: Clicando aqui, você poderá editar a fonte, seja adicionando a fonte que falta ou definindo outra, e o Editor de Fonte A fonte de backup é armazenada na pasta de ativos da fonte, como .yy.gms1 e .png.gms1. Para restaurar esses arquivos, apagar manualmente os arquivos .yy e substituí-los por estes, removendo a extensão .gms1 dos nomes dos arquivos. Importante! Todos os ativos armazenam seus nomes no arquivo .yy, portanto, você deve editar estes e colocar o nome correto do recurso também no campo (se diferente). Escolha automática para o diálogo de regeneração de conteúdo falhado: Quando você vai para regenerar uma textura de recurso de fonte, é mostrado um aviso. A configuração desta opção para "OK" significará que a fonte é regenerada sem que o aviso seja mostrado. Escolha automática para o diálogo de regeneração de fonte falhada: Quando a regeneração de uma textura de recurso de fonte falha, você recebe um aviso. A configuração desta opção para "OK" saltará mostrando a mensagem. Escolha automática para atualização de fontes legadas importadas: Ao importar um projeto legado, você pode receber um aviso sobre as fontes legadas e a oferta de atualização. A configuração desta opção para "OK" dará direito à atualização automática sem mostrando esta mensagem.
