Preferências de linha do tempo As Preferências de Linha do Tempo são usadas para definir certas propriedades para quando você estiver trabalhando com o Editor de Linha do Tempo. As opções disponíveis são: Resposta automática a DnD™conversão para a linguagem GameMaker: Quando você usa o botão direito do mouse opção de menu em um momento da linha do tempo para converter as ações Drag e Drop™em script (código), você receberá uma pergunta rápida se deseja continuar, já que esta é uma conversão de uma só maneira. A mudança desta opção para "Ok" suprimirá a mensagem e prosseguirá com a conversão. como se você tivesse clicado "Ok" no diálogo. A configuração padrão é "Mostrar mensagem". Uma resposta utomática a um aviso de ação permanente: Certas ações dentro de uma linha de tempo não podem ser desfeitas, e quando isso acontece, você receberá uma mensagem de aviso perguntando se deseja continuar. Mudando esta opção para "Ok". suprimirá a mensagem e prosseguirá com a ação como se você tivesse clicado em "Ok" no diálogo. A configuração padrão é "Mostrar mensagem". Abrir novos momentos para editar na criação: Normalmente quando você adiciona um novo momento a uma linha de tempo, o GameMaker Studio 2 abrirá automaticamente o editor para você adicionar seu código ou Arrastar e Soltar para ele. Entretanto, se você desmarcar esta opção, o momento será adicionado sem abrir o editor (você pode abri-lo clicando duas vezes no momento). Por padrão, esta opção está ativada.
