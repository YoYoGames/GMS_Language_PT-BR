Preferências do Navegador de Ativos As Preferências do Navegador de Ativos são usadas para definir certas propriedades para como o Navegador de Ativos e seus elementos funcionam. As principais opções são: Mostrar Acesso Rápido: Quando ativada, esta opção mostrará a seção "Acesso Rápido" no topo do Navegador de Ativos. Esta opção é ativada por padrão. Criar pastas padrão em um novo projeto: Quando ativada, esta opção terá todos os novos projetos começando com as pastas de ativos padrão já criadas, e quando desativada, somente uma pasta de sala será criada com a sala padrão necessária por todos os projetos. Esta opção é ativada por padrão. Número máximo de ativos para abrir simultaneamente sem aviso prévio: Você pode selecionar vários ativos para abrir de cada vez, usando / + para selecioná-los e depois clicando com o botão direito do mouse e escolhendo abrir todos. Se você selecionou mais do que o número especificado nesta opção, então lhe será mostrado um aviso, já que ter muitos ativos abertos ao mesmo tempo pode causar problemas com desempenho em máquinas de extremidade inferior. O valor padrão é 10. Máximo de itens editados recentemente: Esta configuração rege o número de ativos listados na seção "Recente" do Navegador de Ativos. O valor padrão é 10. Tamanho da paleta de cores: Aqui você pode definir o número de cores disponíveis na paleta do navegador de ativos para grupos de ativos. O valor padrão é 16, com um valor mínimo de 16 e um valor máximo de 100. Aplique cores de ativos em Acesso Rápido: Quando marcada, esta opção utilizará as cores definidas para qualquer tipo de ativo para colorir o ativo na seção Acesso Rápido do navegador. A desmarcação desta opção não utilizará a cor do ativo. Isto é ativado por padrão. As cores do grupo substituem as cores infantis: Quando verificado, isto utilizará a cor do grupo de nível superior para colorir qualquer ativo dentro do grupo, ignorando qualquer cor que tenha sido definida para o ativo individual. Isto é desativado por padrão. Criar um estilo de diálogo de ativos: Ao clicar no botão "Adicionar Ativo" no Navegador de Ativos, você será apresentado com uma janela mostrando os diferentes ativos que podem ser adicionados ao seu projeto. Esta janela pode ser apresentada em uma visão de lista ou em uma visão em grade, e esta configuração governa o que é mostrado por padrão. O valor padrão é grid. Ordem de classificação por padrão: Isto pode ser usado para definir a ordem padrão de ordenação dos ativos no navegador, seja por ordem alfabética de A a Z ou de Z a A. A ordem padrão é de A a Z. Pastas de classificação por padrão primeiro: Quando isto estiver ativado, o navegador de ativos ordenará primeiro as pastas, depois quaisquer ativos "soltos" em segundo lugar, enquanto que se isto estiver desativado, então todos os ativos e pastas serão ordenados juntos. Isto é habilitado por padrão. Sprite duplo clique abre o editor de imagens: Por padrão, o duplo clique em qualquer ativo abrirá esse ativo para edição, mas muitas vezes quando se trata de sprites, após a configuração inicial quando você cria o novo ativo, você o abre para editar a imagem em vez das propriedades sprite. Selecionar esta opção abrirá o editor de imagem para o ativo sprite ao invés do próprio ativo sprite. Você ainda pode abrir o editor sprite clicando com o botão direito do mouse em e selecionando Edit. Esta opção está desligada por padrão. O duplo clique do objeto abre todos os eventos de objeto: Por padrão, um duplo clique em em qualquer ativo abrirá esse ativo para edição, mas muitas vezes quando se lida com objetos, após a configuração inicial quando você cria o novo ativo, você o abre para editar apenas os eventos e não as propriedades do objeto. A seleção desta opção abrirá os diferentes eventos do objeto em um novo espaço de trabalho, sendo que cada evento terá sua própria aba dentro do novo espaço de trabalho. Você ainda pode abrir o editor de objetos clicando com o botão direito do mouse em e selecionando Editar. Esta opção está desligada por padrão. Ogrupo vazio é removido por ordem de quarto: Quando ativado, o desagrupamento de todas as salas que pertencem a uma pasta de grupo removerá também a pasta do grupo. Quando desativada, a mesma ação irá desagrupar as salas, mas deixará o grupo vazio. no navegador de ativos. Isto é ativado por padrão. Zoom padrão (%): Isto define a porcentagem padrão de zoom para o Navegador de Ativos quando você inicia um novo projeto. O valor padrão é 100%. Zoom mínimo (%): Isto estabelece o valor mínimo de zoom permitido para o Navegador de Ativos. O valor padrão é 50%. Zoom máximo (%): Isto define o valor máximo de zoom permitido para o Navegador de Ativos. O valor padrão é de 250%. Mostrar linhas de árvores: Aqui você pode ativar/desativar as linhas das árvores que são mostradas ao lado dos ativos no Navegador de Ativos. Isto é verificado por padrão para mostrar as linhas das árvores. Expandir árvore na troca de filtro: Esta opção forçará o Navegador de Ativos a expandir quaisquer nós de árvore colapsados quando você mudar as opções de filtro. A desativação da opção impedirá isto. A opção padrão está ativada. Confirmar Diálogos Aqui você pode selecionar o comportamento dos diálogos de confirmação ao trabalhar com um ativo do Navegador de Ativos. As opções disponíveis são: Resposta automática à eliminação de ativos: Quando você apaga um bem do Navegador de Bens, normalmente recebe um aviso perguntando se você tem certeza de que deseja prosseguir. Ao definir esta opção para "OK", a mensagem será suprimida. como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem". Resposta automática à exclusão do grupo de pedidos de quartos: Quando você exclui uma pasta de grupo contendo salas do Navegador de Ativos, você normalmente recebe um aviso perguntando se você tem certeza e se quer fazer uma procissão (que lista todas as salas do grupo que será apagado). Ao configurar esta opção para "OK", a mensagem será suprimida como se você tivesse clicado no botão OK para prosseguir e excluir todas as salas do grupo, bem como a própria pasta do grupo. O valor padrão é "Mostrar Mensagem". Resposta automática à abertura de múltiplos ativos: Como mencionado acima nas preferências gerais do Navegador de Ativos, quando você abre vários ativos de uma só vez a partir do Navegador de Ativos, você pode receber um aviso se o número exceder o número especificado. Ao definir esta opção para "OK", o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem". Resposta automática a DnD™conversão para a linguagem GameMaker: Ao selecionar a opção de converter um nó DnD™para GML, você será solicitado a confirmar a mudança, pois esta não pode ser desfeita. Ao definir esta opção como "OK". o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem". Resposta automática à configuração de uma sala pai via arrasto do nó: Na seção "Herança" da janela do Room Manager, você pode arrastar uma sala para outra sala para defini-la como uma criança da sala para a qual está sendo arrastada. Ao fazer isto você receberá um aviso perguntando se você tem certeza e se quer fazer um prognóstico. Ao configurar esta opção para "OK", a mensagem será suprimida como se você tivesse clicado no botão OK para prosseguir. O valor padrão é "Mostrar mensagem". Resposta automática a erro de nome de grupo duplicado: Ao criar uma pasta de grupo para organizar seus ativos e nomeá-la como outra pasta no navegador, será exibida uma mensagem perguntando se você deseja continuar e ter duas pastas com o mesmo nome. Ao definir esta opção para "OK" o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem". Resposta automática à configuração de uma sala pai via arrasto do nó: Quando você arrasta uma sala sobre outra sala no Room Manager e depois a libera, será exibida uma mensagem perguntando se você deseja fazer da sala uma "criança" do espaço sendo deixado. Ao definir esta opção para "OK" o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem".
