Seqüências Preferências As Preferências de Seqüências são usadas para definir certas propriedades de como o Editor de Seqüências e seus elementos funcionam. As principais opções são: Mostrar moldura de lona em seqüências recém-criadas: Quando ativado, o guia inteligente da moldura da tela será mostrado na tela do Editor de Seqüências para todas as novas seqüências. Esta opção pode ser habilitada/desabilitada conforme a necessidade a partir das propriedades da tela no próprio editor também. O valor padrão é ativado. Largura da moldura de lona em seqüências recém-criadas: Aqui você pode definir a largura padrão da moldura para todas as novas seqüências. O valor padrão é 1366px. Altura da estrutura da tela em seqüências recém-criadas: Aqui você pode definir a altura padrão da moldura para todas as novas seqüências. O valor padrão é 768px. Pausa ao clicar sobre a tela: Quando esta opção é marcada, clicar em qualquer lugar na tela do Editor de Seqüências fará uma pausa na reprodução da seqüência. Isto é ativado por padrão. Atraso na seleção da sobreposição em milissegundos: Isto controla quanto tempo você precisa esperar antes que a caixa de ferramentas do Item apareça para os itens sobrepostos na tela do Editor de Seqüências. O valor padrão é de 1000ms. Margem de inserção do ponto do caminho de tradução: Na tela do Sequence Editor, quando você tiver adicionado um ativo e depois criado uma faixa de parâmetros de posição, haverá linhas pontilhadas desenhadas para demonstrar o caminho que o ativo tomará quando em movimento. Se você colocar seu mouse próximo a esta linha, então você pode adicionar um novo ponto de faixa de parâmetros a ela, e esta opção define a distância da linha de tradução que o mouse precisa estar antes que o marcador de ponto seja mostrado. O valor padrão é 6px. Criar automaticamente faixas de parâmetros padrão: Quando ativada, esta opção adicionará automaticamente faixas de parâmetros para posição, escala, rotação e origem a quaisquer novas faixas criadas no Painel de Pistas da seqüência. O padrão está desativado. Esticar automaticamente os ativos recém-criados: Esta opção esticará todas as chaves de ativos recém-criadas para corresponder ao comprimento da seqüência. A inadimplência é desativada. Altura da pista em curva animada em folha de dopes: Esta opção pode ser usada para definir quantas pistas de parâmetros de linhas de pista que utilizam curvas irão ocupar a folha de dopw quando na visão do quadro-chave. O padrão é 2, mas você pode selecionar 1 ou 3 também. Confirmar Diálogos O As preferências do Dialogs são usadas para alterar as mensagens que são mostradas no editor de seqüências quando você executa certas ações. As mensagens que você pode alterar são: Escolha automática para incorporar curva editada: Se você tentar incorporar uma curva de animação que você editou, será exibida uma mensagem por padrão para confirmar a ação. Entretanto, definir esta opção para "Sim" não mostrará a mensagem e incorporar a cada vez. Colocá-la em "Não" impedirá a incorporação a cada vez. Escolha automática para converter curva em chaves: Se você tentar converter uma curva de animação em quadros-chave, será exibida uma mensagem por padrão perguntando se você deseja continuar. No entanto, a configuração desta opção para "Ok" não mostrará a mensagem e simplesmente não deixar você converter a curva. Escolha automática para mostrar aviso na edição de curva externa: Se você tentar editar uma curva aplicada a uma faixa de parâmetros e ela for um ativo de Curva de Animação externa, será mostrada uma mensagem de aviso por padrão. Entretanto, a configuração esta opção para "Ok" não mostrará a mensagem e permitirá que você a edite de qualquer forma. Guias inteligentes Aqui você pode selecionar o comportamento dos guias inteligentes quando se trabalha no Editor de Seqüências. As opções disponíveis são: Habilitar guias inteligentes: Aqui você pode ativar ou desativar os guias inteligentes na tela do Editor de Seqüências. Isto é habilitado por padrão. Habilitar as Réguas: Aqui você pode ativar ou desativar as guias inteligentes na tela do Editor de Seqüências. Isto é habilitado por padrão. Réguas de corte para bordas: Esta opção permitirá que as réguas "estalem" até as bordas dos ativos nas seqüências de lona (e vice-versa). O padrão está habilitado. Distância instantânea: Esta é a distância (em pixels) que um item precisa ser para uma régua (ou uma régua para um item) para que ele "estale" para a posição. O valor padrão é 6px.
