Preferências recentes do Windows As preferências aqui contêm opções de como a lista de Windows recentes irá exibir os itens: Mova janelas focadas para o topo: Quando isto for ativado e você clicar em qualquer item da lista Janela Recente, o item selecionado será movido para o topo da lista. Por padrão, isto é desativado. Mostrar apenas janelas abertas atualmente: A ativação disto fará com que a lista de janelas recentes remova aqueles itens que não estão mais abertos em um espaço de trabalho (ou seja: fechar qualquer janela também a removerá da lista). Por padrão, isto é desativado. Janelas máximas a serem exibidas: Isto define o número máximo de janelas previamente abertas que serão exibidas dentro da lista de Janelas Recentes.
