Procure um tampão Esta ação pode ser usada para mover (procurar) para uma posição dentro de um buffer. Você fornece o valor único de identificação do buffer como retornado pela ação Criar buffer e depois o valor de offset em bytes para se mover. O valor de offset será ser aplicado desde o início do buffer, portanto, chamando esta função moverá a posição de leitura/gravação para o início do buffer e, em seguida, moverá o número de bytes dados e posicionará a nova posição de leitura/gravação ali. Entretanto, se você sinalizar o offset como relativo, ele moverá a posição de leitura/escrita para frente (ou para trás se o offset for um valor negativo) a partir da posição atual. Sintaxe de ação: Argumentos: Argumento Descrição Buffer O índice tampão (armazenado em uma variável) offset A compensação (em bytes) em relação à posição atual de busca Exemplo: O código do bloco de ação acima obtém o tamanho do buffer e então usa esse valor para definir a leitura/escrita posição até o final dos dados atualmente armazenados.
