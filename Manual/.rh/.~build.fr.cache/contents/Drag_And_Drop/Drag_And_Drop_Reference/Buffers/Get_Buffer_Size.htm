Obter tamanho de tampão Esta ação pode ser usada para recuperar o tamanho de um buffer, em bytes. Você fornece o valor único de identificação do buffer como retornado pela ação Criar buffer e então dá uma variável alvo para retornar o valor do tamanho. O alvo pode ser sinalizada como sendo uma variável local temporária, neste caso será criada para você e removida da memória novamente no final do script de ação ou evento atual. Sintaxe de ação: Argumentos: Argumento Descrição Buffer O índice tampão (armazenado em uma variável) Meta A variável alvo para manter o valor retornado Exemplo: O código do bloco de ação acima obtém o tamanho do buffer e então usa esse valor para definir o posição de leitura/escrita até o final dos dados atualmente armazenados.
