Se o fim do tampão Esta ação pode ser usada para verificar se a posição de leitura/gravação chegou ao fim do buffer. Você fornece o valor único de identificação do buffer como retornado pela ação Criar buffer e então a ação retornará verdadeira se a posição de leitura/gravação tiver alcançado o final de todos os dados disponíveis no buffer, ou falsa se não tiver alcançado. Você pode marcar "não" para fazer a verificação da ação se a posição de leitura/gravação não estiver no final do buffer.
