Salvar Jogo Esta ação fará uma economia básica do jogo para um determinado arquivo. Você dá um nome de arquivo (como um fio e incluindo a extensão) e a ação fará uma gravação limitada fora do estado atual do jogo. É importante entender que a ação é projetado para tirar uma "foto" do estado atual do jogo e é para ser usado mais como um sistema de ponto de controle do que um sistema universal de salvamento, já que não economiza nenhum dos recursos dinâmicos (como Estruturas de Dados ou Partículas ) . Como tal, a ação deve ser usada com muito cuidado, para não obter erros ou vazamentos de memória. Para uma abordagem mais abrangente para salvar seus dados de jogo, veja as Ações de Arquivo. Sintaxe de ação: Argumentos: Argumento Descrição Nome do arquivo O nome - como um fio e com a extensão - para dar o arquivo a ser salvo (se o arquivo já existir, ele será sobrescrito) Exemplo: O código de bloco de ação acima verificará se há uma colisão com outra instância e, se for encontrado, verificará uma variável de instância.
