Saída Esta ação fará uma de duas coisas, dependendo do contexto dentro do qual é chamada. Se for chamada de dentro de um evento, então sairá do evento e passará para o próximo, pulando qualquer código de bloco de ação que possa ser colocado depois dele. Entretanto, se for utilizada dentro de uma Função de Bloqueio de Ação, sairá da função somente e retornará ao evento que chamou a função, executando quaisquer ações subseqüentes. Note que a chamada isto no meio de uma função não retornará nenhum valor (para isso use a ação de retorno ). Sintaxe de ação: Exemplo: O código de bloco de ação acima verifica uma variável e se ela for maior ou igual a 100, o evento é encerrado, Caso contrário, a variável é incrementada por uma e a instância sprite é dada uma escala aleatória.
