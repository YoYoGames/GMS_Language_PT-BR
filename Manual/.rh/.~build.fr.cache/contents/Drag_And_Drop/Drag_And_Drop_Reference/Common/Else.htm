Ou A ação "outra" só é usada em conjunto com a ação Se Variável (ou qualquer outra ação "se"), e é uma forma de definir um conjunto de ações que devem ser realizadas se a avaliação "se retorna falso. Por exemplo, você poderia ter uma ação "se" que verifica se a variável saúde dos jogadores é menor ou igual a 0, e se isso avalia como verdadeira, você destrói a instância. Entretanto, se você quiser realizar uma ação diferente conjunto de ações de código se a saúde do jogador não for inferior a 0, então você pode adicionar um "outro" e então especificar as ações a serem executadas enquanto o se avalia como falso. Note que para adicionar ações ao bloco "outros", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo: Qualquer ação abandonada em outro lugar será realizada após o "outro". bloco e o seu conteúdo tenha sido executado. Sintaxe de ação: Exemplo: O código de bloco de ação acima verifica uma lista de DS para ver se o valor inicial é "indefinido" ou não. Se ele é então a Lista DS é destruída, caso contrário, o valor é atribuído a uma variável global.
