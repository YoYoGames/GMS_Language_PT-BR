Intervalo Esta ação só pode ser usada dentro de uma ação de loop (por exemplo, Repetir ou Enquanto), e a chamada irá sair do loop naquele momento. Normalmente, um laço será executado um número fixo de vezes ou até uma condição específica é cumprida, mas você pode usar a ação de Intervalo para terminar prematuramente um loop a qualquer momento (ações dentro do loop após o Intervalo não serão executadas). Sintaxe de ação: Exemplo: O código de bloco de ação acima realiza um loop de repetição 1000 vezes. No loop, ele verifica se uma estrutura de dados da Lista está vazio e se não estiver, remove um item da lista, e se for, quebra o laço.
