Obter Vidas Você pode usar esta ação para recuperar o valor da variável de instância criada para as "vidas". Você fornece a variável de destino para armazenar o valor "vidas" retornado (que pode ser marcado como uma variável local temporária), mas observe que se a variável não tiver sido definida anteriormente (usando Set Lives) então a variável será criada para você e a ação retornará 0 (ao contrário das variáveis de instância regular, que darão um erro se você tentar acessá-las sem tê-las definido primeiro). Sintaxe de ação: Argumentos: Argumento Descrição Meta A variável alvo para armazenar o valor "vidas" retornado (pode ser sinalizada como uma variável local temporária) Exemplo: O código de bloco de ação acima recuperará o valor de "vidas" e o armazenará em um bloco de ação temporário variável local. Esta variável é então usada para decidir quantos sprites empilhados devem ser desenhados.
