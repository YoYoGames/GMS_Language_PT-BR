Biblioteca de ação variável de instância As ações das Variáveis de Instância são projetadas para que você comece a criar um jogo e são usadas para criar e manter variáveis específicas de escopo de instância para pontuação, vida e saúde (você pode saber mais sobre as variáveis escopo aqui). Estas ações criarão estas variáveis para você a qualquer momento em que elas forem usadas e se destinam mais para conveniência ou para iniciantes, e se você precisar de mais controle complexo sobre os valores do jogo você deve estar fazendo suas próprias variáveis usando as ações apropriadas (veja aqui) As seguintes ações existem para o uso destas Variáveis de Instância: Pontuação definida Obter pontuação Conjunto de Vidas Ganhe Vidas Conjunto Saúde Obter Saúde Se a pontuação Se Vive Se Saúde Voltar: Índice A seguir: Ações Diversas © Copyright YoYo Games Ltd. 2021 Todos os direitos reservados
