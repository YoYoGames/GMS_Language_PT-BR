Pontuação Com esta ação, você pode criar ou definir uma variável de instância para a pontuação. Você fornece o valor para definir a variável de pontuação ou para adicionar à variável de pontuação (se você marcar a caixa de seleção relativa, então o valor será adicionado à "pontuação" atual. valor). Observe que, ao contrário das variáveis de instância regulares, o nome da variável para "pontuação" é armazenado internamente para recuperar o valor da pontuação que você precisa para usar a ação Obter Pontuação. Sintaxe de ação: Argumentos: Argumento Descrição Pontuação A quantia para definir a pontuação para (ou para adicionar à pontuação se relativo for verificado) Exemplo: O código de bloco de ação acima irá verificar se a sala é a primeira do jogo e se é estabelece as variáveis "pontuação", "vidas" e "saúde", por exemplo.
