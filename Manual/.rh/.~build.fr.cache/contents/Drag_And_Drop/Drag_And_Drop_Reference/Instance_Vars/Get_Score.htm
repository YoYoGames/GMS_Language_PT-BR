Obter pontuação Você pode usar esta ação para recuperar o valor da variável de instância criada para a "pontuação". Você fornece a variável de destino para armazenar o valor da "pontuação" retornada (que pode ser marcada como uma variável local temporária), mas observe que se a variável não tiver sido definida anteriormente (usando Set Score) então a variável será criada para você e a ação retornará 0 (ao contrário das variáveis de instância regular, que darão um erro se você tentar acessá-las sem tê-las definido primeiro). Sintaxe de ação: Argumentos: Argumento Descrição Meta A variável alvo para armazenar o valor da "pontuação" retornada (pode ser marcada como uma variável local temporária) Exemplo: O código de bloco de ação acima recuperará o valor da pontuação atual e o armazenará em um local temporário. variável. Esta variável é então convertida em uma string e armazenada em outra variável local temporária antes de ser atraída para a tela.
