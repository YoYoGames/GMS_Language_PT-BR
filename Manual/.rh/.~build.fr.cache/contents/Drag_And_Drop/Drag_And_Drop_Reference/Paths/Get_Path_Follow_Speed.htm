Obter velocidade no caminho Esta ação terá a velocidade, em pixels por quadro de jogo, de uma instância ser definida para seguir um caminho (note que você deve ter chamado a ação Start Following Path para atribuir um caminho à instância para que esta devolver um valor diferente de 0). O valor retornado será armazenado na variável de destino que você der, ou você pode marcar a variável como uma variável local temporária para criá-la e atribuir o valor. Sintaxe de ação: Argumentos: Argumento Descrição Meta A variável alvo para manter o caminho de retorno segue a velocidade Exemplo: O código de bloco de ação acima obtém a velocidade atual da instância ao longo do e se for inferior a 4, aumenta a velocidade da trajetória em 0,1.
