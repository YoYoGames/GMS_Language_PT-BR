Se a chave for liberada Esta ação verifica quando uma tecla de teclado foi liberada. Ela retornará verdadeiro no passo de jogo único (frame) em que a chave é liberada e falsa em todos os outros momentos. Se você ativar o não modificador, esta ação será revertida, e verificar se a chave não foi liberada, retornando falso na etapa do jogo (frame) em que foi liberada e verdadeira em todos os outros momentos. Observe que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo: Estas ações serão executadas agora se o "se" for avaliado como verdadeiro, enquanto quaisquer ações abandonadas em outro lugar serão executadas após o bloco "se". Sintaxe de ação: Argumentos: Argumento Descrição Chave O nome da chave a ser verificada. Não Negar o cheque (o verdadeiro torna-se falso e vice-versa) Exemplo: O código de bloco de ação acima faz um levantamento do teclado a cada passo e se uma tecla estiver sendo mantida pressionada, ela verifica se a tecla inicial está pressionada. Se a verificação for verdadeira na prensa inicial, um som é tocado, então, enquanto a tecla é mantida pressionada, a mistura de sprite é mudada para vermelho. Se a tecla não estiver sendo mantida pressionada para baixo, é feita uma verificação na liberação da chave para repor a cor da mistura para branco.
