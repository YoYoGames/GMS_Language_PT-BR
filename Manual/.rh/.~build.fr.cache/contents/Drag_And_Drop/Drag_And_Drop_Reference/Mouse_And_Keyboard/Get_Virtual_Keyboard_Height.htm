Obter Altura do Teclado Virtual Esta ação devolverá a altura atual em pixels do teclado virtual, com base no tamanho da tela. Se o teclado não for visível, 0 será retornado. Sintaxe de ação: Exemplo: O código de bloco de ação acima verifica se o teclado virtual do sistema operacional está visível, e se é ele que recebe a altura do teclado (em pixels de tela) e então desenha algum texto acima dele com a entrada do teclado.
