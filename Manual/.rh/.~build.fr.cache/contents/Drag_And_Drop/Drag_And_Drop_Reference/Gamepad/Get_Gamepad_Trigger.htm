Adquira o Gatilho Gamepad Você pode usar esta ação para obter o valor dos diferentes gatilhos a partir de um determinado gamepad. Você fornece o índice do gamepad (este é o número de "slots" que um gamepad ocupa) e seleciona o gatilho a ser verificado (esquerda ou direita). O valor retornado será entre 0 e 1, onde 0 não é pressão no gatilho e 1 pressão total. Este valor retornado será armazenado na variável de destino que você fornecer para uso futuro. Note que esta função não leva em conta a configuração do limiar. Sintaxe de ação: Argumentos: Argumento Descrição Gamepad O índice gamepad. Gatilho O gatilho a ser verificado (esquerda/direita) Meta A variável alvo para armazenar o valor retornado. Exemplo: O código de bloco de ação acima cria uma variável local (temporária) e depois a utiliza para armazenar o valor atual para o gatilho esquerdo. Se o valor do gatilho for superior a 0,5, uma instância é criada na posição da instância chamadora.
