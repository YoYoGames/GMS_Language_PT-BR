Biblioteca de ação Gamepad GameMaker Studio 2 tem uma série de ações dedicadas que podem ser usadas para detectar tanto o analógico quanto o digital controles de múltiplos blocos de jogo conectados. Estas ações exigem que você especifique um valor de índice gamepad, que é um número (contando de 0) que representa o gamepad que está sendo selecionado. Observe que quando um controle de jogo é conectado ao seu dispositivo (ou é removido) e então um Evento de Sistema Assíncrono é acionado (entretanto, para lidar com isso você pode precisar usar código). Os "slots" do gamepad são indexados de 0 com slots 0 - 3 inclusive sendo apenas para os gamepads Xinput , ou seja: controladores Xbox360 e compatíveis. Entretanto, você também pode verificar os slots 4 - 11 inclusive para os gamepads DirectInput, o que significa que você pode detectar muitos outros modelos de controladores quando conectado através destes slots. Windows é totalmente suportado com até um máximo de 11 dispositivos conectados permitidos de uma só vez. Note que no Windows, os primeiros 4 slots de gamepad (0 - 3) são tratados usando a dll XInput, o que significa que somente os controladores XBox podem ser 100% compatíveis e que para outros tipos de controladores você deve verificar o resto dos slots do gamepad (4 - 11). macOS é suportado com até um máximo de 4 dispositivos conectados permitidos de uma vez, e estes dispositivos só podem ser do tipo Playstation3ou Xbox 360. Ubuntu também suporta a entrada do GamePad, mas você pode precisar instalar bibliotecas adicionais a partir do repositório Ubuntu. Você pode fazer isso facilmente abrindo um terminal de linha de comando e digitando o seguinte, que instalará o suporte a GUI para o joystick, assim como o próprio joystick: sudo apt-get install jstest-gtk sudo apt-get install joystick. Os jogosHTML5 suportam gamepads na maioria dos principais navegadores, exceto Safari. O apoio do Gamepad também se estende a iOS com o gabinete iCade. O eixo esquerdo mapeia para o controlador do bastão (embora a entrada seja digital, não analógica), os quatro botões "face" mapeiam para os botões frontais do gabinete, e o quatro botões de ombro mapeados para aqueles na parte de trás do gabinete. Android export suporta controladores NYKO e controladores genéricos Bluetooth, mas somente quando eles estão habilitados, o que significa que você terá que marcar a opção iCade/Bluetooth na seção Geral das Opções de Jogo do Android. Eles requerem API nível 12 para que funcionem plenamente e deve ser observado que o GameMaker Studio 2 registrará como conectado qualquer dispositivo Bluetooth com o qual seu dispositivo esteja emparelhado, quer seja ou não conectado ou não. Portanto, isto deve ser levado em consideração ao atribuir e verificar os "slots". No PS4, se você quiser usar o rastreamento por toque, você precisa usar as funções para o dispositivo_mouse_* botões (não há DnD™ações para isso).
