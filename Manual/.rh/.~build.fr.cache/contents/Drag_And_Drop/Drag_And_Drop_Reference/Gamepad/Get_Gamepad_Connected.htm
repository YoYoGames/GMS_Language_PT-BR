Conectar o Gamepad Você pode usar esta ação para verificar e ver se um determinado "slot" de gamepad tem um gamepad conectado. Você fornece o índice do gamepad (este é o número de "slots" que um gamepad ocupa) e o valor retornado será armazenado na variável de destino que você fornece para uso futuro (isto pode ser sinalizado como uma variável local temporária que você cria para este evento ou roteiro). Sintaxe de ação: Argumentos: Argumento Descrição Gamepad O índice gamepad a ser verificado. Meta A variável alvo para armazenar o valor retornado. Exemplo: O código de bloco de ação acima executa um laço para verificar e ver se um controle de jogo está conectado a qualquer um dos 12 "slots" disponíveis. Se um "gamepad" for descoberto, uma variável global é definida para o número atual do slot e o loop é quebrado.
