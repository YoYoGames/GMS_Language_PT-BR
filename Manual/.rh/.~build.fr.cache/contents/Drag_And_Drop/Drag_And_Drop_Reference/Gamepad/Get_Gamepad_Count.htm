Obter Gamepad Count Você pode usar esta ação para obter o número de controles de jogo conectados ao dispositivo que está executando seu jogo. O número retornado estará entre 0 e 12. Sintaxe de ação: Argumentos: Argumento Descrição Meta A variável alvo para armazenar o valor retornado. Exemplo: O código de bloco de ação acima obtém o número de blocos de jogo conectados e atribui o valor a uma variável local temporária. Isto é então verificado para ver se é mais de 0 e uma variável global definida de acordo.
