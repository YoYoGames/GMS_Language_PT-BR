Rotação da Instância Este bloco de ação define o ângulo_da_imagem (rotação) da instância. O ângulo é medido em graus, com a direita sendo 0°, acima sendo 90°, esquerda sendo 180° e abaixo sendo 270°. Defina esta variável como 0 para reiniciar a instância, o que significa que ela desenhar qualquer duende designado como foi definido no editor do duende. Observe que para que as mudanças nesta ação sejam visíveis, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker Studio 2 irá, por padrão, sortear o duende) ou ser sorteado usando a ação do Draw Self. Sintaxe de ação: Argumentos: Argumento Descrição Ângulo O novo ângulo a ser utilizado (de 0 a 360). Exemplo: O código de bloco de ação acima estabelece um novo duende, bem como uma série de outros propriedades de como esse duende deve ser exibido, inclusive girando-o 180°.
