Se a instância existir Esta ação pode ser usada para verificar se existe uma instância de um objeto específico na sala de jogo, ou se existe uma instância específica em si. Você dá o objeto do explorador de bens, e se alguma instância existir na sala, então a ação retornará verdadeira, caso contrário, retornará falsa. Note que você pode usar uma variável aqui com o ID único para uma instância específica (por exemplo, como retornado pela ação Criar Instância de Objeto) ou mesmo o ID da instância valor conforme retornado pelo editor da sala quando você adicionou a instância. Se você sinalizar o argumento "Não", então a ação verificará se não existe nenhuma instância e se nenhuma for encontrada, ela retornará verdadeira e se mesmo uma existir, ela retornará falsa. Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo: Estas ações serão agora executadas se o "se" for avaliado como verdadeiro, enquanto quaisquer ações descartadas em outro lugar serão executadas após o bloco do "se". Sintaxe de ação: Argumentos: Argumento Descrição Objeto O nome do objeto a ser verificado para uma instância de (ou uma identificação de instância). Exemplo: O código de bloco de ação acima verifica se há uma colisão e retorna o valor de identificação único da instância que está sendo colidida. Este valor é armazenado em uma variável temporária e então verificado para ver se o valor corresponde a uma instância que está atualmente dentro da sala (se não for encontrada nenhuma colisão, então a variável conterá a palavra-chave "noone"), e, se for encontrado, então uma variável é definida na instância com a qual se está colidindo.
