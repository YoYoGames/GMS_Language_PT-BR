Criar instância de objeto Com esta ação você pode criar uma instância de qualquer objeto a partir do Navegador de Ativos. Você dá o nome do objeto e uma posição x/y dentro da sala onde ele deve ser criado, bem como o nome da camada onde a instância será criada (deve ser uma Camada de Instância). Você pode marcar as bandeiras "relativas" para posicionar a instância em relação à instância que está executando os blocos de código de ação. Sempre que você cria uma instância no jogo, ela recebe um valor de identificação único com o qual você pode então identificar a instância em outras ações, portanto, esta ação permite definir uma variável "alvo" para manter este valor de identificação (como mostrado no exemplo abaixo). Note que o alvo pode ser marcado como uma variável temporária (local), que ser criado para a ação e pode ser usado em todas as ações subseqüentes. Sintaxe de ação: Argumentos: Argumento Descrição Objeto O nome do objeto para criar uma instância de. x A posição x na sala para criar a instância (pode ser relativa à instância de chamada). y A posição y na sala para criar a instância (pode ser relativa à instância de chamada). Camada O nome (um fio) da camada da instância para criar a instância sobre. Meta A variável a ser alvo para o valor de retorno. Exemplo: O código de bloco de ação acima cria uma nova instância e armazena seu valor de identificação único em um variável temporária. Esta variável é então utilizada na ação Aplicar a... para definir algumas propriedades da instância criada.
