Obter a contagem de instâncias Com esta ação você pode encontrar o número de instâncias de qualquer objeto atualmente ativo dentro da sala. Você pode selecionar qualquer objeto da lista de recursos ou pode usar a palavra-chave especial todos para obter o número de todas as instâncias ativas. Você também precisa fornecer uma variável de destino para retornar o valor para que possa utilizá-la em outras ações. Se você marcar a variável alvo como "Temp", então isto criará uma nova variável temporária (local) variável para manter o valor de retorno para você, como mostrado no exemplo abaixo. Sintaxe de ação: Argumentos: Argumento Descrição Objeto O nome do objeto a ser verificado. Meta A variável a ser alvo para o valor de retorno. Exemplo: O código de bloco de ação acima cria uma variável local (temporária) e depois atribui o número de instâncias do objeto "obj_Squirrel" a ele. Este valor é então verificado e a instância é destruída se o número de instâncias for maior que 100.
