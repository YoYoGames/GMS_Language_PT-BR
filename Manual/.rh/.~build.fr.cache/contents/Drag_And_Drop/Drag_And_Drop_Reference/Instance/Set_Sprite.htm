Set Sprite Este bloco de ação define o índice sprite_index da instância para outro sprite do Navegador de Ativos. Você seleciona o duende para mudar, e depois dá a imagem (quadro de animação) para mostrar. Observe que se você não quiser que o duende anime depois definindo o quadro, você precisará usar a ação Set Animation Speed e defini-lo como 0, ou se você quiser que o sprite continue animando então defina o valor para usar a variável image_index embutida ao invés disso. A mudança do duende também mudará a máscara de colisão da instância, a menos que você também tenha especificado um índice_de_máscara separado no Editor de Objetos. Favor observar que para mudanças nesta ação para ser visível, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker Studio 2 irá, por padrão, sortear o sprite) ou ser sorteada usando a ação do Draw Self. Sintaxe de ação: Argumentos: Argumento Descrição Sprite O novo duende a ser usado (-1 se você quiser remover o duende). Moldura O quadro de animação para mostrar inicialmente (se não forem usados quadros extras, então use 0). Exemplo: O código de bloco de ação acima estabelece um novo duende, bem como uma série de outras propriedades para a forma como esse duende deve ser exibido.
