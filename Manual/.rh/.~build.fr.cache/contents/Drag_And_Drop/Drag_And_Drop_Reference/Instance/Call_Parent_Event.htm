Chamar Evento dos Pais Este bloco de ação executará o evento do objeto pai de que a instância é uma criança antes de continuar a executar as ações na instância real. Quando você atribui um objeto pai a uma instância, todos os eventos desse objeto são "herdados". pela criança. Entretanto, se você então adicionar um evento ao objeto filho, ele substitui o evento pai. Usando este bloco de ação, você pode forçar um objeto filho a executar tanto o evento pai quanto o evento na própria instância. Consulte a seção Editor de Objetos para obter mais informações sobre os pais.
