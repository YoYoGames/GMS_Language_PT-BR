Evento de chamada ao usuário Com esta ação, você pode chamar qualquer um dos 12 Outros - Eventos do Usuário disponíveis. Um Evento de Usuário é aquele ao qual você pode adicionar scripts ou código de ação e depois ligar a qualquer momento usando este ação, tornando-os essencialmente eventos de objetos personalizados. Sintaxe de ação: Argumentos: Argumento Descrição Evento O número do evento do usuário a ser chamado (0 - 11). Exemplo: O código de bloco de ação acima verifica uma variável e depois chama um Evento de Usuário [0] ou um Evento de Usuário [1] dependendo se o cheque se avalia como verdadeiro ou falso.
