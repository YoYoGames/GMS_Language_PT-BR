Se o arquivo existir Com esta ação, você pode verificar se existe um arquivo com o nome dado. Você seleciona o arquivo a ser verificado (como uma picada e incluindo a extensão) e a função retornará verdadeira se existir, e falsa se não existir. Observe que isto retornará verdadeiro se o arquivo existir no pacote do jogo como um arquivo incluído ou se o arquivo tiver sido salvo enquanto o jogo estava em execução (veja a seção sobre O Sistema de Arquivo para mais informações). Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo: Sintaxe de ação: Argumentos: Argumento Descrição Tipo O tipo de dados a serem lidos, seja um fio (entre aspas ""), ou um númeroreal Exemplo: O código de bloco de ação acima criará uma variável para guardar os dados do buffer e então verificará se um arquivo de proteção de segurança já foi feito anteriormente. Se o arquivo existe, então ele é carregado na variável buffer que criamos, mas se não existir, ele é criado, escrito e então salvo pronto para ser carregado na próxima vez.
