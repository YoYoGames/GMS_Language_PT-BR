Escrever para o arquivo da Ini Com esta ação, você pode escrever um valor numérico ou um fio em um arquivo da Ini. Você deve ter previamente aberto o arquivo Ini para leitura/escrita usando a ação Abrir arquivo Ini, e então com esta ação você dá o tipo de dados para escrever (seja um número real ou uma string incluída em ""), assim como o cabeçalho da seção, a chave dentro dessa seção e o valor para a chave. Para ajudar a visualizar a estrutura de um arquivo da Ini, veja esta imagem: Como você pode ver, os arquivos da Ini são divididos em seções - onde cada seção tem seu próprio nome de cabeçalho - e então cada seção é subseqüentemente dividida em vários pares chave/valor (muito parecido com uma estrutura de dados do mapa). Observe que você mesmo pode criar arquivos Ini usando qualquer software de processamento de texto (como o Bloco de Notas em Windows) e depois inclua-os como um Arquivo Incluído para ler e escrever em seu jogo. IMPORTANTE! Tentar ler dados de um arquivo Ini que não tenha sido aberto anteriormente usando a ação Open Ini File causará um erro grave e travará seu jogo. IMPORTANTE! Isto só grava os dados na memória e para finalmente gravar os dados no arquivo em disco, você deve chamar Close Ini File, caso contrário os dados não serão salvos. Sintaxe de ação: Argumentos: Argumento Descrição Tipo O tipo de dados a escrever, seja um fio (entre aspas ""), ou um númeroreal Seção A seção do arquivo da Ini para escrever (como uma seqüência) Chave A chave dentro da seção para escrever (como um fio) Valor O valor a escrever, seja um fio ou um número real com base no argumento "tipo" (acima) Exemplo: O código de bloco de ação acima abrirá um ini e então escreverá dados para três chaves diferentes em o mesmo cabeçalho de seção (um fio e dois números reais) antes de fechar novamente o arquivo ini (escrevendo assim os dados em disco).
