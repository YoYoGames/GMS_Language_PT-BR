Salve o Buffer Com esta ação você pode salvar um arquivo contendo os dados de um buffer. Você fornece o ID exclusivo do buffer (como retornado quando você criou o buffer com Create Buffer) e dá um nome para o arquivo (como uma string, com a extensão de sua escolha) e o buffer será escrito neste arquivo pronto para ser carregado novamente usando o Load Buffer. Sintaxe de ação: Argumentos: Argumento Descrição Buffer O valor único de identificação do buffer para salvar Nome do arquivo O nome (como um fio) do arquivo para salvar Exemplo: O código de bloco de ação acima criará uma variável para armazenar dados de buffer e então verificará se um arquivo de proteção de segurança já foi feito anteriormente. Se o arquivo existe, então ele é carregado na variável de buffer que criamos, mas se não existir, ele é criado, escrito e então salvo pronto para ser carregado na próxima vez.
