Excluir arquivo Com esta ação, você pode apagar um arquivo que foi criado anteriormente. Você fornece o nome do arquivo (como uma cadeia) do arquivo a ser excluído. Note que você não pode excluir nenhum arquivo que esteja incluído no pacote do jogo, somente aqueles que foram criados usando as ações Close Ini ou Save Buffer (veja a seção sobre O Sistema de Arquivo para mais informações). Sintaxe de ação: Argumentos: Argumento Descrição Nome do arquivo O nome (como um fio) do arquivo a ser excluído Exemplo: O código de bloco de ação acima verificará se o arquivo "checkpoint.sav" existe, e se ele então verifica se o arquivo "checkpoint_OLD.sav" existe. Se esse arquivo também existe, então ele é excluído, e então o arquivo original "checkpoint.sav" é renomeado para "checkpoint_OLD.sav". Finalmente um buffer é salvo como "checkpoint.sav" (essencialmente estamos fazendo backup de um arquivo buffer salvo cada vez que o salvamos, para que haja sempre um arquivo "atual" salvo e um arquivo "antigo" salvo).
