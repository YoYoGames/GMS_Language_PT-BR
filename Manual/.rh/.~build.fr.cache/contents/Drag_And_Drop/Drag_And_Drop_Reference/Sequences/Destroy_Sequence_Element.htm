Elemento de Sequência de Destroy Você pode chamar esta ação sempre que desejar "destruir" um elemento de seqüência previamente criado. Você dá o ID do elemento, como retornado pela ação Criar elemento seqüencial, e esta ação irá destruí-la, retirando-a da sala. Sintaxe de ação: Argumentos: Argumento Descrição Variável A identificação do elemento de seqüência a ser destruído. Exemplo: O código de bloco de ação acima verifica se o elemento de seqüência armazenado na variável "my_seq" existe e, se existir, então a destrói.
