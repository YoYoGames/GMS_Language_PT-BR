Definir posição seqüencial Com esta ação, você pode definir a posição para o elemento de seqüência dado na sala. Você fornece o ID do elemento de seqüência (como retornado pela ação Criar Sequência), e então a nova posição X e Y. Se você marque a caixa de seleção "relativo", então os valores dados para X e Y serão adicionados à posição atual da seqüência, caso contrário, serão uma posição absoluta. Sintaxe de ação: Argumentos: Argumento Descrição Variável A identificação do elemento de seqüência para definir a posição de. X A nova posição x na sala para a seqüência. Y A nova posição y na sala para a seqüência. Exemplo: O código de bloco de ação acima recupera a posição atual do elemento de seqüência ID armazenado na variável "my_seq". Estes valores são então usados para definir a posição da seqüência, movendo-a 10 pixels para a direita.
