Obter Comprimento Sequência Com esta ação você pode recuperar o comprimento (em armações) do elemento de seqüência dado na sala. Você fornece o ID do elemento de seqüência (como retornado pela ação Criar Sequência), e esta ação retornará seu comprimento na variável alvo. Este será o número de quadros para os quais a Sequência corre. Observe que a variável alvo pode ser marcada como temporária (local), que será criada para a ação e pode ser usada em todas as ações subseqüentes para o evento em que a ação estiver. Sintaxe de ação: Argumentos: Argumento Descrição Variável A identificação do elemento de seqüência para obter o comprimento de. Meta A variável alvo para segurar o comprimento. Exemplo: O bloco de ação acima recupera o comprimento do elemento de seqüência ID armazenado na variável "seq_id". Esse valor é então usado para definir a posição da cabeça do mesmo elemento de seqüência para 10 quadros antes de seu fim.
