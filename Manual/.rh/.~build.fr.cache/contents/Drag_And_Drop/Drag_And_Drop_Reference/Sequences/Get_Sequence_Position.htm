Obter posição seqüencial Com esta ação você pode recuperar a posição atual para o elemento de seqüência dado na sala. Você fornece o ID do elemento seqüencial (como retornado pela ação Criar Seqüência), e esta ação retornará sua posição X e Y em variáveis-alvo. Observe que as variáveis-alvo podem ser marcadas como variáveis temporárias (locais), que serão criadas para a ação e podem ser usadas em todas as ações subseqüentes para o evento em que a ação estiver. Sintaxe de ação: Argumentos: Argumento Descrição Variável A identificação do elemento de seqüência para obter a posição de. Alvo X A variável alvo para manter a posição x. Alvo Y A variável alvo para manter a posição y. Exemplo: O código de bloco de ação acima recupera a posição atual do elemento de seqüência ID armazenado na variável "my_seq". Estes valores são então usados para definir a posição da seqüência, movendo-a 10 pixels para a direita.
