Obter índice de azulejos na Pixel Esta ação é utilizada para obter um índice de azulejos em uma determinada camada. A camada é especificada a partir do nome da camada dada (um fio, como usado para nomear a camada no Editor da Sala), e então você dá uma posição x e y para encontrar o azulejo para obter o índice de. O azulejo será aquela em que a posição dada se encaixa na sala, e o índice será devolvido à variável alvo, onde 0 não é azulejo, e um inteiro positivo é o índice do azulejo. A posição x e y pode ser assinalada como sendo relativa a a posição da instância de chamada, e a variável alvo pode ser marcada como uma variável temporária (local) que pode ser usada até o final do evento. Sintaxe de ação: Argumentos: Argumento Descrição Camada A camada para assentar a telha x A posição x para verificar y A posição y para verificar Meta A variável alvo para armazenar o índice de azulejos Exemplo: O código de bloco de ação acima verifica se há um botão esquerdo do mouse e se for detectado, obtém-se o índice da célula do azulejo em que a posição do mouse se encaixa. Este índice de azulejo é então verificado e se não for 0, é ajustado para 0.
