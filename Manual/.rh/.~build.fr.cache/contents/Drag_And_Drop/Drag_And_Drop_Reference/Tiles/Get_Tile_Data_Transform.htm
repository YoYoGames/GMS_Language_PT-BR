Obter transformação de dados do azulejo Com esta ação você pode descobrir se os dados de um azulejo foram transformados de uma de três maneiras: virados, espelhados ou girados. Os dados do azulejo são simplesmente um valor que reflete o índice do azulejo junto com as transformações aplicadas, e você irá normalmente querem obter os dados do azulejo primeiro usando ou Get Tile Data In Cell ou Get Tile Data At Pixel antes de usar esta ação. A ação retornará verdadeiro se a transformação escolhida foi aplicada aos dados do azulejo, ou falsa, e o valor retornado será então armazenado na variável de destino que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você verificar o "Temp" de seleção). Para maiores informações sobre os dados do azulejo, consulte a seção GML sobre Funções do Mapa de Azulejos. Sintaxe de ação: Argumentos: Argumento Descrição Telha Os dados da telha para obter a transformação de Transformar A transformação a ser verificada Exemplo: O código de bloco de ação acima obtém os dados do azulejo na posição do mouse e então verifica se o azulejo teve uma transformação de rotação aplicada a ele. Se tiver, então a transformação é reinicializada e os dados do azulejo colocam o azulejo na posição novamente.
