Definir índice de azulejos na célula Com esta ação, você pode colocar uma telha em uma determinada camada para um novo índice. A camada é especificada a partir do nome da camada dada (um fio, como usado para nomear a camada no Editor de Sala), e então você dá a linha e a coluna da célula dentro da camada para obter o azulejo a ser colocado. Finalmente você fornece o índice de azulejos para ajustar o azulejo (os azulejos são indexados da esquerda para a direita de cima para baixo, com 0 na parte superior esquerda). Note que você pode sempre ajustar o índice de azulejo a 0 para limpar o azulejo. Sintaxe de ação: Argumentos: Argumento Descrição Camada A camada para assentar a telha Coluna A coluna (da esquerda para a direita) da célula Fila A fila (de cima para baixo) da célula Telha O índice de azulejos para definir a célula para Exemplo: O código de bloco de ação acima cria um laço para atravessar a linha superior Se não for o índice de azulejo para essa célula é definido como 0 e a variável temporária (local) é incrementada antes que o laço faça outra conjunto de verificações.
