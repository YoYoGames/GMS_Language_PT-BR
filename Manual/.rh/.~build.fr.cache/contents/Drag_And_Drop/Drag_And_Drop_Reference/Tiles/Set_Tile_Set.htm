Set Tile Set Com esta ação você pode definir uma camada para usar conjuntos de azulejos diferentes daqueles criados no Navegador de Ativos. Você dá o nome da camada (um fio, como definido no Editor de Sala), e então o recurso do conjunto de azulejos a ser utilizado, e todos os azulejos da sala sobre essa camada serão desenhados com o novo conjunto de azulejos. Sintaxe de ação: Argumentos: Argumento Descrição Camada A camada sobre a qual colocar o novo conjunto de azulejos. Tileset O conjunto de azulejos a serem utilizados para a camada. Exemplo: O código de bloco de ação acima verifica se as telhas na camada "Floor_Tiles" está usando o conjunto de azulejos "tl_PalaceRuins", e se não o fizerem, então eles estarão prontos para usá-lo.
