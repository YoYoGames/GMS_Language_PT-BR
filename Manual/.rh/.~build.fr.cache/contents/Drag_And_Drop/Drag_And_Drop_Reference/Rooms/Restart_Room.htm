Sala de Reinício Esta ação reiniciará a sala atual. Note que a sala não reiniciará até o final do evento ou roteiro onde a ação foi chamada, portanto, qualquer código após esta ação ainda será executado. Sintaxe de ação: Exemplo: O código de bloco de ação acima verifica se há uma tecla pressionada e, se for detectado, a sala é reiniciada.
