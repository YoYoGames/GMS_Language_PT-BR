Ir para a próxima sala Esta ação fará com que seu jogo vá para a próxima sala como listado no Gerente de Sala no momento em que o jogo foi compilado. Se esta sala não existir, um erro será lançado e o jogo será forçado para fechar. Note que a sala não mudará até o final do evento ou roteiro onde a ação foi chamada, portanto, qualquer código após esta ação ainda será executado. Sintaxe de ação: Exemplo: O código de bloco de ação acima verifica se há uma colisão com outro objeto e se um é detectado uma variável na instância é verificada e o jogo irá ou para a próxima sala ou para a sala anterior como listado no pedido do Room Manager.
