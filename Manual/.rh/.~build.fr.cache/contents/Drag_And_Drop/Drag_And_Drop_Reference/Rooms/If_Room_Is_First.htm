Se o quarto é o primeiro Esta ação verificará se a sala atual é a primeira sala conforme listado no Room Manager. A ação voltará a ser verdadeira se a sala atual for a primeira no Room Manager e falsa se não for. Você pode marcar a caixa de seleção " Não é a primeira sala" para transformar isto em "Se a sala NÃO é a primeira". Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo: Estas ações serão executadas agora se o "se" for avaliado como verdadeiro, enquanto quaisquer ações descartado em outro lugar será realizado após o bloco "se". Sintaxe de ação: Exemplo: O código de bloco de ação acima verifica se a sala atual é a primeira sala conforme listado no Gerente de Sala e se for ele define uma série de variáveis globais.
