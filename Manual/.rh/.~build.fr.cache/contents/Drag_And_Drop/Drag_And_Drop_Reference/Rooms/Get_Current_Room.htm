Obter Sala de Corrente Esta ação devolverá o valor único do índice de sala para a sala atual. Você precisa fornecer uma variável alvo para manter o valor retornado, e esta variável pode ser marcada como uma variável local temporária. Sintaxe de ação: Argumentos: Argumento Descrição Meta A variável alvo para armazenar o índice de salas devolvidas Exemplo: O código de bloco de ação acima recebe o valor atual do índice da sala e então usa um Troca ação para verificar a sala e definir uma variável global, dependendo do resultado.
