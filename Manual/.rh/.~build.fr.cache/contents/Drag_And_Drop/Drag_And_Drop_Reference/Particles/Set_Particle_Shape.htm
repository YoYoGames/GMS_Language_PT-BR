Definir a forma da partícula Esta ação pode ser usada para definir a forma construída em forma de duende que uma partícula deve usar ao desenhar. Você fornece o valor de identificação único para o tipo para definir a forma (como retornado quando você criou o tipo com a ação Criar tipo de partícula) e a forma a ser usada. As formas das partículas são simplesmente sprites brancos que foram criados para o GameMaker Studio 2 para serem usados especificamente como partículas e consistem das seguintes formas: NOTA: Se você usar esta ação, então você não deve usar a ação Set Particle Sprite, pois você só pode ter um duende ou uma forma, mas não ambos. Sintaxe de ação: Argumentos: Argumento Descrição Tipo O valor de identificação para o tipo para definir a forma de Forma A forma a ser usada para o tipo de partícula Exemplo: O código de bloco de ação acima criará um novo tipo de partícula e atribuirá seu exclusivo valor de identificação para uma variável global. Em seguida, ele passa a definir todas as propriedades para o tipo de partícula.
