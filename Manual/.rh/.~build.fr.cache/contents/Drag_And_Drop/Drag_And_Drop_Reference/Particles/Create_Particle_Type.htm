Criar tipo de partícula Em seus projetos você pode ter vários tipos diferentes de partículas, cada uma com suas próprias propriedades como direção e cor, etc... que são definidas utilizando as diferentes ações do tipo de partícula. Entretanto, primeiro você precisa criar um tipo de partícula usando esta ação, que essencialmente aloca uma pequena quantidade de memória para manter todos os detalhes da partícula, para que você possa usar as partículas a qualquer momento no futuro. Você dá uma variável para manter o valor de identificação único para o tipo de partícula, e então em todas as ações futuras relacionadas a esta partícula você pode chamar a variável. Você pode escolher marcar esta variável como sendo uma variável local temporária, mas isso significará que ela será descartada no final do roteiro de ação ou evento, significando que a partícula tipo ainda existirá, mas o deixará sem meios de acesso, portanto é preciso ter cuidado ao utilizá-lo (em geral, os tipos de partículas nunca precisarão ser armazenados em uma variável local temporária, mas para certas coisas, como desenhar partículas para uma superfície, pode ser necessário). NOTA: Um tipo de partícula, uma vez criada, permanece residente na memória do sistema durante a duração do jogo, e assim se você precisar recriá-la a qualquer momento (por exemplo, se você chamar a ação Reiniciar o Jogo ), então você deve destruir o tipo de partícula atual, caso contrário você terá um vazamento de memória. Isto acontece quando um recurso é adicionado à memória e depois "desreferenciado", significando que o GameMaker Studio 2 não tem como interagir com esse recurso de forma alguma mais. Isto faz com que a memória fique entupida por recursos inacessíveis que, por sua vez, podem causar problemas com seu jogo. Para destruir um tipo de partícula quando não for mais necessário, use a ação Destroy Particle Type. Você também pode definir o modo de mistura a ser usado para as partículas. Por padrão, elas serão desenhadas com um modo de mistura "normal", portanto as cores não serão modificadas pelo que foi desenhado por baixo da partícula, porém quando a opção "Blend é habilitado, eles serão desenhados usando uma mistura aditiva, o que significa que as coisas serão desenhadas com mudanças nas cores com base no que foi desenhado sob a partícula. A mistura aditiva dá efeitos muito brilhantes e "brilhantes", e é ideal para explosões, lasers, fogo, etc. enquanto a mistura normal é melhor para coisas como chuva, fumaça, e efeitos de detritos.
