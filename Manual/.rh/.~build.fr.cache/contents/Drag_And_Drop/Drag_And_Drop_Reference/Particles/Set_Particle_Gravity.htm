Definir a gravidade das partículas Esta ação simulará o efeito da gravidade sobre todas as partículas do tipo dado. Primeiro você especifica o valor de identificação do tipo de partícula (que deve ser o mesmo que o valor usado quando você criou a partícula usando a ação Criar Tipo de Partícula ) e então você pode definir a força da gravidade e a direção. A força é um valor cumulativo e será adicionada ao vetor de velocidade de cada quadro de jogo, portanto uma gravidade de 1 adicionará um ao valor da velocidade de cada quadro de jogo, enquanto o valor da direção é simplesmente a direção na qual a gravidade deve "puxar" a partícula. Note que tanto para a força quanto para a direção você pode usar valores negativos, e essa direção no GameMaker Studio 2 é calculada com 0° para a direita, 90° para cima, 180° para a esquerda e 270° para o fundo. Sintaxe de ação: Argumentos: Argumento Descrição Tipo O valor de identificação para o tipo de partícula para definir a gravidade de Força A força da gravidade (em pixels por frame de jogo) Direção A direção da "tração" da gravidade Exemplo: O código de bloco de ação acima criará um novo tipo de partícula e atribuirá seu exclusivo valor de identificação para uma variável global. Em seguida, ele passa a definir todas as propriedades para o tipo de partícula.
