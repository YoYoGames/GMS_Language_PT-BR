Obter o Draw Alpha Com esta ação você pode obter o desenho alfa geral que está sendo usado. A ação devolverá o alfa atual à variável alvo que você especificar, que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você verificar o "Temp de seleção). Sintaxe de ação: Exemplo: O bloco de ação acima recebe o alfa atual e se não estiver definido para 1 (totalmente opaco) é definido para esse valor.
