Set Draw Alpha Com esta ação você pode definir o alfa a ser usado para qualquer forma (retângulo/elipse/linha) ou para qualquer texto que seja desenhado posteriormente. Note que isto não precisa ser chamado antes de cada ação de desenho, e pode ser definido apenas uma vez no início do jogo. Alfa é o cenário que controla como as coisas "transparentes" serão desenhadas e é baseado em uma escala de 0 a 1, onde 0 é totalmente transparente e 1 é totalmente opaco.
