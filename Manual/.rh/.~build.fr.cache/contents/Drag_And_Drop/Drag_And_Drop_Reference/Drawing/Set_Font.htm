Definir fonte Com esta ação, você pode definir a fonte a ser usada para todas as ações posteriores do texto do sorteio. Clicando no diálogo, será aberta a seguinte janela onde você poderá então selecionar a fonte que deseja utilizar: Note que isto irá definir a fonte a ser usada para todas as chamadas de texto de sorteio subseqüentes, e portanto, pode ser chamado uma vez no início do jogo - em qualquer caso, não apenas no evento do sorteio - e todas as ações de texto do sorteio utilizarão a fonte definida. Entretanto, se você precisar de várias fontes em seu jogo, então você precisará chamar esta ação antes de sortear qualquer texto com a fonte necessária.
