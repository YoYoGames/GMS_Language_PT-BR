Saúde da Instância de Desenho Com esta ação você pode desenhar uma barra colorida para mostrar uma representação gráfica da instância variável saúde (quando você usa qualquer ação "saúde" no GameMaker Studio 2 a instância que a chama terá um novo escopo de instância variável chamada saúde adicionada). O valor a ser exibido é um valor percentual que se enquadra na faixa de 0 - 100. Você pode definir a direção da âncora para desenhar a barra de saúde (da esquerda para a direita ou da direita para a esquerda e de cima para baixo ou de baixo para cima). bem como a posição, seja dentro da sala ou em relação à instância que chama a ação. Você também pode definir a cor a ser misturada de cada um dos quatro cantos da barra sanitária, criando gradientes do vermelho para o verde (por exemplo, mas você pode usar qualquer cor).
