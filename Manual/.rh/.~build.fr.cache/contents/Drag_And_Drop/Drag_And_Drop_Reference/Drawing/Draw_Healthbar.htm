Desenhar Barra de Saúde Com esta ação você pode desenhar uma barra colorida para mostrar uma representação gráfica de um valor. A quantidade a ser exibida é um valor percentual e deve estar dentro da faixa de 0 - 100, porém você ainda pode usar valores maiores com a fórmula (CURRENT_Value / MAXIMUM_value) * 100, Assim, por exemplo, você pode usar isto para exibir poder, saúde, mana, tempo ou qualquer outra coisa que se beneficiaria de uma exibição em um bar. Além do valor para desenhar, você pode definir a âncora de direção para desenhar a barra de saúde também (da esquerda para a direita ou da direita para a esquerda). e de cima para baixo ou de baixo para cima), bem como a posição, seja dentro da sala ou em relação à instância que chama a ação. Além da posição, você também pode definir a cor a ser misturada de cada um dos quatro cantos da barra de saúde, criando gradientes de vermelho a verde (por exemplo, mas você pode usar qualquer cor).
