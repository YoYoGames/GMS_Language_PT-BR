Rectângulo do Gradiente de Desenho Esta ação desenhará um retângulo em uma determinada posição dentro da sala, usando um conjunto de cores de mistura para criar um gradiente. Você dará a posição superior esquerda e a inferior direita e o retângulo será desenhado entre eles, e a posição pode ser uma posição absoluta dentro da sala, ou uma posição relativa à instância que chama a ação. Em seguida, você define as cores para misturar de cada um dos cantos e também pode definir se o retângulo pode ser preenchido ou delineado, verificando o preencha a caixa no fundo. NOTA: Esta ação é apenas para uso nos diversos eventos de sorteio, e não desenhará nada se for usada em outro lugar. Sintaxe de ação: Argumentos: Argumento Descrição À esquerda A posição esquerda para começar a desenhar a partir de Top A posição superior para começar a desenhar a partir de Certo A posição certa para terminar o desenho a partir de Fundo A posição inferior para o desenho final a partir de Superior esquerdo A cor para misturar a partir do canto superior esquerdo Superior direito A cor para misturar a partir do canto superior direito Esquerda inferior A cor para misturar a partir do canto inferior esquerdo Em baixo à direita A cor para misturar a partir do canto inferior direito Exemplo: O código de bloco de ação acima desenha um retângulo de gradiente em relação ao posição de instância com algum texto sobre ela.
