Mostrar mensagem de depuração Com esta ação, você pode enviar uma mensagem de debug para a saída do compilador. Esta mensagem pode ser uma string ou um número real, mas se você desejar usar ambos, então o número deve ser convertido para uma string (veja a ação Número para String). Note que você pode enviar várias mensagens usando esta ação clicando no ícone para expandir a ação com outros campos de saída. Sintaxe de ação: Argumentos: Argumento Descrição Mensagem A mensagem a ser mostrada na janela de saída Exemplo: O código de bloco de ação acima verificará uma variável e se ela retornar verdadeiro, então um par de mensagens de debug será mostrado na janela Saída.
