Biblioteca de Ação Aleatória O Aleatório A biblioteca tem as seguintes ações para lidar com a geração de números aleatórios e operações fortuitas: Obter Número Aleatório Randomise Escolha Vale notar que o GameMaker Studio 2 usará a mesma semente ao gerar números aleatórios cada vez que você executar o jogo a partir da IDE. Isto é para facilitar a depuração e significa que você obterá os mesmos resultados iniciais ao gerar valores aleatórios, mas isto não ocorrerá quando você tiver compilado o jogo em um executável (uma nova semente será gerada cada vez que o jogo for executado). Se você não desejar este comportamento ao testar, então você deve chamar a ação Randomise uma vez no início do jogo, antes de gerar quaisquer valores aleatórios.
