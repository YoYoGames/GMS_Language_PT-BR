Randomise Esta ação define a "semente" para o gerador de números aleatórios para um valor aleatório. Ao utilizar as funções de número aleatório no GameMaker Studio 2, a semente inicial é sempre a mesma, pois isso facilita muito mais o rastreamento de erros e a depuração (compilada jogos não têm este comportamento). Caso você deseje testar com verdadeiro aleatório, você deve chamar esta ação uma vez no início do jogo. Sintaxe de ação: Exemplo: O código de bloco de ação acima define a semente aleatória para um novo valor e então gera um número a partir de 0 a 9 e a atribui a uma variável de escopo global.
