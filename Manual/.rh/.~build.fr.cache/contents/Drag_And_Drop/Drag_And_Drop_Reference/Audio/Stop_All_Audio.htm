Parar todo o áudio Esta ação pode ser usada para parar todo o áudio que está tocando atualmente (note que mesmo que o áudio esteja marcado como pausado, ele será parado com esta função e você não poderá retomá-la mais tarde). Sintaxe de ação: Exemplo: O código de bloqueio de ação acima verifica se o botão do mouse está pressionado e se for detectado, então verifica uma variável global para ver se ela não é verdadeira. Se não for verdade, então um som é tocado e a variável global é definida como verdadeira, caso contrário todo som é parado e a variável global é definida como falsa.
