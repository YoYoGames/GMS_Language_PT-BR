Pausar todo o áudio Esta ação pode ser usada para pausar todos os sons tocados atualmente, mas isto não pausará nenhum som tocado após a ação ter sido chamada. Você pode começar a tocar novamente usando a ação Retomar Todos os Áudios. Sintaxe de ação: Exemplo: O código de bloco de ação acima verificará uma variável global e se ela for avaliada como verdadeira, então todo o som é pausado, caso contrário, tudo é retomado.
