Biblioteca de ação de áudio OGameMaker Studio 2 tem um mecanismo de áudio completo que se baseia no *.ogg, *.mp3 e *.wav formatos de som. Sons destes tipos adicionados à IDE podem então ser usados em seu jogo usando as ações básicas de áudio mostradas abaixo. Para coisas mais complexas do que os efeitos sonoros básicos ou para jogar um único peça de música você pode se referir às funções avançadas de áudio dedicadas ao streaming de áudio, posicionamento de áudio - para dar som 3D - e agrupamento de áudio. IMPORTANTE! Ao usar áudio no alvo HTML5, você deve estar ciente de que nem todos os navegadores suportam Web Audio e, portanto, podem não reproduzir nenhum som para seu projeto quando executado. Você pode ter uma idéia geral do suporte a Web Audio a partir do seguinte link: Posso usar áudio na Web?
