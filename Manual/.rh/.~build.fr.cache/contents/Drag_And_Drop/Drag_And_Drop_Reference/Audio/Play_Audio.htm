Reproduzir áudio Esta ação irá definir o jogo de som dado. Você seleciona o som do Asset Explorer e então define se deseja ou não que o som seja emitido em loop (um som em loop será reproduzido repetidamente até ser interrompido), definindo a bandeira "Loop". Sintaxe de ação: Argumentos: Argumento Descrição Som O bom recurso para jogar Loop Se deve ou não fazer o loop do som (desligado por padrão) Exemplo: O código de bloco de ação acima verifica se o botão do mouse está pressionado e, se for detectado, verifica um variável global para ver se ela não é verdadeira. Se não for verdade, então um som é tocado (e em loop) e a variável global é definida como verdadeira, caso contrário, o som é parado e a variável global é definida como falsa.
