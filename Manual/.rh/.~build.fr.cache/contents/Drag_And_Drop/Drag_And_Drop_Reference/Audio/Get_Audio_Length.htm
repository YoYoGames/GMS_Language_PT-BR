Obter comprimento de áudio Esta ação pode ser usada para recuperar a duração de um som em seu jogo. Você fornece o recurso de som para obter o comprimento de e uma variável alvo para retornar o valor também (ou você pode marcar o alvo como uma variável temporária para que a ação crie a variável para uso até o final do evento ou roteiro). A duração será devolvida à variável alvo (que pode ser temporária) em segundos e pode ser um valor decimal. Sintaxe de ação: Argumentos: Argumento Descrição Som O bom recurso para obter o comprimento de Meta A variável-alvo para armazenar o valor de retorno Exemplo: O código de bloco de ação acima obtém a duração do jogo do recurso sonoro dado e armazena em uma variável local temporária. Isto é então usado para gerar um valor aleatório entre 0 e o comprimento do som, que é então armazenado em uma variável local temporária diferente. Este novo valor aleatório é então usado para definir a posição inicial para a som e o som é então tocado.
