Pausa Áudio Esta ação pode ser usada para pausar o som escolhido se ele estiver tocando no momento. Note que se você tocou várias instâncias do mesmo som, todas elas serão pausadas, e isto não pausa nenhum som tocado após a ação ter sido chamado. Você pode começar um som pausado tocando novamente usando a ação Resume Audio. Sintaxe de ação: Argumentos: Argumento Descrição Som O bom recurso para fazer uma pausa Exemplo: O código de bloco de ação acima verificará uma variável global e, se ela for avaliada como verdadeira, então o som dado é pausado, caso contrário é retomado.
