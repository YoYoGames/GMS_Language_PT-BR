Set Point Direction Esta ação é utilizada para definir uma direção a partir de um vetor formado pela posição atual da instância e pela posição dada. Você fornece a posição x e y para "apontar" - estas podem ser marcadas como relativas à posição da instância - e a posição dada. direção da instância até aquele ponto será usada para a instância. Sintaxe de ação: Argumentos: Argumento Descrição x O componente x do vetor de direção. y O componente y do vetor de direção. Exemplo: O código de bloco de ação acima verifica se uma instância do objeto "obj_Player" existe na sala e, se o fizer, define a direção da instância para apontar para a posição x/y de "obj_Player".
