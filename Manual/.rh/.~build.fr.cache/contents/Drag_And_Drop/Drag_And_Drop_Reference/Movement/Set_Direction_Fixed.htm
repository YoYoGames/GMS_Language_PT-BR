Definir direção fixa Esta ação é utilizada para definir uma direção de movimento fixa. As direções disponíveis são indicadas pelos diferentes botões no bloco de ação, e clicar em qualquer um deles irá definir essa direção. Note que as direções no GameMaker Studio 2 são medidos em graus, sendo a direita 0°, a alta 90°, a esquerda 180° e a baixa 270°. Sintaxe de ação: Exemplo: O código de bloco de ação acima verifica se um botão esquerdo do mouse pressiona a instância e, se for detectado, define a direção para baixo (270°) e a velocidade para 5.
