Definir direção aleatória Esta ação é usada para definir a direção do movimento da instância para uma ou mais direções aleatórias escolhidas a partir dos 8 pontos cardeais. Você pode clicar em uma ou mais das 8 setas direcionais dentro da ação e a ação irá escolher uma delas aleatoriamente para usar como direção para o exemplo. Se você não selecionar nenhuma das setas, a ação não terá efeito sobre a direção atual da instância. Sintaxe de ação: Exemplo: O código de bloco de ação acima escolhe true ou false e armazena o resultado em uma variável temporária (local). Esta variável é então verificada e se for true então um conjunto de direções aleatórias é escolhido e se for false então outro conjunto é escolhido.
