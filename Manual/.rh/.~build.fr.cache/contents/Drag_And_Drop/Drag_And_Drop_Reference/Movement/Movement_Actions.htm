Biblioteca de Ação de Movimento Em qualquer jogo, movimento e posição são de suma importância e por isso o GameMaker Studio 2 tem uma seleção completa de ações para lidar com cada situação. As duas principais maneiras de mover uma instância é definir a posição real (x/y) dentro do espaço, ou para definir um vetor velocidade/direção. As ações aqui mostradas abrangem ambos os métodos e você pode usar a Variável de Atribuição de Ação Comum para definir diretamente a posição x e y.
