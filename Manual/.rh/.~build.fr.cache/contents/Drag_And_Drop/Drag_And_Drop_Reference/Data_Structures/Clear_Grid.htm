Grelha clara Com esta ação você pode limpar uma estrutura de dados da grade para um único valor (cada célula dentro da grade será definida para o valor dado). Você fornece a variável que contém o índice da grade (como retornado pela ação Criar grade) e depois dar o valor para limpar a grade. Sintaxe de ação: Argumentos: Argumento Descrição Grade O índice (armazenado em uma variável) da grade para limpar Valor O valor para limpar a grade para Exemplo: O código de bloco de ação acima limpa a estrutura de dados da grade dada para conter a palavra-chave noone em cada célula, se uma variável global for menor ou igual a 0.
