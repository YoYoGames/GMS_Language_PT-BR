Inserir na lista Com esta ação, você pode recuperar a posição indexada de um item dentro da lista fornecida. Você fornece a variável que armazena o índice da lista (como retornado pela ação Criar Lista) e o valor que você deseja encontrar a posição do índice de dentro da lista, bem como uma variável alvo para armazenar a posição do índice retornado para o item (que pode ser marcada como uma variável local temporária a ser usada até o final do roteiro ou evento). Quando você insere um valor em uma lista, todos os índices acima do índice dado para inserção será movido para cima, portanto, se você tiver uma lista com 4 itens nela (posição de índice 0 a 3) e colocar um novo item na posição de índice 2, o item 2 anterior se tornará item 3 e o item 3 anterior se tornará item 4. Observe que se o valor que você está procurando não existe dentro da lista, então não haverá posição de índice da lista para retornar, e assim o valor de retorno será -1. Sintaxe de ação: Argumentos: Argumento Descrição Lista O índice (armazenado em uma variável) da lista a ser inserida em Índice A posição do índice dentro da lista para inserir um valor Valor O valor a ser armazenado na posição Exemplo: O código de bloco de ação acima usa um loop de repetição para obter o tamanho da lista e gerar uma posição aleatória dentro da lista. Ele então gera um valor aleatório e insere este valor na lista na posição aleatória definida anteriormente.
