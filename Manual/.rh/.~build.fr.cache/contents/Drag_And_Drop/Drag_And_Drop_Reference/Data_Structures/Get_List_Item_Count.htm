Obter lista Contagem de itens Você pode usar esta ação para descobrir quantas entradas existem em uma lista. Você fornece a variável que contém o índice da lista (como retornado pela ação Criar Lista) e a variável alvo para armazenar o número retornado de itens da lista (que podem ser marcados como uma variável local temporária a ser usada até o final do roteiro ou evento). Sintaxe de ação: Argumentos: Argumento Descrição Lista O índice (armazenado em uma variável) para obter a contagem do item de Meta A variável de destino para armazenar a contagem de itens devolvidos Exemplo: O código de bloco de ação acima obtém o tamanho da lista referenciada no e, em seguida, usa um para loop para fazer loop através dele e destruir as instâncias com as IDs armazenadas na lista.
