Adicionar à lista Esta ação pode ser usada para adicionar um novo valor (de qualquer tipo de dado) à lista, e este valor será adicionado no final. Você fornece a variável que armazena o índice da lista (conforme retornado pela ação Criar Lista) e o valor para armazenar. Sintaxe de ação: Argumentos: Argumento Descrição Lista O índice (armazenado em uma variável) da lista para adicionar a Valor O valor a acrescentar à lista Exemplo: O código de bloco de ação acima cria uma variável de escopo global e, em seguida, cria uma nova estrutura de dados da lista, atribuindo seu valor de índice à variável global. O escopo é então alterado para que todas as instâncias do objeto "obj_Enemy_Parent" adicionem seu valor de identificação de instância única na lista.
