Se a estrutura de dados existir Com esta ação você pode verificar uma variável para ver se ela indexa ou não uma estrutura de dados. Primeiro você precisa escolher o tipo de estrutura de dados a ser verificada (uma Lista, um Mapa, uma Grade ou uma Pilha) e então você fornece a variável que contém a estrutura de dados (como retornado pelas ações Criar lista, Criar mapa, Criar grade e Criar pilha). A ação retornará verdadeira se a estrutura de dados existir, e falsa se não existir. Note que você pode marcar "não" na ação para verificar se uma variável não contém um índice da estrutura de dados. Observe que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo: Estas ações serão executadas agora se o "se" for avaliado como verdadeiro, enquanto qualquer ações abandonadas em outro lugar serão realizadas após o bloco "se".
