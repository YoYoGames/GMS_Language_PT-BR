Criar rede Esta ação criará uma nova estrutura de dados da grade e retornará o valor do índice para que você possa posteriormente acessar a grade através das outras ações de Estrutura de Dados. O índice da grade será retornado para a Variável Alvo que você fornecer, que pode ter sido criado anteriormente usando a opção Atribuir Variável ou Declarar Temp, ou você pode marcar a caixa de seleção "Temp" para nomear e criar uma variável local temporária para armazenar o valor até o final do roteiro ou evento. Note que ao contrário de outras estruturas de dados, as grades não são criadas "vazias", e cada célula de uma grade recém-criada terá o valor 0. Cabe a você alterar os dados da célula ou limpar você mesmo a grade para outro valor usando a ação Clear Grid. Observe que você pode criar grelhas DS adicionais clicando no ícone mais ao lado da ação, e selecionando outra variável para segurar a identificação da grade e dando novamente os valores de largura e altura. IMPORTANTE! A criação de qualquer estrutura de dados utiliza a memória na plataforma alvo, e como tal todas as estruturas de dados devem ser livres quando não forem mais necessárias, usando a ação Free Data Structure (Estrutura de dados livre), caso contrário você obtém uma vazamento de memória que pode prejudicar o desempenho de seus jogos ou até mesmo causar a sua queda. Isto é particularmente relevante ao usar variáveis locais temporárias para armazenar índices de estrutura de dados, já que estas variáveis são removidas no final do código ou evento, mas isso não significa que a estrutura de dados seja removida também! A estrutura de dados ainda existirá, só que você não terá como referenciá-la, então ou use uma variável de instância e libere a estrutura em um momento posterior, ou libere a estrutura antes do final de o evento ou função, se seu índice estiver armazenado em uma variável temporária.
