Push Onto Stack Com esta ação, você pode empurrar um valor para uma estrutura de dados em pilha. Você fornece a variável que mantém o índice da pilha (como retornado pela ação Criar pilha) e então dá o valor para empurrar para o topo. Com dados empilhados sempre são adicionados valores no "topo" da pilha, e nunca no final ou no meio (se você precisar desta funcionalidade, então use uma estrutura de lista de dados). Sintaxe de ação: Argumentos: Argumento Descrição Pilha O índice (armazenado em uma variável) da pilha a ser utilizada Valor O valor para empurrar para a pilha Exemplo: O código de bloco de ação acima cria uma variável de instância e uma nova estrutura de pilha de dados. O índice da pilha é armazenado na nova variável, e então é feito um loop que cria 10 instâncias e empurra seus valores de identificação únicos para a pilha.
