ÍndiceDrag And Drop™ Drag e Drop™(também chamado DnD™), é uma linguagem de script visual que se baseia no Idioma GameMaker(GML). Consiste no uso de blocos de ação para construir a lógica do jogo de forma visual e intuitiva, e é uma ferramenta ideal para pessoas que estão aprendendo programação, ou para pessoas que têm uma inclinação mais visual.
