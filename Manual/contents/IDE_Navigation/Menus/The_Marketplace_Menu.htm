O Menu do Mercado

Além de criar seus próprios ativos e código para seus projetos, você também tem acesso ao GameMaker Studio 2 Marketplace onde você pode baixar tanto os ativos gratuitos como os pagos para serem adicionados a seus projetos. Você pode saber mais sobre o Marketplace em a seção sobre o GameMaker Mercadoe as opções deste menu são dadas abaixo:

  • Ir para o Marketplace - Abra a janela do Marketplace onde você pode procurar por ativos.
  • Minha biblioteca - Veja sua biblioteca de conteúdo atual do Marketplace.
  • Meu histórico de compras - Esta opção abrirá um navegador e o levará para a página da conta do usuário logado para que você possa ver o histórico de compras para aquela conta.
  • Update Existing Package Package - Esta opção abrirá o Update Package Wizard para que você possa atualizar um pacote Marketplace existente que você já tenha criado anteriormente.
  • Create Package - Com esta opção você pode abrir o Create Package Wizard para criar um novo pacote de ativos ou tutorial para o Marketplace.
