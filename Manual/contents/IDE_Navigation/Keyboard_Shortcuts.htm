Atalhos de teclado

Há um grande número de atalhos de teclado disponíveis para o GameMaker Studio 2, e embora a maioria deles esteja marcada na IDE ao lado dos itens de menu que eles também relacionam, nem todos eles são ou são óbvios para o usuário no início. Aqui você pode encontrar uma lista completa de todos os atalhos do teclado para um guia de referência fácil. Em geral, os atalhos se enquadram nas três categorias a seguir:

  • Global - Os atalhos globais são aqueles que funcionam não importa onde você esteja no IDE GameMaker Studio 2, a menos que seu foco esteja em uma janela ou gadget específico que o sobreponha.
  • Janela - Os atalhos de janela são aqueles que são específicos para uma única janela ou aba, a menos que um atalho de gadget o sobreponha.
  • Gadget - Os atalhos para gadgets são muito específicos e só funcionam quando se usa um único gadget (o seletor de cores, por exemplo).

Abaixo está a lista completa de atalhos, quebrados em seções baseadas em editores:
