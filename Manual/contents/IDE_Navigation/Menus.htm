Menus

O topo da IDE GameMaker Studio é ocupado pela Barra de Menu. Aqui você encontra uma série de menus suspensos que podem ser usados para navegar a IDE, realize tarefas ou abra janelas adicionais com ferramentas para ajudá-lo a criar seus jogos. Note que listamos aqui os menus padrão que estão disponíveis, mas alguns dos editores de recursos adicionarão menus adicionais dependendo do contexto. Estes menus adicionais Os menus são explicados nas páginas relevantes para o editor que os criou.

Os menus padrão disponíveis são:

  • O Menu Arquivo
  • O menu de edição
  • O Menu Construir
  • O Menu Windows
  • O Menu de Ferramentas
  • O Menu do Mercado
  • O Menu de Layouts
  • O Menu Ajuda
