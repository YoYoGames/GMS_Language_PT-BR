md5_file

Na criptografia, MD5 (Message-Digest algoritmo 5) é uma função de hash criptográfico amplamente utilizada com um valor de hash de 128 bits e tem sido empregada em uma grande variedade de aplicações de segurança. Também é comumente usado para verificar a integridade de arquivos e strings. Esta função pegará o arquivo dado e gerará um MD5 único para esse arquivo que pode então ser armazenado para uso posterior.

NOTA: O MD5 não é completamente seguro e pode ser quebrado. Veja esta página para mais informações.

Sintaxe:

md5_file(filename)

Argumento Descrição
nome do arquivo O arquivo para gerar o hash MD5.

Devoluções:

Cordão

Exemplo:

hash = md5_file(directório_de_trabalho + "game_data.ini")

O código acima irá gerar um hash MD5 para o arquivo especificado e armazenar o valor retornado na variável "hash".
