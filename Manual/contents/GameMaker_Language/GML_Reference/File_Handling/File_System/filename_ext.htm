nome do arquivo_ext

Esta função retorna a parte de extensão do nome do arquivo indicado, incluindo o ponto principal.

Sintaxe:

filename_ext(fname);

Argumento Descrição
fname O arquivo a ser utilizado.

Devoluções:

Cordão

Exemplo:

ext = filename_ext(file_find_first("*.*", 0)));

O código acima recebe a extensão (como uma string) do arquivo especificado.
