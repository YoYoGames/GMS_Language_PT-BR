nome_nome_do_arquivo

Usando esta função retorna a parte do nome do arquivo indicado, com a extensão mas sem o caminho

Sintaxe:

filename_name(fname);

Argumento Descrição
fname O arquivo a ser utilizado.

Devoluções:

Cordão

Exemplo:

nome = nome_do_arquivo(file_find_first("C:/Games/*.doc", 0)));

O código acima recebe o nome (como uma string) do primeiro arquivo do tipo "doc" encontrado no diretório especificado.
