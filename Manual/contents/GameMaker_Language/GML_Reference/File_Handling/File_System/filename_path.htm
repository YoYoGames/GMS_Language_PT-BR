filename_path

Usando esta função, retorna a parte do caminho do arquivo indicado, incluindo a contrabarra final.

Sintaxe:

filename_path(fname);

Argumento Descrição
fname O nome do arquivo e o caminho a ser utilizado.

Devoluções:

Cordão

Exemplo:

caminho = nome_do_caminho_de_arquivo(directório_de_trabalho + "Test.ini");

O código acima recebe o caminho (como um fio) do arquivo especificado.
