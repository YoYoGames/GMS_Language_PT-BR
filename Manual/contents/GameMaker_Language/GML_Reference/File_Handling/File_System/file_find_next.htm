file_find_next

Esta função retorna o nome do próximo arquivo que satisfaz a máscara e os atributos (definidos por file_find_first()). Se não existir tal arquivo, a cadeia vazia é devolvida.

ATENÇÃO! Esta função pode não funcionar como você espera devido ao GameMaker Studio 2 ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

NOTA: Esta função funciona em todas as plataformas alvo C++(Windows, Mac, iOS, Android e WinPhone), MAS as bandeiras de filtro só funcionam no Windows.

Sintaxe:

file_find_next();

Devoluções:

Cordão

Exemplo:

file = file_find_next();

Isto retornará o próximo arquivo que corresponda aos atributos previamente dados.
