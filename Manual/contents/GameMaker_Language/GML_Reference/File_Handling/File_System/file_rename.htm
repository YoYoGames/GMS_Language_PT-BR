file_rename

Esta função irá renomear o arquivo especificado com o nome especificado. A função retornará verdadeiro se o arquivo tiver sido renomeado com sucesso, ou falso em qualquer outra circunstância.

ATENÇÃO! Esta função pode não funcionar como você espera devido ao GameMaker Studio 2 ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

file_rename(nome_do_arquivo, nome_novo);

Argumento Descrição
antigo nome O nome do arquivo a ser mudado.
novo nome O novo nome para dar o arquivo.

Devoluções:

Booleano

Exemplo:

se file_exists("level1.txt")
{
file_rename("level1.txt", "level.txt");
}

Isso verificaria a existência de um arquivo e, se ele existir, será renomeado.
