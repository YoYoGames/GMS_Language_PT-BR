file_find_close

Deve ser chamado após a manipulação de arquivos abertos usando file_find_first() e file_find_next() funções para liberar a memória. O arquivo encontra as funções abertas alças no arquivo e estes ocupam uma quantidade mínima de memória, que (ao longo do tempo) é montada. Portanto, você deve sempre chamar esta função depois de encontrar os arquivos necessários para "fechar" estas alças.

Sintaxe:

file_find_close();

Devoluções

N/A

Exemplo:

file_find_close();

Isto fechará e liberará a memória associada às alças da função de busca de arquivos.
