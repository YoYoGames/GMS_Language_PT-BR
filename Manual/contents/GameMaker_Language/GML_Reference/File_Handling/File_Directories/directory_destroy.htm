diretório_destroy

Esta função removerá um diretório com o nome dado na área de salvamento.

NOTA: Esta função não funcionará em HTML5, pois não é possível criar ou remover diretórios no armazenamento local do navegador.

ATENÇÃO! Esta função pode não funcionar como você espera devido ao GameMaker Studio 2 ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

directory_destroy(dname)

Argumento Descrição
dname O nome do diretório a ser removido.

Devoluções:

N/A

Exemplo:

se o diretório_existe("DLC")
{
directory_destroy("DLC");
}

Isto verificará se o diretório especificado existe na pasta de dados local e, se existir, ele será removido.
