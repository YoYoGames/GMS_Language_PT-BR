Diretórios de arquivos

Ao trabalhar com arquivos, haverá momentos em que você precisará consultar o diretório no qual os arquivos são armazenados e o GameMaker Studio 2 tem algumas funções e variáveis especiais relacionadas aos diferentes diretórios possíveis que você possa precisar:

  • diretório_existe
  • diretório_criar
  • diretório_destroy
  • temp_directório
  • diretorio_de_trabalho
  • program_directory

NOTA: Você não pode criar ou destruir diretórios em HTML5.
