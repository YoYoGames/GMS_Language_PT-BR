file_bin_rewrite

Esta função toma o cabo do nome do arquivo como retornado pela função file_bin_open() e depois reescreve o arquivo, limpando-o de todos os dados anteriores para começar a escrever desde o início do arquivo.

NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.

Sintaxe:

file_bin_rewrite(binfile);

Argumento Descrição
binfile A identificação do arquivo a ser reescrito.

Devoluções:

N/A

Exemplo:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

Isto abriria um arquivo do mesmo diretório do jogo, e atribuiria seu índice à variável "arquivo". Em seguida, ele reescreveria o arquivo (limpando-o de dados).
