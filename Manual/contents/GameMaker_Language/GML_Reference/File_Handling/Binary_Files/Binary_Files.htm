Arquivos binários

Em situações raras você pode precisar ler dados de arquivos binários. Essas funções podem ser usadas para esse fim, embora, em geral, recomendamos que você use as funções de buffer dedicadas para criar, salvar e carregar dados binários, em vez destas funções.

NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.

Para isso, existem as seguintes rotinas de baixo nível:

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
