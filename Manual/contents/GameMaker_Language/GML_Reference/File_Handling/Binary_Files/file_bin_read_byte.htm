file_bin_read_byte

Esta função retornará um byte de dados da posição atual dentro do arquivo com o ID do arquivo dado. Você fornecerá o valor do ID do arquivo, conforme retornado pela função file_bin_open().

NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.

Sintaxe:

file_bin_read_byte(binfile);

Argumento Descrição
binfile A identificação do arquivo para obter os dados.

Devoluções:

Real

Exemplo:

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Isto abriria um arquivo do mesmo diretório do jogo, e obteria um byte de dados dele antes de fechá-lo novamente.
