Desenho

Esta seção contém todas as funções relacionadas ao desenho dentro da sala de jogo, bem como para controlar como as coisas serão desenhadas (mistura, alfa, abate, etc...). Há um grande número de diferentes funções para desenhar, e elas são divididas nas seguintes categorias para facilitar a busca do que você precisa:

  • Cor e Alfa
  • Controle de GPU
  • Mipmapping
  • Formulários básicos
  • Sprites e Ladrilhos
  • Texto
  • Primitivos e formatos de vértices
  • Superfícies
  • Iluminação
  • Partículas
  • Texturas
  • Shaders

Há duas funções especiais disponíveis também, uma que é usada para habilitar ou desabilitar o evento de sorteio para seu projeto, e a outra para a descarga de toda a tubulação de sorteio:

  • draw_enable_drawevent
  • draw_flush
