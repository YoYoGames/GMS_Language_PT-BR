part_system_create

Esta função é usada para criar um novo sistema de partículas e retornará um número índice único que deve ser armazenado e usado em todas as outras funções relacionadas a esse sistema. Ao sistema será atribuída uma camada gerenciada e será definida para ter uma profundidade de 0. camadas gerenciadas não são acessíveis ao usuário e usadas somente para gerenciamento interno quando a profundidade é usada em vez de camadas. Normalmente, o usuário usaria a função part_system_create_layer() em vez de deste aqui.

Sintaxe:

part_system_create();

Devoluções:

Real

Exemplo:

global.Sname = part_system_create();

Isto criará um novo sistema de partículas e armazenará o índice na variável global "Sname".
