part_type_colour_rgb

Com esta função você pode definir a mistura de cores vermelha, verde e azul para todas as partículas criadas do tipo dado. Você fornece um valor mínimo e um valor máximo para cada um dos três componentes e as partículas criadas terão uma cor aleatória baseada na faixa de parâmetros dada. Todos os valores devem estar entre 0 e 255.

Sintaxe:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argumento Descrição
ind O índice do tipo de partícula a ser mudado.
rmin O mínimo que pode ser o componente vermelho da cor final.
rmax O máximo que o componente vermelho da cor final pode ser.
gmin O mínimo que pode ser o componente verde da cor final.
gmax O máximo que o componente verde da cor final pode ser.
bmin O mínimo que pode ser o componente azul da cor final.
bmax O máximo que o componente azul da cor final pode ser.

Devoluções:

N/A

Exemplo:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0, 0, 0);

O código acima define cada partícula emitida do tipo de partícula indexada na variável global "Blood_Part" para ser apenas diferentes tonalidades de vermelho.
