escala_parte_tipo

Esta função ajustará a escala horizontal e vertical da partícula antes que qualquer outra alteração de tamanho seja feita. Esta função funciona na razão do tamanho do duende, portanto um valor de 1 (o valor padrão) é uma razão de 1:1, significando que 0,5 seria metade do valor da partícula. e 2 duplos.

Sintaxe:

part_type_scale(ind, xscale, yscale);

Argumento Descrição
ind O índice do tipo de partícula a ser mudado.
xscale A escala horizontal da partícula de base.
yscale A escalada vertical da partícula base.

Devoluções:

N/A

Exemplo:

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

O código acima estabelecerá vários valores de partículas, incluindo a escala que fixa a escala x em oito vezes o tamanho e a escala y em metade do tamanho.
