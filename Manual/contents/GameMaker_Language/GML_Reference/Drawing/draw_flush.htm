draw_flush

Com esta função, você pode lavar todo o duto de escoamento. Esta é uma função somente de depuração e, em geral, não deve ser usada a menos que seja indicada por um membro do YoYo Games Support. funcionários, pois o uso indiscriminado causará sérios problemas de desempenho com seu jogo.

Sintaxe:

draw_flush();

Devoluções:

N/A

Exemplo:

draw_flush();

O código acima descarrega o duto de escoamento.
