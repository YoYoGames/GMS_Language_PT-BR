gpu_get_texrepeat_ext

Com esta função você pode verificar se a repetição de textura está habilitada (retorna verdadeiro) ou não (retorna falso) para uma dada textura de shader sampler.

Sintaxe:

gpu_get_texrepeat_ext(sampler_id);

Argumento Descrição
sampler_id A identificação do amostrador a partir do shader.

Devoluções:

Booleano

Exemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if !gpu_get_texrepeat_ext(s_tex)
{
gpu_set_texrepeat_ext(s_tex, true);
}

O código acima verifica se a textura está filtrando para uma identificação específica do amostrador (armazenada em uma variável local) e a liga se não estiver.
