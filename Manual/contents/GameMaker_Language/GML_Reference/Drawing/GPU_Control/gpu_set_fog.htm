gpu_set_fog

Esta função pode ser usada para ativar ou desativar o desenho com neblina. O nevoeiro pode ser usado em jogos 3D para fazer com que as instâncias à distância pareçam embaçadas ou mesmo desaparecer, o que ajuda a criar atmosfera, além de mascarar o fato de que você não está desenhando instâncias que estão muito longe. Você define se está habilitado(verdadeiro) ou desabilitado(falso), a cor que o nevoeiro deve usar para misturar, bem como as distâncias de início e fim de tração. Observe que o início da névoa indica o raio relativo ao A câmera de visão onde começa o nevoeiro, e a extremidade do nevoeiro indica em que raio (também relativo) as instâncias serão completamente invisíveis.

A função pode aceitar quatro argumentos individuais (dados acima) ou uma matriz com a seguinte estrutura (o código de exemplo abaixo mostra este método):

  • [0] = alternância ativada (um booleano, verdadeiro ou falso), padrão falso
  • [1] = cor (real), padrão c_preto
  • [2] = distância inicial (real), padrão 0
  • [3] = distância final (real), padrão 1

Sintaxe:

gpu_set_fog(habilitação, cor, início, fim);

Argumento Descrição
habilitar Ativar ou desativar a névoa
cor A cor do nevoeiro
início A distância para começar a aplicar névoa de
fim A distância para acabar com a neblina

Devoluções:

N/A

Exemplo:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

O código acima obtém as configurações atuais de nevoeiro e então define o elemento de cor da matriz para c_red antes de definir o nevoeiro novamente usando a matriz alterada.
