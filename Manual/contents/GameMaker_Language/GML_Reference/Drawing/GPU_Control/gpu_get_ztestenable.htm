gpu_get_ztestenable

Esta função pode ser usada para recuperar se o teste z está ativado (a função retorna verdadeiro) ou não (a função retorna falso). O valor padrão é que o teste z está desabilitado, portanto a função retorna falso.

Sintaxe:

gpu_get_ztestenable();

Devoluções:

Booleano

Exemplo:

if gpu_get_ztestenable() == falso
{
gpu_set_ztestenable(true);
}

O código acima verifica se o teste z está ativado ou não e se está desativado, então ele é ativado novamente.
