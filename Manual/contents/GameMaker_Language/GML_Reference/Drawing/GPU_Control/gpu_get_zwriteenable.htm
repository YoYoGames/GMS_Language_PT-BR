gpu_get_zwriteenable

Esta função pode ser usada para recuperar se a escrita em z está habilitada (a função retorna verdadeiro) ou não (a função retorna falso). O valor padrão é que a escrita em z está ativada, portanto a função retornará verdadeira.

Sintaxe:

gpu_get_zwriteenable();

Devoluções:

Booleano

Exemplo:

if gpu_get_zwriteenable() == falso
{
gpu_set_zwriteenable(true);
}

O código acima verifica se a escrita em z está habilitada ou não e se está desabilitada, então ela é habilitada novamente.
