gpu_get_texrepeat

Com esta função você pode verificar se a repetição de textura está habilitada (retorna verdadeiro) ou não (retorna falso). Para mais informações sobre a repetição de textura, veja a função gpu_set_texrepeat().

Sintaxe:

gpu_get_texrepeat();

Devoluções:

Booleano

Exemplo:

se gpu_get_texrepeat()
{
gpu_set_texrepeat(false);
}
mais
{
gpu_set_texrepeat(true);
}

O código acima verifica se a repetição da textura está ligada ou desligada e a liga em conformidade.
