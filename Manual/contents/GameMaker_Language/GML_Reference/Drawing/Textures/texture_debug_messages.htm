texturas_debug_messages

Esta função pode ser usada para ativar ou desativar as mensagens de depuração de textura. Quando ativada (definida como verdadeira), informações adicionais sobre o uso da página de textura serão enviadas para a janela de saída. Definido como falso para desabilitar esta saída novamente.

Sintaxe:

texture_debug_messages(enable);

Argumento Descrição
habilitar Habilitar ou desabilitar as mensagens de depuração de textura

Devoluções:

N/A

Exemplo:

if global.debug
{
text_debug_messages(true);
}

O código acima verificará o valor de uma variável global e se ela estiver definida como verdadeira, as mensagens de depuração de textura serão ativadas.
