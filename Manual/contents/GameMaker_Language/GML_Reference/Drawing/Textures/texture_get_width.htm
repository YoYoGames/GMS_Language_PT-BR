textura_get_width

Retorna a largura da textura com o id dado, que é sempre um valor dentro da faixa 0 - 1. Isto pode então ser usado ao mapear texturas para modelos ou primitivas.

Sintaxe:

text_get_width(tex);

Argumento Descrição
tex O ponteiro dos ativos da página de textura a usar

Devoluções:

Real

Exemplo:

text_w = text_get_width(sprite_get_texture(spr_Model_tex, 0));

O código acima obterá a largura da textura retirada de um bem patrimonial sprite.
