Texturas

As seguintes funções estão todas relacionadas a "texturas", ou seja: imagens que são armazenadas em VRAM para serem usadas como sprites ou outras coisas dentro de seu jogo. O primeiro conjunto de funções disponíveis são usadas para recuperar informações sobre imagens específicas e sua posição ou tamanho na página de textura, bem como para definir certas características de textura:

  • textura_get_width
  • Textura_esquadre_altura
  • textura_get_texel_width
  • textura_get_texel_height
  • textura_get_uvs
  • textura_set_set_stage
  • escala_global_textura

OGameMaker Studio 2 também tem uma série de funções relacionadas ao gerenciamento da VRAM associadas a texturas individuais, bem como grupos de texturas inteiras:

  • draw_texture_flush
  • textura_prefetch
  • textura_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

Finalmente, temos as seguintes funções, que são principalmente para uso para depurar o projeto e garantir o uso eficiente da memória de textura:

  • texturas_debug_messages
  • texture_is_ready
  • texturas_grupo_get_textures
  • grupo de textura_get_sprites
  • grupo de textura_get_fonts
  • grupos de texturas_get_tilesets
