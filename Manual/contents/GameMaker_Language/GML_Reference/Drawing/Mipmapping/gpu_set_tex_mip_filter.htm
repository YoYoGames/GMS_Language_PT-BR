gpu_set_tex_mip_filter

Com esta função você pode ajustar o modo atual de filtro mip para um dos três tipos suportados. Você dá a constante que se refere ao modo de filtragem mip que você precisa, onde você pode escolher entre filtragem pontual (configuração padrão, ou seja sem filtragem), filtragem linear (também conhecida como filtragem trilinear) ou filtragem aniostrópica (ver a tabela de constantes abaixo).

Constante Descrição
tf_point Isto desativará a filtragem entre os níveis do mipmap, o que pode causar transições de textura visíveis, mas dá o melhor desempenho.
tf_linear Isto permitirá uma filtragem linear entre os níveis do mipmap (também conhecida como filtragem trilinear), que suaviza as transições de textura, mas dará um pequeno golpe no desempenho.
tf_anisotropic Isto permitirá uma filtragem anisotrópica entre os níveis do mipmap, o que melhora muito a qualidade da transição de textura e pode reduzir o embaçamento visível com outros modos de filtragem, mas tem o maior impacto no desempenho.

Sintaxe:

gpu_get_tex_mip_filter(filtro);

Argumento Descrição
filtro O modo de filtro mip a ser usado (uma constante, padrão: tf_point).

Devoluções:

N/A

Exemplo:

if keyboard_check(ck_enter)
{
switch(gpu_get_tex_mip_filter())
{
caso tf_point: gpu_set_tex_mip_filter(tf_linear); pausa;
caso tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); pausa;
caso tf_anisotropic: gpu_set_tex_mip_filter(tf_point); pausa;
}
}

O código acima verifica o teclado e, se a tecla especificada for pressionada, ele obterá o filtro mip atual e alternará o valor para o próximo, pedalando pelos diferentes modos.
