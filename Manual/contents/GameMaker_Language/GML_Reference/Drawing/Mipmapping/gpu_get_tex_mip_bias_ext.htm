gpu_get_tex_mip_bias_ext

Com esta função você pode recuperar o valor de polarização do mipmap para um determinado amostrador de sombreamento. Você fornece o valor do índice para o amostrador de shader (como retornado pela função shader_get_sampler_index(), e a função retornará um valor de 0,0 para nenhum viés, ou um valor maior onde 1 é igual ao primeiro mipmap, 2 é igual ao segundo mipmap etc... Isto controla a taxa na qual o mapa mip é trocado e geralmente fará com que as texturas mais sombreadas se tornem mais confusas. quanto maior o valor e maior a "distância" sendo visualizada. Note que isto também pode retornar valores negativos, em cujo caso as texturas de sombreamento serão mais nítidas sobre uma distância maior quanto menor o valor.

Sintaxe:

gpu_get_tex_mip_mip_bias_ext(sampler_index);

Argumento Descrição
índice_de_amostragem O índice do amostrador de sombreamento para obter

Devoluções:

Real (padrão: 0)

Exemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set (shd_Glass);
if gpu_get_tex_mip_mip_bias_ext(_sampleIndex) != 0
{
gpu_set_tex_mip_mip_bias_ext(_sampleIndex, 0);
}
text_set_set_stage(_sampleIndex , _spriteTex);
shader_reset();

O código acima define o viés do filtro mip como 0 para o amostrador de textura de sombreamento dado, caso ainda não tenha sido definido como 0.
