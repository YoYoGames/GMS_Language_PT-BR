Mipmapping

Esta seção contém todas as funções relacionadas ao uso das funções de mipmapping. Antes de usar estas funções, você deve ter habilitado o mipmapping para o páginas de textura no Editor do Grupo de Textura e/ou habilitou o mipmapping para aqueles sprites que foram definidos para usar apenas uma única página de textura das Opções Gerais de Jogo. A imagem abaixo mostra a diferença que o mipmapping pode fazer ao renderizar seu projeto:

Uma vez ativado o mipmapping para o projeto ou uma página de textura, você pode então usar as seguintes funções para obter e definir valores que mudarão a forma como as coisas ficarão em seu projeto quando forem executadas.

As funções listadas abaixo podem ser usadas para obter os diferentes valores associados ao mipmapping no GameMaker Studio 2:

  • gpu_get_tex_mip_filter
  • gpu_get_tex_mip_filter_ext
  • gpu_get_tex_mip_bias
  • gpu_get_tex_mip_bias_ext
  • gpu_get_tex_min_mip
  • gpu_get_tex_min_mip_ext
  • gpu_get_tex_max_mip
  • gpu_get_tex_max_mip_ext
  • gpu_get_tex_max_aniso
  • gpu_get_tex_max_aniso_ext
  • gpu_get_tex_mip_enable
  • gpu_get_tex_mip_enable_ext

As funções listadas abaixo podem ser usadas para definir os diferentes valores associados ao mipmapping no GameMaker Studio 2:

  • gpu_set_tex_mip_filter
  • gpu_set_tex_mip_filter_ext
  • gpu_set_tex_mip_bias
  • gpu_set_tex_mip_bias_ext
  • gpu_set_tex_min_mip
  • gpu_set_tex_min_mip_ext
  • gpu_set_tex_max_mip
  • gpu_set_tex_max_mip_ext
  • gpu_set_tex_max_aniso
  • gpu_set_tex_max_aniso_ext
  • gpu_set_tex_mip_enable
  • gpu_set_tex_mip_enable_ext
