gpu_set_tex_mip_bias

Com esta função você pode definir o valor do viés do mipmap, onde um valor de 0 é igual a nenhum viés, 1 é igual ao primeiro mipmap, 2 é igual ao segundo mipmap etc... Isto controla a taxa na qual o mapa mip é trocado e geralmente fará com que as texturas tornadas mais borradas quanto maior o valor e maior a "distância" sendo visualizada. Note que esta função também pode assumir valores negativos, em cujo caso as texturas renderizadas serão mais nítidas sobre uma distância maior quanto menor for o valor.

Sintaxe:

gpu_set_tex_mip_bias(bias);

Argumento Descrição
viés O valor de viés do mipmap a usar (padrão: 0)

Devoluções:

N/A

Exemplo:

se gpu_get_tex_mip_bias() != 0
{
gpu_set_tex_mip_bias(0);
}

O código acima verificará o viés atual do mipmap e, se não for 0, será ajustado para 0.
