color_get_green

Esta função retorna a quantidade de verde utilizada para fazer a cor dada, sendo o valor entre 0 e 255, onde 0 não é verde e 255 é todo verde. A imagem a seguir ilustra isto:

Sintaxe:

colour_get_green(col);

Argumento Descrição
col A cor a ser verificada

Devoluções:

Int

Exemplo:

g_comp = color_get_green(c_teal);

O código acima obterá o componente verde da constante de cor "c_teal" e armazenará o valor na variável "g_comp".
