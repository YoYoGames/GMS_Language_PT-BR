merge_colour

Com esta função você pode pegar duas cores e depois fundi-las para fazer uma nova cor. A quantidade de cada uma das cores componentes pode ser definida alterando o argumento "quantidade", onde um valor de 0 retornará a primeira cor (col1), um valor de 1 devolverá a segunda cor (col2) e um valor no meio devolverá a mistura correspondente. Por exemplo, um valor de 0,5 misturará as duas cores igualmente. A imagem seguinte ilustra como isto funciona fundindo as cores vermelho e azul juntos:

Sintaxe:

merge_colour(col1, col2, quantidade);

Argumento Descrição
col1 A primeira cor a fundir
col2 A segunda cor a fundir
valor Quanto de cada cor deve ser fundido. Por exemplo, 0 devolverá col1, 1 devolverá col2, e 0,5 devolverá uma fusão de ambas as cores igualmente

Devoluções:

Int

Exemplo:

col = merge_colour(c_lime, c_orange, 0,5);

O código acima usa a função para criar uma cor fundindo cal e laranja 50/50 e depois armazena seu valor na variável "col" para uso posterior.
