draw_clear

Esta função pode ser usada para limpar a tela inteira (sem mistura alfa) para a cor dada, e é apenas para uso no evento de sorteio de uma instância (não será mostrada se usada em qualquer outro evento). Ela também pode ser útil para limpar superfícies quando elas são criadas recentemente.

Sintaxe:

draw_clear(col);

Argumento Descrição
col A cor com a qual a tela será limpa

Devoluções:

N/A

Exemplo:

draw_clear(c_blue);

Isto irá limpar a tela com a cor azul.
