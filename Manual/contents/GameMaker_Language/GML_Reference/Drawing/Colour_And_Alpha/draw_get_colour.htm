draw_get_colour

Esta função retorna a cor de desenho atual que é usada para desenhar formas, textos, primitivos e modelos 3D não texturizados. Isto pode ser definido com a função draw_set_colour() função.

Sintaxe:

draw_get_colour()

Devoluções:

Real

Exemplo:

if draw_get_colour() != c_black
{
draw_set_colour(c_black);
}

O código acima primeiro verifica a cor atual do desenho e depois, se não estiver definido para c_preto, ele o define para c_preto.
