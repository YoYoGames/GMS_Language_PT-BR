color_get_red

Esta função retorna a quantidade de vermelho utilizada para fazer a cor dada, sendo o valor entre 0 e 255, onde 0 não é vermelho e 255 é todo vermelho. A imagem a seguir ilustra isto:

Sintaxe:

color_get_red(col);

Argumento Descrição
col A cor a ser verificada

Devoluções:

Int

Exemplo:

r_comp = color_get_red(c_teal);

O código acima obterá o componente vermelho da constante de cor "c_teal" e armazenará o valor na variável "r_comp".
