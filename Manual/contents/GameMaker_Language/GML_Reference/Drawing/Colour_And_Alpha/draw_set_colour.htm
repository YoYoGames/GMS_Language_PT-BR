draw_set_colour

Com esta função, você pode definir a cor do sorteio base para o jogo. Este valor afetará todos os outros desenhos onde for apropriado, incluindo fontes, formas, primitivas e 3D. Se algum desses recursos for desenhado com seu próprio valor de cor alterado, este valor será ignorado.

Sintaxe:

draw_set_colour(col);

Argumento Descrição
col A cor a ser definida para o desenho.

Devoluções:

N/A

Exemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "NÍVEL 1"); draw_text(x, y, "NÍVEL 1");

O código acima desenhará algum texto na posição especificada com um efeito de sombra criado pelo desenho alfa e cor modificados.
