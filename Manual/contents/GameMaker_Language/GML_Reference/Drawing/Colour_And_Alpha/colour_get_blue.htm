color_get_blue

Esta função retorna a quantidade de azul utilizada para fazer a cor dada, sendo o valor entre 0 e 255, onde 0 não é azul e 255 é todo azul. A imagem seguinte ilustra isto:

Sintaxe:

color_get_blue(col);

Argumento Descrição
col A cor a ser verificada

Devoluções:

Int

Exemplo:

b_comp = color_get_blue(c_teal);

O código acima obterá o componente azul da constante de cor "c_teal" e armazenará o valor na variável "b_comp".
