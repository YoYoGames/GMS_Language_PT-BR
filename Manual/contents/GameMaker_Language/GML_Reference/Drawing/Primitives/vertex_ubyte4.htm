vértice_ubyte4

Esta função adicionará quatro valores de bytes não assinados (0 - 255) aos dados do vértice. O vértice deve ter sido formatado corretamente para aceitar isto usando a função vértice_formato_add_custom() função.

Sintaxe:

vertex_ubyte4(buffer, byte, byte, byte, byte, byte);

Argumento Descrição
tampão O buffer para escrever as informações.
byte O primeiro valor de entrada.
byte O segundo valor de entrada.
byte O terceiro valor de entrada.
byte O quarto valor de entrada.

Devoluções:

N/A

Exemplo:

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

O código acima acrescentará quatro valores aos dados do vértice que está sendo definido.
