vertex_get_buffer_size

Com esta função, você pode obter o tamanho do buffer de vértice dado em bytes.

Sintaxe:

vertex_get_buffer_size_size(buffer);

Argumento Descrição
tampão O amortecedor para obter o tamanho de.

Devoluções:

Real

Exemplo:

bufferBytes = vertex_get_buffer_size_size(buff);

O código acima obterá o número de bytes utilizados pelo vértice buffer dado e armazenará o valor na variável "bufferBytes".
