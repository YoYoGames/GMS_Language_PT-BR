vértice_color

Se seu formato de vértice definido tomar um valor de cor, você pode usar esta função para adicionar esses dados ao vértice que está sendo definido para o vértice primitivo atual. A função precisa de um buffer para armazenar os dados e tomará ou uma constante de cor, ou um valor hexadecimal (usando o formato padrão GameMaker Studio 2 do BGR, por exemplo: $FF0000 para azul) assim como um valor alfa de 0 (transparente) a 1 (totalmente opaco).

Sintaxe:

vertex_colour(tampão, cor, alfa);

Argumento Descrição
tampão O buffer para escrever as informações.
cor A cor para este vértice (pode ser um valor constante ou hexadecimal).
alfa O valor alfa para o vértice (de 0 a 1).

Devoluções:

N/A

Exemplo:

vértice_colour(b, c_white, 1);

O código acima definirá o vértice a ser definido como branco com um valor alfa de 1.
