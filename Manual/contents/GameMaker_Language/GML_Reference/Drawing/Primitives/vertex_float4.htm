vértice_float4

Esta função adicionará quatro valores de ponto flutuante aos dados do vértice. O vértice deve ter sido formatado corretamente para aceitar isto usando a função vértice_formato_add_custom() função.

Sintaxe:

vertex_float4(tampão, bóia, bóia, bóia, bóia, bóia);

Argumento Descrição
tampão O buffer para escrever as informações.
flutuador O primeiro valor de entrada.
flutuador O segundo valor de entrada.
flutuador O terceiro valor de entrada.
flutuador O quarto valor de entrada.

Devoluções:

N/A

Exemplo:

vertex_float4(buff, 0,05, 0,01, largura_de_quarto / x, altura_de_quarto / y);

O código acima adicionará quatro valores de ponto flutuante aos dados de vértices que estão sendo definidos.
