vértice_formato_add_posição_3d

Diga ao GameMaker Studio 2 para aceitar dados de posicionamento 3D (x, y e z) como parte do novo formato de vértice que está sendo criado.

Sintaxe:

vertex_format_add_position_3d();

Devoluções:

N/A

Exemplo:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position_3d();
my_format = vértice_format_end();

O código acima criará um novo formato de vértice com apenas valores de cor e posição e então armazenará o id de formato na variável "my_format".
