vértice_normal

Esta função adicionará dados normais de superfície ao vértice que está sendo definido atualmente para os primitivos personalizados. Você fornece o buffer para escrever os dados, bem como as partes componentes x, y e z do normal.

Sintaxe:

vertex_normal(buffer, nx, ny, nz);

Argumento Descrição
tampão O buffer para escrever as informações.
nx O componente x da superfície normal.
ny O componente y da superfície normal.
nz O componente z da superfície normal.

Devoluções:

N/A

Exemplo:

vértice_normal(buff, 0, 1, 1);

O código acima definirá a superfície normal do vértice que está sendo definido.
