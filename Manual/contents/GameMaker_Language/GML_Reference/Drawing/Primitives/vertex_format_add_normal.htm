vértice_formato_add_normal

Diga ao GameMaker Studio 2 para aceitar dados normais de superfície (nx, ny e nz) como parte do novo formato de vértice que está sendo criado.

Sintaxe:

vértice_format_add_normal();

Devoluções:

N/A

Exemplo:

vertex_format_begin();
vertex_format_add_textcoord();
vértice_format_add_normal();
my_format = vértice_format_end();

O código acima criará um novo formato de vértice com textura justa e valores normais de superfície e então armazenará o id de formato na variável "my_format".
