draw_vertex

Esta função define a posição de um vértice para um primitivo. O aspecto final do primitivo dependerá do tipo primitivo escolhido para desenhar e da ordem com a qual você adiciona os vértices a ele. Veja draw_primitive_begin() para mais informações. Para terminar e desenhar o primitivo você deve chamar draw_primitive_end().

Sintaxe:

draw_vertex(x, y)

Argumento Descrição
x A coordenada x do vértice.
y A coordenada y do vértice.

Devoluções:

N/A

Exemplo:

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

O código acima desenhará um simples triângulo primitivo.
