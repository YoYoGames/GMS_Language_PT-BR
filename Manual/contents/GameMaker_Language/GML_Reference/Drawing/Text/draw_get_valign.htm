draw_get_valign

Esta função é utilizada para obter o ajuste do alinhamento do texto ao longo do eixo vertical, e retornará uma das constantes listadas abaixo.

Sintaxe:

draw_get_valign();

Devoluções:

Constante

Constante Alinhamento
fa_top
fa_middle
fa_bottom

Exemplo:

if draw_get_valign() != fa_top
{
draw_set_valign(fa_top);
}
draw_text(100, 32, "Pontuação: " + string(pontuação));

O código acima verificará se o alinhamento do texto atual está alinhado por cima, e se não está, então será definido e então algum texto será desenhado.
