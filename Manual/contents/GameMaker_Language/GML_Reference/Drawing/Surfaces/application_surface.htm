aplicação_surface

Este escopo global, variável embutida pode ser usada para acessar a superfície de aplicação usando qualquer uma das funções da superfície. Esta superfície está permanentemente disponível e é onde a maior parte do desenho é feita pelo GameMaker Studio 2.

Sintaxe:

aplicação_surface;

Devoluções:

Apontador

Exemplo:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

O código acima redimensionará a superfície de aplicação para ter uma relação de 1:1 com a camada GUI.
