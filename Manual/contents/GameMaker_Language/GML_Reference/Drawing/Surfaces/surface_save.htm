superfície_save

Esta função salvará uma superfície para o disco usando o nome do arquivo dado. A superfície deve ser salva como um arquivo no formato *.png.

Sintaxe:

surface_save(surface_id, fname);

Argumento Descrição
surface_id A identificação da superfície a ser definida como alvo do desenho.
fname O nome do arquivo de imagem salvo.

Devoluções:

N/A

Exemplo:

se keyboard_check_pressed(ord("S")
{
surface_save(surf, "test.png");
}

O código acima verificará se o usuário pressiona a tecla "S" no teclado e se o faz salvará a superfície indexada na variável "surf" ao disco.
