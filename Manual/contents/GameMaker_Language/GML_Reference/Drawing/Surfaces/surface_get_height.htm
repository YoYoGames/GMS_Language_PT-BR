altura_da_superfície

Esta função simplesmente retorna a altura, em pixels, da superfície indexada. Deve-se notar que se você chamar isto para verificar a superfície da aplicação_ imediatamente após ter mudado seu tamanho usando tamanho_da_superfície() não devolverá o novo valor, pois a mudança precisa de um ou dois passos para ser totalmente processada. Depois de esperar uma etapa, deverá devolver o novo valor corretamente.

NOTA: Quando se trabalha com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido a serem armazenadas na memória de textura. Você deve SEMPRE verificar se uma superfície existe usando superfície_existe() antes de referenciá-los diretamente.

Sintaxe:

surface_get_height(surface_id);

Argumento Descrição
surface_id A identificação da superfície para obter a altura de.

Devoluções:

Real

Exemplo:

sh = surface_get_height(surf);

O código acima irá armazenar a altura da superfície indexada na variável "surf" na variável "sh".
