buffer_save

Com esta função você pode salvar o conteúdo de um buffer em um arquivo, pronto para ser lido de volta à memória usando o buffer_load() função.

NOTA: Em HTML5, o conteúdo do buffer será salvo como cordas codificadas base64 ao usar esta função.

Sintaxe:

buffer_save(tampão, nome do arquivo);

Argumento Descrição
tampão O índice do buffer a ser salvo.
nome do arquivo O nome do arquivo a ser salvo como.

Devoluções:

N/A

Exemplo:

buffer_save(buff, "Player_Save.sav");

Salva o conteúdo atual do buffer com o id armazenado na variável "buff" para um arquivo.
