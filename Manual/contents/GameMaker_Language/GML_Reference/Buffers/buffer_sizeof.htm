tamanho_do_tampão

Esta função retornará o tamanho (em bytes) de qualquer uma das constantes de dados dadas (listadas aqui).

Sintaxe:

buffer_sizeof(tipo);

Argumento Descrição
tipo O tipo de dados que devem ser verificados (ver a lista de constantes aqui).

Devoluções:

N/A

Exemplo:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

O código acima calcula primeiro o tamanho do buffer para criar multiplicando o tipo de dado não assinado de 8 bits por 12 (já que usaremos o buffer para armazenar 12 pedaços de dados), e depois usa este valor para definir um buffer fixo para o tamanho correto.
