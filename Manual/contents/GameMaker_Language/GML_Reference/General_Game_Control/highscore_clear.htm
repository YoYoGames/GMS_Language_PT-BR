highscore_value

Com esta função você pode limpar a lista interna de pontuação alta global, redefinindo todos os nomes para o valor padrão de "Desconhecido" e todas as pontuações para o valor padrão de 0.

Sintaxe:

highscore_clear();

Devoluções:

N/A

Exemplo:

highscore_clear();

Isto limparia a lista de pontuação alta.
