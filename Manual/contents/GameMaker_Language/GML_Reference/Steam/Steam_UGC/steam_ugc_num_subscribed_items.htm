steam_ugc_num_subscribed_items

Esta função pode ser usada para obter o número de itens que o usuário atual subscreveu.

Sintaxe:

steam_ugc_num_subscribed_items();

Devoluções:

Real

Exemplo:

numSub = steam_ugc_num_subscribed_items();

O código acima armazenará o número de itens subscritos em uma variável.
