steam_user_installed_dlc

Se seu jogo tem DLC criado para ele, você pode usar esta função para verificar e ver se o usuário o instalou antes de acessar qualquer arquivo associado a ele. A função retorna verdadeiro se o jogador tiver o conteúdo instalado, e falso se o usuário não tiver, mas note que o usuário também deve ser dono do DLC, portanto você deve usar a função adicional de steam_user_owns_dlc() para verificar se ela é de propriedade também antes de utilizá-la. Note que o vapor As identificações podem ser em grande número e, portanto, você pode precisar lançar seu valor de identificação como um int64() antes de fornecê-lo para a função.

Sintaxe:

steam_user_installed_dlc(dlc_id);

Argumento Descrição
dlc_id O identificador único para o DLC a ser verificado (um número inteiro).

Devoluções:

Booleano

Exemplo:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
if steam_user_installed_dlc(10354)
{
global.Level_max = 200;
}
}

O código acima verificará se o usuário comprou, e instalou, o DLC com o id 10354, e se assim for, definirá uma variável global para um valor diferente.
