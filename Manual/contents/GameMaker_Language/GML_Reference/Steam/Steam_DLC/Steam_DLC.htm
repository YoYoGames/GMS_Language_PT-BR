DLC

Steam suporta tanto o conteúdo gratuito quanto o conteúdo para download pago (DLC), e no cliente Steam, um jogo com conteúdo para download aparece como uma única aplicação na lista de jogos do usuário com o conteúdo para download visualizado através do diálogo de propriedades do jogo. Uma vez que o conteúdo que pode ser baixado é tratado como parte integrante do jogo e Steam atualizará automaticamente o conteúdo quando um patch estiver disponível e instalará o conteúdo quando o usuário instalar o jogo.

Como tudo isso é tratado pelos servidores Steam e a configuração de qualquer DLC é feita através do painel de controle Steamworks, há apenas algumas funções necessárias no GameMaker Studio 2 para verificar se este conteúdo:

  • steam_user_owns_dlc
  • steam_user_installed_dlc
