General Steam API

O seguinte conjunto de funções serve para verificar a disponibilidade de certos aspectos da API do cliente ou servidor Steam e, como tal, retorna verdadeiro ou falso, dependendo de fatores como o servidor estar ocupado ou o usuário estar logado, etc... Isto significa que estas funções devem ser usadas antes de qualquer outra chamada de função Steam API para garantir que a configuração cliente/servidor esteja correta e se comunique com seu jogo:

  • vapor_inicializado
  • vapor_estats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • linguagem_de_jogo_corrente_de_vapor
  • vapor_disponível_línguas
