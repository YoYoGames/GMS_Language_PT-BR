vapor_disponível_línguas

Esta função pode ser usada para recuperar uma lista de todos os idiomas disponíveis para o Steam. O valor retornado é simplesmente uma lista de idiomas separada por vírgulas.

Sintaxe:

vapor_disponível_línguas();

Devoluções:

Cordão

Exemplo:

linguagem = vapor_disponível_línguas();

O acima descrito obtém os idiomas disponíveis para o Steam como um fio e o armazena em uma variável.
