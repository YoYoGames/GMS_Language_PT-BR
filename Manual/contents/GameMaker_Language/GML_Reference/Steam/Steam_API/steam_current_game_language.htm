linguagem_de_jogo_corrente_de_vapor

Esta função é utilizada para recuperar a linguagem atual que o Steam está usando (como um fio), por exemplo, "inglês".

Sintaxe:

linguagem_de_jogo_vapor_corrente();

Devoluções:

Cordão

Exemplo:

linguagem = vapor_corrente_língua_do_jogo();

O código acima obtém o idioma utilizado para o jogo atual.
