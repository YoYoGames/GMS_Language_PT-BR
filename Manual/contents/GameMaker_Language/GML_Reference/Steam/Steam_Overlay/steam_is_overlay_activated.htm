vapor_é_activado_sobre_camada

Esta função pode ser usada para descobrir se o usuário tem o Steam Overlay ativo ou não. Se a sobreposição estiver ativa e visível para o usuário a função retornará verdadeira, e se não estiver, então retornará falsa. Um exemplo do que Esta função pode ser usada para sondar o Steam API para a sobreposição, para que você possa pausar seu jogo enquanto a sobreposição está sendo mostrada.

Sintaxe:

vapor_é_a_sobrepelado_ativado();

Devoluções:

Booleano

Exemplo:

se vapor_é_sobrepelado_ativado()
{
global.Pausa = verdadeiro;
}

O código acima verificará se a sobreposição de vapor está ativa e se a variável global "Pausa" será definida como verdadeira.
