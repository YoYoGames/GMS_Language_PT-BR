steam_file_exists

Com esta função você pode verificar se um arquivo do Steam Cloud existe ou não, com um valor de retorno de verdadeiro se ele existe, ou falso de outra forma.

Sintaxe:

steam_file_exists(filename);

Argumento Descrição
nome do arquivo O nome do arquivo a ser verificado.

Devoluções:

Booleano

Exemplo:

se o steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt");
}

O código acima verifica se um arquivo existe no Steam Cloud e se existe, ele o abre e lê seu conteúdo na variável "save_str".
