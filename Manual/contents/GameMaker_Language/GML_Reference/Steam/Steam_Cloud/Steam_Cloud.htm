Nuvem devapor

O Steam Cloud fornece um sistema de armazenamento remoto de arquivos fácil e transparente para seu jogo. Todos os arquivos escritos em disco usando as funções de nuvem do GameMaker Studio 2 (listadas abaixo) serão replicados para os servidores Steam após a saída do jogo. Se o usuário então muda de computador, os arquivos serão então baixados para o novo computador antes do lançamento do jogo, o que significa que o jogo pode então acessar os arquivos lendo-os usando as funções apropriadas do Steam. O cliente Steam faz o trabalho de garantir que os arquivos sejam mantidos sincronizados em todos os computadores que o usuário possa estar acessando.

NOTA: Por padrão, a Nuvem não está habilitada para um jogo no Steamworks. ela deve ser habilitada anteriormente na guia "Nuvem" do administrador do jogo Steamworks, onde você deve definir o byte e a cota de arquivo. A próxima vez que você publicar seu jogos Configuração Steamworks, o armazenamento em nuvem estará pronto para uso.

As seguintes funções podem ser usadas para acessar o Steam Cloud de dentro do GameMaker Studio 2

  • vapor_é_activado_para_apo_a_aplicação
  • vapor_é_activado_para_conta
  • steam_get_quota_total
  • steam_get_quota_free
  • steam_file_exists
  • tamanho_do_arquivo_de_vapor
  • vapor_file_persisted
  • steam_file_write
  • steam_file_write_file
  • steam_file_read
  • steam_file_share
  • steam_file_delete
