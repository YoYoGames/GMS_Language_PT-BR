tamanho_do_arquivo_de_vapor

Com esta função você pode verificar o tamanho de um arquivo armazenado no Steam Cloud. O número real retornado é o tamanho, em bytes, do arquivo.

Sintaxe:

steam_file_size(filename);

Argumento Descrição
nome do arquivo O nome do arquivo para verificar o tamanho do mesmo.

Devoluções:

Real

Exemplo:

file_bytes = steam_file_size("Save.txt");

O código acima armazena o tamanho de um arquivo do Steam Cloud na variável "file_bytes".
