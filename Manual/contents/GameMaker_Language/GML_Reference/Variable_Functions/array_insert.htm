array_insert

Com esta função você pode inserir um valor (ou valores) em uma matriz em qualquer posição. A função requer que você forneça uma variável que mantenha o array, o índice (posição) no array a ser inserido, bem como pelo menos um valor para inserir, embora opcionalmente você possa fornecer outros argumentos e todos eles serão inseridos na matriz em ordem consecutiva a partir do índice dado.

Observe que se o índice dado estiver fora do comprimento da matriz, os valores serão adicionados ao índice dado, e quaisquer espaços vazios na matriz entre seu último valor e os valores inseridos serão definidos para um valor padrão de 0.

Sintaxe:

array_insert(variável, índice, valor, [valor], [valor], [valor], [etc...])

Argumento Descrição
variável A variável que sustenta a matriz.
índice O índice (posição) na matriz para inserir o(s) valor(es)
valor O valor a ser inserido
[valor], [valor], [valor], [etc...] (Opcional) Outros valores a serem inseridos na matriz

Devoluções:

N/A

Exemplo:

array_insert(score_array, score_rank, current_score);

O código acima irá inserir a pontuação atual na matriz de pontuação, no índice definido pela variável score_rank.
