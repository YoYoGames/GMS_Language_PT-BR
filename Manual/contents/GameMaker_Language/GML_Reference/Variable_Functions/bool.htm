bool

Esta função tentará converter um determinado valor em um tipo de dado booleano, onde o valor será devolvido como verdadeiro se for maior que 0,5 e falso caso contrário.

Sintaxe:

bool(n);

Argumento Descrição
n O valor a converter.

Devoluções:

Booleano

Exemplo:

if !is_bool(val)
{
val = bool(val);
}

O código acima verifica a variável "val" para ver se é um booleano e se não é, então é convertido em um.
