Funções variáveis

As funções desta página são projetadas para lidar com todas as diferentes variáveis e tipos de variáveis ao usar a linguagem GameMaker em seus jogos. Para mais informações sobre variáveis, consulte a página sobre Variáveis e Escopo de Variáveis.

As seguintes funções variáveis estão disponíveis:

  • variável_instância_existe
  • nomes_de_instância_variável_get_names
  • nomes_de_instância_variável
  • variable_instance_get
  • conjunto_de_instância_variável
  • variável_global_existe
  • variável_global_get
  • variável_global_set
  • variável_estruto_existe
  • variável_struct_get
  • variável_set_struct_set
  • variável_struct_remove
  • nome_da_estrutura_variável_get_names
  • nomes_de_estrutura_variável
  • exemplo de

Há algumas funções associadas ao uso de Arrays também. Elas são projetadas para lhe dar flexibilidade em seu código e lhe permitirão criar arrays mais funcionais e dinâmicos ao fazer seus jogos. Estas funções são:

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(depreciado)
  • array_length_2d(depreciado)
  • array_height_2d(depreciado)

Estas próximas funções estão todas relacionadas ao uso de Variáveis de Método:

  • método
  • método_get_self
  • método_get_index
