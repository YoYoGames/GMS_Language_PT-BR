array_height_2d(Deprecated)

ADVERTÊNCIA! Esta função é depreciada, pois as matrizes não são mais limitadas a apenas 1 ou 2 dimensões, e como tal, esta função é fornecida apenas para apoio a projetos legados. Todos os novos projetos devem usar a forma atual de criar e acessar projetos multi-dimensionais arrays (veja aqui para mais informações).

Com esta função você pode obter a altura (número de entradas) de uma primeira dimensão de uma matriz 2D. Você fornece a matriz para verificar e a saída da função informa quantas entradas iniciais ela contém. Você pode obter o número de entradas para a segunda dimensão da matriz usando a função array_length_2d.

Sintaxe:

array_height_2d(array_index);

Argumento Descrição
array_index O índice da matriz a ser verificada.

Devoluções:

Real

Exemplo:

para (var i = 0; i < altura_da_matriz_2d(a); ++i;)
{
for (var j = 0; j < array_length_2d(a, i); ++j;)
{
a[i, j] = -1;
}
}

O código acima passará por uma matriz 2D e definirá cada entrada como -1.
