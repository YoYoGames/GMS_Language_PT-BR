int64

Esta função tentará converter um determinado valor em um inteiro de 64 bits, onde o valor deve ser ou um real, uma string, um int64, um int32, ou um ptr. Qualquer coisa caso contrário, o jogo se chocará com uma mensagem de erro. Você pode verificar se uma variável contém uma int64 usando a função is_int64().

Sintaxe:

int64(val);

Argumento Descrição
val O valor a converter.

Devoluções:

64bit inteiro

Exemplo:

steam_handle= int64(global.fileReadString);

O código acima converte o valor mantido na variável global para um inteiro de 64bit.
