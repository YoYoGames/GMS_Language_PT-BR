variável_struct_remove

Com esta função, você pode remover uma variável de uma estrutura. Você fornece a identificação da estrutura para remover a variável e o nome da string da variável a ser removida.

Sintaxe:

variável_struct_set(estrutura, nome);

Argumento Descrição
estrutura A referência estrutural para remover a variável de
nome O nome da variável a ser removida (como uma string)

Devoluções:

N/A

Exemplo:

if variable_struct_exists(mystruct, "shields")
{
variable_struct_remove(mystruct, "shields");
}

O código acima verificará se a variável dada existe na estrutura dada e se ela existe, então ela é removida.
