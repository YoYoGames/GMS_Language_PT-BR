variável_global_existe

Com esta função, você pode verificar se existe ou não uma variável de escopo global. Você fornece o nome da variável global a ser verificada como uma string (ver código de exemplo abaixo) e a função retornará verdadeira se existe uma variável global com o nome dado ou falsa de outra forma.

Sintaxe:

variável_global_existe(nome);

Argumento Descrição
nome O nome da variável global a ser verificada (como uma string)

Devoluções:

Booleano

Exemplo:

se !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

O código acima verificará se a variável global chamada "enemy_num" existe e se ela não existe é criada e define um valor.
