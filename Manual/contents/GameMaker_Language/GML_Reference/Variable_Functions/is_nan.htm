is_nan

Esta função retorna se uma determinada variável é NaN (não um número) ou não, retornando verdadeira se for, e falsa se não for.

Sintaxe:

is_nan(n);

Argumento Descrição
n O argumento para verificar.

Devoluções:

Booleano

Exemplo:

if is_nan(global.value)
{
show_debug_message("Valor não é um número");
}

O código acima verifica o "valor" da variável global para ver se é um número ou não e mostra uma mensagem de debug se não for.
