is_method

Esta função pode ser usada para verificar e ver se uma variável é uma variável de método (ela retornará verdadeira) ou não (neste caso, retornará falsa).

Sintaxe:

is_method(n);

Argumento Descrição
n A variável a ser verificada.

Devoluções:

Booleano

Exemplo:

if is_method(get_vec)
{
show_debug_message("Método variável!");
}

O código acima verifica uma variável para ver se é um método, e se a função retorna verdadeiro, então uma mensagem de debug é emitida para o console.
