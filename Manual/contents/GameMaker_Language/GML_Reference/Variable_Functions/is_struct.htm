is_struct

Esta função verifica se o valor fornecido é uma estrutura. Retorna verdadeiro se for, caso contrário, retorna falso.

Observe que as variáveis do método também retornarão verdadeiras, e as instâncias de objeto retornarão falsas.

Sintaxe:

is_struct(val);

Argumento Descrição
val O valor a ser verificado.

Devoluções:

Booleano

Exemplo:

if is_struct(a)
{
delete(a);
}

O código acima verifica uma variável para ver se é uma estrutura, e se a função retorna verdadeira, a estrutura é apagada.
