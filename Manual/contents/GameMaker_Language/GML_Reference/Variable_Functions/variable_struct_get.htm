variável_struct_get

Com esta função você pode obter o valor de uma determinada variável nomeada dentro de uma estrutura. Você fornece a referência da estrutura bem como o nome da variável para obter o valor de uma string (veja o código de exemplo abaixo). A função irá retornar o valor detido pela variável ou indefinido se a variável nomeada não existir.

IMPORTANTE! Se a variável que você está recebendo não existir, então a função retornará a palavra-chave indefinida e você poderá receber erros que impedirão o funcionamento do jogo, portanto, em caso de dúvida, use a função variável_struct_exists() primeiro.

Sintaxe:

variável_struct_get(estrutura, nome);

Argumento Descrição
estrutura A referência estrutural a ser usada
nome O nome da variável a ser obtida (como um fio)

Devoluções:

Valor (qualquer tipo de dado) ou indefinido

Exemplo:

if variable_struct_exists(mystruct, "shields")
{
var ss = variable_struct_get(mystruct, "shields");
}
mais
}
var ss = -1;
}

O código acima verificará se uma variável existe na estrutura dada e se existe, então o valor que ela detém é recuperado e armazenado em uma variável local. Se não existir, a variável local é definida como -1.
