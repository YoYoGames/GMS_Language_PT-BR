is_matrix

Esta função retorna se uma determinada variável se refere a uma matriz ou não. Em alguns casos, você deseja verificar e ver que tipo de dados uma variável contém no GameMaker Studio 2 e é quando você usaria esta função. Ela retorna verdadeiro ou falso, dependendo se o valor é ou não uma matriz.

Sintaxe:

is_matrix(n);

Argumento Descrição
n O argumento para verificar.

Devoluções:

Booleano

Exemplo:

if !is_matrix(val)
{
show_debug_message("Não é uma matriz válida!");
}

O código acima verifica a variável "val" para ver se ela contém uma matriz e se não contém, então ela mostra uma mensagem no console de depuração.
