array_length_1d(Deprecated)

ADVERTÊNCIA! Esta função é depreciada, pois as matrizes não são mais limitadas a apenas 1 ou 2 dimensões, e como tal, esta função é fornecida apenas para apoio a projetos legados. Todos os novos projetos devem usar a forma atual de criar e acessar projetos multi-dimensionais arrays (veja aqui para mais informações).

Com esta função você pode obter o comprimento (número de entradas) de uma matriz 1D. Para arrays 2D você deve estar usando a array_height_2d e array_length_2d funções.

IMPORTANTE! Se a matriz tiver mais de 32.000 entradas, esta função retornará um valor errado e não deve ser usada.

Sintaxe:

array_length_1d(array_index);

Argumento Descrição
array_index O índice da matriz a ser verificada.

Devoluções:

Real

Exemplo:

para (var i = array_length_1d(a) - 1; i > -1; i--;)
{
a[i] = -1;
}

O código acima passará por uma matriz e definirá cada entrada como -1.
