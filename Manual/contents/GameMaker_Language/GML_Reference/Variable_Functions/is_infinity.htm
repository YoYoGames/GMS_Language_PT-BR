is_infinity

Esta função retorna se uma determinada variável é infinita (um número infinito) ou não, retornando verdadeira se for, e falsa se não for.

Sintaxe:

is_infinity(n);

Argumento Descrição
n O argumento para verificar.

Devoluções:

Booleano

Exemplo:

if is_infinity(global.value)
{
show_debug_message("Valor é infinito!");
}

O código acima verifica o "valor" da variável global para ver se ela é infinita ou não e mostra uma mensagem de debug se ela é.
