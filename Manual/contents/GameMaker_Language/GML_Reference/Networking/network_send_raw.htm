network_send_raw

Com esta função, você pode enviar um pacote de dados "bruto" através da rede. A função leva a identificação do soquete para conectar e então você deve fornecer a identificação do buffer que contém os dados a serem enviados (para mais informações sobre buffers ver Referência - Buffers) e finalmente o tamanho (em bytes) do pacote de dados. Os dados enviados não são formatados pelo GameMaker Studio 2 de forma alguma e os dispositivos receptores receberão os dados como um fluxo que significa que você mesmo terá que lidar com isso. A função retornará o número de bytes de dados enviados, ou um número inferior a 0 se o envio falhar.

Sintaxe:

network_send_raw(socket, buffer, tamanho);

Argumento Descrição
soquete A identificação do soquete a ser utilizado.
tampão A identificação do buffer de onde obter os dados.
tamanho O tamanho (em bytes) dos dados.

Devoluções:

Real

Exemplo:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

A informação acima carrega um dado de buffer previamente gravado na memória e retorna a identificação do buffer para ser armazenado na variável "buff". Este buffer completo é então enviado como um pacote de dados brutos através da rede usando o soquete identificado pelo variável "meia".
