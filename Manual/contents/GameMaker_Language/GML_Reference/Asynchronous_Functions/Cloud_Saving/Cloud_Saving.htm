Economia de nuvens

Ao fazer jogos, muitas vezes é necessário armazenar informações sobre o estado do jogo em um arquivo de algum tipo, mas armazenar essas informações no dispositivo nem sempre é a melhor opção, pois se o jogador excluiu o jogo e depois reinstalou, essas informações pode ser perdido. Para evitar isso, você pode usar vários serviços diferentes de nuvem, que oferecem armazenamento de dados através da Internet para recuperação e modificação a qualquer momento. Você deve observar que esta função é limitada a um único blob" de dados por jogo, então toda vez que você enviar uma nova gravação para o serviço de nuvem, seja uma string ou um arquivo, ele irá sobrescrever qualquer dado armazenado anteriormente.

OBSERVAÇÃO: Atualmente apenas o Android O alvo usa estas funções, e você precisa marcar a caixa de seleção Ativar o Google Cloud Saving na seção Social das opções de jogo do Android. Isto o levará a baixar a extensão dos Serviços Google, que é necessário para que a economia de nuvens no Android funcione. Para mais informações sobre a economia de nuvens, consulte o seguinte artigo do helpdesk: Economia de nuvens noAndroid

  • cloud_string_save
