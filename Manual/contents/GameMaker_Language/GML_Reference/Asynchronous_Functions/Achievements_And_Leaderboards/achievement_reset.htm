achievement_reset

Essa função redefinirá todas as conquistas de volta aos seus valores iniciais para o jogo. Esta função é fornecida como uma função de depuração e não é recomendado que você permita que o usuário final faça isso em seus jogos.

Sintaxe:

achievement_reset()

Devoluções:

N/A

Exemplo:

se debug_mode
{
if keyboard_check_pressed(vk_space)
{
achievement_reset()
}
}

O código acima verifica se o jogo está em modo de depuração e, se estiver pressionando "espaço", reinicializará todas as conquistas.
