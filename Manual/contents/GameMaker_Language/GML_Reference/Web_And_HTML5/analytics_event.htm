evento_analítico

Esta função enviará o texto especificado para o provedor analítico que você configurou através das Opções de Jogo HTML5. Esta função pode ser usada para criar uma análise personalizada para rastrear algo fora do escopo do fornecedor que está sendo utilizado.

Sintaxe:

analytics_event(string);

Argumento Descrição
corda Um fio a ser enviado ao fornecedor

Devoluções:

N/A

Exemplo:

se hs_new
{
analytics_event("New hiscore of " + string (score));
}

O código acima verificará uma variável para ver se ela é verdadeira e se é então um evento analítico especial será acionado com a string especificada.
