url_get_domain

Você pode usar isto para obter o domínio onde seu jogo HTML5 está sendo jogado.

Sintaxe:

url_get_domain();

Devoluções:

corda

Exemplo:

dom_name=url_get_domain();

Isto irá armazenar o domínio de seu jogo na variável "dom_name".
