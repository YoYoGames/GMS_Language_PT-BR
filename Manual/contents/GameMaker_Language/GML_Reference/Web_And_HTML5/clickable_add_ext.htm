clickable_add_ext

Com esta função você pode criar um ícone personalizado, clicável, que atua como um "botão" fora da tela do jogo, mas dentro da própria janela do navegador. A função retornará um valor de id que deve ser armazenado em uma variável para ser usada em todos outras funções do botão que fazem referência ao novo ícone. Esta função é muito útil para evitar que o navegador crie uma janela pop-up ao clicar no botão, pois tudo isso é feito através do próprio GameMaker Studio 2.

Esta função funciona exatamente da mesma forma que a clickable_add() função, onde você deve definir a posição relativa ao (0,0) da janela do navegador, dar a entrada da página de textura para o sprite que você deseja o botão ícone a ter (veja: sprite_get_tpe()), especificar um URL para abrir ou um script (do Navegador de Ativos do GameMaker Studio 2) a ser aberto ou executado então definir a meta e os parâmetros a serem utilizados. Entretanto, esta função também tem duas opções adicionais que permitem definir o ícone alfa de 0 (totalmente transparente) a 1 (totalmente opaco) e a escala do ícone, que é um valor multiplicador aplicado ao fada utilizada. Portanto, um valor de 1 é a escala padrão, enquanto (por exemplo) 0,5 seria a metade da escala e 2 seria o dobro.

Também vale a pena notar que se o argumento URL que você passa é o nome de uma função do script (ou seja: não começa com "http://") E o script é chamado gmcallback_*. então, você pode obter feeds diretos em um evento html5. Os nomes das funções do GameMaker Studio 2 não são ofuscados se começarem com gmcallback_.

Sintaxe:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Argumento Descrição
x A posição x dentro da janela.
y A posição y dentro da janela.
tpe A entrada da página de textura para o duende a ser utilizado.
url O URL (endereço do site) para o link, ou o roteiro a ser executado.
alvo Esta é a área de destino para abrir a URL em (ver descrição).
param Os diversos parâmetros para o ícone (ver descrição).
alfa O alfa da imagem do ícone (padrão 1).
escala A escala do ícone (padrão 1).

Devoluções:

Real

Exemplo:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

O código acima cria um ícone DOM clicável na posição (32, 32) da página em que a tela do jogo está rodando. O ícone usa o sprite referenciado da página de textura como "spr_MS_Home" e quando clicado, o ícone abrirá uma nova janela para a URL especificada e com as propriedades definidas para aquela janela. o botão também será escalado para o dobro do tamanho original do duende e metade de uma imagem alfa de 1.
