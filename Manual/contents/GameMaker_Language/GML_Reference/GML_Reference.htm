Referência delinguagem GameMaker

Esta seção do manual é um guia de referência para a GameMaker Studio 2 Idioma(GML). Você pode encontrar todas as funções disponíveis documentadas aqui junto com os argumentos e exemplos de código necessários para mostrar como elas podem ser usadas. Para informações sobre como usar o GML, consulte o Visão geral daGML seção do manual.

NOTA: Em qualquer lugar onde o manual tenha "N/A" como valor de retorno para uma função, isso significa que a função não deve retornar nenhum valor como parte de sua operação. Se você tentar verificar um valor de retorno destas funções, então você terá o valor undefined.
