view_get_visible

Esta função pode ser usada para verificar a visibilidade do porto de vista dado. A função retornará verdadeira se for visível e falsa se não o for.

Sintaxe:

view_get_visible(view_port)

Argumento Descrição
view_port O porto de visão para o alvo (0 - 7)

Devoluções:

Booleano

Exemplo:

if !view_get_visible(0)
{
view_set_visible(0, true);
}

O código acima verificará se a porta de visualização[0] está visível e se não está definida para visível.
