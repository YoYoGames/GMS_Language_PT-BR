camera_get_default

Esta função pode ser usada para recuperar o valor único de identificação da câmera padrão (a câmera que o GameMaker Studio 2 usa quando não há nenhuma visualização ou porta de câmera ativa em uma sala de jogos).

Sintaxe:

camera_get_default()

Devoluções:

Índice da câmera (real).

Exemplo:

var def = camera_get_default();
view_camera[0] = def;

O código acima obtém o ID da câmera para a câmera padrão e define a câmera de visualização para a porta [0] para usá-la.
