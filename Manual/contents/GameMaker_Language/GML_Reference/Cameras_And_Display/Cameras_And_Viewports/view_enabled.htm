view_enabled

Esta variável controla se qualquer porta de visualização visível dentro da sala está habilitada ou não. Se você tiver portas de visualização definidas para visível e depois desabilitar esta opção, toda a sala será atraída para a tela escalonada para o tamanho da janela, em vez da diferentes câmeras sendo desenhadas através dos portos de visão.

Sintaxe:

view_enabled;

Devoluções:

Booleano(verdadeiro: habilitado, falso: desabilitado)

Exemplo:

se !view_enabled
{
view_visible[0] = true;
view_enabled = true;
}

O código acima verifica se as vistas estão habilitadas para a sala, e se não estiverem, assegura que a porta de vista[0] esteja visível e então habilita as vistas para a sala.
