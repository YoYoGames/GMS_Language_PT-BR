window_mouse_get_x

Com esta função você pode obter a posição x do cursor do mouse (em pixels) dentro do navegador se for um jogo HTML5 ou dentro da tela se for um jogo Windows, Ubuntu(Linux) ou macOS.

NOTA: Para funções regulares do mouse, veja a seção sobre Entrada do Rato

Sintaxe:

window_mouse_get_x();

Devoluções:

Real

Exemplo:

wx = window_mouse_get_x();

O código acima armazena a posição atual da janela do eixo x do mouse na variável "wx".
