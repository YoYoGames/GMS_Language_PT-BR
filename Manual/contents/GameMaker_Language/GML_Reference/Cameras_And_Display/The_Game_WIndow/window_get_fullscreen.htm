janela_get_fullscreen

Esta função retorna se a janela do jogo está em modo de tela cheia(verdadeiro) ou não(falso).

Sintaxe:

window_get_fullscreen();

Devoluções:

Booleano

Exemplo:

se window_get_fullscreen()
{
draw_text(32, 32, "Fullscreen is ON");
}
else {
draw_text(32, 32, "tela cheia está desligada");
}

O código acima verificará se a janela está no modo tela cheia ou não e desenhará uma mensagem de texto, dependendo do valor retornado.
