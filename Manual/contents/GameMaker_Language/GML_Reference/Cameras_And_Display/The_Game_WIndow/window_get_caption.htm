window_get_caption

Esta função retorna a legenda da janela (este é o texto que aparece no topo da janela, ao lado de seu ícone) e por padrão isto mostra a legenda da sala em que você está atualmente.

Sintaxe:

window_get_caption();

Devoluções:

Cordão

Exemplo:

se window_get_caption() != ""
{
window_set_caption("");
}

O código acima verificará a legenda da janela para ver se ela tem texto ou não, e se a coloca em uma cadeia vazia para que nenhuma legenda seja exibida.
