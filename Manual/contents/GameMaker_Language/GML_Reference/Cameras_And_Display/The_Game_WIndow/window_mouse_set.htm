conjunto_de_ratos_da_janela

Com esta função você pode mudar ou definir a posição do mouse dentro da janela do jogo que pode ser útil para jogos FPS, por exemplo. A função só funcionará enquanto o jogo estiver em foco e, usando alt + tabulação, o mouse será desbloqueado.

NOTA: Para funções regulares do mouse, veja a seção sobre Entrada do Rato

Sintaxe:

window_mouse_set(x, y);

Argumento Descrição
x A coordenada x na janela.
y A coordenada y na janela.

Devoluções:

N/A

Exemplo:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

O código acima centralizaria o mouse na janela do jogo.
