window_device

Esta função devolverá o ponteiro do dispositivo d3d atual, que você poderá então (por exemplo) passar para uma DLL ou Dylib no Windows e macOS.

NOTA: Esta função foi depreciada no GameMaker Studio 2 em favor de os_get_info().

Sintaxe:

window_device();

Devoluções:

Apontador

Exemplo:

gfx_pointer = window_device()

O código acima irá armazenar o ponteiro do dispositivo d3d em uma variável.
