janela_get_width

Com esta função você pode obter a largura atual (em pixels) da janela do jogo.

Sintaxe:

window_get_width();

Devoluções:

Real

Exemplo:

ww = window_get_width();

O código acima irá armazenar a largura atual da janela na variável "ww".
