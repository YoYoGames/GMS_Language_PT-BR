mp_grid_clear_all

Com esta função você pode limpar uma grade MP de todas as células "proibidas".

Sintaxe:

mp_grid_clear_all(id);

Argumento Descrição
id Índice da rede mp_ que deve ser usada

Devoluções:

Booleano

Exemplo:

se !instance_exists(obj_Player)
{
mp_grid_clear_all(grid);
}

O código acima limpará o mp_grid indexado na variável "grid", marcando todas as células como livres, se uma instância do objeto "obj_Player" não existir mais na sala.
