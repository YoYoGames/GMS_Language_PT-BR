place_empty

Você pode usar esta função para verificar e ver se a instância de chamada colidiria com qualquer outra instância de um objeto ou todas as instâncias em seu jogo. Para que esta colisão seja resolvida corretamente, a instância que executa o código deve ter uma colisão válida. máscara (seja para o próprio duende, seja através do máscara_index) e só registrará colisões com aquelas instâncias que também possuem uma máscara válida.

A função é testar se não há colisões caso a instância de chamada seja colocada em uma posição específica, e você pode fornecer um argumento opcional para refinar a verificação para verificar apenas se uma posição está livre de colisões com instâncias do dado tipo. Observe que se não for fornecido um ID de objeto opcional, a verificação será feita contra todas as instâncias dentro da sala. A verificação da colisão será precisa ou baseada na caixa de delimitação da instância, dependendo do tipo de colisão A máscara foi selecionada, mas para que colisões precisas funcionem corretamente, todas as instâncias na verificação devem ter máscaras de colisão precisas.

Observe que as coordenadas x/y dadas serão niveladas com o inteiro mais próximo antes da verificação ser realizada.

Sintaxe:

place_empty(x, y, [object_id]);

Argumento Descrição
x A posição x a ser verificada.
y A posição y a ser verificada.
[object_id] Opcional! O objeto a ser verificado.

Devoluções:

Booleano

Exemplo:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
x = mouse_x;
y = mouse_y;
}

O código acima verificará uma colisão com qualquer outra instância do objeto "obj_Enemy", como se a instância de chamada fosse colocada na mesma posição que o mouse. Se não for detectada nenhuma colisão, então a instância tem suas coordenadas x/y ajustado para os do mouse.
