roda_do_mouse_up

Esta função retorna verdadeiro se a roda do mouse estiver sendo girada para cima e falsa caso contrário.

Sintaxe:

mouse_wheel_up();

Devoluções:

Booleano

Exemplo:

se mouse_wheel_up()
{
y -= 10;
}

Isto move a instância atual para cima da tela se a roda do mouse for girada para cima.
