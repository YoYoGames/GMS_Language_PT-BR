Entrada do mouse

A entrada do mouse é aceita em todas as plataformas (em dispositivos móveis é aceita como um único toque na tela - se você precisar usar multi-toque, você deve estar usando as funções específicas do dispositivo) e tem algumas constantes que são usados para especificar os botões que estão sendo pressionados. Estas constantes são mostradas na tabela a seguir:

Constante Descrição
mb_esquerda O botão esquerdo do mouse
mb_middle O botão central do mouse (isto pode não ser válido para todas as plataformas alvo)
mb_right O botão direito do mouse
mb_side1* Botão do lado do mouse 1
mb_side2* Botão do lado do mouse 2
mb_any Qualquer um dos botões do mouse
mb_none Sem botão do mouse


* NOTA: Os botões mb_side1 e mb_side2 estão disponíveis apenas no Windows, macOS, Ubuntu e HTML5.

As seguintes funções existem para os controles padrão do botão do mouse:

  • mouse_button
  • mouse_check_button
  • botão_de_marcação_pressionado_do_mouse
  • mouse_check_button_released
  • mouse_clear
  • último botão do mouse
  • roda_do_mouse_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

Note que também há um conjunto de funções de janela relacionadas ao uso do mouse em alvos de desktop:

  • window_mouse_get_x
  • window_mouse_get_y
  • conjunto_de_ratos_da_janela
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
