teclado_última tecla

Esta variável se refere ao valor que teclado_chave estava no quadro anterior, devolvendo o código chave dessa chave (todas as constantes de código chave padrão são devolvidas). Esta variável não é somente lida e você pode alterá-lo, por exemplo, para defini-lo como -1, se você já o manipulou.

Sintaxe:

última tecla_do_ teclado;

Devoluções:

Real

Exemplo:

if (última tecla do teclado != -1)
{
str += keyboard_lastchar;
keyboard_lastkey = -1;
}

O código acima verifica se a última variável chave não é igual a -1, e se é ela adiciona qualquer que seja a última chave como uma string à variável "str", então ela redefine a última variável chave do teclado para aceitar mais entradas.
