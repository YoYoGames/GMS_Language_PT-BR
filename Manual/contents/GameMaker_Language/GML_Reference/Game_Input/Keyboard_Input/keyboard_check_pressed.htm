teclado_check_pressed

Com esta função você pode verificar se uma tecla foi pressionada ou não. Ao contrário da função teclado_check() esta função só funcionará uma vez para cada vez que a tecla for pressionada, de modo que ela possa ser acionada novamente, a chave deve ser primeiramente liberada e depois pressionada novamente. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer um dos constantes vk_* listadas na página principal de Entrada do Teclado.

Sintaxe:

keyboard_check_pressed(key);

Argumento Descrição
chave A chave para verificar o estado de pressão.

Devoluções:

Booleano

Exemplo:

if keyboard_check_pressed(vk_anykey)
{
room_goto_next();
}

O código acima avançará para a próxima sala se o jogador pressionar qualquer uma das teclas do teclado (funcionando como um prompt "Pressione qualquer tecla para continuar").
