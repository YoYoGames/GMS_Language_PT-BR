teclado_set_numlock

Você pode usar esta função para ligar ou desligar o teclado numérico (ajustado para true for on, e false for off).

NOTA: Esta funcionalidade só está disponível no alvo do Windows e não funcionará em nenhum outro dispositivo.

Sintaxe:

teclado_set_numlock(valor);

Argumento Descrição
valor Coloque isso como verdadeiro para "ligado" e falso para "desligado".

Devoluções:

N/A

Exemplo:

se teclado_get_numlock()
{
keyboard_set_numlock(false);
}
mais
{
keyboard_set_numlock(true);
}

O código de exemplo acima obterá o estado da chave de bloqueio numérico e, se estiver ligado (verdadeiro), o fará disparar (falso) e vice-versa.
