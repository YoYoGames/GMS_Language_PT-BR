gesto_drag_distância

Esta função é utilizada para definir a distância necessária para que um evento de gesto de arrastamento seja acionado pelo movimento de um toque ou clique. A distância é medida em polegadas e tem um valor padrão de 0,1.

Sintaxe:

gesture_drag_distance(distância);

Argumento Descrição
distância A distância (em polegadas) a ser ajustada para a detecção de eventos de gestos de arrasto.

Devoluções:

N/A

Exemplo:

if gesture_get_drag_distance() != 0.1
{
gesture_drag_distance(0.1);
}

O código acima verifica se a distância de arrasto para gestos está ajustada a 0,1 polegadas e se não está ajustada a esse valor.
