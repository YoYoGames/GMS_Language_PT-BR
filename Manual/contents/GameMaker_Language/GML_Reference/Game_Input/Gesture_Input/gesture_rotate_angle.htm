gesto_rotate_angle

Esta função é usada para definir o ângulo que um par de toques deve exceder para acionar um gesto de partida rotativo. O ângulo é medido em graus e tem um padrão valor de 5°.

Sintaxe:

gesto_rotate_angle();

Devoluções:

Real (polegadas)

Exemplo:

se gesto_get_rotate_angle() != 5
{
gesture_rotate_angle(5);
}

O código acima verifica se o ângulo de rotação dos gestos está ajustado a 5° e se não está ajustado a esse valor.
