gesture_get_pinch_angle_towards

Esta função é utilizada para obter o ângulo dentro do qual um toque deve estar se movendo em direção a outro toque antes que um Pinch In Gesture possa ser potencialmente iniciado. O ângulo é medido em graus e tem um valor padrão de 45°.

Sintaxe:

gesture_get_pinch_angle_towards();

Devoluções:

Real (polegadas)

Exemplo:

se gesture_get_pinch_angle_towards() != 45
{
gesture_pinch_angle_towards(45);
}

O código acima verifica se a pitada em ângulo para gestos está ajustada a 45° e se não está ajustada a esse valor.
