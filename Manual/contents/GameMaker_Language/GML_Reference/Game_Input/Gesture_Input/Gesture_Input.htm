Entrada por gestos

As funções de Entrada do Dispositivo são todas boas e bem para lidar com simples toques na tela ou obter orientação do dispositivo, mas às vezes você vai querer que seus projetos detectem coisas mais complexas, como ser capaz de arrastar ou "flick" uma instância por uma sala. Estas coisas podem ser feitas com as funções do Dispositivo, mas para facilitar as coisas, o GameMaker Studio 2 tem uma gama completa de eventos de gestos para detectar torneiras duplas, arrastamento, movimentos, pitadas e rotações, tanto a nível de instância como a nível global, como bem como informações de retorno sobre o evento. As funções listadas aqui são todas projetadas para lhe dar maior controle sobre como e quando estes eventos serão acionados.

IMPORTANTE! Estas funções não funcionarão na plataforma de destino HTML5. Elas não darão um erro se usadas, mas serão simplesmente ignoradas.

As seguintes funções de gestos diferentes estão disponíveis:

  • gesture_get_drag_time
  • gesture_get_drag_distance
  • gesture_get_flick_flick_speed
  • gesture_get_double_tap_time
  • gesto_get_double_tap_distance
  • gesto_get_tap_count
  • gesto_get_pinch_distance
  • gesture_get_pinch_angle_towards
  • gesture_get_pinch_angle_away
  • gesture_get_rotate_time
  • gesto_get_rotate_angle
  • gesto_drag_time
  • gesto_drag_distância
  • gesto_flick_flick_speed
  • gesto_double_tap_time
  • gesto_douplo_tap_distância
  • gesto_tap_count
  • gesto_picch_distance
  • gesto_pino_ângulo_para_a frente
  • gesto_picch_angle_away
  • gesto_rotate_time
  • gesto_rotate_angle
