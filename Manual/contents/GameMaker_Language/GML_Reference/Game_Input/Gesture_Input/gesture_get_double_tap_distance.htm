gesto_get_double_tap_distance

Esta função é utilizada para obter a distância dentro da qual você tem que tocar/clicar a tela novamente após um único toque a fim de acionar um gesto de duplo toque. A distância é medido em polegadas e tem um valor padrão de 0,1.

Sintaxe:

gesture_get_double_tap_distance();

Devoluções:

Real (polegadas)

Exemplo:

if gesture_get_double_tap_distance() != 0.1
{
gesture_double_tap_distance(0.1);
}

O código acima verifica se a distância de toque duplo para gestos está ajustada a 0,1 polegadas e se não está ajustada a esse valor.
