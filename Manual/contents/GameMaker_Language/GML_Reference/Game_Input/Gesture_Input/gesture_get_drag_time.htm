gesture_get_drag_time

Esta função é usada para obter o tempo que leva para que um evento de Arrastar o gesto de partida seja acionado por um toque ou clique. Este tempo também afetará a forma como o Evento de Arrastar acionado como um toque/clique e liberação antes que este tempo acabe, será considerado um Tap. O tempo é medido em segundos e tem um valor padrão de 0,16.

Sintaxe:

gesture_get_drag_time();

Devoluções:

Real (tempo em segundos)

Exemplo:

if gesture_get_drag_time() != 0.16
{
gesture_drag_time(0.16);
}

O código acima verifica se o tempo de arrastamento dos gestos está configurado para 0,16 segundos e se não está configurado para esse valor.
