gesture_get_pinch_angle_away

Esta função é utilizada para obter o ângulo dentro do qual um toque deve estar se afastando de outro toque antes que um gesto de Pinch Out possa ser potencialmente iniciado. O ângulo é medido em graus e tem um valor padrão de 45°.

Sintaxe:

gesture_get_pinch_angle_away();

Devoluções:

Real (polegadas)

Exemplo:

if gesture_get_pinch_angle_away() != 45
{
gesture_pinch_angle_away(45);
}

O código acima verifica se o ângulo de aperto dos gestos está ajustado a 45° e se não está ajustado a esse valor.
