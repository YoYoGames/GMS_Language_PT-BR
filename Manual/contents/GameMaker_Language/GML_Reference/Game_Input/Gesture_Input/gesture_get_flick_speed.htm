gesture_get_flick_flick_speed

Esta função é utilizada para obter a velocidade necessária para que um evento Flick Gesture seja acionado quando um toque ou clique é liberado. A velocidade é medida em polegadas por segundo. e tem um valor padrão de 2,0.

Sintaxe:

gesture_get_flick_flick_speed();

Devoluções:

Real (polegadas por segundo)

Exemplo:

se gesture_get_flick_speed() != 2
{
gesture_flick_speed_speed(2);
}

O código acima verifica se a velocidade dos gestos está ajustada para 2 polegadas por segundo e se não está ajustada a esse valor.
