gamepad_set_colour

Esta função pode ser usada para definir a cor dos LEDs dentro de um controlador PlayStation. Você especifica o slot do dispositivo a ser configurado, e então dá uma cor, que pode ser qualquer uma das constantes de cor ou um valor de cor criado usando as funções de cor específicas ou um valor HEX (como $FFFFFFF).

Sintaxe:

gamepad_set_colour(dispositivo, cor);

Argumento Descrição
dispositivo Qual dispositivo "slot" do gamepad a ser definido.
cor A cor a ser usada.

Devoluções:

N/A

Exemplo:

se saúde < 10
{
gamepad_set_colour(0, c_red);
}

O código acima irá definir os LEDs do controlador PlayStation para vermelho se a variável saúde cair abaixo de 10.
