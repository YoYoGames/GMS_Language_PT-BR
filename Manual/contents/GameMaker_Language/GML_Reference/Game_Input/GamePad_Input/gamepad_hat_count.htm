gamepad_hat_count

Esta função devolverá o número total de chapéus disponíveis para o gamepad conectado ao "slot" do dispositivo em questão. Os chapéus geralmente referem-se aos botões para cima/baixo/esquerda/direita, e note que no alvo do Windows, os chapéus só estão disponíveis no DirectInput controladores (portanto, do slot 4 para cima).

Sintaxe:

gamepad_hat_count(dispositivo);

Argumento Descrição
dispositivo Qual dispositivo de controle de jogo "slot" a ser verificado.

Devoluções:

Real

Exemplo:

h_num = gamepad_hat_count(4);

O código acima retornará o número de chapéus disponíveis no controle de jogo conectado ao dispositivo "slot" 4 e armazenará o valor na variável "h_num".
