string_replace

Você pode usar esta função para analisar uma corda procurando uma peça específica, que pode então ser substituída pela nova corda que você especificou.

Sintaxe:

string_replace(str, substrato, newstr);

Argumento Descrição
str O fio a ser copiado.
substr O substrato dentro do cordão a ser substituído.
newstr O novo substrato para substituir o anterior.

Devoluções:

Cordão

Exemplo:

str1 = "Olá Terra";
str2 = string_replace(str1, "Terra", "Mundo");

Isto colocará str2 em str1, mas com seu exemplo de "Terra" substituído por "Mundo", resultando em str2 sendo "Olá Mundo ".
