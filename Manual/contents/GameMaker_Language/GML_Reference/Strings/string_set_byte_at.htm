string_set_byte_at

Esta função define um byte diretamente em uma string (baseado no formato UTF8) e retorna uma cópia da string com as mudanças.

NOTA: Esta função é incrivelmente lenta, portanto, considere cuidadosamente se é necessária e onde você a utiliza.

Sintaxe:

string_set_byte_at(str, pos, byte);

Argumento Descrição
str O fio para mudar o byte de.
pos A posição dentro da cadeia (começando em 1) para mudar o byte de.
byte O novo valor do byte.

Devoluções:

Cordão

Exemplo:

str = string_set_byte_at("olá", 2, 97);

O código acima mudaria o valor do byte da segunda letra da corda, e assim definir a corda variável para segurar "hallo".
