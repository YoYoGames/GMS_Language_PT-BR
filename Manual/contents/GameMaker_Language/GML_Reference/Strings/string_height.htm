altura_de_filtro

Esta função retornará a altura (em pixels) da linha de entrada, levando em conta a separação de linhas e quaisquer quebras de linha que o texto possa ter. É muito útil para calcular as distâncias entre os elementos do texto com base na mais alta das letras que compõem o cordel como seria desenhado com draw_text() usando a fonte atualmente definida.

Sintaxe:

string_height(string);

Argumento Descrição
corda O fio para medir a altura de.

Devoluções:

Real

Exemplo:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

O código acima obterá a altura da corda dada e então desenhará duas linhas de texto, usando a altura da corda retornada como separador.
