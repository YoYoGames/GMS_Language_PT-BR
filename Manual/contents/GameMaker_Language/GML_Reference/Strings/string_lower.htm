string_lower

Com esta função você pode forçar um fio a conter apenas caracteres em minúsculas.

NOTA: Esta função detecta apenas o alfabeto inglês de 26 letras de A a Z.

Sintaxe:

string_lower(string);

Argumento Descrição
corda O fio para converter para minúsculas.

Devoluções:

Cordão

Exemplo:

str1 = "Olá Mundo";
str2 = string_lower(str1);

O código acima definirá str2 como "olá mundo".
