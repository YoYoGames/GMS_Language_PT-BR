string_lettersdigits

Esta função devolverá uma cópia de uma determinada cadeia com tudo menos suas letras e dígitos removidos, o que significa que ela pode ser usada para remover quaisquer caracteres indesejados (como "#" ou "?") de, por exemplo, um nome de login ou uma senha.

NOTA: Esta função só retorna o alfabeto inglês de 26 letras de A a Z (maiúsculas ou minúsculas) e os números 0 a 9.

Sintaxe:

string_lettersdigits(string);

Argumento Descrição
corda O fio para converter em letras e dígitos.

Devoluções:

Cordão

Exemplo:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
draw_text(32,32, "Senha inválida! Somente cartas e números, por favor");
}

O código acima verificará o comprimento de uma corda em relação ao comprimento da mesma corda, mas com todos os símbolos removidos. Se eles não forem os mesmos, uma mensagem é desenhada na tela.
