string_hash_to_newline

Esta função retorna um fio onde o símbolo "#" foi convertido em uma nova linha. Portanto, uma string formatada como:

"Olá#World"

Seria impresso para a tela como:

Olá
Mundo

IMPORTANTE! Esta função é fornecida para compatibilidade de importação entre o GameMaker Studio 2 e versões anteriores do software e como tal, esta função não deve ser usada, exceto quando necessário para replicar funcionalidades obsoletas. Em vez disso, você deve estar usando os caracteres de escape para marcar uma nova linha. Para mais informações sobre compatibilidade de importação, veja aqui.

Sintaxe:

string_hash_to_newline(string);

Argumento Descrição
corda O fio para converter em várias linhas

Devoluções:

Cordão

Exemplo:

var str = string_hash_to_newline("Olá#World");
draw_text(32, 32, str);

O código acima converte a corda com o símbolo hash em uma corda dividida em duas linhas e depois imprime-a para a tela.
