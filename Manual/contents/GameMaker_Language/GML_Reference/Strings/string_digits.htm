dígitos_de_filtro

Você pode usar esta função para analisar uma determinada cadeia e obter qualquer número dela. Por exemplo, diga que você tem este texto - "Eu tenho 81 anos". Com esta função, você obteria uma cadeia de retorno de "81".

Sintaxe:

string_digits(string);

Argumento Descrição
corda O fio para obter os dígitos.

Devoluções:

Cordão

Exemplo:

var t_str = string_digits(input_str);
age = real(t_str);

O código acima pegará a cadeia de caracteres de entrada, desnudará todos os caracteres que não sejam números e então definirá a idade da variável para manter o valor real do número da cadeia de caracteres de retorno (assim, se a cadeia de caracteres de entrada foi - por exemplo - "Eu tenho 18 anos", o função retornaria "18").
