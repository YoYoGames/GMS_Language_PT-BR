string_ord_at

Você pode usar esta função para retornar um código de caractere específico em uma posição específica dentro de uma string, com o índice começando em 1 para o primeiro caractere. Se nenhum caractere for encontrado ou se a string for menor que o valor dado ao índice, -1 é retornado.

Sintaxe:

string_ord_at(str, index);

Argumento Descrição
str O fio a ser verificado.
índice A posição de onde obter o código de caracteres.

Devoluções:

Real

Exemplo:

str = "Hello World";
char_code = string_ord_at(str, 7);

Isto irá obter o código do sétimo caractere (onde "H" conta como o primeiro) na string "str" e armazená-lo na variável "char_code".
