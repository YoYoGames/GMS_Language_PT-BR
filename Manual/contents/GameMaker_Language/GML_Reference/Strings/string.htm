corda

Com esta função, você pode transformar qualquer número real em um fio. Se o número real for um número inteiro, ele será salvo sem casas decimais, caso contrário, ele será salvo com duas casas decimais. Se você precisar de mais casas decimais, então use a função string_format(). Observe também que a utilização desta função em uma variável que armazena uma matriz, uma estrutura de dados, ou uma estrutura irá converter o conteúdo destas variáveis em uma string que pode então ser emitida para o console ou salva em um arquivo para depuração.

Sintaxe:

cordão(val);

Argumento Descrição
corda O número real a ser transformado em um fio.

Devoluções:

Cordão

Exemplo:

draw_text(100, 100, "Pontuação: " + string(pontuação) + " / Saúde: " + string(saúde));

O código acima usa a função de cordas para desenhar tanto números reais quanto cordas juntas (o desenho só aceitará uma corda ou um real, mas não ambos).
