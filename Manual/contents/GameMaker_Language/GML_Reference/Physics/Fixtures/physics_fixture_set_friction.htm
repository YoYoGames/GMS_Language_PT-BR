física_fixture_set_fricção_física

O atrito é a força que resiste ao movimento relativo dos elementos materiais que deslizam um contra o outro, que no mundo da física do GameMaker Studio 2, se traduz como a perda de momento causada pela colisão de duas instâncias com os dispositivos ligados a eles. Assim, quando duas instâncias colidem, seu movimento é afetado por este valor, com um alto atrito causando uma maior perda de impulso do que um valor mais baixo. Observe que o atrito geralmente é definido para um valor entre 0 e 1, mas você pode usar qualquer valor não negativo valor, se necessário.

Sintaxe:

fisic_fixture_set_friction(fixação, fricção)

Argumento Descrição
fixação o índice da fixação
fricção o atrito da fixação

Devoluções:

N/A

Exemplo:

physics_fixture_set_friction(fix_brick, 0.1);

O código acima definirá o atrito do acessório indexado em "fix_brick" para 0,1.
