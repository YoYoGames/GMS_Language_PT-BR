fisica_fixture_set_awake

Quando você amarra um acessório a uma instância usando fisics_fixture_bind() isto retorna um "id" para a fixação do limite. Você pode usar este id para obter o valor de fricção da fixação vinculada(não a fixação "base") a qualquer momento usando esta função.

Sintaxe:

physics_get_friction(fixture)

Argumento Descrição
fixação a identificação da fixação do limite

Devoluções:

Real

Exemplo:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

O código acima obtém o valor de atrito atual para as propriedades físicas vinculadas da instância e depois as define para um valor diferente.
