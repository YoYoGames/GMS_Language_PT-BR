physics_set_friction

Quando você amarra um acessório a uma instância usando physics_fixture_bind() isto retorna um "id" para a fixação do limite. Você pode usar este id para definir o valor de fricção da fixação vinculada, não a fixação "base", a qualquer momento usando esta função. Note que o atrito é normalmente definido para um valor entre 0 e 1, mas você pode usar qualquer valor não negativo, se necessário.

Sintaxe:

physics_set_friction(fixture, fricção)

Argumento Descrição
fixação a identificação da fixação do limite
fricção o novo valor de fricção a aplicar

Devoluções:

N/A

Exemplo:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

O código acima obtém o valor de atrito atual para as propriedades físicas vinculadas da instância e depois as define para um valor diferente.
