fitocinemático

Esta variável somente de leitura retornará verdadeira se a instância for classificada como sendo um objeto cinemático, ou falsa se não o for. Uma instância cinemática é aquela que tem massa infinita (uma densidade de 0), mas que pode se mover. Portanto, para fazer uma instância cinemática, primeiro você criaria uma instância estática e depois definiria uma ou mais variáveis de instância relacionadas ao movimento (ou seja, uma ou mais variáveis de instância relacionadas ao movimento): phy_speed_x, phy_speed_you f_velocidade_angular)

Sintaxe:

fitocinemático;

Devoluções:

Booleano (ou indefinido se a instância não estiver habilitada para física)

Exemplo:

if !phy_kinematic
{
phy_speed_x = 5;
}

O código acima verifica se a instância é cinemática e, se não for, define a velocidade horizontal para 5.
