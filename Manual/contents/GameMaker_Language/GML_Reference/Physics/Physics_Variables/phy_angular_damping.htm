f_angular_damping

Esta variável pode ser usada para definir o amortecimento angular da instância, ou pode ser usada para obter o amortecimento angular atual. O amortecimento é a quantidade de "resistência" à rotação angular que a instância habilitada pela física tem, com um valor mais baixo permitindo que a instância gire mais rápido após uma colisão (por exemplo) e um valor mais alto, o que exige um empurrão mais forte e uma rotação mais lenta

Sintaxe:

f_angular_damping;

Devoluções:

Real (valor de ponto flutuante de precisão único, ou indefinido se a instância não estiver habilitada fisicamente)

Exemplo:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_angular_damping = 10;
}
mais
{
phy_angular_damping = 3;
}

O código acima verificará se há uma colisão entre a instância de chamada e instâncias de "obj_Water" e alterará o amortecimento angular de acordo.
