phy_speed_y

Esta variável pode ser usada para obter ou alterar o componente y do vetor de velocidade linear da instância e é definida em pixels por etapa (para pixels por segundo, veja f_linear_velocidade_y). Alterando isto para uma instância estática (ou seja: uma instância com densidade 0) a transformará em uma instância cinemática.

Sintaxe:

phy_speed_y;

Devoluções:

Real (valor de ponto flutuante de precisão único, ou indefinido se a instância não estiver habilitada fisicamente)

Exemplo:

if phy_speed_y != 0
{
phy_speed_y = 0;
}

O código acima verificará o componente y do vetor de velocidade linear e se for algo diferente de 0, ele o colocará em 0.
