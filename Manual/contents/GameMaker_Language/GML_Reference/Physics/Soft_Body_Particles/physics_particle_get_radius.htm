physics_particle_get_radius

Com esta função você pode descobrir qual é o raio atual (em pixels) das partículas na simulação física (você pode definir este valor usando physics_particle_set_radius()).

Sintaxe:

physics_particle_get_radius()

Devoluções:

Real

Exemplo:

if (physics_particle_get_radius() < 32)
{
physics_particle_set_radius(physics_particle_get_radius() + 1);
}

O código acima verificará o raio atual das partículas na simulação e se for inferior a 32, então aumentará seu tamanho em 1.
