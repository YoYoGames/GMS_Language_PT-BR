bandeiras_de_grupo_de_partículas_físicas

Com esta função você pode mudar as bandeiras do grupo para um grupo de partículas. O valor do grupo é aquele que foi retornado quando você criou o grupo de partículas usando a função física_particle_group_end(), enquanto as bandeiras são o valor de retorno de uma combinação das seguintes constantes:

Constante Descrição
phy_particle_group_flag_solid Um grupo de partículas sólidas impede que outros dispositivos se alojam dentro dele. Se algo penetrar nele, o grupo de partículas sólidas empurra o dispositivo ofensivo de volta para sua superfície, fazendo com que um grupo de partículas sólidas possua um força repulsiva.
phy_particle_group_flag_rigid Os grupos rígidos de partículas são aqueles cuja forma não muda, mesmo quando colidem com outros dispositivos.

Estas bandeiras de grupo utilizam a bit-masking para criar um valor de saída final que é então verificado para definir as diferentes propriedades básicas do grupo, além daquelas definidas pelas bandeiras de partículas. Você pode usar nenhuma, uma ou ambas estas constantes usando o bitwise ou "|" para mascarar os bits apropriados (como mostrado no exemplo abaixo).

Sintaxe:

physics_particle_set_group_flags(grupo, bandeiras)

Argumento Descrição
grupo O grupo de partículas a definir.
bandeiras As bandeiras a serem colocadas sobre a partícula.

Devoluções:

N/A

Exemplo:

var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);

O código acima criará uma variável para armazenar o valor das bandeiras e depois a utilizará para definir as bandeiras de todas as partículas criadas no grupo indexado pela variável "grupo".
