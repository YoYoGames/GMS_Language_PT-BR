física_particle_group_begin

Com esta função você pode criar um grupo de partículas em uma sala. Um grupo de partículas é aquele em que as partículas necessárias para criar uma forma específica são criadas de uma só vez como um grupo, permitindo-lhe criar corpos moles simulados com várias propriedades colocadas pelas bandeiras utilizadas. As "bandeiras" são o valor de retorno de uma combinação das constantes das partículas (ver a função physics_particle_create() para detalhes destes), enquanto as "groupFlags são uma combinação das seguintes constantes de grupo (ou 0 para nenhuma bandeira):

Constante Descrição
phy_particle_group_flag_solid Um grupo de partículas sólidas impede que outros dispositivos se alojam dentro dele. Se algo penetrar nele, o grupo de partículas sólidas empurra o dispositivo ofensivo de volta para sua superfície, fazendo com que um grupo de partículas sólidas possua um força repulsiva.
phy_particle_group_flag_rigid Os grupos rígidos de partículas são aqueles cuja forma não muda, mesmo quando colidem com outros dispositivos.

Estas bandeiras de grupo utilizam a bit-masking para criar um valor de saída final que é então verificado para definir as diferentes propriedades básicas do grupo, além daquelas definidas pelas bandeiras de partículas. Você pode usar nenhuma, uma ou ambas estas constantes usando o bitwise ou "|" para mascarar os bits apropriados (como mostrado no exemplo abaixo).

Além das bandeiras, você também pode definir a posição na sala para criar o grupo de partículas (a posição dada será o centro de massa do grupo acabado), a velocidade horizontal e vertical inicial, bem como a velocidade angular do grupo de partículas (a posição dada será o centro de massa do grupo acabado), a velocidade horizontal e vertical inicial, bem como a velocidade angular do grupo de partículas (a posição dada será o centro de massa do grupo acabado). grupo, permitindo a criação de grupos móveis e rotativos.

Você também pode definir a cor inicial e alfa de todas as partículas do grupo, com os valores sendo usados ao desenhar as partículas usando a função physics_particle_draw(). Depois disso, você estabelece o coeso força das partículas do grupo, que é outra forma de dizer o quanto elas se "colam" umas às outras. A força pode ser um valor entre 0 e 1, onde 0 não é coesão, e 1 é coesão máxima.

Finalmente, você tem que dar a categoria para seu grupo de partículas. Este valor é um valor inteiro arbitrário que você dá a cada grupo de partículas que você deseja que tenha propriedades similares, e pode ser usado para definir bandeiras mais tarde também para aquela categoria específica. como uma série de outras coisas.

NOTA: Você pode usar qualquer valor inteiro para a categoria , exceto 0 que é reservado pela simulação e é usado para selecionar todas as categorias em outras funções (como para desenho).

É importante notar que chamar esta função não cria o grupo na sala de física. Esta função simplesmente inicia o processo e define as propriedades da partícula base para o grupo. Para criar de fato o grupo de partículas é necessário dar é um formulário (veja o exemplo abaixo) e depois chamar a função física_particle_group_end()e, em seguida, será criado.

OBSERVAÇÃO: As bandeiras do tipo partícula que você colocar também influenciarão a coesão do grupo de partículas, de modo que se você sinalizar as partículas como sendo do tipo água (por exemplo), quando o grupo colide com outro dispositivo, ele se separará, enquanto que se você sinalizar a partícula como um tipo de mola, então ela manterá sua coesão em caso de colisão.

Sintaxe:

physics_particle_group_begin(bandeiras, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Argumento Descrição
bandeiras As bandeiras a serem colocadas sobre a partícula.
groupflags As bandeiras do grupo a serem colocadas sobre a partícula (ver descrição abaixo).
x A posição x para criar o grupo de partículas em.
y A posição y para criar o grupo de partículas em.
ang O ângulo (em graus) para girar o grupo para criar.
xv A velocidade horizontal inicial.
yv A velocidade vertical inicial.
ang_velocidade A velocidade angular inicial.
col A cor base a ser utilizada para o grupo de partículas.
alfa O alfa base a ser utilizado para o grupo de partículas.
força Estabelecer a força de coesão entre as partículas do grupo.
categoria A categoria definida pelo usuário à qual o grupo pertence.

Devoluções:

N/A

Exemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

O código acima armazena as bandeiras para o tipo de partícula e as propriedades do grupo de partículas em variáveis e depois as utiliza para criar um grupo de partículas circular com um raio de 100px na posição do mouse.
