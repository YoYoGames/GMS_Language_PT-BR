dados_de_agrupamento_de_particula_física

Esta função retorna várias informações sobre um grupo de partículas na simulação física usando as bandeiras verificadas. O índice do grupo (sua identificação) é o que foi retornado pela função physics_particle_group_end(), e o buffer utilizado deve ter sido criado anteriormente usando a função buffer_create(). Deve ser do tipo "crescer", com o tamanho aproximado do retorno esperado. dados. As bandeiras em si são definidas usando as constantes dadas abaixo, e você usaria o bitwise ou "|" para criar um único valor de bandeira para obter as informações desejadas.

Constante Descrição Tipo de dados
bandeiras tipo phy_particle_data_flag O valor das bandeiras para a partícula. buffer_u32
phy_particle_data_flag_position A posição x e y da partícula. 2 x buffer_f32
phy_particle_data_flag_velocity A velocidade horizontal e vertical. 2 x buffer_f32
phy_particle_data_flag_colour A cor e o valor alfa (hexadecimal). buffer_f32
phy_particle_data_category A categoria de partícula (como definida quando você criou o grupo ao qual pertence). buffer_u32

Sintaxe:

physics_particle_group_get_data(group, buffer, flags)

Argumento Descrição
grupo O índice do grupo de partículas (ID) do grupo de partículas para obter os dados.
tampão O buffer (anteriormente criado) a ser usado para armazenar os dados.
bandeiras As bandeiras a serem usadas para extrair dados sobre tipos específicos de partículas.

Devoluções:

Dados tampão

Exemplo:

var count = physics_particle_group_count(gp);
var flags = phy_particle_data_flag_position | phy_particle_data_flag_colour;
if (contagem > 0)
{
var buffer = buffer_create(count * 12, buffer_grow, 4);
physics_particle_group_get_data(gp, buffer, flags);
for (var n = 0; n < count; n++;)
{
var xx = buffer_read(buffer, buffer_f32);
var yy = buffer_read(buffer, buffer_f32);
var argb = buffer_read(buffer, buffer_u32);
var alfa = (argb >> 24) & 255;
draw_sprite_ext(sprBlob, 0, xx, yy, 1, 1, 0, c_green, alpha);
}
buffer_delete(tampão);
}

O código acima obtém a posição e a velocidade de cada partícula do grupo indexada pela variável "gp", armazena os dados buffer em uma série de variáveis, e depois usa isso para desenhar um sprite na posição de cada partícula do grupo.
