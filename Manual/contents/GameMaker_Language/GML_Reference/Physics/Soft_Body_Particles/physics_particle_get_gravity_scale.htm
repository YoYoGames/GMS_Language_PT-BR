escala_física_particular_agravidade_agravidade

Com esta função você pode descobrir qual é o fator de escala de gravidade atual para partículas na simulação física (você pode definir este valor usando escala_de_gravura_de_partículas_físicas()).

Sintaxe:

escala_física_particular_agravidade_agravidade()

Devoluções:

Real

Exemplo:

physics_particle_set_gravity_scale(physics_particle_get_gravity_scale() + 0.1);

O código acima estabelecerá a escala de gravidade.
