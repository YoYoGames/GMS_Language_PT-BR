densidade_de_particula_esquema_física

Com esta função você pode descobrir qual é a densidade atual das partículas na simulação física (você pode definir este valor usando densidade_de_set_física_de_partícula()).

Sintaxe:

physics_particle_get_damping()

Devoluções:

Real

Exemplo:

if (physics_particle_get_density() < 1)
{
physics_particle_set_density(physics_particle_get_density() + 0.01);
}

O código acima verificará o valor da densidade atual para todas as partículas do sistema e se for inferior a 1, então acrescentará 0,01 a ele.
