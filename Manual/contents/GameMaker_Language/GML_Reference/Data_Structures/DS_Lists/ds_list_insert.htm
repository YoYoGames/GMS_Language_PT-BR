ds_list_insert

Esta função adicionará o valor dado à lista na posição dada. se a lista contiver mais valores após a posição dada, sua posição será deslocada para cima uma para dar espaço para que a lista seja maior por uma.

Sintaxe:

ds_list_insert(id, pos, val);

Argumento Descrição
id A identificação da lista a ser acrescentada.
pos A posição para adicionar o valor, onde 0 corresponde ao início da lista e a posição final é ds_list_size(id)-1.
val O valor a ser acrescentado à lista.

Devoluções:

N/A

Exemplo:

ds_list_insert(lista, 9, pontuação);

O código acima adicionará o valor armazenado na variável "pontuação" na 10ª posição da lista indexada pela variável "lista" (listas são contadas a partir de 0, portanto uma lista de dez valores é numerada de 0 a 9).
