ds_list_replace

Esta função substituirá o valor na posição dada por outro.

Sintaxe:

ds_list_replace(id, pos, val);

Argumento Descrição
id O id da lista para mudar.
pos A posição para substituir o valor, onde 0 corresponde ao início da lista e a posição final é ds_list_size(id)-1.
val O novo valor para substituir o valor dado por.

Devoluções:

N/A

Exemplo:

ds_list_replace(n_list, 3, nome);

O código acima substituirá o valor armazenado na posição 3 da lista por aquele armazenado na variável "name".
