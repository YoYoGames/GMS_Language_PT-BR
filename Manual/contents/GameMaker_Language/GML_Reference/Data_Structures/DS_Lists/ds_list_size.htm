ds_list_size

Esta função retornará o "tamanho" da lista, ou seja: o número de itens que foram adicionados a ela.

Sintaxe:

ds_list_size(id);

Argumento Descrição
id A identificação da estrutura de dados a ser verificada.

Devoluções:

Real

Exemplo:

if !ds_list_empty(control_list)
{
num = ds_list_size(control_list);
}

O código acima verifica uma lista DS para ver se ela está vazia ou não, e se não estiver, recebe o número de itens que contém e armazena o valor em uma variável.
