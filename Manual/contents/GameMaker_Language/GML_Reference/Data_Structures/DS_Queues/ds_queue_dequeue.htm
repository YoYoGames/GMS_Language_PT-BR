ds_queue_dequeue

Esta função irá desmarcar o valor da cabeça da fila DS, retirando-a da fila e retornando o valor a ser armazenado em uma variável. Se a fila estiver vazia, a função retornará a constante indefinido, caso contrário, ele retornará o valor real ou de cordel contido na fila.

Sintaxe:

ds_queue_dequeue_dequeue(id);

Argumento Descrição
id A identificação da fila de espera a partir da fila de espera.

Devoluções:

Valor do tipo de dados armazenados na fila ou indefinidos

Exemplo:

if !ds_queue_empty(move_queue)
{
var xx = ds_queue_dequeue(move_queue);
var yy = ds_queue_dequeue(move_queue);
move_towards_point(xx, yy, 4);
}

O código acima verifica a fila DS indexada na variável "move_queue" para ver se ela está vazia, e se não estiver, ela então retira os dois valores da cabeça da fila e os usa para definir uma direção para o movimento.
