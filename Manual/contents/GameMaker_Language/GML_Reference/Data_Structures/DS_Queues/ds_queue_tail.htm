ds_queue_tail

Esta função só lerá o último valor da fila (aquela que está "na cauda"). Ela não irá decifrar o valor, o que significa que ainda poderá ser lido no futuro por esta função ou pelo ds_queue_dequeue().

Sintaxe:

ds_queue_tail(id);

Argumento Descrição
id A identificação da estrutura de dados a ser lida.

Devoluções:

Valor do tipo de dados armazenados na fila

Exemplo:

num = ds_queue_tail(control_queue);

O código acima irá ler o valor da fila indexada na variável "control_queue" e armazenar o valor de retorno na variável "num".
