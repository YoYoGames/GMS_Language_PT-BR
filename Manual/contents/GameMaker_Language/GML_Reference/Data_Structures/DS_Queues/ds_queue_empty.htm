ds_queue_empty

Com esta função você pode verificar a fila de DS dada para ver se ela está vazia (retorna verdadeira) ou não (retorna falsa).

Sintaxe:

ds_queue_empty(id);

Argumento Descrição
id A identificação da estrutura de dados a ser verificada.

Devoluções:

Booleano

Exemplo:

if (count === 15) && (!ds_queue_empty(command_queue))
{
ds_queue_clear(command_queue);
alarm[0] = room_speed;
ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a fila DS indexada na variável "command_queue", define um alarme e redefine a variável para 0.
