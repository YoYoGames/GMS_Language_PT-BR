ds_grid_destroy

Esta função removerá da memória a estrutura de dados da grade dada, liberando os recursos que ela estava usando e removendo todos os valores que ela continha. Esta função deve ser sempre utilizada quando você terminar de usar a grade DS para evitar vazamentos de memória. que pode diminuir a velocidade e quebrar seu jogo.

IMPORTANTE! Quando você cria uma estrutura de dados, o valor do índice para identificá-lo é um valor inteiro começando em 0. Isso significa que estruturas de dados de diferentes tipos podem ter o mesmo valor de índice, portanto, em caso de dúvida, você deve estar usando o ds_exists() antes de acessá-los. Observe também que os índices são reutilizados, de modo que um valor de índice de estrutura de dados destruída pode ser usado por um índice recém-criado posteriormente, por isso recomendamos sempre definindo a variável que manteve o índice DS em -1 após a destruição.

Sintaxe:

ds_grid_destroy(index);

Argumento Descrição
índice Este índice da grade a ser destruída.

Devoluções:

N/A

Exemplo:

se (vidas == 0)
{
ds_grid_destroy(Wall_Grid);
Wall_Grid = -1;
room_goto(rm_Menu);
}

O código acima verificará o valor da variável global "vidas" construída e, se for 0, destruirá a grade DS indexada na variável "Wall_Grid" e, em seguida, mudará de sala.
