ds_grid_shuffle

Esta função pode ser usada para aleatorizar as posições de todos os valores em todas as células dentro de uma grade.

NOTA: Esta função embaralhará as células da grade para as mesmas posições toda vez que o jogo for executado novamente devido ao fato de que o GameMaker Studio 2 gera a mesma semente inicial aleatória toda vez para fazer do código de depuração uma tarefa muito mais fácil. Para evitar que este comportamento usa aleatoriedade() no início de seu jogo. Isto só é verdade ao testar e depurar o jogo, como o executável final não mostrará este comportamento e será aleatório a cada jogada.

Sintaxe:

ds_grid_shuffle(index);

Argumento Descrição
índice O índice da grade para embaralhar.

Devoluções:

N/A

Exemplo:

ds_grid_shuffle(grid)

Isto pegaria todos os valores da grade DS indexados na variável "grid" e os embaralharia para obter uma ordem diferente.
