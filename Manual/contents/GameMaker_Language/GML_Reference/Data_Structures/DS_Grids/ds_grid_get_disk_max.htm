ds_grid_get_disk_max

Esta função pode ser usada para encontrar o valor máximo para todas as células encontradas dentro da área definida do disco de uma grade, como mostrado na imagem abaixo:

Sintaxe:

ds_grid_get_disk_max(index, xm, ym, r);

Argumento Descrição
índice O índice da grade.
xm A posição x do centro do disco na grade.
ym A posição y do centro do disco na grade.
r O raio do disco na grade.

Devoluções:

Real ou String

Exemplo:

val = ds_grid_get_disk_max(grid, 5, 5, 2)

O código acima definirá a variável "val" para o valor máximo contido dentro do disco dado da grade DS indexada na variável "grid".
