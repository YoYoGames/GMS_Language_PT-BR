ds_grid_add_disk

Esta função pode ser usada para adicionar um determinado valor (real ou string) a todos os valores das células encontradas dentro da área definida do disco de uma grade. O valor a ser adicionado deve ser do mesmo tipo do que é mantido dentro das células da grade, ou seja: não se pode adicionar uma string a um real ou vice-versa, e para as cordas isto corresponde à concatenação.

Sintaxe:

ds_grid_add_disk(index, xm, ym, r, val);

Argumento Descrição
índice O índice da grade.
xm A posição x do disco na grade.
ym A posição y do disco na grade.
r O raio do disco na grade.
val O valor a ser acrescentado às células dentro do disco.

Devoluções:

N/A

Exemplo:

ds_grid_add_disk(grid, 7, 6, 5, 2)

Isto adicionaria 2 a todos os valores mantidos nas células dentro da área definida do disco da grade DS referenciada pela variável "grid".
