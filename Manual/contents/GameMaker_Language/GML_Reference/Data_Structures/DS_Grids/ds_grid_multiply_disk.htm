ds_grid_multiply_disk

Esta função tomará todos os valores em uma determinada região em forma de disco da grade DS, e multiplicará cada um deles pela quantidade determinada.

NOTA: Esta função só funcionará com números reais, não com cordas.

Sintaxe:

ds_grid_multiply_disk(index, xm, ym, r, val);

Argumento Descrição
índice O índice da grade.
xm A posição x do disco na grade.
ym A posição y do disco na grade.
r O raio do disco na grade.
val O valor para multiplicar as células dentro do disco com.

Devoluções:

N/A

Exemplo:

ds_grid_multiply_disk(mygrid, 5, 5, 5, 5, 2)

O código acima tomará todos os valores encontrados dentro da área da grade circular e multiplicará cada um por 2.
