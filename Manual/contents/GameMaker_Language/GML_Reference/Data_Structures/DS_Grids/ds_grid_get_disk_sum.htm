ds_grid_get_disk_sum

Esta função pode ser usada para somar todos os valores de todas as células encontradas dentro da área definida do disco de uma grade, como mostrado na imagem abaixo:

Sintaxe:

ds_grid_get_disk_sum(index, xm, ym, r);

Argumento Descrição
índice O índice da grade.
xm A posição x do disco na grade.
ym A posição y do disco na grade.
r O raio do disco na grade.

Devoluções:

Real ou String

Exemplo:

val = ds_grid_get_disk_sum(grid, 5, 5, 2)

O código acima definirá a variável "val" para a soma de todos os valores contidos dentro do disco dado da grade DS indexada na variável "grid".
