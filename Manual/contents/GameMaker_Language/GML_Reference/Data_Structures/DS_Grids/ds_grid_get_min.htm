ds_grid_get_min

Esta função pode ser usada para encontrar o valor mínimo para todas as células encontradas dentro da região definida de uma grade, como mostrado na imagem abaixo:

Sintaxe:

ds_grid_get_min(index, x1, y1, x2, y2);

Argumento Descrição
índice O índice da grade.
x1 A coluna da célula esquerda da região.
y1 A fileira de células superiores da região.
x2 A coluna da célula da direita da região.
y2 A fila inferior da célula da região.

Devoluções:

Real ou String

Exemplo:

val = ds_grid_get_min(grid, 0, 0, 5, 5)

O código acima definirá a variável "val" para o valor mínimo contido dentro da região determinada da grade DS indexada na variável "grid".
