Pilha de DS

Uma pilha DS é uma estrutura de dados chamada LIFO (last-in-first-out), semelhante a uma fila DS. Você pode empurrar valores para uma pilha e depois retirá-los novamente, estalando-os da pilha, onde o valor que foi empurrada para a pilha mais recentemente é a primeira a ser retirada dela novamente (basta pensar em uma pilha de moedas, onde cada moeda que você adiciona tem que ser removida novamente antes que você possa chegar ao resto das moedas abaixo). As pilhas são freqüentemente usadas quando há interrupções para manejar, ou quando há funções recursivas, ou mesmo na construção de uma IA rudimentar para seus jogos.

NOTA: Como acontece com todos os recursos dinâmicos, as estruturas de dados ocupam a memória e assim devem ser sempre destruídas quando não são mais necessárias para evitar vazamentos de memória, o que irá retardar e, eventualmente, quebrar seu jogo.

NOTA: Se você precisar verificar se existe uma estrutura de dados, você pode usar o ds_exists() função.

Existem as seguintes funções que lidam com pilhas de DS.

  • ds_stack_create
  • ds_stack_destroy
  • ds_stack_clear
  • ds_stack_empty
  • ds_stack_size
  • ds_stack_copy
  • ds_stack_top
  • ds_stack_pop
  • ds_stack_push
  • ds_stack_read
  • ds_stack_write
