ds_stack_top

Esta função só lerá o primeiro valor da pilha (aquilo que está "em cima"). Ele não irá estourar o valor, o que significa que ainda poderá ser lido no futuro por esta função ou pelo ds_stack_pop(). Se a pilha estiver vazia, então a função retornará a constante indefinida, caso contrário, retornará o valor real ou string contido na pilha.

Sintaxe:

ds_stack_top(id);

Argumento Descrição
id A identificação da estrutura de dados a ser lida.

Devoluções:

Valor do tipo de dados armazenados na pilha ou indefinidos

Exemplo:

num = ds_stack_top(control_stack);

O código acima irá ler o valor da pilha indexada na variável "control_stack" e armazenar o valor de retorno na variável "num".
