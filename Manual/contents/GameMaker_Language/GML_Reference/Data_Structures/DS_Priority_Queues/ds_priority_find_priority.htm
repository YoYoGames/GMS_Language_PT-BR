ds_priority_find_priority

Com esta função, você pode recuperar a prioridade de qualquer valor dado. Se o valor não existir na fila de prioridade, será retornado indefinido.

Sintaxe:

ds_priority_find_priority(id, val);

Argumento Descrição
id A identificação da fila prioritária a ser usada.
val O valor para encontrar a prioridade de.

Devoluções:

Real ou indefinido

Exemplo:

p = ds_priority_find_priority(ai_priority, "inteligência");

O código acima armazenará a prioridade de retorno para o valor dado na variável de instância "p".
