ds_priority_clear

Com esta função, você pode limpar todos os dados da estrutura de dados da fila prioritária dada. Isto NÃO destrói a estrutura de dados (para isso você deve usar ds_priority_destroy()) ele só limpa tudo dados dele e retorna uma fila de prioridade vazia.

Sintaxe:

ds_priority_clear(id);

Argumento Descrição
id A identificação da estrutura de dados a limpar.

Devoluções:

N/A

Exemplo:

if (count = 15) && (!ds_priority_empty(command_queue))
{
ds_priority_clear(command_queue);
alarm[0] = room_speed;
ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a fila de prioridade DS indexada na variável "command_queue", define um alarme e redefine a variável para 0.
