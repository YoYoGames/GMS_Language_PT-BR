ds_priority_change_priority

Esta função terá um determinado valor e mudará sua prioridade dentro da fila de prioridade referenciada.

Sintaxe:

ds_priority_change_priority(id, val, priority);

Argumento Descrição
id A identificação da fila prioritária para a mudança.
val O valor para mudar a prioridade de.
prioridade A nova prioridade do valor.

Devoluções:

N/A

Exemplo:

if (global.Game_Time < 1000)
{
ds_priority_change(ai_priority, AI_Search, 1);
}

O código acima verifica uma variável global e, se ela estiver abaixo de um determinado valor, alterará a prioridade do índice de função do script mantido na fila de prioridade.
