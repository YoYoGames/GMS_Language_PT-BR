ds_priority_size

Esta função devolverá o "tamanho" da fila prioritária, ou seja: o número de itens que foram priorizados nela.

Sintaxe:

ds_priority_size(id);

Argumento Descrição
id A identificação da estrutura de dados a ser verificada.

Devoluções:

Real

Exemplo:

if !ds_priority_empty(control_priority)
{
num = ds_priority_size(control_priority);
}

O código acima verifica uma fila de prioridade DS para ver se ela está vazia ou não, e se não estiver, recebe o número de itens que contém e armazena o valor em uma variável.
