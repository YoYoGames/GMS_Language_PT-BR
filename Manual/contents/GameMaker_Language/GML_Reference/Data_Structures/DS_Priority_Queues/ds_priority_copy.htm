ds_priority_copy

Esta função pode ser usada para copiar o conteúdo de uma fila prioritária para outra. Note que isto NÃO remove o conteúdo da fila de prioridade original, nem destrói a fila de prioridade original. Ao usar esta função, a prioridade A fila que está sendo copiada deve ter sido previamente criada e se ela contivesse algum item antes da cópia, então estes serão limpos primeiro (significando que esta informação será perdida).

Sintaxe:

ds_priority_copy(id, fonte);

Argumento Descrição
id A identificação da fila prioritária para a qual copiar.
fonte A identificação da fila prioritária de onde copiar.

Devoluções:

N/A

Exemplo:

com (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
p_queue = ds_priority_create();
ds_priority_copy(p_queue, other.p_queue);
}

A função acima cria uma nova instância e, nessa instância, cria uma nova fila de prioridade DS e copia o conteúdo da fila de prioridade na instância que executa o bloco de código, para a fila de prioridade de instância recém-criada.
