ds_map_values_to_array

Com esta função você pode recuperar todos os valores que um mapa DS contém para cada chave no mapa. Você fornece o DS map ID para obter as chaves de (como retornado por ds_map_create()) e o retornará um array onde cada entrada no array é um valor do mapa DS (por exemplo, se o mapa tiver 10 chaves, então um array de 10 entradas será retornado com cada entrada correspondente a um valor para uma chave). A função tem um segundo argumento opcional onde você pode fornecer um array que você criou, caso em que os dados do valor da chave do mapa serão anexados a qualquer dado existente no array. Note que a função modificará a matriz fornecida diretamente, mas também retornará uma referência a ela (ou uma referência a uma nova matriz se nenhuma for fornecida).

Sintaxe:

ds_map_values_to_array(id, [array])

Argumento Descrição
id A identificação do mapa a ser utilizado.
[matriz] OPCIONAL! A matriz para anexar os valores do mapa DS.

Devoluções:

Array

Exemplo:

var _values = ds_map_values_to_array(lvl_score);
