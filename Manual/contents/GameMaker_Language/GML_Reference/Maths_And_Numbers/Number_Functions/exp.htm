exp

Esta é a função power(e,n), onde e é aproximadamente 2,718281828 (também conhecido como Euler's Number), e n é o número de vezes que ele deve ser multiplicado por si mesmo. Isto basicamente diz "quanto crescimento eu obtenho após n unidades de tempo (e crescimento 100% contínuo)?". Assim, exp(3) retornaria 20,08, o que significa que após 3 unidades de tempo temos 20,08 vezes a quantidade com a qual começamos.

Sintaxe:

exp(n);

Argumento Descrição
n O poder do e.

Devoluções:

Real

Exemplo:

val = exp(2);

Isto colocará val para e^2, por volta de 7,4.
