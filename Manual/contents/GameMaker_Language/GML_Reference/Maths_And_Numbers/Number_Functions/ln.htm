ln

O logaritmo natural ln(n ) é a quantidade de tempo necessária para alcançar um certo nível de crescimento contínuo, onde n é o nível alcançado. Portanto, se quisermos descobrir quantas unidades de tempo precisamos para obter 20 unidades de crescimento, usaríamos ln(20 ) que devolve 2,99 unidades de tempo para obter essa quantidade de crescimento.

Sintaxe:

ln(n);

Argumento Descrição
n O valor de entrada.

Devoluções:

Real

Exemplo:

alarme[0] = ln(age) * room_speed;

O código acima usa o logaritmo natural do valor armazenado na variável "idade" para definir um alarme.
