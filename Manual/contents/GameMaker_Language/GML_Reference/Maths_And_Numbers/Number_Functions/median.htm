mediana

Esta função retorna a mediana dos valores de entrada, ou seja, o valor médio. Quando o número de argumentos é igual, o menor dos dois valores médios é retornado e a função pode ter tantos argumentos quantos forem necessários (note que mais argumentos significará que a função será mais lenta a analisar) que devem ser todos valores reais. Isto significa que, por exemplo, a mediana (43, 12, 25, 19, 6) retornaria 19, pois é o valor médio entre todos os demais.

Sintaxe:

median(val1, val2, ... max_val);

Argumento Descrição
val0 ... max_val Os valores a serem comparados.

Devoluções:

Real

Exemplo:

x = mediana( 0, x, largura_de_quarto );
y = mediana( 0, y, altura_de_quarto );
