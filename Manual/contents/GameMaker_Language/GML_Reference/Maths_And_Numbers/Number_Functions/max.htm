max

Esta função retorna o máximo dos valores de entrada, dos quais ela pode ter até 16. Por exemplo, o máximo (12, 96, 32, 75) retornará 96, pois este é o maior de todos os valores de entrada.

Sintaxe:

max(val1, val2, ... max_val);

Argumento Descrição
val0 ... max_val Os valores a serem comparados.

Devoluções:

Real

Exemplo:

x = máximo(x, 0);

Isto impedirá que o jogador saia pela esquerda da sala. Isto funciona ajustando constantemente seu x para si mesmo ou 0, o que for maior. Se o jogador sair pela esquerda, seu x será menor que 0 (ou seja, negativo), então ele ficará reto. voltar.
