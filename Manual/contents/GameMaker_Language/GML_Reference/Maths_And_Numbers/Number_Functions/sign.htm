assine

Esta função retorna se um número é positivo, negativo ou nenhum dos dois e retorna 1, -1, 0 respectivamente. Por exemplo - sinal(458) retorna 1, sinal(-5) retorna -1 e sinal(0 ) retorna 0.

Sintaxe:

sinal(n);

Argumento Descrição
n O número para obter o sinal de.

Devoluções:

Inteiro

Exemplo:

y += sinal(y - mouse_y);

O código acima adicionará 1, -1 ou 0 em y, dependendo do resultado de y - mouse_y.
