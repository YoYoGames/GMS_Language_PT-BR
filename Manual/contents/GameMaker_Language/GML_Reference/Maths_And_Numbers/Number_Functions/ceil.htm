ceil

Esta função pega qualquer número real e o arredonda para o inteiro mais próximo. Deve-se tomar cuidado com esta função, pois um erro comum é usá-la arredondando um valor aleatório e esperar que seja sempre maior que 1, ou seja, que seja sempre maior que 1:

int = ceil(aleatório(5));

Agora, você esperaria que este código sempre desse um número inteiro entre 1 e 5, mas este pode não ser sempre o caso, pois há uma possibilidade muito pequena de que a função aleatória retorne 0, e arredondar 0 ainda lhe dá 0. Este é um código remoto possibilidade mas deve ser levada em conta ao utilizar esta função.

Sintaxe:

ceil(val);

Argumento Descrição
x O número a ser mudado.

Devoluções:

Inteiro

Exemplo:

val = ceil( 3,4 );

Isto colocará Val a 4.
