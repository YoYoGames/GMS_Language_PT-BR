log2

Esta função basicamente pergunta "quantos 2's você precisa multiplicar para obter n?". Um exemplo comum de uso em programação seria calcular o número de bits que são necessários para representar um número. Para isso, calculamos o log2(x ) do número, arredondamos para baixo e adicionamos 1 - por exemplo log2(100) retorna 6,643856, que arredondado para baixo é 6, ao qual adicionamos um e nos dá 7. Portanto, precisamos de 7 bits para representar 100. Ele também pode ser usado para descrever o crescimento exponencial ou decadência, e é freqüentemente chamado de logaritmo binário.

Sintaxe:

log2(n);

Argumento Descrição
n O valor de entrada.

Devoluções:

Real

Exemplo:

colorido = piso(log2(cor)) - 1;

O código acima armazena o número de bits necessários para representar o valor armazenado na variável "cor".
