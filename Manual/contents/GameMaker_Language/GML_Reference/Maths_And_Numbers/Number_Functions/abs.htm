abs

Esta função retorna o valor absoluto do argumento de entrada, portanto, se for um valor positivo, permanecerá o mesmo, mas se for negativo, será multiplicado por -1 para torná-lo positivo.

Sintaxe:

abs(val)

Argumento Descrição
val O número para se tornar absoluto.

Devoluções:

Real

Exemplo:

x += abs (x - mouse_x)

Isto adicionará uma quantia igual ao valor absoluto da posição x atual da instância e da posição x do mouse.
