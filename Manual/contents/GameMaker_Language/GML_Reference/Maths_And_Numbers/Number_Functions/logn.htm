logn

Esta função é similar à função de log2(n) e log10(n) somente você estipula o valor base de registro. Por exemplo, o logn(5,25) retornará quantos 5's precisamos para multiplicar para conseguir 25 (que é 2).

Sintaxe:

logn(n, val);

Argumento Descrição
n A base de toros.
val O valor de entrada.

Devoluções:

Real

Exemplo:

logval = logn(5, num);

O código acima recebe o log5 do valor armazenado em "num".
