Funções da Matriz

Uma matriz é um conjunto de números dispostos em um número fixo de linhas e colunas. Normalmente os números são números reais, mas isso nem sempre tem que ser o caso. A imagem a seguir mostra em geral como uma matriz é construída com quatro filas e quatro colunas:

A linha superior é a linha 1, a coluna mais à esquerda é a coluna 1, e esta matriz é uma matriz 4x4 porque tem quatro filas e quatro colunas (outras matrizes de tamanho podem ser construídas). Na descrição das matrizes, o formato é sempre linhas / colunas, e cada número que compõe uma matriz é chamado de um elemento da matriz. Os elementos de uma matriz têm locais específicos, descritos por sua posição de linha e coluna, onde o canto superior esquerdo da matriz é a linha 1 coluna 1. No exemplo da matriz acima, o elemento na linha 1 col 1 é o valor "1". O elemento na linha 2 coluna 3 é o valor "1". valor "4,8".

As matrizes são muito importantes para muitas coisas, mas são mais usadas para lidar com o espaço 3D, por exemplo, podem ser usadas para definir a visão da câmera ou para traduzir ou transformar um modelo. As seguintes funções existem para lidar com matrizes no GameMaker Studio 2:

  • matrix_get
  • matriz_set
  • matrix_build
  • matrix_multiply
  • matrix_build_identity
  • matrix_build_lookat
  • matrix_build_projection_ortho
  • matrix_build_projection_perspective
  • matrix_build_projection_perspective_fov
  • matrix_transform_vertex
  • is_matrix

As seguintes funções são para uso de uma pilha matricial, que é semelhante a uma pilha DS, mas projetada para uso apenas com matrizes. Esta é uma maneira útil de aplicar operações com matrizes múltiplas (como transforma) um após o outro ao criar uma cena 3D.

  • matrix_stack_is_empty
  • matrix_stack_clear
  • matrix_stack_set
  • matrix_stack_push
  • matrix_stack_pop
  • matrix_stack_top
