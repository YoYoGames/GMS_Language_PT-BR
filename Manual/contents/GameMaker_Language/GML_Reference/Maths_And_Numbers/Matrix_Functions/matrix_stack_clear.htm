matrix_stack_clear

Esta função limpa a pilha matricial.

Sintaxe:

matrix_stack_clear();

Devoluções:

N/A

Exemplo:

if !matrix_stack_is_empty()
{
matrix_stack_clear();
}

O código acima verifica a pilha matricial para ver se está vazia e se não está limpa.
