dia_semana atual

Esta variável lida somente retornará o dia da semana como um valor, onde domingo é 0 e sábado é 6.

Sintaxe:

dia_semana atual;

Devoluções:

Real

Exemplo:

var dia;
switch(current_weekday)
{
caso 0: dia = "domingo"; intervalo;
caso 1: dia = "segunda-feira"; intervalo;
caso 2: dia = "terça-feira"; intervalo;
caso 3: dia = "quarta-feira"; intervalo;
caso 4: dia = "quinta-feira"; pausa;
caso 5: dia = "sexta-feira"; pausa;
caso 6: dia = "sábado"; pausa;
}
draw_text(32, 32, "Hoje é " + dia +");

O código acima usa o valor do dia_semana atual para definir uma variável com o dia correto no texto, depois desenha isso para que o usuário veja.
