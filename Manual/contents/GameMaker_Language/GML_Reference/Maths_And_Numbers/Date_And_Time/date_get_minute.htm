data_get_minute

Esta função retorna o minuto do valor da data e hora determinados.

Sintaxe:

data_get_minute(data);

Argumento Descrição
data A data a ser verificada.

Devoluções:

Real

Exemplo:

myminute = date_get_minute(date_current_datetime()));

Isto definiria "myminute" para o minuto atual.
