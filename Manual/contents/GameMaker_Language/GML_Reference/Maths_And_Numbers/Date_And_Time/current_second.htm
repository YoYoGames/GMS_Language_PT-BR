segundo_actual

Esta variável lida somente retornará os segundos que correspondem ao momento atual.

Sintaxe:

segundo_actual;

Devoluções:

Real

Exemplo:

draw_text(32, 32, "O tempo é " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

O código acima desenharia a hora internacional atual na tela.
