data_get_year

Esta função retorna o ano da data indicada.

Sintaxe:

data_get_year(data);

Argumento Descrição
data A data a ser verificada.

Devoluções:

Real

Exemplo:

myyear = date_get_year(date_current_datetime()));

Isto definiria "meu ano" para o ano atual.
