data_apoio_ano

Esta função retornará verdadeiro se o componente de ano do valor da data/hora dado for um ano bissexto ou falso de outra forma. Esta pode ser uma função útil para coisas como Ovos de Páscoa em seus jogos, ou para desbloquear conteúdo especial.

Sintaxe:

date_leap_year(date);

Argumento Descrição
data A data a ser usada.

Devoluções:

Booleano

Exemplo:

if date_leap_year(date_current_datetime())
{
if !global.ExtraContent
{
global.ExtraContent = true;
}
}

O código acima verificará a data atual para ver se o ano é um ano bissexto ou não. Se for, definirá uma variável global.
