get_string

Isto cria uma janela pop-up mostrando uma mensagem padrão, com um botão chamado "Ok", que solicita ao usuário a entrada de uma string. A função retornará a string de entrada, ou o valor padrão, se nada tiver sido inserido.

OBSERVAÇÃO: Esta função é para uso de depuração somente em alvos Desktop, mas é depreciada em todos os outros alvos.

Sintaxe:

get_string(string, def);

Argumento Descrição
str O fio a ser mostrado na mensagem pop-up.
def A cadeia padrão na caixa de texto.

Devoluções:

Cordão

Exemplo:

global.test_name = get_string("Test highscore name:", "Anônimo");

O código acima solicitará que o usuário dê um nome que será então armazenado na variável global "test_name". Se nada for inserido e o usuário apenas pressionar "Ok", então o valor padrão, "Anônimo", será retornado.
