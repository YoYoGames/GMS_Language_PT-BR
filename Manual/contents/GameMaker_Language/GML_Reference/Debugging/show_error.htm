show_error

Esta função mostrará uma string personalizada como uma mensagem de erro e dará a opção de abortar o jogo posteriormente, se necessário.

NOTA: Esta função é apenas para uso de depuração.

Sintaxe:

show_error(str, abort);

Argumento Descrição
str O fio a ser mostrado na mensagem pop-up.
abortar Se abortar (verdadeiro) ou não (falso).

Devoluções:

N/A

Exemplo:

if (room != rm_Dungeon)
{
show_error("Error: Went to wrong area. Aborting game.", true);
}

O código acima verificará se a sala em que o jogador está é "rm_Dungeon" e, caso contrário, mostrará uma mensagem de erro e forçará o usuário a abortar o jogo.
