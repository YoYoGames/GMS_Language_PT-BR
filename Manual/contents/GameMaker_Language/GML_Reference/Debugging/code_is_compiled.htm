code_is_compiled

Esta função retornará verdadeiro se a plataforma se compilar fora da máquina virtual, tal como para as plataformas YYC e JS.

Sintaxe:

code_is_compiled();

Devoluções:

Booleano

Exemplo:

se code_is_compiled()
{
show_debug_message("Compilador ok!");
}

O código acima verifica se o jogo foi compilado usando o Compilador YoYo e mostra uma mensagem de debug se foi.
