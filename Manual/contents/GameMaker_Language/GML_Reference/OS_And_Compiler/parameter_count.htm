parâmetro_contagem

Os parâmetros de linha de comando são aqueles comandos extras que você pode adicionar a um exe para mudar a forma como o programa é executado. Você pode encontrar o número de parâmetros para o jogo atual usando esta função, onde o primeiro parâmetro tem índice 1 e o último tem o índice retornado pela função (um valor de 0 é especial, pois é o nome do arquivo do jogo executável, incluindo o caminho). Deve-se notar que esta função funcionará na plataforma HTML5, recuperando os parâmetros da url.

Sintaxe:

parâmetro_count();

Devoluções:

Real

Exemplo:

p_num = parameter_count();

O código acima irá armazenar o número de parâmetros de linha de comando que foram usados para o jogo na variável "p_num".
