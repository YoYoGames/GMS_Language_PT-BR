os_version

Esta variável lhe dirá o número da versão do sistema operacional que está rodando seu jogo. Por exemplo, se você estiver executando no Windows 10, os_version será igual a 655360.

NOTA: Caso você precise de mais informações sobre o sistema operacional Windows, você pode usar o ambiente_get_variable().

Sintaxe:

os_version

Devoluções:

Real

Exemplo:

if (os_type == os_android) && (os_version > 10)
{
global.GFX = 1;
}

O código acima verifica o tipo e o número da versão e ambos estão corretos, então a variável global é definida como 1.
