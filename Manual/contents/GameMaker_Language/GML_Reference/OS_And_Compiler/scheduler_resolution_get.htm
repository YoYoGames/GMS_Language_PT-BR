agendador_resolução_get

Esta função é utilizada para recuperar a resolução do agendador de fios do Windows em milissegundos. Se a resolução do agendador for definida para o valor padrão (como definido pelo Windows), a função retornará -1.

Para obter informações sobre o agendador de fios, consulte a página para scheduler_resolution_set().

NOTA: Esta função é para Windows somente.

Sintaxe:

programador_resolução_get();

Devoluções:

Real (ou -1 para padrão)

Exemplo:

agendador_resolução = agendador_resolução_get();

Este exemplo recupera a resolução do agendador de fios do Windows, e a armazena na variável "scheduler_resolution".
