os_get_config

Esta função retorna o nome (como um fio) da configuração atualmente selecionada para seu jogo. Para mais informações sobre as configurações, consulte a seção Configurações.

Sintaxe:

os_get_config()

Devoluções:

Cordão

Exemplo:

if os_get_config() = "Free_Version"
{
global.Ads = true;
}
else global.Ads = falso;

O código acima irá verificar qual configuração está sendo usada e se for a chamada "Free_Version", os anúncios serão habilitados no jogo.
