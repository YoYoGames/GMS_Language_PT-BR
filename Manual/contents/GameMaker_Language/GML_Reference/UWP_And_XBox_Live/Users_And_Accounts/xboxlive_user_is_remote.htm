xboxlive_user_is_remote

Esta função verificará o ID do usuário dado e retornará verdadeiro se o jogador for um jogador remoto, ou falso de outra forma.

Sintaxe:

xboxlive_user_is_remote(user_id);

Argumento Descrição
user_id O ponteiro de identificação do usuário a ser verificado.

Devoluções:

Booleano

Exemplo:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
var _uid = xboxlive_get_user(i);
se xboxlive_user_is_active(_uid)
{
if !xboxlive_user_is_remote(_uid)
{
global.Remote_Player_ID[global.PlayerNum++] = _uid;
}
}
}

O código acima percorre as contas de usuário e depois verifica se alguma delas está ativa. Se estiverem, seu ID de usuário é adicionado a uma matriz global, somente se não forem usuários remotos.
