Usuários e contas

Esta seção é para usuários direcionados às plataformas UWP ou Xbox One e descreve as funções do XBox Live disponíveis para lidar com usuários e contas de usuários.

O Xbox One permite que vários usuários se conectem a um console simultaneamente. Cada usuário tem um controlador específico associado a ele e qualquer dado de jogo salvo também é específico do usuário. Para aproveitar ao máximo este sistema, o GameMaker Studio 2 introduz um número da XBox One somente funções relacionadas com a associação entre controladores e usuários, permitindo-lhe consultar informações sobre o usuário e também controlar como são tratados os dados salvos por usuário. Você pode encontrar mais informações do desenvolvedor sobre este sistema do white paper oficial no site do desenvolvedor Microsoft.

NOTA: Note que o ID de usuário é único para cada indivíduo e é perpetuado em consoles diferentes, portanto, se o mesmo usuário assinar em dois dispositivos diferentes, este ID de usuário será o mesmo.

As seguintes funções existem para você usar com as contas XBox Live, tanto na XBox One como com a exportação regular UWP (note que esta é a funcionalidade mínima permitida para UWP sob o Programa de Criadores, conforme descrito neste artigo do helpdesk):

  • xboxlive_user_is_signed_in
  • xboxlive_user_is_signing_in
  • xboxlive_gamertag_for_user
  • xboxlive_show_account_picker

As seguintes funções só são válidas quando o XBox One é direcionado ao UWP com uma conta ID@XBox válida:

  • xboxlive_get_user_count
  • xboxlive_get_user
  • xboxlive_get_activating_user
  • xboxlive_user_is_guest
  • xboxlive_user_is_active
  • xboxlive_user_is_remote
  • xboxlive_id_de_utilizador_para_utilizador
  • xboxlive_sponsor_para_utilizador
  • xboxlive_set_rich_presence
  • xboxlive_gamedisplayname_para_utilizador
  • xboxlive_user_for_pad
  • xboxlive_pad_para_utilizador
  • xboxlive_pad_count_for_user
  • xboxlive_agegroup_for_user
  • xboxlive_gamerscore_para_utilizador
  • xboxlive_show_profile_card_for_user
  • xboxlive_reputation_for_user
  • xboxlive_sprite_add_from_gamerpicture
  • xboxlive_generate_player_id_session_id
