xboxlive_get_activating_user

Com esta função você pode recuperar o ponteiro de identificação do usuário para o usuário que lançou o jogo do console.

NOTA: Como você usa este ponteiro dependerá do que você quer fazer no jogo, mas você não deve contar com ele para ser o ID do usuário que está jogando atualmente, pois o usuário que está ativando pode fazer logout enquanto o jogo está sendo executado.

Sintaxe:

xboxlive_get_activating_user();

Devoluções:

Apontador

Exemplo:

global.activUser = xboxlive_get_activating_user();

O acima armazena o ponteiro de identificação do usuário ativador em uma variável global
