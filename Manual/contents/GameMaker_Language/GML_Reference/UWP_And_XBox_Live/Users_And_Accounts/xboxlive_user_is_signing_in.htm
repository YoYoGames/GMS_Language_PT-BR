xboxlive_user_is_signing_in

A função verificará se um usuário está em processo de login e retornará verdadeiro se for esse o caso e falso caso contrário.

Sintaxe:

xboxlive_user_is_signing_in();

Devoluções:

Booleano

Exemplo:

if !xboxlive_user_is_signed_in()
{
if !xboxlive_user_is_signing_in()
{
xboxlive_show_account_picker();
}
}
else
{
global.GamerTag = xboxlive_gamertag_for_user();
}

O código acima verifica se um usuário está conectado ao XBox Live e se ele está conectado, o código irá obter o gamertag para o usuário e armazená-lo em uma variável global para uso futuro. Se não estiverem assinados, o código verificará se eles estão no processo de assinatura, caso em que nada mais acontecerá, e se não estiverem, então abrirá o selecionador de contas, levando-os a assinar.
