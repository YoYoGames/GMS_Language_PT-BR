UWPAnd XBox Live

Esta seção delineia as funções especificamente para uso quando se visa a plataforma alvo UWP. O conjunto inicial de funções é para controlar o comportamento geral do jogo ao criar um aplicativo UWP:

  • uwp_suspend
  • uwp_is_suspending
  • uwp_is_constrained
  • uwp_was_terminated
  • uwp_was_closed_by_user
  • uwp_show_help
  • uwp_license_trial_version
  • uwp_license_trial_user
  • uwp_license_trial_time_remaining
  • uwp_check_privilege

O UWP também permite criar jogos para a plataforma XBox One usando a XBox Live API usando as seguintes funções. Estas funções exigem que você tenha verificado a opção Enable XBox Live nas Opções de Jogo UWP. Algumas destas funções exigem que o usuário esteja conectado ao XBox Live, o que o GameMaker Studio 2 tentará fazer automaticamente quando você executar o projeto. Este login "silencioso" gerará um Evento Assíncrono de Sistema que pode ser verificado para ver se foi bem-sucedido ou não antes de prosseguir. Note que, ao contrário da exportação do Xbox One Console, a exportação UWP para XBox Live permite apenas que um usuário seja conectado por vez.

IMPORTANTE! O alvo UWP só tem um mínimo de suporte para o XBox Live quando configurado como parte do Programa Criador, que é limitado a algumas funções básicas de usuário e conta (veja aqui para mais informações sobre a configuração desta configuração). Para poder usar todas as funcionalidades listadas aqui, você precisará ter uma conta ID@XBox.

Você pode encontrar todas as várias funções do XBox Live para uso com o alvo de exportação UWP nas seções listadas abaixo:

  • Usuários e contas
  • Salvando dados
  • Estatísticas e quadros de líderes
  • Fazer fósforos
