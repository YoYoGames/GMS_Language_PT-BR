xboxlive_get_savedata_user

Esta função retorna o ponteiro de identificação do usuário atualmente associado à gravação de arquivos (ou o ponteiro constante_nulo se nenhum usuário estiver sendo usado no momento). Veja xboxlive_set_set_savedata_user() para mais detalhes.

Sintaxe:

xboxlive_get_savedata_user();

Devoluções:

Pointer ou ponteiro_null

Exemplo:

if (xboxlive_get_savedata_user() != user_id[0])
{
xboxlive_set_set_savedata_user(user_id[0]);
}

O código acima verifica se um usuário está atualmente designado como alvo a ser salvo, e se não estiver, então ele é designado.
