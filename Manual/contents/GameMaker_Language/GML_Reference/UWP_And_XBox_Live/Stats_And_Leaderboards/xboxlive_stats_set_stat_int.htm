xboxlive_stats_set_stat_int

Esta função pode ser usada para definir o valor de uma estatística para o ID do usuário dado. O usuário fornece o ID de usuário conforme retornado pela (por exemplo) função xboxlive_get_user()então a cadeia estatutária para definir(se a cadeia de estatísticas ainda não existir, então uma nova estatística será criada e definida para o valor dado) e um valor (um número inteiro) para definir a estatística. Note que o nome do stat deve ser apenas alfanumérico, sem símbolos ou espaços.

Ao definir o valor stat, qualquer valor anterior será substituído, portanto cabe a você determinar se o valor stat deve ser atualizado ou não (ou seja, verificar se a pontuação mais alta é realmente a mais alta), comparando com o valor stat atual com o novo antes de defini-lo.

A função retornará -1 se houver um erro ou a identificação do usuário for inválida, ou qualquer outro valor se a função foi chamada com sucesso.

Sintaxe:

xboxlive_stats_set_stat_int(user_id, stat, value);

Argumento Descrição
user_id O ponteiro de identificação do usuário para definir a estatística para
stat A estatística a definir (um fio)
valor O valor para definir o estatuto (um número inteiro)

Devoluções:

Real

Exemplo:

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

O código acima recupera o valor para a estatística "GamesPlayed", então define a mesma estatística para esse valor mais 1.
