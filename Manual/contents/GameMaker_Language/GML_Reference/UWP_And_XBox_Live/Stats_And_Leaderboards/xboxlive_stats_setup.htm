xboxlive_stats_setup

Esta função precisa ser chamada antes que você possa usar qualquer uma das outras funções estatísticas do Xbox Live, e simplesmente inicializa as bibliotecas necessárias no sistema. O argumento "user_id" é a identificação bruta do usuário como retornada pela função xboxlive_get_user(), enquanto o "service_config" e o "title_id" são a identificação única para seu jogo no Xbox Live Dev Center.

Sintaxe:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Argumento Descrição
user_id O ponteiro de identificação do usuário a ser verificado.
service_config_id Esta é a seqüência config_id para o jogo.
title_id A identificação única para o seu jogo no Xbox Dev Center.

Devoluções:

N/A

Exemplo:

var user = xboxlive_get_user(0); xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

O código acima inicializa o sistema de estatísticas para o ID do usuário dado.
