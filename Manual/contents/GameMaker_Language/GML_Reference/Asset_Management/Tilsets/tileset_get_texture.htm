tileset_get_texture

Esta função retorna um ponteiro especial para a página de textura do conjunto de azulejos. Este valor pode então ser usado em outras funções de desenho, particularmente nas funções primitivas 2D, como bem como o Shader funções. Você pode obter mais informações sobre a página de textura retornada usando as diferentes funções de textura_ encontradas aqui.

Sintaxe:

tileset_get_texture(tileset);

Argumento Descrição
conjunto de azulejos O índice do conjunto de azulejos a ser utilizado.

Devoluções

Ponteiro (identificação da página de textura)

Exemplo:

var tex;
tex = tileset_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

O código acima desenhará uma faixa triangular de 4 vértices texturizados com a textura mantida na variável "tex".
