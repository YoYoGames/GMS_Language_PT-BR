path_end

Chamando esta função terminará o caminho atual que a instância está seguindo, como definido quando a função path_start() foi chamada...

Sintaxe:

path_end();

Devoluções:

N/A

Exemplo:

if place_meeting(x, y, obj_Blocker)
{
path_end();
}

O código acima terminará o caminho atual se a instância detectar qualquer colisão com qualquer instância do objeto em questão.
