caminho_get_kind

Os caminhos podem ser lisos ou retos (um caminho suave calcula um caminho curvo ao redor dos pontos definidores, enquanto um caminho reto vai direto de um ponto a outro). Este pode ser usada para descobrir se o caminho dado é suave(verdadeiro) ou não(falso).

Sintaxe:

path_get_kind(index);

Argumento Descrição
índice O índice do caminho a ser verificado.

Devoluções:

Booleano

Exemplo:

path_kind = caminho_get_kind(pth_Patrol);

Isto definirá a variável "path_kind" como verdadeira ou falsa, dependendo do tipo de caminho que o índice de caminho dado for.
