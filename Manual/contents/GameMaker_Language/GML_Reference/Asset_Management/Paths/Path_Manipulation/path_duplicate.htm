caminho_duplicar_duplicar

Esta função toma um caminho e o copia para um novo caminho. O novo caminho é criado no processo, e o índice é devolvido para ser usado em todas as chamadas posteriores para usar este novo caminho.

Sintaxe:

path_duplicate(index);

Argumento Descrição
índice O índice do caminho existente para duplicar.

Devoluções:

Real

Exemplo:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

O código acima escolhe um dos quatro recursos do caminho e o duplica, armazenando o índice do novo caminho na variável "mypath".
