Manipulação do caminho

No GameMaker Studio 2 você não está limitado apenas aos caminhos que você faz usando o Path Editor. Há uma série de funções que podem ser usadas para manipular e mudar esses caminhos em tempo de execução, ou mesmo para criar novos e depois mudar e manipular esses caminhos.

As seguintes funções são usadas para mudar os caminhos:

  • caminho_existe
  • caminho_add
  • caminho_ponto_add
  • caminho_mudança_ponto
  • caminho_ponto_de_inserção
  • caminho_ponto_de_apagamento
  • pontos_de_caminho
  • path_append
  • path_assign
  • path_delete
  • caminho_duplicar_duplicar
  • path_flip
  • caminho_espelho
  • path_reverse
  • caminho_rota
  • caminho_escala
  • caminho_de_série_fechada
  • caminho_tipo_de_série
  • caminho_set_precisão
  • path_shift
