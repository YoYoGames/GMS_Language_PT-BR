path_get_number

Esta função pode ser usada para retornar o número de pontos em um caminho.

Sintaxe:

path_get_number(index);

Argumento Descrição
índice O índice do caminho a ser verificado.

Devoluções:

Real

Exemplo:

if (path_get_number(pth_AI) > 1)
{
path_start( pth_AI, 4, 3, 0);
}

O código acima verifica se um caminho tem mais de um ponto nele e, em caso afirmativo, inicia o movimento da instância ao longo desse caminho.
