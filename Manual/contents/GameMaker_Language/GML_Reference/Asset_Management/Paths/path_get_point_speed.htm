caminho_escolher_pontar_velocidade

Com esta função você pode obter a velocidade do ponto (conforme definido no Editor de Caminhos ou quando você adiciona dinamicamente um ponto de caminho usando caminho_add_ponto()) expresso como uma porcentagem. Portanto, se você tiver um ponto de caminho definido para 50 no editor de caminhos, esta função retornaria 50 quando utilizada.

Sintaxe:

path_get_point_speed(index, n);

Argumento Descrição
índice O índice do caminho a ser verificado.
n A identidade do ponto a ser verificado.

Devoluções:

Real

Exemplo:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
var _px = path_get_point_x(pth_Patrol, 0);
var _py = path_get_point_x(pth_Patrol, 0);
path_change_point(pth_Patrol, 0, _px, _py, 100);
}

O código acima verifica o ponto "0" do caminho indexado por "pth_Patrol" para ver se a velocidade não é igual a 100. Se não for, então ele define a velocidade desse ponto para 100.
