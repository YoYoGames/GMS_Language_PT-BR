Variáveis de caminho

As seguintes variáveis são incorporadas a todas as instâncias quando criadas e podem ser usadas para obter informações sobre um caminho, ou para alterar certas propriedades:

  • path_index
  • posição_do_caminho
  • caminho_posição anterior
  • caminho_velocidade
  • escala_do_caminho
  • orientação_pista
  • path_endaction

Observe que você pode encontrar mais informações sobre as variáveis de instância aqui: Variáveis de instância.
