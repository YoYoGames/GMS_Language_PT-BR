posição_do_caminho

Esta função pode ser usada para obter ou definir a posição de uma instância ao longo de um caminho. O valor é normalizado de 0 - 1, portanto, se você defini-lo para, por exemplo, 0,5, a instância será movida exatamente para o meio do caminho.

Sintaxe:

posição_do_caminho;

Devoluções:

Real

Exemplo:

path_position = random(1);

O código acima coloca a instância em uma posição aleatória em qualquer lugar no caminho atual.
