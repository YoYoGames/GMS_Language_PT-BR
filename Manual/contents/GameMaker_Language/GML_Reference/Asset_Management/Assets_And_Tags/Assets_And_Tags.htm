Ativos e etiquetas

As funções listadas aqui são especificamente projetadas para obter informações de qualquer um dos ativos definidos no Navegador de Ativos, assim como as etiquetas que lhes foram atribuídas. As etiquetas são uma ferramenta muito poderosa, tanto no Navegador de Ativos como em seu código de jogo real, pois podem ser lidos e criados em tempo de execução, dando-lhe maneiras adicionais de filtrar ativos quando seu jogo está sendo executado e criar verificações condicionais ou do tipo "pai sistemas fora do sistema baseado em objetos.

Para os bens em geral, temos as seguintes funções:

  • asset_get_index
  • asset_get_type

Para lidar com o sistema de "tags" do navegador de ativos, temos as seguintes funções:

  • tag_get_asset_ids
  • tag_get_assets
  • asset_get_tags
  • asset_add_tags
  • asset_remove_tags
  • ativos_etags_de_benefícios
  • ativo_ tem_qualquer_e_etiqueta
  • marcas_de_activo
