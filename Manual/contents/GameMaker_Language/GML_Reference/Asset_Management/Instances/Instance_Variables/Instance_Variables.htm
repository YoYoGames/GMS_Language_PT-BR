Variáveis de instância

Quando você cria um novo objeto, ele virá com certas variáveis já inicializadas com valores padrão. Estas variáveis chamamos variáveis de instância, já que serão criadas para cada instância que você colocar do objeto em suas salas de jogo, e uma vez criada uma instância, os valores dessas variáveis são exclusivos para essa instância e somente para essa instância. Algumas destas variáveis você usará muito, como x e y, enquanto outras são requeridas menos dependendo do que você deseja para fazer, mas em geral elas são muito úteis e, sempre que possível, você deve procurar usá-las em vez de criar suas próprias variáveis personalizadas.

A maioria das variáveis de instância podem ser definidas, bem como lidas, permitindo-lhe alterar as propriedades e o comportamento de uma instância simplesmente ajustando o valor de uma determinada variável - você pode evitar que uma instância desenhe, por exemplo, simplesmente ajustando o visível variável embutida a falsa. Abaixo você pode encontrar as diferentes variáveis que são inicializadas para todas as instâncias de todos os objetos em seu jogo.

IMPORTANTE! Se uma instância estiver em uma seqüência, então algumas destas variáveis - como image_xscale / image_yscale / image_angle / x / y - será sobregravada quando a seqüência atualiza cada passo depois de começar a ser jogado.

Estas variáveis tratam de propriedades de instância geral:

  • id
  • visível
  • sólido
  • persistente
  • profundidade
  • camada
  • alarme

Estas variáveis tratam da posição e do movimento da instância:

  • direção
  • fricção
  • gravidade
  • direção_da_gravidade
  • hspeed
  • vspeed
  • velocidade
  • xstart
  • ystart
  • x
  • y
  • xprevious
  • yprevious

Propriedades do objeto

A variável a seguir contém a identificação única do índice do objeto a partir do qual a instância foi criada:

  • object_index

Todas estas variáveis estão relacionadas ao duende atribuído à instância e podem ser usadas para mudar o que é desenhado e como:

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale

Estas variáveis lidam com a máscara de colisão:

  • máscara_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top

Estas variáveis estão relacionadas a caminhos e como a instância interage com uma, se designada:

  • path_index
  • posição_do_caminho
  • caminho_posição anterior
  • caminho_velocidade
  • escala_do_caminho
  • orientação_pista
  • path_endaction

Estas variáveis servem para definir uma instância para utilizar uma linha do tempo:

  • timeline_index
  • linha do tempo_executada
  • linha do tempo_velocidade
  • posição_da_linha do tempo
  • timeline_loop

Sequências

As seguintes variáveis armazenam informações sobre a Sequência que pode estar controlando a instância em um determinado momento:

  • in_sequence
  • sequence_instance

  • Física
