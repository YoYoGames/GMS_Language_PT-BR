direção

Todas as instâncias no GameMaker Studio 2 têm certas propriedades "embutidas" que você pode usar e definir para governar o seu aspecto e comportamento. A direção é uma dessas propriedades e pode ser usada para definir a direção do movimento do instância quando a instância tem um velocidade Note que as direções no GameMaker Studio 2 são normalmente calculadas como 0° para a direita, 90° para cima, 180° para a esquerda e 270° para baixo, e que o gravity e gravity_direction podem modificar o valor direction quando são utilizados em seus jogos.

Sintaxe:

direção;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

se direção keyboard_check(vk_left) += 5;
se direção keyboard_check(vk_right) -= 5;

O código acima mudará a direção de movimento da instância com base em qual tecla (esquerda ou direita) é pressionada.
