ystart

Esta variável armazena a posição y inicial da instância quando ela é criada na sala pela primeira vez. Esta não é uma variável somente de leitura e pode ser definida tão bem quanto de leitura.

Sintaxe:

ystart;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

if place_meeting(x, y, obj_spike)
{
pontuação -= 100;
x = xstart;
y = ystart;
}

O código acima verificará uma colisão com uma instância de "obj_spike" e se houver uma, ela deduz 100 da pontuação e move a instância de volta para sua posição inicial.
