gravidade

gravidade é uma das variáveis "embutidas" que todas as instâncias têm e, quando definida, aplicará uma força constante no direção_da_gravidade da instância, influenciando tanto a instância velocidade e direction . Note que a gravidade é uma força cumulativa e irá acelerar o objeto se você optar por não limitar a velocidade final, e é comum que você defina esta variável para pequenas casas decimais valores como 0,01. Se você definir o gravity como 0, então nenhuma gravidade será aplicada à instância (este é o valor padrão).

Sintaxe:

gravidade;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

if !place_meeting(x, y + 1, obj_Ground)
{
gravidade = 0,01;
}
mais
{
gravidade = 0;
}

O código acima só aplicará a gravidade se a instância não encontrar nenhuma instância de "obj_Ground" abaixo dela.
