fricção

O atrito é uma das propriedades "embutidas" que as instâncias podem ter e pode ser usada para retardar a instância ao longo do tempo, quando o velocidade é diferente de zero. Funciona simplesmente pela subtração de uma quantia desde a velocidade de cada passo até que o objeto tenha uma velocidade 0, então se o atrito for ajustado, por exemplo, para 0,1 e a velocidade da instância for 1 (1 pixel por passo), ele irá diminuir a velocidade e parar após 10 passos terem passado. Observe também que o atrito é aplicado a velocidades positivas e negativas igualmente, sendo o resultado líquido sempre que o objeto tem uma velocidade 0 após um determinado tempo.

Sintaxe:

fricção;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

se abs(speed) > 0
{
fricção = 0,05;
}
mais
{
fricção = 0;
}

O código acima só aplicará o atrito se a velocidade absoluta da instância estiver acima de 0.
