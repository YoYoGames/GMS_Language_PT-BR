yprevious

Esta variável embutida retorna a posição y anterior para a instância. Esta variável será definida para a posição x atual imediatamente antes do início do evento da etapa inicial, mas também pode ser definida através de código a qualquer momento, o que significa que você pode dar-lhe seu próprio valor personalizado, caso seja necessário.

Sintaxe:

yprevious;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

se x != xprevious ||| y != yprevious
{
moved = true;
}

O código acima verifica se há alguma diferença entre os valores xprevious e yprevious e os valores x e y atuais. Se houver, define uma variável como verdadeira.
