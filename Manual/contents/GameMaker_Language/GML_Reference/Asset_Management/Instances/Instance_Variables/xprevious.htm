xprevious

Esta variável incorporada retorna a posição x anterior para a instância. Esta variável será definida imediatamente antes do início do evento da etapa inicial, mas também pode ser definida através de código a qualquer momento, o que significa que você pode dar a ela seu próprio valor personalizado caso isso seja necessário.

Sintaxe:

xprevious;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

se x != xprevious ||| y != yprevious
{
moved = true;
}

O código acima verifica se há alguma diferença entre os valores xprevious e yprevious e os valores x e y atuais. Se houver, define uma variável como verdadeira.
