instance_id_get

Com esta função você pode obter o valor de identificação único de qualquer instância a partir da lista de instâncias atualmente ativa. Você fornece o índice na lista de instâncias para obter o ID e a função retornará o valor para armazenamento em uma variável.

Sintaxe:

instance_id_get(index);

Argumento Descrição
índice O índice dentro da lista de instâncias de 0 - (contagem de instâncias - 1).

Devoluções:

Valor de identificação da instância (real)

Exemplo:

para (var i = 0; i <conto_de_exemplo; ++i;)
{
var temp_id = instance_id_get(i);
com (temp_id)
{
speed += 0.1;
}
}

O código acima passará por todas as instâncias dentro da sala e adicionará 0,1 à sua velocidade.
