font_delete

Com esta função, você pode excluir um recurso de fonte do jogo. Esta é uma remoção permanente, e mudar de sala ou reiniciar o jogo não trará a fonte removida de volta. Para isso, o jogador precisaria sair do jogo e reinicie dessa forma, portanto, tenha cuidado ao usar esta função. Em geral, ela só é necessária para liberar a memória que foi utilizada por uma fonte adicionada ao jogo através das funções font_add() ou font_add_sprite().

Sintaxe:

font_delete(ind);

Argumento Descrição
ind O índice da fonte a ser apagada.

Devoluções:

N/A

Exemplo:

font_delete(global.Font);

O código acima excluirá a fonte indexada na variável global "Font".
