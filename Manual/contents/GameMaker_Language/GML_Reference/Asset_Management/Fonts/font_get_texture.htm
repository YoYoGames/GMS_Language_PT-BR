font_get_texture

Esta função retorna um ponteiro especial para a página de textura da fonte. Este valor pode então ser usado em outras funções de desenho, particularmente no desenho em geral quando se usa primitivos como bem como o Shader funções. Você pode obter mais informações sobre a página de textura retornada usando as diferentes funções de textura_ encontradas aqui.

Sintaxe:

font_get_texture(font);

Argumento Descrição
fonte O índice da fonte a ser usada.

Devoluções:

Ponteiro (identificação da página de textura)

Exemplo:

tex = font_get_texture(fnt_Main);

O código acima terá o ponteiro de textura para a fonte indexada como "fnt_Main".
