font_get_fontname

Com esta função, você pode obter o nome real do sistema do ativo de fonte selecionado. Esta função retorna uma string e não um índice, e o nome retornado dependerá da fonte que está sendo usada. Por exemplo, o nome de uma fonte, você pode ter um recurso de fonte chamado "fnt_Main" no GameMaker Studio 2 Asset Browser, e a fonte em si pode ser a fonte do sistema Windows Arial. Neste caso, a função retornará "Arial". pois esse é o nome do sistema da fonte. Caso você precise do nome como ele aparece no Navegador de Ativos, você deve usar font_get_name().

Sintaxe:

font_get_fontname(ind);

Argumento Descrição
ind Índice da fonte a ser verificada.

Devoluções:

Cordão

Exemplo:

fnt_Name = font_get_fontname(font0);

O código acima obterá o nome do sistema de um recurso de fonte e o armazenará como uma string na variável "fnt_Name".
