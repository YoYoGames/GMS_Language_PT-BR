object_get_sprite

Esta função lhe dirá se o objeto que você está verificando tem um duende ou não, e se tiver, então ele retornará o índice desse duende, ou -1 se não tiver. Note que esta não é uma função de instância! Você pode ter um objeto que não tenha sprite enquanto uma instância desse mesmo objeto pode ter um e vice-versa, ou podem até mesmo ter diferentes sprites. Você pode definir uma instância individual de duende usando o sprite_index variável de instância.

Sintaxe:

object_get_sprite(obj);

Argumento Descrição
obj O índice do objeto a ser verificado

Devoluções:

Real (ID do ativo sprite)

Exemplo:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
sprite_index = _spr;
}

O exemplo acima verificará o sprite_index da instância contra o duende do object_index da instância. Se eles não forem os mesmos, então atribuirá o mesmo duende que o do índice de objetos para a instância.
