object_get_parent

Esta função lhe dará o índice do objeto_de qualquer pai que tenha sido atribuído ao objeto especificado, ou então retornará -100 para mostrar que o objeto não tem pai atribuído a ele, ou -1 se o objeto sendo verificado não existir. Para maiores informações sobre pais ver a seção sobre o Editor de Objetos.

Sintaxe:

object_get_parent(obj);

Argumento Descrição
obj O índice do objeto a ser verificado

Devoluções:

Real

Exemplo:

par = object_get_parent(object_index);

O código acima obterá o pai do índice do objeto para a instância executando o código e armazenará o valor de retorno na variável "par".
