objeto_existe

Esta função retorna se um objeto com o índice especificado existe ou não. Observe que isto verifica se um objeto está presente no Navegador de Ativos, e não na sala de jogos. Para isso você deve usar a função instance_exists().

Sintaxe:

object_exists(obj);

Argumento Descrição
obj O índice do objeto a ser verificado.

Devoluções:

Booleano

Exemplo:

if object_exists(obj_temp)
{
com (obj_Temp) scr_add_event();
}

O código acima verifica a variável "obj_temp" para certificar-se de que armazena o índice de um objeto e então, se o fizer, faz com que o objeto execute um script.
