audio_criar_grupo_sync

Cria um grupo de sincronização e retorna um valor de identificação único para ele que deve então ser usado em todas as outras chamadas de funções de áudio para este grupo. Se você quiser que o grupo faça um loop, então passe em true, caso contrário passe em false, mas note que se você Se quiserem que eles façam loop, todas as faixas adicionadas posteriormente precisam ter o mesmo comprimento. Note que quando você cria um grupo de sincronização, você precisará liberar a memória e os sons associados a ele quando não estiver em uso, usando o audio_destroy_sync_group() função - por exemplo, nos eventos Room End ou Destroy.

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_create_sync_group(loop);

Argumento Descrição
loop Se as pistas do grupo devem ser laçadas(verdadeiras) ou não(falsas).

Devoluções:

índice

Exemplo:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

O acima exposto cria um novo grupo de sincronização e atribui o índice do grupo à variável "sg". Quatro sons são então adicionados ao grupo, com o ganho para três deles definidos como 0. Finalmente, o grupo de sincronização é tocado.
