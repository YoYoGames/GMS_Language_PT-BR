audio_group_stop_all

Esta função interromperá todos os sons de um determinado grupo de áudio que estão tocando no momento.

Sintaxe:

audio_group_stop_all(groupID);

Argumento Descrição
groupID O índice do grupo de áudio a parar (como definido na Janela Grupos de Áudio)

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(vk_space)
{
audio_group_stop_all(audiogroup_level1);
}

O código acima verifica a tecla "espaço" e então pára toda a reprodução de áudio do grupo em questão.
