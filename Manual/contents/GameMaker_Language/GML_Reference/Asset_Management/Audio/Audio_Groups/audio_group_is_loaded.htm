grupo_de_áudio_é_carregado

Esta função verificará um grupo de áudio específico para ver se ele foi carregado na memória, pronto para uso.

Sintaxe:

audio_group_is_loaded(groupID);

Argumento Descrição
groupID O índice do grupo de áudio a verificar (como definido na Janela Grupos de Áudio)

Devoluções:

Booleano

Exemplo:

if audio_group_is_loaded(audiogroup_level1)
{
audio_group_unload(audiogroup_level1);
}

O código acima verifica se um grupo de áudio foi carregado e se ele o descarrega.
