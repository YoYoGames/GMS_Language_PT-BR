audio_pause_sound

Com esta função você pode fazer uma pausa em qualquer som que esteja tocando no momento. O som pode ser uma única instância de um som (o índice para sons individuais que estão sendo tocados pode ser armazenado em uma variável quando se usa o audio_play_sound() ou audio_play_sound_at() funções) ou um bem sonoro, em cujo caso todas as instâncias do som dado serão pausadas.

Sintaxe:

audio_pause_sound(index);

Argumento Descrição
índice O índice do som a pausar.

Devoluções:

N/A

Exemplo:

se keyboard_check_pressed(ord("P"))
{
global.Pause = !global.Pause;
if global.Pause
{
audio_pause_sound(snd_Waterfall);
}
else
{
audio_resume_sound(snd_Waterfall);
}
}

O código acima verifica se a tecla "P" está pressionada e se detecta uma, define a variável global "Pause" como verdadeira ou falsa e , em seguida, ou pausa o som indexado na variável "snd_Waterfall". ou retoma o som a partir de seu estado de pausa.
