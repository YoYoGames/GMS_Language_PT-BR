audio_get_type

Ao adicionar áudio ao GameMaker Studio 2, ele pode ser transmitido em streaming ou em memória. Se você precisa saber se um determinado índice de som é para áudio streamed ou não, você pode use esta função que retornará 1 para streamed, 0 para som na memória, e -1 se houver algum erro ou se o índice não apontar para um recurso sonoro válido.

Sintaxe:

audio_get_type(index);

Argumento Descrição
índice O índice do som a ser verificado.

Devoluções:

Real

Exemplo:

type = audio_get_type(snd_Music_1);

O código acima verifica o tipo de áudio indexado na variável "snd_Music_1" e armazena o valor retornado na variável "type".
