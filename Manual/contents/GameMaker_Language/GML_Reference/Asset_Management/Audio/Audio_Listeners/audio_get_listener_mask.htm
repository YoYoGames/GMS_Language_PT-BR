audio_get_listener_mask

Esta função retornará os dados da máscara de bits que define a máscara padrão (global) atual para os ouvintes de áudio.

Sintaxe:

audio_get_listener_mask();

Devoluções:

Real

Exemplo:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
var num = audio_get_listener_count();
global.Audio_Mask = 0; for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var m = audio_listener_get_data(info[? "máscara"]);
global.Audio_Mask = global.Audio_Mask | m;
ds_map_destroy(info);
}
audio_set_listener_mask(máscara);
}

O código acima obtém os dados atuais da máscara do ouvinte e os compara com os dados armazenados em uma variável global. Se não forem os mesmos, o código verifica o número de ouvintes disponíveis e, em seguida, faz um loop através deles e recebe seus pedaços de máscara, que são combinados para criar uma máscara única que é aplicada ao sistema de áudio para definir os ouvintes globais.
