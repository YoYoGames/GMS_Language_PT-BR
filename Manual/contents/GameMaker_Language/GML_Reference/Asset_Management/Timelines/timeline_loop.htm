timeline_loop

Esta variável retornará se a linha de tempo está em looping (verdadeiro) ou não (falso). Você pode mudar esta variável para ligar ou desligar o looping e ela funciona com uma velocidade negativa da linha do tempo (se a posição da linha do tempo for inferior a 0, ela começará novamente na linha do último momento definido).

Sintaxe:

timeline_loop;

Devoluções:

Booleano

Exemplo:

if !timeline_loop
{
timeline_loop = true;
}

O código acima verifica a linha de tempo atualmente atribuída a uma instância e, se não for o caso, o código o define como verdadeiro.
