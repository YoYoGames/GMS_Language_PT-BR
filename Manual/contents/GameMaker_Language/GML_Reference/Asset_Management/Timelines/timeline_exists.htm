linha do tempo_existe

Com esta função você pode verificar e ver se existe uma linha de tempo (retorna verdadeiro) ou não (retorna falso). Isto é particularmente útil ao criar linhas de tempo dinamicamente usando a linha de tempo_add() função, mas você deve observar que se você procurar por uma variável não inicializada (que de outra forma seria atribuída a um índice de uma linha de tempo), um erro será lançado.

Sintaxe:

timeline_exists(ind);

Argumento Descrição
ind O índice da linha do tempo a ser verificado.

Devoluções:

Booleano

Exemplo:

se a linha do tempo_existe(global.tl)
{
timeline_delete(global.tl);
}

O código acima verifica se existe uma linha de tempo e se ela está indexada na variável global "tl" e, em caso afirmativo, apaga essa linha de tempo.
