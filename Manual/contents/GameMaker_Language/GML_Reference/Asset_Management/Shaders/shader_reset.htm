shader_set_uniform_matrix_array

Esta função restabelece o alvo de sorteio e deve ser chamada quando não se deseja mais usar o sombreador atual (definido usando shader_set()).

Sintaxe:

shader_reset();

Devoluções:

N/A

Exemplo:

shader_set(shader_Glass);
draw_self();
shader_reset();

O código acima definirá um shader top a ser usado para desenhar, depois desenhar o sprite atual usado para o exemplo usando-o.
