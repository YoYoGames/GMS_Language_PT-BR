shader_get_name

Com esta função você pode recuperar o nome de um recurso de sombreamento. Você fornece o valor de identificação único para o shader obter o nome do recurso e a função retornará o nome do recurso como uma cadeia.

Sintaxe:

shader_get_name(shader);

Argumento Descrição
shader O índice do sombreador para obter o nome de.

Devoluções:

Cordão

Exemplo:

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Presentemente Rendering = " + _name);

O código acima receberá o nome do sombreador dado e o desenhará para a tela.
