nome_get_name_do_script

Esta função retornará o nome como uma seqüência do roteiro especificado. Este nome é o que foi especificado para o script no Navegador de Ativos da janela principal do GameMaker Studio 2. Para maiores informações sobre scripts, veja The Script Editor.

Sintaxe:

script_get_name(scr);

Argumento Descrição
scr O índice do roteiro do qual você quer obter o nome.

Devoluções:

Cordão

Exemplo:

scr_name = script_get_name(Help_File);

O código do exemplo acima armazenará o nome do índice do script indicado na variável "scr_name".
