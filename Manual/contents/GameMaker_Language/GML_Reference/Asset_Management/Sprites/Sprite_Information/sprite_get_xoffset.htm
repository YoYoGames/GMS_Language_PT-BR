sprite_get_xoffset

Quando você define um duende no editor do duende, você precisa definir a origem para esse duende. Este é o ponto em que o duende será "anexado" ou "desenhado" quando usado com uma instância. Esta função retorna o parente offset para o eixo x da origem com base no canto superior esquerdo sendo a posição 0,0, sendo um +x a direita e um -x a esquerda desta posição. A imagem a seguir ilustra isto:

Sintaxe:

sprite_get_xoffset(index);

Argumento Descrição
índice O índice do duende para encontrar o xoffset de.

Devoluções

Real

Exemplo:

if x - sprite_get_xoffset(sprite_index) < 0
{
x = sprite_get_xoffset(sprite_index);
}

O código acima garantirá que uma instância seja mantida dentro da sala com base no duende (para que o duende esteja sempre visível).
