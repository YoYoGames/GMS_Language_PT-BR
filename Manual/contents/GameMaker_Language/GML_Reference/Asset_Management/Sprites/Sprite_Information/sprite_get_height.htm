sprite_get_height

Com esta função, você pode encontrar a altura do bem básico sprite, sem transformar-se, em pixels.

Sintaxe:

sprite_get_height(índice);

Argumento Descrição
índice O índice do duende para encontrar a altura de.

Devoluções

Real

Exemplo:

se sprite_height != sprite_get_height(sprite_index)
{
image_yscale = 1;
}

O código acima verifica a altura do duende como está na instância atual e se houver uma diferença entre isso e o duende base original, ele repõe a escala do eixo y.
