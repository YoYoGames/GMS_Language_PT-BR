sprite_get_width

Com esta função você pode encontrar a largura do duende de recursos de base, sem transformações, em pixels.

Sintaxe:

sprite_get_width(index);

Argumento Descrição
índice O índice do duende para encontrar a largura de.

Devoluções

Real

Exemplo:

se sprite_width != sprite_get_width(sprite_index)
{
image_xscale = 1;
}

O código acima verifica a largura do duende como está na instância atual e, se houver uma diferença entre isso e o duende de base original, ele repõe a escala do eixo x.
