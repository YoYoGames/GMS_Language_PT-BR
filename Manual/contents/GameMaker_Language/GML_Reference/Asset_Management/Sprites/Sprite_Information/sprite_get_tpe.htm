sprite_get_tpe

Esta função devolverá um valor especial relacionado à posição do duende e da subimagem dada na página de textura utilizada por seu jogo. Desta forma, você pode passar o valor para extensões para uso fora do jogo compilado, contornando efetivamente o GameMaker Studio 2 funciona e permite que o sprite seja usado no conteúdo DOM que pode ser desenhado em qualquer lugar dentro da janela que contém a tela do jogo. Esta função é de particular interesse para aqueles que desejam criar botões e outras mídias interativas fora do elemento de tela do GameMaker Studio 2 em sua página principal usando a função clickable_add().

NOTA: Esta função é apenas para HTML5.

Sintaxe:

sprite_get_tpe(sprite, índice);

Argumento Descrição
sprite O índice do sprite para encontrar a entrada da página de textura.
índice O índice de imagem do duende.

Devoluções

Real

Exemplo:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

O código acima cria um botão DOM na posição (32, 32) da página em que a tela do jogo está rodando. O botão usa o sprite referenciado da página de textura como "spr_MS_Home" e quando clicado, o botão abrirá uma nova janela para o URL especificado e com as propriedades definidas para aquela janela.
