sprite_get_name

Esta função devolverá o nome como um fio do duende especificado. Este nome é o que foi especificado para o duende na árvore de recursos da janela principal do GameMaker Studio 2. Observe que isto é apenas uma cadeia de caracteres e não pode ser usado para referenciar diretamente o duende - para isso você precisaria do índice sprite. Você pode, entretanto, usar esta string para obter o índice sprite usando a string retornada junto com a função asset_get_index().

Sintaxe:

sprite_get_name(index);

Argumento Descrição
índice O índice do duende para obter o nome de.

Devoluções

Cordão

Exemplo:

str = sprite_get_name(sprite_index);

O código acima obterá o nome do índice sprite para a instância que executa o código e armazenará o valor de retorno na variável "str".
