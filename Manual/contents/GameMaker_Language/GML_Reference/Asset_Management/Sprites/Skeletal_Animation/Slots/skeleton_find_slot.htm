esqueleto_de_squadra_esqueleto

Com esta função, você pode encontrar quais ranhuras estão em uma posição específica de espaço na espinha dorsal associada à instância atual. Você cria uma lista DS e fornece sua identificação ao longo com uma posição x/y para verificação e a lista será preenchida com uma cadeia de nomes para cada um dos slots de anexo disponíveis para o sprite (incluindo quaisquer modificações de anexo). Note que a lista é sempre ordenada em ordem decrescente a partir de a ranhura mais alta.

Sintaxe:

skeleton_find_slot(x, y, list);

Argumento Descrição
x A posição x na sala a ser verificada.
y A posição y na sala a ser verificada.
lista A identificação da lista DS a ser preenchida com os mapas DS.

Devoluções:

N/A

Exemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

O código acima cria uma lista DS e depois a preenche com os nomes dos slots encontrados na posição do mouse na sala.
