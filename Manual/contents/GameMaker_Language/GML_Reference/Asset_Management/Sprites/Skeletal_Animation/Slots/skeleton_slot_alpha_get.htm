esqueleto_esqueleto_alpha_get

Com esta função, você pode obter um valor alfa do slot de fixação. Você fornece o nome do slot (um fio) e a função retornará um valor real para o valor alfa entre 0 e 1.

Sintaxe:

skeleton_slot_alpha_get(slot);

Argumento Descrição
ranhura O nome do slot a verificar, um fio

Devoluções:

Inteiro

Exemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
para (var i = 0; i < ds_list_size(slot_list); ++i;)
{
if skeleton_slot_alpha_get(slot_list[| i]) != 1
{
skeleton_slot_colour_set(slot_list[| i], c_white, 1);
}
}
}

O código acima cria uma lista DS e depois a preenche com os nomes dos slots encontrados na posição do mouse na sala. Em seguida, ele percorre a lista de slots e repõe a cor dos slots encontrados para branco com um alfa de 1 se o alfa não for já 1.
