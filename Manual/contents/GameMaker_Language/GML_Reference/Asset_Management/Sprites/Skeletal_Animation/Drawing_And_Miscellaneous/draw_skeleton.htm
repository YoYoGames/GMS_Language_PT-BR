draw_skeleton

Esta função é apenas para uso com sprites que foram criados usando um programa de animação esquelética como Spine. Enquanto você pode desenhar estes sprites usando as funções normais de desenho (como draw_self()), há momentos em que você pode querer desenhar uma única moldura ou mostrar uma mudança na textura da pele sem realmente mudar as propriedades do duende. Nestes casos, você pode desenhar o duende com esta função, onde você dá o índice do duende e depois o nome da animação para obter o quadro de (um fio, como definido dentro do programa usado para fazer o duende). Em seguida você dá o nome da pele a ser usada (novamente, como definido quando o duende foi feito), e um valor de frame. O valor da moldura é para dizer ao GameMaker Studio 2 que parte da animação mostrar e qual o valor entre 0 e o image_number. O resto dos argumentos são os mesmos usados para o desenho normal de duendes e mudarão a escala, ângulo, cor da mistura e alfa do quadro de animação final que está sendo desenhado.

ADVERTÊNCIA! Devido à forma como as animações esqueléticas são interpoladas entre vários quadros "chave", há uma sobrecarga adicional associada a esta chamada e, portanto, é recomendável que você evite desenhar sprites com base no esqueleto usando este método, a menos que seja absolutamente necessário.

Sintaxe:

draw_skeleton(sprite, animname, skinname, frame, x, y, xscale, yscale, podridão, cor, alfa);

Argumento Descrição
sprite O índice do duende a desenhar.
animname O nome da animação para obter o quadro de (um fio).
skinname O nome da pele a ser usada (um fio).
quadro O quadro de animação a desenhar (de 0 a image_number - 1).
x A coordenada x de onde desenhar o duende.
y A coordenada y de onde desenhar o duende.
xscale A escalada horizontal do duende, como um multiplicador: 1 = escalada normal, 0,5 é metade, etc...
yscale A escalada vertical do duende, como um multiplicador: 1 = escalada normal, 0,5 é metade, etc...
podridão A rotação do duende. 0=normal, 90=voltada 90 graus no sentido anti-horário, etc.
cor A cor com a qual se deve misturar o duende.
alfa O alfa do duende (de 0 a 1 onde 0 é transparente e 1 opaco).

Devoluções:

N/A

Exemplo:

draw_skeleton(spr_Jump, "jump", "skin1", 0, x, y, image_xscale, image_yscale, image_angle, c_white, 0,5);

O código acima desenhará o primeiro quadro do duende de animação esquelético dado, usando o conjunto de animação "salto", a pele definida como "skin1" e na mesma escala e posição que as propriedades da instância definem, mas com um alfa de 0,5.
