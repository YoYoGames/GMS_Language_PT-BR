esqueleto_animação_get

Com esta função você pode obter o conjunto de animação atual sendo usado por seu duende de animação esquelética. O valor de retorno é uma string, que será o nome do conjunto como você o definiu em seu programa de animação esquelética.

Sintaxe:

esqueleto_animação_get();

Devoluções:

Cordão

Exemplo:

if keyboard_check_pressed(vk_space)
{
if skeleton_animation_get() != "pular"
{
skeleton_animation_set("salto");
}
}

O código acima mudará o conjunto de animação sendo usado para o conjunto de "salto" quando a tecla de espaço for pressionada, mas somente se o conjunto atual sendo usado ainda não estiver "pulando".
