image_speed

Esta variável determina a velocidade na qual o GameMaker Studio 2 irá circular através das sub-imagens para o sprite da instância atual. O valor da velocidade dada é um multiplicador, sendo 1 o valor padrão, e definindo-o como 0,5 irá metade do velocidade de animação - como definido no Editor Sprite ou Editor de Imagem - enquanto o ajuste para 2 irá dobrá-la. Se o sprite utilizado tiver sem sub-imagens, esta variável não terá efeito.

Sintaxe:

image_speed;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

com (instance_create_layer(x, y, "Effects", obj_Explosion))
{
image_speed = 0,5;
}

O código acima cria uma instância do objeto "obj_Explosion" e define sua velocidade_de_imagem para 0,5 (efetivamente reduzindo pela metade a velocidade de execução).
