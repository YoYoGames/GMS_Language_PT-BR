máscara_index

Quando você define um objeto no GameMaker Studio 2, você pode atribuir-lhe uma máscara para ser usada em caso de colisões, em vez daquela que corresponde ao duende definido. Esta variável pode ser usada para encontrar o sprite_index dessa máscara (ou retornará -1 se nenhum duende tiver sido designado) ou para definir a máscara para uma instância para o duende escolhido. Definir o índice da máscara significa que você pode ter, por exemplo, um duende para a instância com uma forma irregular, mas ainda assim dar-lhe uma máscara de colisão circular que é obtida de um duende diferente.

Sintaxe:

máscara_index;

Devoluções:

Real (número inteiro - valor do índice de recursos)

Exemplo:

mask_index = spr_Round;

O código acima define a máscara da instância como a do duende "spr_Round".
