Variáveis de Sprite Instance

Estas variáveis estão incluídas por padrão com GameMaker Studio 2 objetos e estão todas relacionadas ao duende designado para a instância na sala. Estas variáveis podem ser acessadas a qualquer momento dentro do código ou ações de uma instância e utilizadas para mudar o que é desenhado e como.

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale
  • máscara_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
