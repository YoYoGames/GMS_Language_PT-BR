image_xscale

Este valor define a escala horizontal aplicada ao duende que foi atribuída à instância atual. Uma escala de 1 indica que não há escala (1:1), valores menores serão escalados para baixo (0,5, por exemplo, será metade da largura do duende), valores maiores escalará para cima e os valores negativos inverterão o duende e o escalarão a menos que o valor utilizado seja exatamente -1 (neste caso, o duende é simplesmente invertido sobre sua origem sem escalar).

Sintaxe:

image_xscale;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo:

se image_xscale < 5
{
image_xscale += 0.2;
image_yscale = image_xscale;
}
mais
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy();
}

O código acima escalona o duende e então, uma vez escalado para 5 vezes seu tamanho original, uma nova instância de outro objeto é criada e a instância é destruída.
