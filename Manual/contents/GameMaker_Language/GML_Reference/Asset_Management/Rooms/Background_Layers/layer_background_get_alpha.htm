fundo_da_camada_get_alpha

Esta função pode ser usada para obter o valor alfa do elemento de fundo. Você dá o ID do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e a função retornará um valor entre 0 (totalmente transparente) e 1 (totalmente opaco).

Sintaxe:

layer_background_get_alpha(background_element_id);

Argumento Descrição
background_element_id O valor único de identificação do elemento de fundo para obter as informações de

Devoluções:

Real (de 0 a 1)

Exemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_alpha(back_id) < 0.1
{
layer_background_destroy(back_id);
}

O código acima obterá a identificação da camada chamada "Background_sky" e depois a usará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para verificar o elemento alfa e se for inferior a 0,1, então o elemento da camada é destruído.
