fundo_camada_cortado

Usando esta função, você pode mudar se o elemento de fundo dado em uma camada deve ser ladrilhado horizontalmente ou não. Você dá a identificação do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e depois definir o valor do azulejo. Se definido como verdadeiro, então o elemento será exibido em mosaico horizontalmente em toda a sala, e quando definido a falso, o elemento não será azulejado. Esta função é apenas para projetos 2D, e não funcionará corretamente quando uma projeção de câmera 3D for usada.

Sintaxe:

layer_background_htiled(background_element_id, tiled);

Argumento Descrição
background_element_id O valor de identificação único do elemento de fundo a ser mudado
visível A alternância horizontal de azulejos, que pode ser verdadeira ou falsa

Devoluções:

N/A

Exemplo:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
layer_background_htiled(back, true);
layer_background_vtiled(back, true);
}

O código acima verificará o elemento de fundo atribuído à camada em que a instância que está executando o código está e, se não estiver ladrilhada em nenhuma das direções, ela o colocará em azulejo vertical e horizontalmente.
