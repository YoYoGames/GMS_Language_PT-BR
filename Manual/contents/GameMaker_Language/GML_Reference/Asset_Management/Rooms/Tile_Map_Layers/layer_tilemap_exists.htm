layer_tilemap_existe

Você pode usar esta função para verificar e ver se existe um elemento do mapa de azulejos em qualquer camada. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um fio - isso terá um impacto no desempenho) e o ID do elemento do mapa de azulejos (que você obtém ao criar um elemento do mapa de azulejos usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()) e a função retornará ou verdadeira se o elemento existir, ou falsa se não existir.

NOTA: Esta função funciona dentro do escopo da sala alvo atual - por padrão a sala na qual a função é chamada - que pode ser definida usando a função layer_set_target_room().

Sintaxe:

layer_tilemap_exists(layer_id, tilemap_element_id)

Argumento Descrição
layer_id O valor de identificação único da camada a ser alvo (ou o nome da camada como uma corda)
tilemap_element_id O valor de identificação único do elemento do mapa de azulejos a verificar

Devoluções:

Booleano

Exemplo:

var lay_id = layer_get_id("tilemap_trees");
if layer_tilemap_exists(lay_id, global.Treestilemap)
{
layer_tilemap_destroy(lay_id, global.Treestilemap);
}

O código acima verifica a camada "tilemap_trees" para ver se o elemento dado do mapa de azulejos existe e se existe, então ele é destruído (mas não a camada).
