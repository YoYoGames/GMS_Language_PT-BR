tile_set_index

Esta função pode ser usada para definir o índice de azulejos (a posição do azulejo dentro da imagem do conjunto de azulejos) para um conjunto de dados de azulejos. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e então o valor inteiro que representa a posição do azulejo (seu índice) dentro do conjunto de azulejos. A função retornará um conjunto modificado de dados do azulejo que pode então ser aplicado usando o tilemap_set() função.

Sintaxe:

tile_set_index(tiledata, index)

Argumento Descrição
tiledata os dados da telha a definir
índice O valor inteiro do índice do conjunto de azulejos a serem usados

Devoluções:

Real (tileata)

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

O código acima obtém o ID do mapa de azulejos da camada dada e depois usa-o para obter os dados do azulejo para a célula sob a posição do mouse. Estes dados são então usados para definir o índice de azulejos para um número aleatório.
