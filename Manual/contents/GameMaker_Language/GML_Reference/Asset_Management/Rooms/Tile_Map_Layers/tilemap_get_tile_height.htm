mapa de azulejos_get_tile_height

Usando esta função você pode recuperar a altura (em pixels) de cada célula de azulejo do elemento do mapa de azulejos. Você dá o ID do elemento do mapa de azulejo (que você obtém quando cria um elemento do mapa de azulejo usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e a função devolverá a altura da célula do conjunto de azulejos.

Sintaxe:

tilemap_get_tile_height(tilemap_element_id);

Argumento Descrição
tilemap_element_id O valor de identificação único do elemento do mapa de azulejos para obter a altura da célula do azulejo de

Devoluções:

Real

Exemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

O código acima usa o ID do mapa de azulejos recuperado para obter a largura e a altura da célula do azulejo e usá-los para definir duas variáveis globais.
