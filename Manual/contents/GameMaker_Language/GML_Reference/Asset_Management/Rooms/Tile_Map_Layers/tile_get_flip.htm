tile_get_flip

Esta função pode ser usada para verificar se em um determinado conjunto de dados da telha a telha foi virada ou não. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e a função retornará verdadeiro se o azulejo estiver virado, ou falso se não estiver.

Sintaxe:

tile_get_flip(tiledata)

Argumento Descrição
tiledata os dados dos azulejos a serem verificados

Devoluções:

Booleano

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_flip(data);
data = tile_set_flip(data, bool);
tilemap_set(map_id, data, mx, my);

O código acima obtém o ID do mapa de azulejos da camada dada e depois obtém a posição das células x e y para o azulejo sob o mouse. Esta posição é então usada para obter os dados do azulejo que é virado e depois usado para definir o azulejo novamente.
