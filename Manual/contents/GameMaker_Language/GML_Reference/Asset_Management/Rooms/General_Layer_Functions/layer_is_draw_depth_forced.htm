layer_is_draw_depth_forced

Esta função pode ser usada para verificar e ver se a profundidade Z forçada está habilitada para renderizar as camadas no projeto. Ver layer_force_draw_depth() para mais informações.

Sintaxe:

layer_is_draw_depth_forced()

Devoluções:

Booleano

Exemplo:

se !layer_is_draw_depth_forced()
{
layer_force_draw_depth(true, 0);
}

O código acima verifica se a profundidade da camada Z é forçada ou não e, se não for, fixa a profundidade Z em 0 e a habilita.
