largura_da_sala

Esta variável mantém a largura da sala atual em pixels. Você pode mudar esta variável para mudar a largura da sala a qualquer momento.

Sintaxe:

largura_da_sala;

Devoluções:

Real

Exemplo:

se bbox_right > room_width
{
x += room_width - bbox_right;
}

O código acima verifica se a caixa delimitadora da instância atual é maior do que a largura da sala, e se é ela que move a instância para cima.
