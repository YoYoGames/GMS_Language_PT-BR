camada_escrita_escala_x

Usando esta função você pode mudar se o elemento sprite dado em uma camada deve ou não ser escalado ao longo do eixo x. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir o valor da escala. Uma escala de 1 indica que não há escala (1:1), valores menores serão escalados para baixo (0,5, por exemplo, terá a metade da largura do sprite usado), valores maiores serão escalados, e valores negativos virarão o duende e o escalarão a menos que o valor usado seja exatamente -1 (neste caso o duende usado é apenas virado da direita para a esquerda sobre sua posição de origem sem escalar).

Sintaxe:

layer_sprite_xscale(sprite_element_id, xscale);

Argumento Descrição
sprite_element_id O valor de identificação único do elemento sprite a mudar
xscale O valor em escala x (padrão é 1)

Devoluções:

N/A

Exemplo:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
layer_sprite_xscale(asset_sprite, 1);
layer_sprite_yscale(asset_sprite, 1);
}

O código acima verificará o elemento sprite atribuído à camada em que a instância que está executando o código está e, se for escalado em qualquer direção, ajustará tanto a escala do eixo x quanto a escala do eixo y para 1.
