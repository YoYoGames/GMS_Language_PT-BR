layer_sprite_exists

Você pode usar esta função para verificar e ver se existe um elemento sprite em qualquer camada. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o (como uma string - isto terá um impacto no desempenho) e o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()) e a função retornará ou verdadeira se o elemento existir, ou falsa se não existir.

NOTA: Esta função funciona dentro do escopo da sala alvo atual - por padrão a sala na qual a função é chamada - que pode ser definida usando a função layer_set_target_room().

Sintaxe:

layer_sprite_exists(layer_id, sprite_element_id)

Argumento Descrição
layer_id O valor de identificação único da camada a ser alvo (ou o nome da camada como uma corda)
sprite_element_id O valor de identificação único do elemento sprite a ser verificado

Devoluções:

Booleano

Exemplo:

var lay_id = layer_get_id("Asset_Trees");
se layer_sprite_exists(lay_id, global.TreesSprites)
{
layer_sprite_destroy(lay_id, global.TreesSprites);
}

O código acima verifica a camada "Asset_trees" para ver se o elemento sprite dado existe e se existe, então ele é destruído (mas não a camada).
