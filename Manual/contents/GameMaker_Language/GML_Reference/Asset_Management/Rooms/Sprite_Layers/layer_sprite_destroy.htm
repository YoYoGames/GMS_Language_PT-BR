layer_sprite_destroy

Esta função destruirá o elemento sprite dado. Você fornece o ID do duende (que você obtém quando cria o duende usando layer_sprite_create() ou quando você usa a identificação da camada junto com layer_get_sprite_id()) e isto irá removê-lo. Note que isto não remove a camada, apenas o duende dela, e se o duende for um que foi adicionado no editor da sala, então na próxima vez que você sair da sala e depois retornar, o duende será recriado. novamente. Entretanto, se o quarto for persistente, o duende será removido a menos que a persistência do quarto seja desligada novamente.

Sintaxe:

layer_sprite_destroy(sprite_element_id)

Argumento Descrição
sprite_element_id O valor de identificação único do duende a ser destruído

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Asset_Trees");
se layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
layer_sprite_destroy(lay_id);
}

O código acima verifica a camada "Asset_Trees" para ver se o elemento sprite dado existe e se existe, então ele é destruído (mas não a camada).
