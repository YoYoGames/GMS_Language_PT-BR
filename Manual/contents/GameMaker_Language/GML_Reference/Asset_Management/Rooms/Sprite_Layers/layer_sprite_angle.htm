layer_sprite_angle

Usando esta função você pode mudar o ângulo para o elemento sprite dado em uma camada. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usar a função layer_sprite_get_id()), e então definir o valor do ângulo, de 0 a 359 no sentido anti-horário, onde 0 é para a direita, 90 é para cima, 180 é para a esquerda e 270 é para baixo. Se você definir um valor superior a 360 este valor será looped para colocá-lo dentro da faixa 0 - 359.

Sintaxe:

layer_sprite_angle(sprite_element_id, angle_id);

Argumento Descrição
sprite_element_id O valor de identificação único do elemento sprite a mudar
ângulo O ângulo do duende (padrão é 0)

Devoluções:

N/A

Exemplo:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
layer_sprite_angle(asset_sprite, 0);
}

O código acima verificará o elemento sprite atribuído à camada em que a instância que executa o código está e, se seu ângulo não for 0, ele o fixará em 0.
