nome_do_quarto_get_name

Esta função pode ser usada para retornar o nome da sala especificada como um fio. Observe que isto é apenas um fio e não pode ser usado para referenciar diretamente a sala - para isso você precisaria do índice da sala. Você pode, entretanto, usar isto para obter o índice da sala usando a corda retornada junto com a função asset_get_index().

Sintaxe:

room_get_name(index);

Argumento Descrição
índice O índice da sala para verificar o nome da.

Devoluções:

Cordão

Exemplo:

var roomname = room_get_name(room);
draw_text(32, 32, roomname);

O código acima obterá o nome da sala atual e o desenhará na tela.
