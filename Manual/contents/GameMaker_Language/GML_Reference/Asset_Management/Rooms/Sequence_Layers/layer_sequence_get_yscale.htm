camada_seqüência_agenda_escala_de_série

Com esta função você fornece a identificação do elemento de seqüência - como retornado por layer_sequence_create() ou por uma das funções dos elementos de camada - e retornará a escala atual ao longo do eixo Y do elemento de seqüência na sala de jogos.

Sintaxe:

layer_sequence_get_yscale(sequence_element_id)

Argumento Descrição
sequence_element_id O valor de identificação único do elemento de seqüência para o alvo

Devoluções:

Real

Exemplo:

var _ys = layer_sequence_get_yscale(title_sequence)
if _ys < 1
{
_ys += 0.01;
layer_sequence_yscale(title_sequence, _ys);
}

O código acima recupera a escala atual ao longo do eixo Y do elemento de seqüência com o ID armazenado na variável "title_sequence", e se for menor que 1, então 0,01 é adicionado à escala Y atual.
