layer_sequence_get_length

Com esta função você fornece a identificação do elemento de seqüência - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e ele retornará o comprimento da seqüência. Este é o número de quadros para os quais a seqüência será executada.

Sintaxe:

layer_sequence_get_length(sequence_element_id)

Argumento Descrição
sequence_element_id O valor de identificação único do elemento de seqüência para o alvo

Devoluções:

Inteiro

Exemplo:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

O código acima recupera o número de quadros que uma seqüência executará e então usa este valor para definir um alarme.
