layer_sequence_get_angle

Com esta função você fornece a identificação do elemento de seqüência - como retornado por layer_sequence_create() ou por uma das funções dos elementos de camada - e retornará o ângulo atual do elemento de seqüência na sala de jogos. Note que os ângulos são retornados em graus, e 0º é para a direita, 90º é para cima, 180º é para a esquerda e 270º é para baixo.

Sintaxe:

layer_sequence_get_angle(sequence_element_id)

Argumento Descrição
sequence_element_id O valor de identificação único do elemento de seqüência para o alvo

Devoluções:

Real

Exemplo:

var _ang = layer_sequence_get_angle(title_sequence)
if _ang > 0
{
_ang -= 1;
layer_sequence_angle(title_sequence, _ang);
}

O código acima recupera o ângulo atual do elemento de seqüência com o ID armazenado na variável "title_sequence", e se não for 0, então 1 é subtraído do ângulo atual.
