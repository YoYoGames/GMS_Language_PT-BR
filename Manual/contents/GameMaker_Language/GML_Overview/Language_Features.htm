Características lingüísticas

A linguagem GameMaker(GML) tem uma série de características que fazem parte da estrutura essencial de todos os programas escritos com a linguagem. Cada um deles será usado em qualquer código que você escreva várias vezes, portanto você deve saber o que eles são e como eles trabalhar antes de começar a programar um projeto.

  • se/outro e Operadores Condicionais
  • repita
  • enquanto
  • fazer / até
  • para
  • interruptor
  • intervalo
  • continuar
  • saída
  • com
  • retornar
  • início / fim
  • tentar / apanhar / finalmente
  • jogar
  • novo
  • excluir
