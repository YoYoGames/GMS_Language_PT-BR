argumento

Esta variável contém um array que é usado junto com a contagem de argumentos de variáveis somente de leitura em funções ou métodos de script. Cada posição de array contém um valor de entrada para a função e é especificamente para uso com funções de argumento de variável .

Note que há também uma série de variáveis independentes de escopo global que também podem ser usadas em funções definidas pelo usuário para referenciar os diferentes argumentos de entrada: argument0, argument1, argument2, etc... até .argument15

NOTA: Isto não é mais estritamente necessário e é fornecido, mais do que qualquer outra coisa, para o apoio ao legado. Todas as funções definidas pelo usuário aceitarão argumentos variáveis, e qualquer argumento passado para uma função que não faça parte da função nomeada. as entradas serão inicializadas para indefinidas por padrão, o que pode ser verificado usando a função is_undefined().

Sintaxe:

argumento[n]
argumento1
argumento2
...
argumento15

Devoluções:

Valor (pode ser de qualquer tipo de dado fornecido para a função)

Exemplo:

function my_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
if argument_count > i
{
arg[i] = argumento[i]
}
mais
{
arg[i] = -1;
}
}
}

O código acima usa a variável argument_count junto com o array argument para inicializar variáveis, e é um exemplo de como permitir que uma função definida pelo usuário aceite um número de argumentos de uma variável.
