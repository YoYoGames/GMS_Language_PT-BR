pontuação

Esta variável é global no escopo e é usada para manter um valor numérico que normalmente é usado para a pontuação do jogador. Esta variável é projetada apenas para suportar projetos legados de versões anteriores de GameMaker e deve não ser utilizado em novos projetos como pode ser depreciado no futuro.

Sintaxe:

pontuação;

Devoluções:

Real (valor de ponto flutuante de precisão único)

Exemplo

switch (object_index)
{
case obj_Enemy_Fighter:
score += 10;
break;
case obj_Enemy_Mage:
score += 25;
break;
case obj_Enemy_Boss:
score += 100;
break;
}
instance_destroy();

O código acima verifica o índice de objetos da instância executando o código usando um switch, e então adiciona diferentes valores à variável score, dependendo do objeto que é.
