argument_count

Esta variável somente de leitura retorna o número de "argumentos" que são passados para uma função de roteiro ou um método. Normalmente usada em conjunto com um conjunto de argumentos (argumento[0....15]) para permitir diferentes argumentos de entrada para uma determinada função.

NOTA: Isto não é mais estritamente necessário e é fornecido, mais do que qualquer outra coisa, para o apoio ao legado. Todas as funções definidas pelo usuário aceitarão argumentos variáveis, e qualquer argumento passado para uma função que não faça parte da função nomeada. as entradas serão inicializadas para indefinidas por padrão, que podem ser verificadas usando a função is_undefined().

Sintaxe:

argument_count;

Devoluções:

Inteiro

Exemplo:

function my_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
if argument_count > i
{
arg[i] = argumento[i]
}
mais
{
arg[i] = -1;
}
}
}

O código acima usa a variável argument_count junto com o array argument para inicializar variáveis, e é um exemplo de como permitir que uma função definida pelo usuário aceite um número de argumentos de uma variável.
