O Micro Servidor Web

Esta pequena ferramenta tem como função principal conectar e servir arquivos a algumas das plataformas alvo disponíveis para o GameMaker Studio 2 (como o alvo HTML5, por exemplo). Quando você executa um jogo em um alvo que requer o Micro Web Server, você obterá uma nova abertura de janela à medida que o jogo é servido para o dispositivo escolhido:

NOTA: A primeira vez que você executa um jogo em um alvo que requer o Micro Web Server, é possível que você receba um alerta de firewall. Você deve permitir que o servidor web se comunique através do firewall, caso isso aconteça.

O servidor Web mostrará uma certa quantidade de informações de depuração relacionadas aos arquivos que foram servidos, e tem um botão para limpar isso na parte inferior. Ao lado do botão "Clear" está o caminho para o arquivo JS raiz do projeto, enquanto por baixo que você tem a opção Network Speed. Esta está definida como "Sem limite" por padrão, mas clicando nela, será aberta uma lista de diferentes opções de rede:

A mudança desta configuração também mudará a opção de tamanho do pacote para a direita. Você também pode editar isto manualmente, Neste caso, a velocidade da rede será definida como "Personalizada". Configurá-la para qualquer outra coisa que não seja o valor máximo irá efetivamente acelerar a velocidade de transferência da rede e permitir que você teste jogos em um ambiente de baixa largura de banda.

Você também pode ver a lista de IPs permitidos junto com o número da porta sendo usada, mas estes não podem ser editados no Micro Web Server e devem ser definidos nas configurações do Device Manager. para o alvo do dispositivo em questão.
