Gerador de imagens do projeto

O Gerador de Imagens do Projeto é acessado do Menu de Ferramentas em o topo da IDE GameMaker Studio 2. Esta ferramenta foi projetada para criar rapidamente todas as diferentes telas de splash, imagens promocionais e ícones que são exigidos por todas as diferentes plataformas alvo a partir de um único conjunto de imagens. Você fornece a imagem base e o ícone base e depois clique no botão Gerar , e suas Opções de Jogo para cada plataforma serão atualizadas com as novas imagens.

Para que isto funcione corretamente, você precisa fornecer uma tela de alta resolução e uma imagem de ícone de alta resolução. Os tamanhos recomendados seriam:

  • Splash Screen: pelo menos 1920x1080px, ou 1080x1920px para retrato, e de autoria de 24bit *.png, sem transparências.
  • Ícones: pelo menos 1024x1024px (que é o tamanho do maior ícone necessário), e de autoria de 24bit *.png que pode ter transparências

Uma vez fornecidas estas imagens, elas serão mostradas nas janelas de pré-visualização e você poderá então definir o tipo de "ajuste" a ser usado ao aplicá-las nas diferentes plataformas. Você pode escolher entre as seguintes opções:

  • Keep Aspect Ratio - isto irá dimensionar a imagem para se ajustar mantendo a relação de aspecto e adicionando uma cor de fundo para preencher os lados se o ajuste não for 1:1
  • Stretch To Fit - isto esticará as imagens para caber no tamanho total da imagem/índice.
  • Crop To Fit - isto levará a imagem de base e a fará caber melhor ao longo do eixo horizontal ou vertical e, em seguida, cortar o restante. Por exemplo, se você tiver uma imagem 640x480 que tem que ser usada como tela de respingo para 480x800, então a imagem será escalada para se ajustar ao eixo vertical (480 -> 800) e o resto da imagem será cortada. Da mesma forma, se você tiver uma imagem de 1080x1920 que tem que caber 800x800, então a imagem será escalada para baixo para caber no eixo horizontal eixo e o resto cortado.

Você também pode definir a cor de fundo tanto para os ícones quanto para as imagens. Esta cor será usada para "almofadar" a imagem quando dimensionada com relação de aspecto. A imagem a seguir mostra como isto ficará uma vez que tudo tenha sido configurado:

Agora basta clicar em Generate e o GameMaker Studio 2 irá preencher todas as Opções de Jogo de Imagem e Configurações de ícones com imagens para cada plataforma.

OBSERVAÇÃO: A exceção à declaração acima é o Android Adaptive Icons. Estes não serão gerados usando esta ferramenta devido à forma como estão estruturados.

IMPORTANTE! Embora esta ferramenta gere todas as imagens que você precisa, isso não significa que seja perfeita! Ela tentará fazer o melhor para que as imagens se adaptem aos diferentes tamanhos e resoluções para cada plataforma, mas às vezes isso pode não Veja como você realmente quer, já que não existe uma solução "tamanho único". Portanto, após executar esta ferramenta, recomendamos que você revise todas as imagens em cada plataforma Opções de Jogo e ajuste aquelas que podem não parecer exatamente como você querer manualmente.
