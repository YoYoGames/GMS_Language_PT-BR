Opções de Repositório

Tendo criado as credenciais GIT e ativado o Controle de Fontes para seu projeto, você agora precisa ligar a IDE GameMaker Studio 2 a um repositório, o que é feito de uma de três maneiras:

  • Criar um Repositório de Projetos
  • Empurrar o projeto atual para um Repositório Externo
  • Clonar um Repositório existente

No primeiro caso, criando um repositório de projetos, isto criará um repositório GIT diretamente onde seu projeto está. Na linguagem GIT, este é um "repositório não nu", então Push/Pull make não faz sentido neste caso, mas você ainda pode se comprometer e acompanhar as mudanças do projeto, bem como reverter as mudanças e revisões, se necessário. Para configurar isto, você só precisa clicar em Source Control > Create Project Repository no menu principal.

Entretanto, você pode já ter um projeto existente e querer colocá-lo em um repositório externo para rastreamento e compartilhamento entre sua equipe. Isto requer que um repositório externo já tenha sido configurado, assim como o nome de usuário e senha corretos definidos em suas preferências de identidade para acessá-lo. Finalmente, como uma função de conveniência, você pode clonar um repositório externo através da IDE. Novamente, você precisará do nome de usuário e senha para o repositório que você vai conectar para configurar em suas preferências de identidade, e então ele é apenas um caso de clicar em Source Control > Clone Repository.
