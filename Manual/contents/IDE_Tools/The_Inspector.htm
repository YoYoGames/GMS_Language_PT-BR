A Janela do Inspetor

A janela do inspetor é uma janela que mostra todas as propriedades associadas a um elemento selecionado na IDE. A janela mostrará informações sobre o elemento que está sendo inspecionado, bem como todas as propriedades editáveis para aquele elemento e permitir que você as altere conforme necessário, atualizando o bem que está sendo inspecionado para corresponder.

IMPORTANTE! Atualmente a Janela do Inspetor está disponível apenas para inspecionar as propriedades da Sequência, mas em futuras atualizações ela será expandida para incluir outros ativos.

Você pode abrir várias janelas do Inspetor a qualquer momento, e cada uma delas pode ser ligada a um elemento específico na IDE. Para garantir que cada vez que você clicar em um elemento, todos os inspetores abertos não mudem para mostrar a nova seleção, você deve primeiro bloquear a janela do inspetor. Para fazer isso, você abriria um inspetor, selecionaria um elemento IDE para inspecionar e, em seguida, clicaria no botão de bloqueio no inspetor para bloqueá-lo no elemento (mostrado na imagem abaixo). Uma vez trancado o inspetor, você pode abrir outra e depois repetir o processo.

Note que uma janela de inspetor trancada não tranca a edição de nenhum valor para o elemento a ser inspecionado. Ela simplesmente bloqueia o inspetor para um elemento e você pode continuar a editá-lo a qualquer momento, mesmo que ele não seja o foco da IDE principal.
