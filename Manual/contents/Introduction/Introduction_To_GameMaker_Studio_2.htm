Introdução ao GameMaker Studio 2

OGameMaker Studio 2 é uma ferramenta projetada para capacitar você e sua equipe a fazer jogos novos e inovadores, bem como idéias de protótipos da maneira mais rápida e intuitiva possível em múltiplas plataformas alvo. Ele é destinado principalmente como uma ferramenta para fazer Jogos 2D - embora jogos 3D sejam perfeitamente realizáveis - e vem com uma série de ferramentas e editores para ajudá-lo a realizar seus sonhos e idéias, com seu projeto final sendo portado através de múltiplas plataformas a partir dos mesmos recursos básicos iniciais.

Para aqueles que são novos no mundo da programação ou que nunca usaram nenhuma ferramenta de criação de jogos antes, o GameMaker Studio 2 oferece uma ferramenta intuitiva e fácil de usar Arrastar e Soltar ( DnD™) interface de ícones de ação que permitem você para começar a criar seus próprios jogos muito rapidamente usando o roteiro visual:

Se você é um programador mais experiente - ou menos visual - então você pode usar a linguagem proprietária de scripting GML para criar seus jogos:

Você pode até mesmo combinar ambas as funções GML e misturá-las com DnD™ações, dando-lhe uma grande flexibilidade quando se trata de como você escreve as coisas. Isto é tudo disposto de forma intuitiva, com diferentes aspectos de seu jogo "encadeado " juntos na IDE para lhe dar uma representação única e muito visual do fluxo de trabalho.

OGameMaker Studio 2 vem com uma série de ferramentas poderosas, como um Editor de Imagem completo para criar seus próprios sprites e a capacidade de usar conjuntos de azulejos e você também pode importar imagens que você fez em qualquer editor externo, incluindo animações de esqueleto de espinha e arquivos vetoriais em formato SWF. Uma vez que você tenha suas imagens, é simplesmente uma questão de atribuí-las a objetos em o Editor de Objetos, adicionando algum código ou ações do DnD™, e depois colocando instâncias destes objetos em uma sala de jogos no Editor de Sala. Há também ferramentas adicionais para a criação de Caminhos, Seqüências, Shaders e muito mais. Você verá que a IDE também é incrivelmente flexível, e pode ser personalizada de acordo com suas necessidades, caso a configuração padrão da IDE não cubra algo que você precise, usando as diferentes Preferências.

Este arquivo de ajuda lhe dirá tudo o que você precisa saber sobre o GameMaker Studio 2 e como você pode criar seus próprios jogos com ele. Entretanto, por favor, perceba que mesmo com um programa como o GameMaker Studio 2, projetar jogos de computador não é completamente sem esforço. - portanto, nada de botão "faça de mim um MMORPG" aqui! Esteja ciente também que existem muitos aspectos para fazer jogos e que todos eles são igualmente importantes: jogabilidade, gráficos, sons, interação do usuário, etc... Mas, o aspecto mais importante de tudo, é a paciência. Embora o GameMaker Studio 2 seja uma ferramenta poderosa, ele não pode fazer tudo por você ao toque de um botão, então comece pequeno com exemplos simples e você descobrirá que não só está criando jogos muito divertidos, mas que aprender a usar o GameMaker Studio 2 para fazer é também muito gratificante!
