Biblioteca do seu mercado

A janela Biblioteca mostra os pacotes que você comprou e lhe dá a opção de baixá-los e/ou adicioná-los ao seu projeto.À esquerda estão os diferentes filtros de categoria que você pode usar para ver o que você tem em cada um, e depois no meio você tem a lista de ativos em sua biblioteca. Cada um deles pode ser selecionado de modo que você possa ver uma visão geral do que ele faz (no direito) e todos eles têm ícones para visualizar, baixar ou instalar o pacote de bens em um projeto. Observe que você tem dois filtros adicionais no topo da janela principal onde você pode selecionar para ver apenas os ativos que foram criados especificamente para GameMaker Studio 2, bem como um filtro para mostrar apenas os ativos que você já baixou para sua máquina. Você também pode atualizar a lista de ativos mostrados a qualquer momento, clicando no botão Atualizar .

Para adicionar um pacote de ativos ao seu projeto atual, você deve primeiro baixá-lo clicando no botão Download e quando o download estiver concluído, você pode clicar no botão Importar para adicioná-lo ao seu projeto. Adicionando um pacote desta forma criará uma extensão dentro de seu projeto atual, e lhe pedirá para selecionar entre os diferentes recursos do pacote de ativos para escolher quais você deseja adicionar (ou você pode importar todos eles). Você também pode selecionar a opção Importe todos os recursos para um novo projeto e, em vez de adicionar os recursos ao seu projeto atual, o GameMaker Studio 2 irá solicitar que você dê um novo nome ao projeto e, em vez disso, salvará o recurso dentro desse projeto. Observe que se você for diretamente à Biblioteca a partir da página inicial, você encontrará a maioria dos bens que não podem ser adicionados individualmente e, em vez disso, você será forçado a usar o botão Adicionar Tudo. Isto é porque você não tem um projeto para adicionar os ativos do GameMaker Studio 2 devem criar um novo projeto para que você possa adicionar todos os ativos do pacote (ao clicar no botão Importar, você será solicitado a salvar o novo projeto com o nome de sua escolha).Com os ativos adicionados ao seu Navegador de Ativos, você pode agora ir em frente e usá-los em seu jogo como usaria aqueles ativos que você mesmo criou. Por favor, observe, que os bens baixados do Marketplace são para uso individual e não podem ser compartilhados com terceiros - embora você possa fazer backup deles e copiá-los para outros locais por segurança - a menos que o bem tenha um EULA personalizado que declare o contrário.

NOTA: As pessoas que criam pacotes de ativos para o Marketplace trabalham muito para criar um grande recurso para você usar, portanto, considere avaliar os ativos que você compra quando os tiver testado ou se os usar. Isto encoraja o e é também um guia útil para outros usuários quanto à qualidade dos ativos disponíveis. Você pode avaliar facilmente seus ativos baixados da janela Biblioteca.
