Configurações

Devido às exigências do desenvolvimento de plataformas cruzadas e à necessidade de que um produto esteja disponível e personalizado para diferentes clientes ou dispositivos ou plataformas, o GameMaker Studio 2 inclui configurações para tornar as coisas mais simples. Usando isto é possível alternar rapidamente entre diferentes telas splash, ícones e arquivos incluídos e testar ou exportar apenas aqueles que são relevantes para a plataforma ou cliente alvo a qualquer momento, tudo a partir do mesmo código base.

Para poder lidar com as diferentes demandas dos clientes e plataformas você pode usar o Editor de Configuração que é acessível a partir do menu Extras no Navegador de Ativos:

Todos os jogos feitos no GameMaker Studio 2 começam com uma configuração "padrão", que é listada no editor quando você abre pela primeira vez isso:

Para configurar suas configurações, recomenda-se que você primeiro prepare esta configuração "padrão", incorporando todos os ativos configuráveis para o projeto nele envolvido. Para fazer isso, basta passar pela seguinte lista de opções e ajustá-las ao que é exigido por padrão:

  • Opções de extensão: Configurar os vários campos "Copiar para..." para as plataformas necessárias para a configuração. Qualquer arquivo "Proxy" criado aqui terá sua plataforma "Target". ligado à configuração também.
  • Editor Sprite: Defina o grupo de textura ao qual você quer que os sprites sejam adicionados.
  • Editor de Ladrilhos: Defina o grupo de textura ao qual você deseja que os tilesets sejam adicionados.
  • Editor de fontes: Defina o grupo de textura ao qual você quer que as fontes sejam adicionadas.
  • Gerente do Grupo de Textura: Aqui você pode definir qualquer uma das propriedades em uma base por configuração, incluindo o grupo pai ao qual pertence um grupo de textura.
  • Gerente do Grupo de Áudio: Os sons que são designados para os diferentes grupos de áudio são confiúveis aqui.
  • Arquivos incluídos: Defina o campo "Copiar para..." para as plataformas necessárias para a configuração.

Depois de configurar a configuração padrão, você pode então adicionar uma nova configuração ao Editor de Configuração, clicando com o botão direito do mouse em na pasta de configurações e selecionando Adicionar Config, ou clicando no botão "Criar" na parte inferior da janela. A nova configuração será mostrada como uma "criança" da configuração padrão - todas as configurações definidas pelo usuário são crianças. do padrão - e você pode dar-lhe um nome apropriado.

Para editar a configuração, você deve clicar duas vezes sobre ela ou usar o botão direito do mouse opção do menu "SetConfig", que selecionará a configuração no editor e, uma vez que isso tenha sido feito, você pode ir em frente e personalizá-la. Isto é feito simplesmente percorrendo as opções listadas acima e mudando para os ajustes desejados para a configuração recém-criada. O GameMaker Studio 2 armazenará estes valores automaticamente para a configuração, e você pode mudar entre as configurações a qualquer momento simplesmente usando a opção do menu RMB ou clicando duas vezes em .

Outras configurações podem ser adicionadas como filhos da configuração padrão, ou filhos das configurações definidas pelo usuário que você criou, dependendo de qual configuração foi destacada no editor quando você a criou. Vale notar que você pode usar qualquer configuração com qualquer plataforma de destino ao compilar ou testar seu projetos. Isto é feito selecionando a configuração necessária dentro da seção Config do Gerente de destino:

Você também pode abrir o Editor de Configuração em qualquer tme do Gerenciador de destino clicando no botão de menu no canto superior direito (como destacado na imagem acima).

Uma vez definida uma configuração a ser utilizada desta forma, a configuração escolhida será utilizada sempre que você compilar para um alvo, seja para testes ou para criar um pacote final de ativos. Para mais informações sobre a Janela de Alvos, consulte a seção na Compilação.
