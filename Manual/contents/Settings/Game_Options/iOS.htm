Opções de jogoiOS

Esta seção descreve as diferentes opções disponíveis para você que controlam como seus projetos de jogo iOS serão compilados. As diferentes seções são:

Geral

A primeira parte da seção Geral cobre os Build Settings para seu jogo. Para começar, defina as Orientações que seu jogo pode ser executado para obter a melhor experiência de usuário. As opções disponíveis são:

  • Retrato
  • Retrato virado
  • Paisagem
  • Paisagem virada

Você pode então ir para a seção Suporte do Dispositivo, com as seguintes opções disponíveis para definir o tipo de dispositivo a ser visado:

  • iPhone ou iPod Touch: iPhone 6 e superior ou dispositivos iPod Touch
  • iPad: dispositivos iPad
  • Ambos: tanto iPhone/iPod quanto iPad

A segunda coisa a fazer é preencher as Informações do produto sobre seu jogo, incluindo seu Nome de exibição, seu ID do pacote e as Informações da versão. Observe que o Bundle ID deve estar no formato URL invertido para que a construção final do jogo funcione corretamente, EG: com.[Company].[GameName]. Aqui você também pode dar ao Team Identifier que você deseja usar para assinar a aplicação final que O GameMaker Studio 2 cria para você. Colocá-lo aqui substituirá o ID da equipe que você forneceu nas PreferênciasiOS.

A seguir, você pode verificar ou desmarcar a opção de Deferir a Saída do Indicador de Casa. Quando isto for verificado, a barra de casa será sempre mostrada com um alfa baixo, e o primeiro deslize acorda a barra e o segundo deslize o leva para casa. Se a opção não é verificado, então a barra não é mostrada no jogo, mas o primeiro deslize de borda o levará para casa. O padrão é desmarcado.

Finalmente, você tem a opção de gerar um Podfile e um Podfile.lock. Estes arquivos são necessários se você estiver adicionando CocoaPods a uma extensãoiOS usando o Editor de Extensão. Os arquivos contêm macros específicas exigidas pelo GameMaker Studio 2 para configurar qualquer CocaoPodslistado ao construir seu jogo. Clicando nos botões, os arquivos serão criados com as macros pré-geradas, e também abrir um explorador de arquivos sobre a localização dos arquivos, caso você deseje editá-los manualmente. Há também um botão que o levará ao seguinte guia:

  • iOS e tvOS: usando cápsulas de cacau

Gráficos

Aqui você pode alterar os seguintes detalhes relacionados a como seu jogo será exibido, com as seguintes opções disponíveis:

  • Interpolar as cores entre os pixels: Ativa a interpolação linear, que basicamente "suaviza" os pixels. Para gráficos de pixel nítidos, deve ser mas se você tiver misturas alfa agradáveis e gráficos alisados é melhor deixá-lo ligado. Isto está desativado por padrão.
  • Metade das texturas do iPad 1: O iPad 1 tem muito menos memória de textura do que os dispositivos mais recentes, portanto você pode habilitar esta opção para reduzir o tamanho das texturas pela metade, se você quiser direcionar este dispositivo ou obter o máximo de compatibilidade.
  • Escala: Aqui você pode selecionar a escala da tela de desenho mantendo a relação de aspecto dentro da tela do dispositivo - adicionando "acolchoamento" ao redor das bordas para caber na tela - ou você pode selecionar ter a tela de desenho esticada para caber a tela do dispositivo.

Finalmente, há a opção de definir o tamanho da página de textura. O tamanho padrão (e o mais compatível) é 2048x2048, mas você pode escolher entre 256x256 até 8192x8192. Lá é também um botão marcado Preview que irá gerar as páginas de textura para esta plataforma e depois abrir uma janela para que você possa ver como elas ficam. Isto pode ser muito útil se você desejar ver como as páginas de textura são estruturadas e para evitar ter páginas de textura maiores (ou menores) do que o necessário. Para maiores informações sobre páginas de textura, por favor, veja aqui.

NOTA: Esteja ciente de que quanto maior o tamanho da página de textura, menos compatível seu jogo será com diferentes navegadores e dispositivos.

Ícones

Esta seção permite adicionar os vários ícones diferentes exigidos pelos vários dispositivos iPhone e iPad e as diferentes partes do iOS App Store. Estes ícones devem ser de autoria de 24bit *.png imagens no tamanho especificado para cada uma.

Vale notar que o GameMaker Studio 2 possui uma ferramenta Gerador de Imagens de Projeto que pode ser usada para criar automaticamente todas as imagens necessárias para todas as diferentes plataformas alvo de seu jogo está sendo compilado para. Se você usar esta ferramenta, você deve revisar as imagens criadas para garantir que elas sejam o que você precisa. A ferramenta gerará tanto ícones quanto imagens (veja a seção abaixo).

NOTA: Ao importar um projeto mais antigo, você poderá receber o seguinte aviso:Isto ocorre porque o objetivo atual do iOS mudou o formato dos ícones necessários desde que o projeto foi previamente carregado. Você será solicitado a recriar todos os ícones, pois eles terão sido substituídos pelo genérico GameMaker Studio 2 (você pode usar os ícones acima mencionados Ferramenta Gerador de Imagens do Projeto para fazer isso rapidamente).

Imagens

Esta seção permite adicionar gráficos separados como Telas de Lançamento (a imagem que será mostrada brevemente enquanto suas cargas de aplicação) para cada uma das orientações do dispositivo, bem como a Cor da Borda que será usada como fundo para as telas de lançamento que não estão configuradas para preencher a tela inteira. Estas imagens devem ser de autoria individual 24bit *.png imagens sem transparências e em tamanho apropriado. Note que isto não é considerado uma tela de respingo e a Apple tem algumas diretrizes bastante rígidas sobre como isto deve ser configurado e utilizado. Veja a página do desenvolvedor em Diretrizes de Interface Humana para mais informações.

Você também pode definir como a tela de lançamento deve ser exibida a partir das seguintes opções:

  • Mantenha a relação de aspecto: Isto manterá a relação de aspecto da imagem em comparação com a tela do dispositivo, adicionando "acolchoamento" ao redor das bordas onde for necessário.
  • Cortar à medida: Isto manterá a imagem 1:1 com a tela do dispositivo e cortará (ou almofada) as bordas, conforme necessário.
  • Esticar: Isto esticará a imagem para preencher a tela do dispositivo (e pode distorcer a imagem mostrada).

Social

Esta seção é para permitir diferentes características sociais em seu jogo iOS. As seguintes opções podem ser escolhidas:

  • Facebook: Se você precisar das funções do Facebook, você precisa selecionar esta caixa de seleção e então fornecer a ID do aplicativo do Facebook e o Nome do aplicativo do Facebook. Observe que todas as funções do Facebook são acessadas através de um e quando você marcar a caixa Usar Facebook pela primeira vez, você será solicitado a baixar e instalar a extensão do Facebook a partir do Mercado.
  • Habilitar notificações de Push: Para que seu jogo possa utilizar as Notificações Push, elas devem ser habilitadas aqui primeiro, caso contrário, não funcionarão.
  • Habilite o login com a Apple: Esta opção é para usar a Extensão Apple Sign In no iOS. Você pode obter esta extensão no Jogos YoYo Mercado, e a própria extensão contém instruções completas de uso. Se você não estiver usando esta extensão, então não deve assinalar esta opção.
