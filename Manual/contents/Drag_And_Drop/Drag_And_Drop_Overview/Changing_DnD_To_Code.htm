Mudando DnD™para código

Outra característica de DnD™é que você pode convertê-lo em código GML(GameMaker Language), e desta forma ver o que exatamente está acontecendo nos bastidores quando você usa ações. Antes de realmente mudar as ações para GML, no entanto, você pode primeiro visualizá-las usando o botão direito do mouse opção de menu Live Preview:

O que abrirá uma nova janela:

Conforme você adiciona, altera ou remove ações DnD™, o GML Live Preview será atualizado para mostrar o código real que é sendo criados "por trás dos bastidores". O código na visualização ao vivo não pode ser editado diretamente, mas você pode selecionar seções e copiá-las para colar em scripts GML ou Ações de Código (por exemplo).

Se você decidir que quer converter o DnD™em código após ver a visualização, então isso pode ser feito novamente clicando no botão direito do mouse em qualquer espaço de trabalho para eventos com ações e selecionando Converter para GML.

A primeira vez que você fizer isso, receberá uma mensagem de aviso dizendo que esta é uma conversão de uma só maneira, já que você pode converter ações em código, mas não se pode convertê-las novamente em ações individuais mais tarde. Clicando em "ok" aqui, você fará a conversão para você.

O código resultante usará {} para delimitar as ações individuais, e você pode ver claramente quais ações relacionar com quais funções ou declarações variáveis dentro do código. Se o DnD™for mais complexo, então o código também será, mas as mesmas regras gerais se aplicam e o código será estruturas sequencialmente exatamente as mesmas que você escreveu o DnD™. Ao aprender a programar usando Drag and Drop, esta pode ser uma ferramenta importante para passar a usar GML em uma etapa posterior, mas não é de forma alguma obrigatória e você ainda pode fazer grandes jogos usando DnD™!Também vale a pena notar que enquanto a conversão é uma maneira, após converter ações em código, você pode voltar a usar DnD™novamente, usando o menu no editor de código e selecionando "Converter para DnD™".
