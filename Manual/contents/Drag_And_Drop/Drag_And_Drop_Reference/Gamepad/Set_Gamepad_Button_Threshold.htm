Ajustar o Limiar do Botão Gamepad

Esta ação pode ser usada para definir o "limiar" dos botões analógicos do gamepad. Você especifica o índice do gamepad a ser definido, e então define um valor de 0 a 1 e se a quantidade de entrada do botão analógico for menor que o valor dado, o botão gamepad Note que este é um cenário global que afetará todos os botões analógicos conectados ao gamepad. Este valor será usado em todas as verificações para baixo, pressionado e liberado para o gamepad dado, mas será ignorado pela ação Get Gamepad Trigger.
