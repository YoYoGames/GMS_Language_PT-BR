Jogar Sequência

Com esta ação, você pode pausar uma seqüência de jogo. Você fornece o ID do elemento de seqüência (como retornado pela ação Criar Sequência), e a ação fará uma pausa na jogada.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Variável A identificação do elemento de seqüência a ser pausado.

Exemplo:

O código de bloco de ação acima verifica se uma variável global é verdadeira. Se for então o elemento de seqüência referenciado na variável "my_seq" é pausado, caso contrário é definido para jogar.
