Jogar Sequência

Com esta ação, você pode iniciar uma seqüência (previamente pausada) de reprodução. Você fornece o elemento de seqüência ID (como retornado pela ação Criar Sequência), e a ação começará a jogar.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Variável A identificação do elemento de seqüência a ser tocado.

Exemplo:

O código de bloco de ação acima verifica se uma variável global é verdadeira. Se for então o elemento de seqüência referenciado na variável "my_seq" é pausado, caso contrário é definido para jogar.
