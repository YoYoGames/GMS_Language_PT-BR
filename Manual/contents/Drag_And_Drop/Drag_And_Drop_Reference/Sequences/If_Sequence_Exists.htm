Se a sequência existir

Esta ação pode ser usada para verificar se o elemento de seqüência dado existe na sala de jogos. Você dá o ID do elemento, como retornado pela ação Criar Seqüência, e se o elemento seqüencial existe na sala então a ação retornará verdadeira, caso contrário, retornará falsa.

Se você sinalizar o argumento "Não", então a ação verificará se não existe tal elemento de seqüência e se nenhum for encontrado, ele retornará verdadeiro.
