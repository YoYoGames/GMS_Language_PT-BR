Definir posição seqüencial

Com esta ação, você pode definir a posição para o elemento de seqüência dado na sala. Você fornece o ID do elemento de seqüência (como retornado pela ação Criar Sequência), e então a nova posição X e Y. Se você marque a caixa de seleção "relativo", então os valores dados para X e Y serão adicionados à posição atual da seqüência, caso contrário, serão uma posição absoluta.
