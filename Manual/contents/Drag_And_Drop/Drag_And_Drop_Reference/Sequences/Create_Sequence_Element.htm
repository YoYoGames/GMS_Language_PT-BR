Criar Elemento de Sequência

Com esta ação você pode criar um elemento de seqüência de qualquer seqüência a partir do Navegador de Ativos. Você dá o nome da seqüência e uma posição x/y dentro da sala onde ela deve ser criada, bem como a camada na qual ela deve ser criada (esta camada deve ser uma camada patrimonial). Você pode marcar as bandeiras "relativas" para posicionar a seqüência em relação à instância que executa os blocos de código de ação. Sempre que você cria uma seqüência no jogo desta forma, é dado um valor de identificação de elemento único com o qual você pode identificar a seqüência em outras ações, portanto, esta ação permite definir uma variável "alvo" para manter este valor de identificação (como mostrado no exemplo abaixo). Note que o alvo pode ser marcado como temporário (local). variável, que será criada para a ação e pode ser usada em todas as ações subseqüentes para o evento em que a ação estiver.
