Se o arquivo existir

Com esta ação, você pode verificar se existe um arquivo com o nome dado. Você seleciona o arquivo a ser verificado (como uma picada e incluindo a extensão) e a função retornará verdadeira se existir, e falsa se não existir. Observe que isto retornará verdadeiro se o arquivo existir no pacote do jogo como um arquivo incluído ou se o arquivo tiver sido salvo enquanto o jogo estava em execução (veja a seção sobre O Sistema de Arquivo para mais informações).

O código de bloco de ação acima criará uma variável para guardar os dados do buffer e então verificará se um arquivo de proteção de segurança já foi feito anteriormente.
