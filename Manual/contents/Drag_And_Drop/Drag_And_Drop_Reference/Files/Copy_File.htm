Copiar arquivo

Com esta ação, você pode copiar qualquer arquivo que tenha sido salvo por seu jogo ou que tenha sido empacotado como parte dos Arquivos Incluídos. Você dá o nome do arquivo a ser copiado (como uma string) e depois o nome do arquivo a ser criado como uma cópia do arquivo original (também como um fio e incluindo a extensão do arquivo). O dado arquivo será duplicado e receberá o novo nome.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Nome do arquivo O nome (como um fio) do arquivo a ser copiado
Copiar para O nome (como um fio) do arquivo a ser criado como uma cópia

Exemplo:

O código de bloco de ação acima irá verificar se um arquivo ini não existe e se não então é criado através da cópia de outro arquivo.
