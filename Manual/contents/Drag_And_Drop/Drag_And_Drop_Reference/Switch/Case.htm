Caso

Esta ação criará um novo caso dentro de um Troca. Você dá o valor para o caso e ele será comparado com o valor de entrada para o Switch, e se eles corresponderem às ações adicionadas sob o caso, será realizado. As ações que são adicionadas a um Caso a ser realizado somente nesse Caso devem ser descartadas para o lado da ação, assim como você adicionaria uma ação a um "se":

Note que um valor de Caso deve ser uma constante, que pode ser qualquer tipo de dado (real, string, etc...), desde que se avalie a um único valor na compilação - de modo que não se possa usar expressões que dependam de variáveis. Observe também que Case não pode ser usado fora de um Switch e tentar fazer isso lhe dará um erro.
