Troca

Esta ação criará uma nova declaração Switch, à qual você poderá então acrescentar outros casos para avaliação. Você dá o valor que deve ser usado para comparação com os casos no Switch e então adiciona as diferentes ações de Caso, muito como você acrescentaria uma ação a um "se", ou seja: deixá-la cair para o lado da ação em vez de sob ela:

Note que você não pode adicionar qualquer outro tipo de ação em um interruptor sem antes ter adicionado pelo menos um Estojo para contê-lo.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Valor O valor a ser usado pela chave para avaliação

Exemplo:

O código de bloco de ação acima obtém o valor armazenado em uma variável e depois o compara com os diferentes possíveis casos. Se qualquer um dos casos for igual ao valor, então a ação sob esse caso será executada, caso contrário, o próximo caso será avaliado. Depois que a chave tiver funcionado, um alarme é ajustado para 30.
