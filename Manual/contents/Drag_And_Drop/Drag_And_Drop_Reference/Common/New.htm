Novo

Esta ação é utilizada apenas em quando se cria estruturas usando uma função que foi marcada como uma função de construtor (ver Declarar uma nova função para mais informações). Isto A ação toma o método de função para uma função previamente definida, bem como uma série de argumentos, que você pode expandir para adicionar mais, se necessário, usando o ícone , à esquerda. Estes argumentos devem corresponder aos inputs exigidos pela função, e serão usados para povoar a estrutura que está sendo criada. A estrutura será devolvida à variável alvo, que pode ser sinalizada como uma variável local temporária variável ou não.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Roteiro O nome do script ou função definida pelo usuário a ser chamada.
Argumento0 ... Argumento3 Os diferentes argumentos (valores) que devem ser passados ao roteiro ou função (argumentos não utilizados podem ser deixados em branco)
Meta O nome da variável que deve ser alvo de qualquer valor retornado (pode ser deixado em branco)
