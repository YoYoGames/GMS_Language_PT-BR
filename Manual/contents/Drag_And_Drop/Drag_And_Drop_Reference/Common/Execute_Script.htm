Executar Roteiro

Esta ação pode ser usada para chamar uma função de script ou uma função definida pelo usuário. Em geral, esta ação seria usada apenas para suporte legado e não deveria realmente ser usada em projetos atuais ou novos. Para usar a função ação, você dá o recurso do script para chamar - ou o nome da variável da função definida pelo usuário para chamar - e então fornece quaisquer argumentos que ela requeira, ou seja: quaisquer valores (strings, inteiros, índices de recursos, etc...) que você deseja inserir no script ou funcionar para que possa usá-los. Você pode adicionar argumentos adicionais, conforme necessário, clicando no ícone mais ao lado da ação.

Alguns scripts e funções definidas pelo usuário também podem retornar um valor para uso posterior na lista de blocos de ação, e assim você pode especificar a variável alvo que deve armazenar este valor de retorno. Isto pode ter sido criado anteriormente ou pode ser um novo valor temporário local (se você marcar a caixa de seleção "Temp"). Tanto o campo alvo quanto o campo de argumento podem ser deixados em branco se não forem necessários.

Note que geralmente é melhor usar a ação Chamada de função para chamar funções definidas pelo usuário, e geralmente não deve precisar usar esta ação, ou chamar um recurso de script diretamente em código.

Argumentos:

Argumento Descrição
Roteiro O nome do script ou função definida pelo usuário a ser chamada.
Argumento0 ... Argumento3 Os diferentes argumentos (valores) que devem ser passados ao roteiro ou função (argumentos não utilizados podem ser deixados em branco)
Meta O nome da variável que deve ser alvo de qualquer valor retornado (pode ser deixado em branco)
