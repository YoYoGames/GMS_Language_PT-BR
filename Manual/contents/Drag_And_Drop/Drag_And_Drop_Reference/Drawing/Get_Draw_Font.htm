Obter a fonte Draw Font

Com esta ação, você pode obter o valor de identificação da fonte atualmente designada para o desenho. O valor retornado será ou -1 para nenhuma fonte atribuída, ou um valor >= 0 que representa uma constante de fonte do Navegador de Ativos. O valor retornado será armazenado na variável alvo que você especificar, que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você marcar a caixa de seleção "Temp").
