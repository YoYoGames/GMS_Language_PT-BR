Se Mouse Down

Com esta ação você pode sondar o mouse para ver se o botão selecionado está sendo pressionado no momento. A ação retornará verdadeiro a cada passo que o botão estiver sendo mantido pressionado, ou falso, embora se você clicar no não modificador, poderá então verificar se o botão não está sendo mantido pressionado, ou seja: a ação retornará verdadeiro enquanto nenhum botão estiver pressionado e falso se houver. Se você só precisa verificar se há um único botão, pressione então use a ação Se pressionado pelo mouse.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão executadas agora se o "se" for avaliado como verdadeiro, enquanto quaisquer ações abandonadas em outro lugar serão executadas após o bloco "se".

Exemplo:

O código de bloco de ação acima pesquisa o botão do mouse a cada passo e se ele está sendo mantido pressionado, verifica a pressão inicial do mouse para baixo. Se a verificação for verdadeira na descida inicial pressione a velocidade da instância está definida, então, enquanto o botão está pressionado, a mistura de sprite é mudada para vermelho. Se o mouse não estiver sendo pressionado para baixo, um A verificação é feita na liberação do mouse para repor a cor da mistura para branco e ajustar a velocidade para 0.
