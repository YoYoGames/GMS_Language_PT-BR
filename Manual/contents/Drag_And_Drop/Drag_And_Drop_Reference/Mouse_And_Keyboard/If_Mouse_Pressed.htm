Se o mouse for pressionado

Esta ação verifica se o botão inicial do mouse foi pressionado. Retornará verdadeiro no único passo do jogo (quadro) que o botão é inicialmente pressionado e falso em todos os outros momentos. Se você ativar o não modificador, esta ação será revertida, e verifique se o botão não foi pressionado, retornando falso no passo do jogo (moldura) que foi pressionado e verdadeiro em todos os outros momentos. Se você precisar verificar se um botão foi mantido pressionado para baixo e não apenas o evento inicial da imprensa, então use a ação If Mouse Down.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão executadas agora se o "se" for avaliado como verdadeiro, enquanto quaisquer ações abandonadas em outro lugar serão executadas após o bloco "se".

Argumento Descrição

Exemplo:

O código de bloco de ação acima pesquisa o botão do mouse em cada passo e se for sendo mantido pressionado, ele verifica se o mouse está pressionado inicialmente. Se a verificação for verdadeira na descida inicial pressione a velocidade da instância é definida, então, enquanto o botão está pressionado, a mistura de sprite é mudada para vermelho. Se o mouse não estiver sendo pressionado para baixo, uma verificação é feita na liberação do mouse para repor a cor da mistura para branco e ajustar a velocidade para 0.
