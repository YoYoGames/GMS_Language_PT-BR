Se o mouse for liberado

Esta ação verifica quando um botão do mouse foi solto. Ela retornará verdadeiro no único passo do jogo (frame) em que o botão do mouse for solto e falso em todos os outros momentos. Se você ativar o não modificador, esta ação será revertida, e verifique se o botão do mouse não foi liberado, retornando falso na etapa do jogo (frame) em que foi liberado e verdadeiro em todos os outros momentos.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão agora executadas se o "se" for avaliado como verdadeiro, enquanto quaisquer ações descartadas em outro lugar serão executadas após o bloco do "se".

Sintaxe de ação:

Argumentos:

Argumento Descrição
Botão do Rato O nome do botão do mouse para verificar (esquerda, meio, direita).
Não Negar o cheque (o verdadeiro torna-se falso e vice-versa)

Exemplo:

O código de bloco de ação acima pesquisa o botão do mouse a cada passo e se ele está sendo mantido pressionado, ele verifica a pressão inicial do mouse para baixo. Se a verificação for verdadeira na descida inicial pressione a velocidade da instância está definida, então, enquanto o botão está pressionado, a mistura de sprite é mudada para vermelho. Se o mouse não estiver sendo pressionado para baixo, uma verificação é feita na liberação do mouse para repor a cor da mistura para branco e ajustar a velocidade para 0.
