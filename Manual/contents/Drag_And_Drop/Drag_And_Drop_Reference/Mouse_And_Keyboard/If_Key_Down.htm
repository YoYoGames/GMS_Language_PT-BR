Se a chave estiver em baixo

Com esta ação você pode pesquisar o teclado para ver se alguma tecla está sendo pressionada no momento. A ação retornará verdadeira a cada passo que a tecla do teclado estiver sendo mantida pressionada, ou falsa, embora se você clicar no não modificador, poderá então verificar se a tecla não está sendo ajudada, ou seja: a ação retornará verdadeira enquanto nenhuma tecla estiver pressionada e falsa se houver. Se você só precisa verificar se há uma única tecla pressionada, então use a ação Se a tecla for pressionada.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Qualquer ação abandonada em outro lugar será executada após o "se". bloco.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Chave O nome da tecla do teclado a ser verificado.
Não Negar o cheque (o verdadeiro torna-se falso e vice-versa)

Exemplo:

O código de bloco de ação acima pesquisa o estado do teclado a cada passo e se uma tecla está sendo mantido pressionado, verifica se a tecla inicial está pressionada. Se a verificação for verdadeira na prensa inicial para baixo, um som é tocado, então, enquanto a tecla é mantida pressionada, a mistura de duendes é mudada para vermelho. Se a tecla não estiver sendo mantida pressionada, uma verificação é feito na liberação da chave para repor a cor da mistura para branco.
