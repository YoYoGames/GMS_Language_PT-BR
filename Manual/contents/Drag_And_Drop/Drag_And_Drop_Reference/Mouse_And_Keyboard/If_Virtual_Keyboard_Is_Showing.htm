Se o Virtual Keyboard estiver aparecendo

Esta ação pode ser usada para verificar se o teclado virtual do SO está sendo mostrado ou não. A ação avaliará se é verdade quando o teclado está aberto ou aberto, ou falso se o teclado virtual está fechado ou fechado. Se você ativar o não modificador, esta ação será revertida e você poderá armazenar o valor de retorno verdadeiro / falso em uma variável alvo, que pode ser marcada como temporária (local).

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão executadas agora se o "se" avaliar a verdade, enquanto quaisquer ações descartadas em outro lugar serão executadas após o bloco "se".
