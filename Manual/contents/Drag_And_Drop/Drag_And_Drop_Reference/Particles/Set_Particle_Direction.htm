Definir a direção das partículas

Esta ação é utilizada para definir a direção das partículas, que pode ser estática ou definida para aumentar / diminuir ao longo do tempo. Você fornece o valor de identificação único para o tipo para definir o tamanho de (como retornado quando você criou o tipo com a ação Criar tipo de partícula) e depois dar uma gama de direções iniciais de partida, depois a quantidade de aumento e finalmente a "sacudida". A direção na qual uma partícula é criada é calculada de modo que 0 é para a direita, 90 é para cima, 180 é para a esquerda e 270 é para baixo, portanto uma configuração mínima de -45 e um ajuste máximo de 45 emitirá partículas que se moverão apenas para a direita, por exemplo.

A opção de aumentar a direção de uma partícula permite que você insira um valor que será adicionado ao valor da direção base de cada quadro de jogo da vida útil da partícula. Assim, definir este valor para, por exemplo, 0,1 adicionará 0,1 à direção de cada jogo. moldura. Note que aqui também é possível usar valores negativos para fazer a partícula se mover na direção oposta para baixo também, assim os valores positivos farão a partícula espiralar no sentido anti-horário, e negativos no sentido horário. Note que o decréscimo mínimo de direção aceito é de -0,01 e o incremento mínimo é de 0,01.

Finalmente, você pode definir a opção "sacudir". Este é um valor que será gerado aleatoriamente entre 0 e o valor que você dá e então adicionado ou subtraído à direção de cada etapa da vida útil das partículas, fazendo com que a imagem desenhada flutue em direção. Obviamente os valores maiores são mais pronunciados do que os menores, e este valor pode até mesmo ser negativo, sendo a faixa máxima entre -20 e 20.

Argumentos:

Argumento Descrição
Tipo O valor de identificação único para o tipo de partícula para definir a direção de
Min Dir A direção mínima para o tipo de partícula (de -360 a +360, padrão 0)
Max Dir A direção máxima para o tipo de partícula (de -360 a +360, padrão 360)
Aumentar O valor a ser acrescentado à direção do tipo de partícula em cada quadro de jogo (pode ser negativo, mínimo +/-0,01)
Wiggle O valor máximo a ser adicionado ou subtraído aleatoriamente da direção das partículas por etapa (+/-20).
