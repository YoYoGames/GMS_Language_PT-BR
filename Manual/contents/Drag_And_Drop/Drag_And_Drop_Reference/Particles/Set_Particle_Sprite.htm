Conjunto Sprite de Partículas

Esta ação pode ser usada para definir o duende que uma partícula pode usar ao desenhar. Você fornece o valor de identificação único para o tipo de duende (como retornado quando você criou o tipo com a ação Criar tipo de partícula). e o recurso sprite a ser utilizado. Se o duende tiver sub-imagens, então a partícula animará através dos quadros na velocidade definida no Editor de Imagens para o duende. Observe também que se você definir um valor de cor para a partícula (usando a ação Set Particle Colour ) então esta cor será misturada com o duende utilizado, assim como definir a mistura da imagem em uma instância irá misturar uma cor com o duende de instância (use branco para não misturar).

NOTA: Se você usar esta ação, então você não deve usar a ação Set Particle Shape, pois você só pode ter um duende ou uma forma, mas não ambos.

Argumentos:

Argumento Descrição
Tipo O valor de identificação para o tipo a definir o duende de
Sprite O duende a atribuir ao tipo de partícula
