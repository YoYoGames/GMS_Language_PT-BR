Definir a forma da partícula

Esta ação pode ser usada para definir a forma construída em forma de duende que uma partícula deve usar ao desenhar. Você fornece o valor de identificação único para o tipo para definir a forma (como retornado quando você criou o tipo com a ação Criar tipo de partícula) e a forma a ser usada.
