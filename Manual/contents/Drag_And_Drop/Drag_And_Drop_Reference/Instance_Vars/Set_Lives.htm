Vidas do conjunto

Com esta ação você pode criar ou definir uma variável de instância para vidas. Você fornece o valor para definir a variável vidas ou para adicionar à variável vidas (se você marcar a caixa de seleção relativa, então o valor será adicionado às "vidas" atuais. valor atual). Observe que ao contrário das variáveis de instância regulares, o nome da variável para "vidas" é armazenado internamente, de modo que para recuperar o valor da pontuação, você precisa usar a ação Obter vidas.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Vive A quantidade para definir as vidas (ou para adicionar às vidas se parente for verificado)
