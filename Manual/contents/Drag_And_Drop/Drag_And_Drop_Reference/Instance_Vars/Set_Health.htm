Definir Saúde

Com esta ação você pode criar ou definir uma variável de instância para a saúde. Você fornece o valor para definir a variável de saúde ou para adicionar à variável de saúde (se você marcar a caixa de seleção relativa, o valor será adicionado à variável de saúde atual valor "saúde"). Observe que, se você desejar usar a ação da Barra de Saúde de Draw Instance, este valor deve estar entre 0 e 100, caso contrário, pode ser qualquer quantia que você precisar. Observe também que, ao contrário de variáveis de instância regular o nome da variável "saúde" é armazenado internamente para recuperar o valor da pontuação que você precisa para usar a ação Obter saúde .

Argumentos:

Argumento Descrição
Saúde A quantia para definir a saúde (ou para adicionar à saúde se parente for verificado)
