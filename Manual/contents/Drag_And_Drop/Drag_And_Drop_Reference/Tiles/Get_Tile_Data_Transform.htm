Obter transformação de dados do azulejo

Com esta ação você pode descobrir se os dados de um azulejo foram transformados de uma de três maneiras: virados, espelhados ou girados. Os dados do azulejo são simplesmente um valor que reflete o índice do azulejo junto com as transformações aplicadas, e você irá normalmente querem obter os dados do azulejo primeiro usando ou Get Tile Data In Cell ou Get Tile Data At Pixel antes de usar esta ação. A ação retornará verdadeiro se a transformação escolhida foi aplicada aos dados do azulejo, ou falsa, e o valor retornado será então armazenado na variável de destino que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você verificar o "Temp" de seleção). Para maiores informações sobre os dados do azulejo, consulte a seção GML sobre Funções do Mapa de Azulejos.
