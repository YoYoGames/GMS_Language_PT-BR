Definir índice de azulejos no Pixel

Com esta ação, você pode colocar uma telha em uma determinada camada para um novo índice. A camada é especificada a partir do nome da camada dada (uma corda, como usada para nomear a camada no Editor de Sala), e então você dá uma posição x e y para que o azulejo se ajuste. O x e o y a posição pode ser assinalada como sendo relativa à posição da instância de chamada, e o azulejo que está sendo colocado será aquele em que a posição dada se encaixa na sala e o conjunto de azulejos usado será aquele atribuído à camada na sala editor. Finalmente você fornece o índice de azulejos para ajustar o azulejo (os azulejos são indexados da esquerda para a direita de cima para baixo, com 0 na parte superior esquerda). Note que você sempre pode ajustar o índice de azulejos a 0 para limpar o azulejo.

Argumentos:

Argumento Descrição
Camada A camada para assentar a telha
x A posição x para verificar
y A posição y para verificar
Telha O novo índice do azulejo a ser utilizado
