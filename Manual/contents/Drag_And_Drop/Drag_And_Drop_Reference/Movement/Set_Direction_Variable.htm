Definir variável de direção

Esta ação é utilizada para definir uma direção para a instância que não é fixada em incrementos de 45°. Você fornece o valor da direção de 0° a 360° - onde direita é 0°, para cima é 90°, esquerda é 180° e para baixo é 270° - e você também pode definir a bandeira relativa para adicionar/subtrair o valor que você dá à direção atual.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Direção Direção de 0° a 360° (valores fora da faixa serão embrulhados).

Exemplo:

O código de bloco de ação acima verifica se o botão esquerdo do mouse está sendo mantido para baixo e, se for, define a direção para um valor aleatório de 0 a 360 e a velocidade para 5.
