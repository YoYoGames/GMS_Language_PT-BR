String To Number

Esta ação pode converter um fio em um número. Você fornece a string a ser mudada e depois dá uma variável de destino para retornar o número real (esta variável pode ser marcada como uma variável local temporária). Observe que se a cadeia de caracteres tiver algum caractere exceto números e um ponto decimal, ela não será convertida, embora se a cadeia começar com um número, estes serão convertidos e o resto da cadeia será excluído. É importante notar que ao passar os decimais para este ação, o fio retornado será arredondado para apenas duas casas decimais. Abaixo está uma tabela de valores de entrada e o que você esperaria que fosse retornado

Cordão Valor retornado
"123" 123
"123 Olá Mundo" 123
"Olá Mundo" 0 (por padrão, se a cadeia não puder ser convertida, 0 será devolvido)
"Olá Mundo 123" 0
"123.45678" 123.46

Argumentos:

Argumento Descrição
Valor O fio para converter
Meta A variável alvo para manter o valor retornado

Exemplo:

O código de bloco de ação acima inicia uma variável global e depois carrega um arquivo com um buffer salvo nele. Duas cordas são lidas do buffer e armazenadas em uma variável local temporária antes que uma delas seja convertida em um número real e armazenada em outra variável local temporária. Estas são então atribuídas a variáveis globais, e o buffer é eliminado.
