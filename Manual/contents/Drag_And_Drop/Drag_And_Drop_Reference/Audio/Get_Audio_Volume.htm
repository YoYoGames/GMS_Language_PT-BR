Obter volume de áudio

Esta ação devolve o volume do som dado. Você escolhe o som do explorador de ativos e dá uma variável alvo para retornar o valor do volume (você pode sinalizar isto como sendo uma variável local temporária). O valor retornado estará entre 0 e 1, onde 0 é silencioso e 1 é volume completo.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Som O bom recurso para obter o volume de
Meta A variável alvo para armazenar o volume retornado

Exemplo:

O código de bloco de ação acima verifica uma variável global para ver se ela é verdadeira ou falsa. Se for verdade, então o volume do som dado é recuperado e armazenado em uma variável local temporária. O valor é então verificado para ver se não é igual a 0,5, e se não for, então o volume do som é ajustado para 0,5. Se a variável global avalia como falsa, então o som tem seu volume ajustado para 1.
