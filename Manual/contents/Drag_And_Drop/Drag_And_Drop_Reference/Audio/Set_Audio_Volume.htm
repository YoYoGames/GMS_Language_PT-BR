Definir volume de áudio

Com esta ação, você pode definir o volume de um determinado som. Você seleciona o som do explorador de ativos e então define o valor do volume. O volume pode estar entre 0 (silencioso) e 1 (volume completo) e a escala é linear, de modo que um valor de 0,5 seria meio volume. Note que esta ação afetará todas as instâncias do som que estão tocando atualmente na sala, e afetará todas as instâncias futuras do som tocado.

Argumentos:

Argumento Descrição
Som O bom recurso para definir o volume de
Volume O novo volume para o som (de 0 a 1, padrão 1)

O código de bloco de ação acima verifica uma variável global para ver se ela é verdadeira ou falsa. Se for verdade, então o volume do som dado é recuperado e armazenado em uma variável local temporária. O valor é então verificado para ver se não é igual a 0,5, e se não for, então o volume do som é ajustado para 0,5. Se a variável global avalia como falsa, então o som tem seu volume ajustado para 1.
