Definir volume principal

Com esta ação, você pode definir o volume para todo o sistema de som. O valor do volume que você dá pode estar entre 0 (silencioso) e 1 (volume completo) e a escala é linear, de tal forma que um valor de 0,5 seria metade do volume. Note que esta ação afetará todos os instâncias de todos os sons que estão tocando atualmente na sala, e afetará todas as instâncias futuras de todos os sons tocados. Esta ação também é independente do volume definido para qualquer som individual, portanto, se você tiver um som definido para tocar a 0,5 volume e depois ajustar o volume principal para 0,5, o som tocará a 0,25 volume.

O valor é então verificado para ver se não é igual a 0,5, e se não for, então o volume mestre do sistema de som está ajustado para 0,5. Se a variável global for avaliada como falsa, então o volume mestre é definido como 1.
