Obter o passo de áudio

Esta ação recuperará o tom do recurso sonoro que você fornece. O valor de retorno será um multiplicador para o tom real gravado da amostra, onde um valor de 1 não é alterado. Para mais informações sobre o tom, veja a ação Definir tom de áudio.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Som O recurso sólido para obter o tom de
Meta A variável alvo para armazenar o passo devolvido

Exemplo:

O código de bloco de ação acima recebe primeiro o tom para o som dado e o armazena em um variável local. Em seguida, verifica se há uma tecla pressionada e, se for detectada, o valor do pitch é verificado para ver se é inferior a 2. Se for, então o som é parado, a variável pitch tem 0,1 adicionado a ela, e então esta variável é usada para definir o tom do som novamente antes de ser tocado.
