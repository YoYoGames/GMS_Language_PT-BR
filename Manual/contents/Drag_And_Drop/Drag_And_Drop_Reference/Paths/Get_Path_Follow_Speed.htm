Obter velocidade no caminho

Esta ação terá a velocidade, em pixels por quadro de jogo, de uma instância ser definida para seguir um caminho (note que você deve ter chamado a ação Start Following Path para atribuir um caminho à instância para que esta devolver um valor diferente de 0). O valor retornado será armazenado na variável de destino que você der, ou você pode marcar a variável como uma variável local temporária para criá-la e atribuir o valor.
