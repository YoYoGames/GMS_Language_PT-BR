Pare de seguir o caminho

Esta ação acabará com o movimento ao longo do caminho atual que a instância está seguindo.

Sintaxe de ação:

Exemplo:

O código de bloco de ação acima verifica o valor de um alarme e se ele for inferior a 0 um Switch é verificado. Dependendo da string armazenada na variável de instância que é passada para o switch, a instância parará o caminho atual que está ligada, ou iniciará um novo caminho antes de ajustar o alarme novamente.
