Obter posição ao longo do caminho

Esta ação colocará a posição de uma instância ao longo de um caminho. O valor retornado é normalizado de 0 - 1, portanto, se você obtiver um resultado de, por exemplo, 0,5, a instância exatamente no meio do caminho. Note que você deve ter chamado a ação Start Following Path para atribuir um caminho à instância para que esta ação retorne um valor diferente de 0. O valor retornado será armazenado na variável de destino que você der, ou você pode marcar a variável como uma variável local temporária para criá-la e atribuir o valor.
