Definir o caminho seguir a velocidade

Esta ação irá definir a velocidade, em pixels por quadro de jogo, que uma instância deve seguir um caminho (note que você deve ter chamado a ação Start Following Path para atribuir um caminho à instância para que esta tenha qualquer efeito). Você pode usar valores negativos para significar que a instância deve seguir o caminho ao contrário e vale a pena notar que a velocidade que você definir aqui pode ser modificada pelo modificador de velocidade definido no editor de caminho para cada ponto do caminho, portanto, se você definir uma velocidade de caminho para 2 e no caminho um dos pontos tem um modificador de velocidade de 200% a velocidade real por frame de jogo nesse ponto será de 4 pixels.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Velocidade A velocidade (em pixels por frame de jogo) que a instância deve seguir o caminho

Exemplo:

O código de bloco de ação acima inicia uma instância seguindo um caminho, depois gera um entre 2 e 4 antes de usar este valor para definir a velocidade em que a instância se move ao longo do caminho.
