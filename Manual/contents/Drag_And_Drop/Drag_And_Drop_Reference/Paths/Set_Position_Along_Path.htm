Definir posição ao longo do caminho

Esta ação colocará a posição de uma instância ao longo de um caminho. O valor é normalizado de 0 a 1, portanto, se você defini-lo para, por exemplo, 0,5, a instância será movida exatamente para o meio do caminho. Note que você deve ter chamado a ação Start Following Path para atribuir um caminho à instância para que isto tenha qualquer efeito.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Posição A posição (de 0 a 1) ao longo do caminho para colocar a instância

Exemplo:

O código de bloco de ação acima inicia a instância ao longo de um caminho e depois gera um valor decimal aleatório entre 0 e 1. Este valor é armazenado em uma variável local temporária e usado para definir a posição da instância ao longo do caminho.
