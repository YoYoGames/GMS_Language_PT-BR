Copiar Buffer

Esta ação pode ser usada para copiar todos os dados armazenados em um buffer para outro. Ao usar dois buffers e copiar de um para o outro, ambos os buffers devem ter sido criados anteriormente usando a ação Criar buffer, e você fornece a identificação exclusiva do buffer retornado como o de/para argumentos.

NOTA: Você não pode copiar para o mesmo buffer.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Buffer O índice tampão (armazenado em uma variável) para copiar de
Copiar para O índice tampão (armazenado em uma variável) para copiar para

Exemplo:

O código de bloco de ação acima irá criar um novo buffer e armazenar sua identificação em uma variável temporária. Em seguida, ele verifica se o buffer indexado na variável global em questão existe, e se existe, então é copiado para o buffer temporário e a posição de leitura/gravação é redefinida de volta ao início, e se não existe, então ele é criado.
