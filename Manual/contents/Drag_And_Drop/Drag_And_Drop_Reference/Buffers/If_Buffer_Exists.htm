Se o Buffer existir

Esta ação pode ser usada para verificar se uma determinada identificação de buffer é válida e se o buffer realmente existe na memória. Se a identificação do buffer for válida e existir, a ação retornará verdadeira, caso contrário, retornará falsa. Você pode marcar "não". para fazer a verificação da ação se um buffer não existir.
