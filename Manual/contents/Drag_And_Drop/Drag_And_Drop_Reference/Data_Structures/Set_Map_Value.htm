Definir valor do mapa

Com esta ação você pode adicionar um par chave/valor na estrutura de dados do mapa. Você fornece a variável que contém o índice do mapa (como retornado pela ação Criar Mapa) e então dá uma "chave" (que é o identificador dentro do mapa para um valor), e então o "valor" a ser associado a essa chave. Ambas as chaves podem ser valores inteiros (como 0, 200, 36712, etc...) ou cordas (como "Nome", "Escudo", etc...), mas o valor pode ser qualquer tipo de dado válido.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Mapa O índice (armazenado em uma variável) do mapa a ser definido
Chave A chave a ser definida (inteiro ou string)
Valor O valor a ser armazenado com a chave dada
