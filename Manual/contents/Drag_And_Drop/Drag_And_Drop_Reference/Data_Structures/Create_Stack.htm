Criar Pilha

Esta ação criará uma nova estrutura de pilha de dados e retornará o valor do índice para que você possa mais tarde acessar a pilha através das outras ações de Estrutura de Dados. O índice da pilha será retornado para a Variável de destino que você fornecer, que pode ter foi criado anteriormente usando a opção Atribuir Variável ou Declarar Temp, ou você pode marcar a caixa de seleção "Temp" para nomear e criar um para armazenar o valor até o final do roteiro ou evento. Uma estrutura de pilha de dados recém-criada é considerada "vazia", ou seja: não contém entradas. Observe que você pode criar pilhas adicionais de DS clicando no ícone mais ao lado de a ação, e selecionando outra variável para segurar a identificação da pilha.

IMPORTANTE! A criação de qualquer estrutura de dados utiliza a memória na plataforma alvo, e como tal todas as estruturas de dados devem ser livres quando não forem mais necessárias, usando a ação Free Data Structure (Estrutura de dados livre), caso contrário você obtém uma vazamento de memória que pode prejudicar o desempenho de seus jogos ou até mesmo causar a sua queda. Isto é particularmente relevante ao usar variáveis locais temporárias para armazenar índices de estrutura de dados, já que estas variáveis são removidas no final do código ou evento, mas isso não significa que a estrutura de dados seja removida também! A estrutura de dados ainda existirá, só que você não terá como referenciá-la, então ou use uma variável de instância e libere a estrutura em um momento posterior, ou libere a estrutura antes do final de o evento ou função, se seu índice estiver armazenado em uma variável temporária.
