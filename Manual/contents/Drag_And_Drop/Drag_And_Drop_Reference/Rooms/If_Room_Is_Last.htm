Se o quarto for o último

Esta ação verificará se a sala atual é a última sala, conforme listado no Room Manager. A ação voltará a ser verdadeira se a sala atual for a última sala no Room Manager e falsa se não for. Você pode marcar a caixa de seleção " não" para transformar isto em "Se a sala NÃO for a última".

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão executadas agora se o "se" for avaliado como verdadeiro, enquanto quaisquer ações descartado em outro lugar será realizado após o bloco "se".
