Ir para o quarto

Esta ação encerrará a sala atual e irá para a sala especificada pelo índice de entrada. A sala deve existir ou então um erro será dado, e se você usar a mesma sala que a sala atual terá o mesmo efeito que usar a Sala de Reinício. Note que a sala não mudará até o final do evento ou roteiro onde a ação foi chamada, portanto, qualquer ação após esta ainda será executada.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Sala A sala para ir para

Exemplo:

O código de bloco de ação acima recupera uma variável global e depois verifica se o valor que ela contém é menor ou igual a 0. Se for, então o jogo irá para outra sala.
