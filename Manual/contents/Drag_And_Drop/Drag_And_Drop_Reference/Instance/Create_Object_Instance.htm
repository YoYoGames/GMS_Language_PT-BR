Criar instância de objeto

Com esta ação você pode criar uma instância de qualquer objeto a partir do Navegador de Ativos. Você dá o nome do objeto e uma posição x/y dentro da sala onde ele deve ser criado, bem como o nome da camada onde a instância será criada (deve ser uma Camada de Instância). Você pode marcar as bandeiras "relativas" para posicionar a instância em relação à instância que está executando os blocos de código de ação. Sempre que você cria uma instância no jogo, ela recebe um valor de identificação único com o qual você pode então identificar a instância em outras ações, portanto, esta ação permite definir uma variável "alvo" para manter este valor de identificação (como mostrado no exemplo abaixo). Note que o alvo pode ser marcado como uma variável temporária (local), que ser criado para a ação e pode ser usado em todas as ações subseqüentes.
