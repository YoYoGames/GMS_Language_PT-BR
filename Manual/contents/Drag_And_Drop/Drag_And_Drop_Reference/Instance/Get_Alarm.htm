Obter Alarme

Este bloco de ação recupera o valor de uma contagem regressiva de alarme e o armazena na variável alvo dada. Se você marcar a variável alvo como "Temp", isto criará uma nova variável temporária (local) para manter o valor de retorno para você, como mostrado abaixo no exemplo abaixo.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Alarme O alarme a receber.
Meta A variável alvo a ser usada para armazenar o valor do alarme.

Exemplo:

O código de bloco de ação acima cria uma variável local (temporária) e armazena o valor atual de o alarme[0]. Isto é então verificado e se for inferior a 0, o alarme é ajustado para 60.
