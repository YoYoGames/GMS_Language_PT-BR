Definir Velocidade de Animação

Este bloco de ação define a velocidade da imagem (velocidade de animação) para o duende atribuído à instância. Você define os frames ideais por segundo no Editor Sprite e então pode usar este valor para modificá-lo. A velocidade de animação padrão é definida para 1, o que significa que o jogo tentará manter o número de frames por segundo que você definiu para o sprite, mas você pode usar valores maiores que 1 para animar mais rapidamente (os frames podem ser pulados), ou valores decimais para animar mais lentamente (os quadros serão mostrados em várias etapas) ou mesmo valores negativos para inverter a velocidade da animação.
