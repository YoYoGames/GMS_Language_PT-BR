Set Instance Alpha

Este bloco de ação define o valor da imagem_alfa para a instância. O alfa da imagem é o valor "transparência" para desenhar qualquer duende atribuído à instância e tem um valor entre 0 e 1, onde 0 seria totalmente transparente e 1 seria ser totalmente opaco. Observe que se o duende tiver pixels transparentes ou semi-transparentes, então sua transparência será escalonada pela quantidade, então se você tiver cerca de 50% de pixels transparentes no duende e definir a instância alfa para 0,5, a instância alfa final para os pixels transparentes seria de 25%. Você pode definir este valor como sendo um valor relativo

Favor observar que para que as mudanças nesta variável sejam visíveis, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker Studio 2 irá, por padrão, sortear o sprite) ou ser sorteada usando a ação Draw Self .
