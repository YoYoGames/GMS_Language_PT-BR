Preferências do editor Sprite

As Preferências do Editor Sprite são usadas para definir a aparência e o funcionamento do Editor Sprite. Existem as seguintes opções:

  • Origem padrão dos duendes: Esta opção permite que você defina a posição de origem padrão para qualquer novo recurso sprite que você criar. O valor padrão é o Top Left.
  • Máscara de colisão tipo sprite padrão: Esta opção permite definir o tipo padrão de máscara de colisão a ser usada para novos sprites. As escolhas são retângulo, retângulo girado, elipse, diamante ou preciso / preciso por moldura. O retângulo padrão é o retângulo.
  • Modo de animação sprite predefinido: Esta opção permite definir o modo de animação padrão para sprites no editor sprite. Você pode escolher entre " Frames por segundo" ou" Frames por frame de jogo ", com a configuração padrão sendo " Frames por segundo".
  • Velocidade padrão de animação sprite: Esta opção permite que você defina o valor padrão da velocidade de animação para quando você criar um novo ativo sprite. Note que este valor está ligado ao modo de animação e - por exemplo - 3 quadros por segundo. não é o mesmo que 30 quadros por quadro de jogo, portanto, certifique-se de definir este valor corretamente em referência ao modo preferido. O valor padrão é 30.
  • Largura padrão do duende: Esta opção permite definir a largura padrão para todos os novos sprites criados. O valor padrão é 64.
  • Altura padrão do duende: Esta opção permite que você defina a altura padrão para todos os sprites novos. O valor padrão é 64.
  • Aplicar redimensionamento e escala na origem do duende: Quando marcada esta opção mudará a origem do duende quando o duende for redimensionado ou escalonado, com a nova origem sendo movida/escalonada para corresponder aos novos valores para o duende. Observe que isto só é aplicável para as origens de duendes personalizados que não estão trancados no editor sprite. Se a origem estiver bloqueada ou o duende usar uma das posições padrão - por exemplo: "Center" ou "Top Left", etc... - então esta opção ser cinzento e indisponível para mudança. O padrão está habilitado.

A seguinte sub-categoria também existe para o Editor Sprite:

Confirmar Diálogos

As opções disponíveis aqui controlam as mensagens que aparecem quando são necessárias para confirmar uma ação. Você pode alterar estas configurações para mostrar sempre a mensagem ou rejeitá-la com uma ação específica (Sim ou Não) :

  • Resposta automática à importação de arquivos: Quando você importa uma nova imagem e uma já existe, por padrão você será solicitado a continuar importando e sobrescrever a imagem existente ou cancelar a operação. Esta opção permite que você suprima esta mensagem e sempre cancelar a importação selecionando "Não", ou sobrescrever o sprite atual selecionando "Sim". O valor padrão é "Mostrar mensagem".
  • Resposta automática à falha de carga de imagem: Quando você importa uma nova imagem e a importação falha, por padrão será exibida uma mensagem de aviso informando que houve uma falha. Esta opção lhe permite suprimir esta mensagem selecionando "Ok". O valor padrão é "Mostrar mensagem".
