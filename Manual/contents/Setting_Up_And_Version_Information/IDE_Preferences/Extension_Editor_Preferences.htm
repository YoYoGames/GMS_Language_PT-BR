Preferências do Editor de Extensão

As preferências do Extension Editor cobrem algumas opções relacionadas com as mensagens que você é mostrado ao utilizá-lo, com os detalhes exatos descritos abaixo:

  • Resposta automática à eliminação de arquivos: A configuração padrão aqui é mostrar uma mensagem perguntando se você tem certeza de que deseja excluir, mas você pode definir isto como "OK" para que todas as exclusões sejam concluídas sem esta mensagem como se você clique em "OK" já.
  • Resposta automática à exclusão de arquivos por procuração: Como acima, isto controla a mensagem que lhe é mostrada na exclusão de qualquer arquivo proxy.
