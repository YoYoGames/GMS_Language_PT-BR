Preferências de Plugin

As Preferências de Plugins são usadas para controlar quaisquer plugins extras que tenham sido adicionados à IDE GameMaker Studio 2. Por padrão, esta tem uma seção dedicada ao plugin de Controle de Fonte para integrar o GameMaker Studio 2 projetos com Git.

Controle da fonte(Git)

Esta seção trata das preferências que você é obrigado a definir para obter o Git SCM plugin para trabalhar com o GameMaker Studio 2. Primeiro você tem o conjunto de G opções enérgicas :

  • Janela de saída de controle de fonte aberta automaticamente: Quando isto for verificado, a janela de saída de controle de fonte será mostrada automaticamente quando algo mudar. Isto está ligado por padrão.
  • Registro de Verbose: Permitir o registro verboso para o registro de saída Git. Isto está desativado por padrão.

A seguir é o Seção de Identidade do Usuário com os seguintes campos que devem ser preenchidos:

  • Nome de usuário para se comprometer como: Aqui você deve digitar o nome de usuário que você estará usando para fazer qualquer compromisso com Git (este deve ser o mesmo que o nome de usuário especificado quando você configurou seu repositório Git ).
  • Email para se comprometer como: O e-mail que está associado a esta identidade Git.

Após configurar a Identidade do Usuário, você pode então adicionar uma nova combinação de nome de usuário/senha para autenticação, clicando no botão etiquetado Adicione uma nova autenticação Usuário/Pass, que abrirá a seguinte janela com diferentes campos para você preencher:

Você também pode Adicione uma nova autenticação do par de chaves SSH clicando no botão com essa etiqueta. Isto abrirá a seguinte janela para que você possa preencher:

Abaixo destes botões você pode então configurar o Ferramenta de fusão e o Ferramenta Diff para SCM com estas opções:

  • Ferramenta Caminho para fundir: Aqui você especifica o caminho completo do arquivo para a ferramenta de fundir Git.
  • Opções para a ferramenta Merge: Neste campo você pode adicionar qualquer comando extra a ser executado sempre que você usar a Ferramenta de Integração.
  • Caminho para a ferramenta de difusão: Aqui você especifica o caminho completo do arquivo para a ferramenta de difusão Git.
  • Opções para a ferramenta Diff: Neste campo você pode adicionar qualquer comando extra a ser executado sempre que você usar a Ferramenta Diff.

Por fim, depois de configurar essas instalações, você tem uma última opção:

  • Habilitar automaticamente o controle da fonte: Quando isto for verificado, o controle de fonte será ativado automaticamente para todos os novos projetos. Isto está desativado por padrão.
