Preferências gerais

As Preferências Gerais são usadas para definir o comportamento geral da IDE GameMaker Studio 2. As mais gerais estão listadas na página principal enquanto muitas outras estão divididas em várias subcategorias, explicadas mais abaixo nesta página.

  • Linguagem IDE: Esta opção permite selecionar o idioma para a IDE a partir da lista de opções fornecidas. O idioma padrão é o inglês.
  • Pele IDE: Aqui você pode selecionar entre as diferentes peles que a IDE pode utilizar. Por padrão, o GameMaker Studio 2 vem com duas peles, uma clara e outra escura, e a configuração inicial utiliza a pele escura.
  • Framerate máximo do Editor do GameMaker Studio 2: Esta opção permite que você defina o frame máximo que a IDE GameMaker Studio 2 atualizará, permitindo que você defina o frame para corresponder à velocidade de atualização de seu monitor. Padrão valor é 60.
  • Dispositivo de áudio padrão: Aqui você pode escolher o dispositivo de áudio padrão para o IDE GameMaker Studio 2 a ser utilizado. Você pode atualizar a lista clicando no Botão Atualizar .
  • Resposta Automática à Confirmação de Saída: Esta opção permite que você defina se uma mensagem de confirmação deve ser mostrada ao sair do GameMaker Studio 2. Por padrão, esta opção é definida para "mostrar mensagem", mas pode ser definida para pular a mensagem como se você tivesse clicado OK.
  • Resposta automática ao diálogo de mudança de fonte do sistema: Esta opção permite definir se uma mensagem de confirmação deve ser exibida quando uma fonte do sistema tiver sido adicionada, removida ou editada. Por padrão, isto é definido para "mostrar mensagem", mas pode ser definida para saltar a mensagem como se você tivesse clicado em OK.
  • Habilitar notificações: Ao verificar isso, a IDE poderá mostrar notificações sobre novos tempos de execução, etc... Isto está ativado por padrão.
  • Notificação de tempo limite (ms): Define o período de tempo (em milissegundos) para o qual uma notificação é mostrada. O tempo padrão é 6000.
  • Logout na saída: Se esta opção for marcada, então a IDE sairá automaticamente da sessão do usuário quando o GameMaker Studio 2 for fechado. Isto é destinado principalmente aos usuários de Educação, pois - quando marcada - isto significa que sempre que o programa desliga o usuário seria desconectado e assim liberaria o assento da sessão para que outros usuários pudessem usar. Isto é desligado por padrão.
  • Habilitar o DPI Override: Esta opção permite que você sobreponha a configuração padrão de DPI com a sua própria. Você deve primeiro ativar a opção e então definir o valor Raw DPi Override (o valor padrão é 96) ou definir a porcentagem do DPi nativo (valor padrão 100%), e uma vez alterado você será solicitado a reiniciar o IDE, o que pode exigir que você salve qualquer trabalho que esteja sendo feito no projeto atualmente aberto. Observe que o DPi tem uma porcentagem mínima de 50% e máxima de 500%, e se você você passar o mouse sobre o ícone de informação, você pode ver a configuração máxima recomendada para o monitor que está sendo usado:

    Note que se você for acima da configuração DPi recomendada para o monitor, você será mostrado uma notificação que lhe permite reiniciar o DPi.
  • Mostrar tela de boas-vindas na inicialização: Isto está ligado por padrão e simplesmente diz ao GameMaker Studio 2 para mostrar uma breve tela de boas-vindas quando você iniciar o programa. Desmarcando isto evitará que a tela de início apareça.
  • Permitir o fechamento de abas vazias via atalho: Você pode fechar qualquer aba aberta usando o atalho do teclado / + W, mas se você desativar esta opção, o atalho não funcionará mais. Isto é ativado por padrão.
  • Abrir novos recursos para edição no arraste de arquivos para o IDE: Alguns recursos (como sprites ou sons) podem ser arrastados para o espaço de trabalho principal na IDE GameMaker Studio 2 e serão detectados e adicionados como um novo recurso no Ativo Navegador. Por padrão, o editor de recursos correspondente também será aberto para edição quando isto acontecer, mas a desmarcação desta opção impedirá que os editores sejam abertos e você será obrigado a abri-los manualmente.
  • Abrir novos recursos para edição na criação: Se isto for verificado, toda vez que você criar um novo recurso no Navegador de Recursos, o editor de recursos correspondente será aberto automaticamente. Isto é verificado por padrão e desmarcado. significará que você terá que abrir manualmente todos os recursos recém-criados.
  • Recarregar automaticamente os arquivos alterados: Se isto for verificado, a IDE verificará automaticamente os arquivos para ver se algum deles foi alterado fora do GameMaker Studio 2 e os recarregará se tiverem sido alterados.
  • Desativar as animações de transição IDE: O GameMaker Studio 2 IDE tem uma série de transições animadas para coisas como abrir janelas, etc... no entanto, elas não são do gosto de todos e você pode experimentar um aumento de desempenho se elas são desativados em máquinas mais velhas ou com baixa potência, se estiverem desativadas. Você pode desativá-las aqui, e elas estão ligadas por padrão.
  • Opt-in para pesquisas de feedback: Quando esta opção é marcada, você está dizendo ao GameMaker Studio 2 que deseja participar de pesquisas de feedback, o que significa que ocasionalmente, enquanto trabalha, uma mensagem será mostrada sobre a área em que você está trabalhando e solicitar uma avaliação e feedback. Esta opção está ativada por padrão.

  • Depuração
  • Vigilante de arquivos
  • Ir para
  • Ajuda
  • Entrada
  • Janelas desaída
  • Caminhos
  • Economia de energia
  • Janelasrecentes
  • Recursos
  • Espaços de trabalho
