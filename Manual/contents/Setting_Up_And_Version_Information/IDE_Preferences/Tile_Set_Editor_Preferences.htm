Preferências do editor do conjunto de azulejos

O Tile Set Editor Preferences tem a seguinte opção para o Auto Tiling:

Cor de fundo do azulejo automático: Esta opção define a cor de fundo dentro do editor do conjunto de azulejos na seção de criação de azulejos automáticos. A cor padrão é (rgba) #808080FF.

As preferências do Tile Set Editor também têm uma subseção Confirm Dialogs :

A opção disponível aqui controla a mensagem que aparece quando você muda a célula do conjunto de azulejos e isto afetará a forma como as camadas do mapa de azulejos da sala são renderizadas (resultando em camadas cortadas). Por padrão, você será solicitado a continuar ou cancelar a operação, e esta opção permite suprimir esta mensagem, selecionando "Ok". que irá adiante com a mudança de célula do conjunto de azulejos, independentemente do impacto sobre as camadas do mapa de azulejos da sala. O valor padrão é "Mostrar mensagem".
