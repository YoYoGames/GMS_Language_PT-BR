Preferências de espaço de trabalho

As preferências aqui afetarão a forma como você se movimenta dentro dos diferentes GameMaker Studio 2 espaços de trabalho. As opções são:

  • Escala de graus de navegação do teclado do espaço de trabalho: Dentro de um espaço de trabalho você pode ter múltiplas janelas abertas para diferentes objetos e recursos, e assim para se mover rapidamente entre eles você pode usar o atalho de teclado / + + <Arrow Keys> para se mover entre eles. Dado que as janelas dentro de um espaço de trabalho não estão alinhadas em grade, a IDE tem que escolher qual janela você quer dizer quando pressiona qualquer uma das 8 direções possíveis, e esta opção permite definir o "ângulo de visão" que é usado. O valor padrão é 40°.
  • Forro da coluna da cadeia do espaço de trabalho (px): Os elementos do espaço de trabalho são freqüentemente "encadeados" uns aos outros (por exemplo, o Editor de Código é encadeado ao Editor de Eventos é encadeado ao Editor de Objetos), e esta opção permite que você defina a distância horizontal entre quaisquer duas janelas encadeadas. O valor padrão é 30px.
  • Forro de corrente do espaço de trabalho (px): Como mencionado acima, os elementos do espaço de trabalho são frequentemente "encadeados" um ao outro e esta opção permite definir a distância vertical entre quaisquer duas janelas encadeadas. O valor padrão é 20px.
  • Contagem do segmento de renderização da cadeia de espaço de trabalho: Aqui você pode definir o número máximo de segmentos para renderização de cadeias. A redução deste valor pode ajudar no desempenho de máquinas de menor porte. O valor padrão é 20.
  • Cor do elo da cadeia do espaço de trabalho: Esta é a cor que será usada para exibir cadeias entre os elementos do espaço de trabalho. A cor padrão é ARGB $99DD00.
  • A navegação do teclado do espaço de trabalho repõe o zoom: Como mencionado acima, você pode pular de janela em janela dentro do espaço de trabalho usando atalhos de teclado. Você também pode aumentar e diminuir o espaço de trabalho usando o / + a roda do mouse . Esta opção significa que se você tiver o espaço de trabalho ampliado e usar os atalhos para pular para outra janela, então o espaço de trabalho será retornado para uma visualização 1:1 e centrado na janela selecionada. Esta opção está desligada por padrão.
  • Cadeias de espaços de trabalho podem se sobrepor: Quando ativada, esta opção permitirá que as cadeias de espaço de trabalho se cruzem e se sobreponham. Esta opção está desligada por padrão.
