Preferências de recursos

As preferências aqui contêm opções de como certos recursos serão utilizados em seus projetos:

  • Cor inicial padrão de desenho para novos projetos: Quando você usa uma fonte ou desenha uma forma usando as funções do GameMaker Studio 2 ou ações do DnD™sem especificar uma cor, a cor definida aqui é a que será usada. A cor padrão é branco ( ARGB: $FFFFFFFFFFFF).
  • Cor inicial da janela padrão para novos projetos: Esta opção é usada para definir a cor de fundo da página HTML para o alvo HTML5, assim como definir a cor mostrada fora das salas em todos os alvos (incluindo HTML5). A cor padrão é preto (ARGB: $000000FF).
