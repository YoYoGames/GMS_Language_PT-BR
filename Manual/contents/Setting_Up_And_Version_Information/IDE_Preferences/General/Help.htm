Preferências de ajuda

O arquivo de ajuda é projetado para operar como uma página web e mesmo que os arquivos sejam armazenados localmente, é necessário o uso de um navegador externo para visualizá-los. As seguintes configurações relacionadas com o arquivo de ajuda podem ser alteradas aqui:

  • Linguagem Manual: Isto define a linguagem a ser utilizada pela IDE. O padrão é "inglês".
  • Porta manual de ajuda: Isto define o número da porta para o micro servidor web manual a ser utilizado. O valor padrão é 51291.
