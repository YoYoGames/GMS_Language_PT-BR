Confirmar as Preferências de Diálogo

As preferências do Confirm Dialogue tratam de alguns dos pop-ups de confirmação geral que aparecerão ao realizar determinadas ações na IDE GameMaker Studio 2. As opções são:

  • Resposta automática ao fechamento de um espaço de trabalho através de atalho de teclado: O atalho de teclado mencionado acima pode ser usado para fechar qualquer aba aberta, mas às vezes você pode acidentalmente fechar uma aba vazia que você precise. Para evitar isso, você pode definir esta opção para "Mostrar mensagem" (a opção padrão) para perguntar se você realmente quer fechar aquela aba vazia. As outras configurações são "Não", para simplesmente não fazer nada se o atalho for usado, ou "Certo" para fechar a aba sem qualquer mensagem.
  • Resposta automática para reiniciar uma construção em funcionamento: Quando você reinicia uma construção e outra anterior ainda está em funcionamento, você será perguntado se quer parar a construção atual primeiro. Ao verificar esta mensagem, você suprimirá que mensagem e forçar o GameMaker Studio 2 a assumir que você clicou em "Ok" como resposta. Esta opção está definida como "Mostrar mensagem" por padrão.
  • Resposta automática ao aviso ao tentar depurar o YYC: O alvo YYC compila para código nativo e como tal não pode ser depurado usando o depurador embutido. Quando você tenta usar o depurador no alvo YYC, você receberá um aviso informando sobre isso, e o fato de que o projeto será compilado usando o VM em seu lugar. Esta opção está definida como "Mostrar Mensagem" por padrão, mas definindo-a como "OK" irá suprimir este aviso.
  • Resposta automática para redefinir o layout do projeto: Quando você seleciona reiniciar o layout do projeto (no Menu Layouts), você receberá, por padrão, uma mensagem de aviso sobre como isto fechará qualquer janela de recursos aberta. No entanto, você pode suprimir esta mensagem e redefinir automaticamente o layout selecionando "Ok", ou fazer com que nada seja feito selecionando "Não". O valor padrão é ter "Mostrar Mensagem".
