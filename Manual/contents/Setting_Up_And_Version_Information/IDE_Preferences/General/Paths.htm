Preferências de caminho

As preferências de caminho controlam os diferentes caminhos de arquivo exigidos pelos elementos da IDE GameMaker Studio 2, bem como para qualquer editor externo que você necessite. As seguintes opções existem para a IDE:

  • Diretório Temp: O local para salvar todas as pastas temporárias. Por padrão no Windows, este é

    \Usuários Nome de usuário Dados locais

    E em macOS é:

    /var/folders/<Path_Hash>/GameMakerStudio2

  • Diretório de cache IDE: O local para salvar o cache IDE. Por padrão no Windows, este é

    \Usuários Nome de usuário >AppDataRoaming2Cache

    E em macOS é:

    /Usuários/nome de usuário>/.config/GameMakerStudio2/Cache

  • Diretório de cache de ativos: O local para salvar o cache de ativos para cada projeto. Por padrão no Windows, este é

    \Usuários Nome de usuário >AppDataRoaming2Cache

    E em macOS é:

    /Usuários/nome de usuário>/.config/GameMakerStudio2/Cache

  • Localização de "Meus Projetos": O local onde o GameMaker Studio 2 irá inicialmente criar novos projetos. Por padrão no Windows, este é

    \Usuários Nome de usuário>Documentos_GameMakerStudio2

    E em macOS é

    /usuário/nome do usuário>/GameMakerStudio2

  • Apagar automaticamente o diretório temporário no fechamento: Ao verificar isso forçará o GameMaker Studio 2 a apagar automaticamente a pasta Temp que ele cria por projeto para compilação, etc... Esta configuração está ativada por padrão, e a desmarcação será desligá-lo (o que significa que você terá que remover manualmente quaisquer arquivos temporários mais tarde).
  • Eliminar automaticamente o cache de ativos no fechamento: Se isto for verificado, a pasta de cache do compilador de ativos será removida quando você sair do GameMaker Studio 2. Isto está desativado por padrão, e ativá-lo significará que toda vez que você carregar e executar qualquer projeto, o cache precisará ser reconstruído (o que pode levar tempo, dependendo do tamanho do jogo).
  • Eliminar automaticamente o cache IDE no fechamento: Se isto for verificado, a pasta de cache do compilador IDE será removida quando você sair do GameMaker Studio 2. Isto está desativado por padrão, e ativá-lo significará que toda vez que você iniciar o GameMaker Studio 2 o cache precisará ser reconstruído.
  • Apagar Pasta Temp: Clicando neste botão, a pasta temp para o projeto será apagada.
  • Apagar Cache de Ativos: Clicando neste botão, o cache de ativos do compilador para o projeto será excluído.
  • Excluir Cache IDE: Clicando neste botão, o cache IDE será apagado.

As seguintes opções existem para definir caminhos para os editores externos:

  • Caminho para o editor/visualizador externo de arquivos SWF: Se você estiver trabalhando com sprites em formato SWF, você pode definir este caminho para o caminho de seu visualizador/editor preferido e ao clicar no botão Edit Image no Editor Sprite, então ele abrirá o programa em questão. O valor padrão aqui é não ter caminho.
  • Caminho para o editor/visualizador externo de arquivos da Coluna vertebral: Se você estiver trabalhando com sprites no formato Spine, você pode definir este caminho para o caminho de seu visualizador/editor preferido e quando você clicar no botão Edit Image no Editor Sprite, então abrirá o programa em questão. O valor padrão aqui é não ter caminho.
