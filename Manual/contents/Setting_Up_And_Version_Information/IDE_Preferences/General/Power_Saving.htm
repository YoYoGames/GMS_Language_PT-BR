Preferências de economia de energia

As preferências de economia de energia aqui afetarão quanta energia de CPU e GPU será usada para gerar a IDE GameMaker Studio 2 (essencialmente isto diminuirá o quadro base no qual o programa é executado e assim diminuirá o consumo de energia).

As opções aqui são:

  • Quadro de potência AC (%): O modificador de frame IDE de base quando conectado a uma entrada de energia. O padrão é 100%.
  • Bateria de potência (%): O modificador de frame IDE de base ao usar a bateria para alimentar a máquina. O padrão é de 35%.
  • Quadro de fundo (%): O modificador de frame IDE básico quando o GameMaker Studio 2 não está em foco (independentemente da fonte de alimentação). O padrão é 15%.
