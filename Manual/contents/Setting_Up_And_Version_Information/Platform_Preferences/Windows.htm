Preferências doWindows

As preferências do Windows têm as seguintes opções:

  • Localização doVisual Studio: Esta opção é para apontar o GameMaker Studio 2 para a instalação do Microsoft Visual Studio no PC construído.
  • Tipo de embalagem padrão: Ao construir um pacote executável final para Windows, você pode escolher entre criar um instalador (usando NSIS) ou um arquivo comprimido ZIP que terá o jogo e quaisquer arquivos adicionais dentro dele. Por padrão será mostrado um diálogo para você escolher entre estas duas opções de compilação, mas você pode definir esta preferência para usar sempre uma ou outra. Note que quando esta opção for definida para Mostrar Diálogo, na janela de diálogo real que aparece na compilação, permitindo "Lembrar Opção de Embalagem" irá definir esta preferência para a opção que você selecionar.
