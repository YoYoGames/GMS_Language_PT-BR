Preferências devapor

O Steam SDK é usado para construções Windows, MacOS e Ubuntu(Linux) que devem ser distribuídas através da plataforma Steam. Esta seção contém as opções de configuração necessárias para que ele funcione dentro de seus projetos:

  • Caminho para o SDK a vapor: O caminho do sistema até onde você instalou o Steam SDK. Uma vez que você tenha definido este caminho para o SDK, você precisará então ativar o Steam nas Opções de Jogo e então para cada plataforma adicionar a ID do aplicativo.
