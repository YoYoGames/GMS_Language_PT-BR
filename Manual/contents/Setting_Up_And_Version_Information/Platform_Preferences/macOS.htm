As preferências macOS têm as seguintes opções:

  • Identificador padrão da equipe: Aqui você pode adicionar seu identificador de equipe padrão, conforme lhe foi designado pela Apple. Este ID de equipe será usado quando seus arquivos de jogo forem enviados para Xcode para construir o aplicativo, e permitirá que Xcode gere os certificados de assinatura necessários. Note que esta configuração será aplicada por padrão a todos os jogos construídos para macOS, mas pode ser superada com base em cada projeto do General macOS Opções de jogo.
  • Escolha automática de embalagem: Aqui você pode definir uma preferência padrão para a criação de um executável para o alvo macOS. A opção "Mostrar Diálogo" sempre mostrará um diálogo antes de criar um executável, pedindo que você escolha entre DMG e Zip, mas você pode definir esta opção para qualquer um dos que estão nas Preferências para pular esse diálogo. Note que ativar a opção "Lembrar opção de pacote" no diálogo também definirá esta opção para o tipo de pacote selecionado.
