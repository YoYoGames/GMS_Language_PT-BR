Bem-vindo ao manual do usuário do GameMaker Studio 2! Este documento é dividido em três partes com o objetivo de apresentar a interface e o funcionamento básico do GameMaker Studio 2 e a programação geral antes de passar para uma utilização mais avançada e as funções disponíveis através de nossa linguagem proprietária GameMaker scripting GML ou nossa ferramenta visual de scripting Drag e Drop™.

Para facilitar, criamos um Guia de Início Rápido, e mesmo que você esteja familiarizado com outras ferramentas de criação de jogos, recomendamos que você comece por lá para ter uma visão geral da maneira como o GameMaker Studio 2 funciona. Você encontrará o GameMaker Studio 2 para ser fácil e intuitivo de usar e que, qualquer que seja seu nível de habilidade, você logo estará fazendo aqueles jogos com os quais sempre sonhou.

NOTA: Devido à natureza modular do GameMaker Studio 2 há certas coisas que não estão incluídas neste manual do usuário, em particular os aspectos de configuração e conexão dos diferentes dispositivos que podem ser suportados. Para informações atualizadas e dicas de solução de problemas relacionados às diferentes plataformas-alvo, consulte o Centro de Ajuda dos Jogos YoYo.

Referência Roteiro

Esta seção trata das diferentes linguagens de script disponíveis para você programar no GameMaker Studio 2. A linguagem ou método que você utiliza para criar seus projetos dependerá de sua habilidade e de seu histórico anterior, e - uma vez que todos são diferentes - o GameMaker Studio 2 visa ser o mais adaptável possível às suas diferentes necessidades, oferecendo duas opções diferentes para que você possa criar jogos.

NOTA: Estas não são formas mutuamente exclusivas de fazer seus jogos, e você pode misturar usando GML com DnD™como desejar, dependendo de sua habilidade e exigências com qualquer uma delas.

Drag And Drop™(DnD™)

Esta é a linguagem GameMaker Studio 2 Visual Scripting que utiliza blocka de ações que são encadeadas para criar sua lógica de jogos. As seções seguintes explicam como usar essas ações:

  • Drag And Drop™ Visão geral
